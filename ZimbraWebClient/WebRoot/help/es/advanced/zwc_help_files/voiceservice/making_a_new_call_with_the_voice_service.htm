Efectuar una llamada nueva con el servicio de voz

Nota: Esta función se aplica sólo para ZCS Network Edition y está disponible únicamente en la versión avanzada del Cliente web.

Puedes efectuar llamadas desde todos los dispositivos telefónicos registrados, incluidos los clientes de softphone como Cisco o Mitel.

  1. En la página Voz, haz clic en Nueva llamada. A continuación, se mostrará un nuevo cuadro de diálogo de llamada.

  2. Selecciona el número De en el menú desplegable. Los números enumerados son los registrados para ti.

  3. Introduce o selecciona el número Para para el que deseas efectuar la llamada. El número puede obtenerse de tus libretas de direcciones o tarjetas de contactos, o puedes introducir un número manualmente.

  4. Haz clic en Aceptar para efectuar la llamada. Utiliza el dispositivo telefónico asociado con el número de teléfono que estás utilizando para efectuar la llamada.

Finalización de una llamada

Para finalizar una llamada, cuelga el teléfono o haz clic en el cuadro de diálogo para colgar.
