Personalizar las opciones para redactar mensajes

Desde tus preferencias, puedes optar por crear mensajes simples de texto sin formato o crear mensajes con estilo HTML; además, puedes configurar firmas y perfiles que se apliquen a los mensajes que envíes.

Además de las opciones de redacción de mensajes, entre las demás opciones que puedes configurar se incluyen: solicitudes de notificación de lectura, redacción de mensajes en una nueva ventana y pasar el corrector ortográfico en los mensajes antes de enviarlos.

  1. Ve a la página Preferencias>Correo, sección Redactar mensajes.

  2. Selecciona cómo deseas que se envíen los mensajes, Como HTML o Como texto.

  3. Si seleccionas Como HTML, en los menús desplegables selecciona el estilo de Fuente, Tamaño y Color.

  4. En Configuración, selecciona las opciones para gestionar la redacción de mensajes.

    5. Opción

    Descripción

    Guardar borradores automáticamente al redactar los mensajes

    A medida que redactas un mensaje, periódicamente se guarda una copia en la carpeta Borradores.

    Solicitar siempre notificación de lectura

     Todos los mensajes que envíes solicitarán que se te envíe una notificación de lectura cuando el destinatario abra el mensaje.

    Cuando configures esta opción como valor predeterminado, puedes cambiarla para mensajes de correo individuales desde el menú desplegable Opciones en la barra de herramientas de redacción.

    Responder/reenviar usando el formato del mensaje original

    Incluso si el editor de textos predeterminado es diferente al del mensaje recibido, activa esta casilla para responder al mensaje con el mismo formato que el del mensaje original.  Esto resulta útil para asegurarse de que el destinatario puede obtener el mensaje en el mejor formato que pueda leer.

    Siempre redactar en una ventana nueva

    Se abrirá una ventana de redacción nueva. Podrás ver y navegar por tu buzón mientras esté abierta esta ventana.

    Corrección ortográfica obligatoria antes de enviar un mensaje

    Se corrige la ortografía de todos los mensajes antes de enviarlos.

    Guardar una copia en la carpeta Enviados

    Esta opción está habilitada de modo predeterminado. Puedes anular la selección de esta opción.

  5. (Opcional) Haz clic en Página de cuentas para ir a la página Preferencias de las cuentas y configurar tus perfiles.

  6. (Opcional) Haz clic en Administra tus firmas ... para ir a la página Preferencias de firmas y añadir firmas que se puedan utilizar al enviar mensajes.

  7. Haz clic en Guardar.
