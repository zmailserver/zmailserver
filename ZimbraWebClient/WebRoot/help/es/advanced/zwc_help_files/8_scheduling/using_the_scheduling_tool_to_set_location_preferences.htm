Usar la herramienta de programación para ver la disponibilidad de las ubicaciones

La herramienta de programación de agendas puede ayudarte a encontrar las horas libres más próximas y las ubicaciones para reuniones disponibles en tu organización. Puedes configurar las preferencias de las horas, los edificios, las ubicaciones y el tamaño de sala preferidos para las reuniones.

A medida que introduzcas los nombres de los asistentes, la herramienta de programación comenzará a sugerir horas y ubicaciones en el panel de resumen. También tienes la opción de buscar una ubicación para la reunión antes de añadir a los asistentes. Si configuras las preferencias de ubicación, sólo se mostrarán estas ubicaciones.

Usar la herramienta de programación para configurar las preferencias de ubicación

Si siempre quieres reservar salas que se encuentran en una ubicación específica, puedes utilizar la herramienta Ubicaciones sugeridas en la aplicación Agenda para sugerir la disponibilidad de esas salas cuando crees una cita.

  1. En la página Nueva cita, haz clic en Sugerir una ubicación.

  2. En el panel que se abre, haz clic en el icono de engranajes.

  3. En el cuadro de diálogo Preferencias de sugerencia, en la sección Preferencias de ubicación, introduce los datos de la ubicación. Puedes introducir tan solo el nombre de la ubicación o puedes ser más específico.

  4. Haz clic en Aceptar.

La columna de la derecha muestra las ubicaciones que se ajustan a tus requisitos. Las ubicaciones de las salas se mostrarán si se han configurado en el directorio de la empresa.

Una vez que hayas configurado estas opciones para una cita, esta información se guarda y estará disponible para cuando hagas otras citas.
