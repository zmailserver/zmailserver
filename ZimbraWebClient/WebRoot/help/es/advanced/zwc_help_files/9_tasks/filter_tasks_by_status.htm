Filtrar las tareas por estado

Puedes filtrar las tareas de la lista para verlas rápidamente por estado.

•       En el menú desplegable Filtrar por de la barra de herramientas Tarea, selecciona el tipo de tareas que deseas ver.

Opción

Descripción

No se ha iniciado

Se muestran las tareas que no se han iniciado.

Completada

Se muestran las tareas completadas.

Procesando

Se muestran las tareas que están en curso.

Esperando a otra persona

Se muestran las tareas que están esperando a otra persona.

Diferida

Se muestran las tareas diferidas.

Todas

Se muestran todas las tareas, independientemente del estado.

Lista de tareas

Se muestran las tareas con el estado Procesando, No se ha iniciado o, si la tarea ha caducado, Esperando a otra persona.
