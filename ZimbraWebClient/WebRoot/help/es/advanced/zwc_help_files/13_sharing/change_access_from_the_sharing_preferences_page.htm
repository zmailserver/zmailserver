Cambiar el acceso desde la página de preferencias para compartir

  1. En la sección Carpetas compartidas por mí en la página Compartir, selecciona el nombre del usuario que deseas cambiar.

  2. En la columna Acciones, puedes editar, revocar o reenviar un elemento compartido.

Opción

Descripción

Editar

Para abrir el cuadro de diálogo Propiedades compartidas y cambiar el rol.

Revocar

Para abrir el cuado de diálogo Revocar compartir y cancelar la carpeta compartida.

Reenviar

Para reenviar el aviso de elemento compartido.
