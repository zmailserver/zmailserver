Adjuntar archivos a un mensaje

Los mensajes pueden incluir archivos adjuntos. Puedes adjuntar documentos, hojas de cálculo, fotos, presentaciones y otros tipos de archivos.

  1. Redacta el mensaje tal como se describe en Redactar un mensaje nuevo.

  2. Haz clic en Añadir archivos adjuntos. Se abrirá una nueva ventana donde podrás cargar hasta 10 archivos adjuntos.

  3. Para cargar un archivo, haz clic en Examinar... para encontrar el archivo.

  4. Selecciona el archivo y haz clic en Abrir.

  5. Carga más archivos siguiendo los mismos pasos.

  6. Cuando se hayan añadido todos los archivos adjuntos, haz clic en Hecho. Los archivos adjuntos se detallarán debajo del asunto del mensaje.

  7. Cuando estés listo para enviar el mensaje, haz clic en Enviar para mandar el mensaje y los archivos adjuntos.

Nota: Los destinatarios de tu mensaje deberán tener el software apropiado para poder abrir y leer el archivo adjunto. Los usuarios con una configuración de sistema típica podrán abrir los archivos creados con programas de Microsoft Office. También se pueden usar diversos programas para abrir otros formatos comunes, como los archivos de texto, HTML e imágenes como .GIF o .JPG. Sin embargo, depende del tipo de sistema que tenga cada destinatario.

Para eliminar un archivo adjunto, anula la selección de la casilla situada al lado de ese archivo antes de enviar el mensaje.
