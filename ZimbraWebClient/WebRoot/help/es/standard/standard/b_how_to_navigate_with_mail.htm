Navegar por la pestaña Correo

Al abrir la pestaña de correo, verás una lista con tus mensajes. Desde esta vista podrás abrir y leer tus mensajes.

  1. Tienes las opciones siguientes:

  • Haz clic en Actualizar para ver si tienes nuevos mensajes de correo.

  • Haz clic en Redactar para escribir un nuevo mensaje. Se abrirá en el panel de contenido.

  • Haz clic en Imprimir para imprimir el mensaje o la conversación.

  • Haz clic en Eliminar para eliminar un mensaje. El mensaje se moverá a la carpeta Papelera.

  • Utiliza la barra de acción Mover a para mover los mensajes a la carpeta seleccionada.

  • Utiliza la barra de acción Acciones para marcar los mensajes seleccionados como leídos o no leídos, para añadir o quitar una marca de seguimiento o para etiquetarlos con una etiqueta que ya has creado o quitarles la etiqueta.

  • Selecciona Organizado por: para ver tus mensajes por Fecha o Asunto.

  1. El encabezado del panel de contenido. Si activas la casilla en el encabezado, activarás todos los mensajes en esa página.

  2. En el panel de contenido, tus mensajes se visualizan como conversaciones o mensajes individuales. Usa la casilla de verificación para seleccionar el mensaje o la conversación cuando quieras llevar a cabo la tarea "Mover a" o "Acciones".
