Usar las versiones avanzada y estándar del cliente

Hay dos versiones disponibles de @ClientNameLong@.

La versión Avanzada ofrece todas las funciones para la colaboración en la web. Funciona con la mayoría de los navegadores actuales y conexiones a Internet más rápidas.

Si tienes una conexión a Internet lenta o prefieres un programa basado en HTML, puedes utilizar la versión Estándar, que tiene muchas de las mismas funciones que la versión avanzada pero permite acceder de manera diferente. Puedes hacer que la versión predeterminada del Cliente Web sea la versión Estándar desde la página Preferencias>General después de iniciar sesión.

Herramientas del navegador

•El botón Atrásdel navegador te llevará a la página que veías anteriormente. También puedes usar el botón Adelante.

•No utilice el botón Recargar(Actualizar) del navegador. Si lo haces, se reiniciará la sesión.
