Aceptar el acceso a una carpeta de correo compartida

Nota: Disponible únicamente en las cuentas de Zimbra.

Si recibes una notificación por correo electrónico de que se te ha otorgado acceso para ver una carpeta de correo compartida de otra persona, puedes aceptar o rechazar la oferta.

Si decides aceptar, se añadirá la carpeta compartida a tu lista de carpetas en el panel de resumen. Si estás trabajando en Zimbra Desktop, deberás estar conectado a Internet para ver el contenido de las carpetas compartidas. El contenido de las carpetas compartidas no se descarga a tu ordenador.

Para aceptar

  1. Haz clic en Aceptar compartir en el mensaje de correo electrónico. Se abrirá un cuadro de diálogo donde verás el rol que se te ha otorgado.

  2. Antes de aceptar la carpeta, puedes cambiarle el nombre y seleccionar un color para resaltarla.

  3. (Opcional) Puedes enviar un mensaje de confirmación al dueño de la carpeta.

  4. Pulsa en Sí. Se añadirá la carpeta de correo a tu lista de carpetas. Se moverá el mensaje a la carpeta Papelera.
