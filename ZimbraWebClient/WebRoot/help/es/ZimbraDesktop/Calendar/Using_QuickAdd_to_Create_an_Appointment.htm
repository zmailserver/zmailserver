Usar el Añadido rápido para crear una cita

El cuadro de diálogo de añadido rápido hace que sea fácil y rápido crear una cita. En tu agenda, selecciona la fecha de inicio y haz doble clic o arrastra el cursor sobre la hora para abrir el cuadro de diálogo de añadido rápido.

El añadido rápido es una opción. Si prefieres que siempre se abra la página Cita al hacer doble clic o arrastrar el cursor en la agenda, desactiva esta opción desde las preferencias de tu agenda.

Para crear una cita rápida

  1. Selecciona la agenda para la cita. Haz clic en la hora de la cita o haz clic y arrastra desde la hora de inicio a la de finalización de la cita.  Se abrirá el cuadro de diálogo Añadir cita rápida.

Nota: Para crear una cita para todo el día, desde la vista diaria o semanal disponibles en la parte superior de la agenda, haz clic y arrastra el cursor sobre uno o más días.

  1. Introduce el asunto y su ubicación y, si se trata de una cita recurrente, selecciona un modo de repetición.

  2. Configura cuándo deseas recibir un recordatorio de la hora de inicio de la reunión.

  3. Haz clic en Aceptar. Verás la cita en la agenda.

Si necesitas invitar a otros asistentes o cambiar las opciones de repetición, haz clic en Más detalles para abrir la página Cita.
