Control de versiones de los archivos del Maletín

El control de versiones permite gestionar cambios en los archivos del Maletín. El número de versión se muestra en la lista de documentos. Si se visualiza por historial de versiones, puedes ver versiones anteriores del archivo y el autor. Puedes restablecer una versión anterior del archivo y eliminar versiones que ya no necesitas.

Nota: Cada versión de un archivo resta espacio de almacenamiento disponible para la cuenta.  

Los archivos que cargas pueden enviarse para evitar que más de una persona trabaje en el mismo archivo en el Maletín. Estos archivos se descargan en tu escritorio. Cuando cargas el archivo actualizado, se comprueba y la versión se modifica.
