Salvando critérios de busca

Se você criar uma busca que acha que usará novamente, poderá salvá-la.

  1. Crie o critério de busca, seja inserindo texto na barra Busca ou criando um critério com a busca Avançada.

  2. Para salvar a busca, clique em na barra Buscar. A caixa de diálogo Salvar busca é exibida.

  3. Digite o nome para sua busca salva. As buscas são salvas como uma pasta de busca na seção Buscas do painel Visão geral por padrão, mas você pode selecionar outra pasta.

  4. Clique em OK. Sua busca é salva.

Para usar uma busca salva, clique na pasta de busca. Os resultados da busca são exibidos imediatamente no painel Conteúdo.
