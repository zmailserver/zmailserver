Restaurando contas a partir de arquivos de backup

Se você fizer backup das contas periodicamente, será fácil restaurá-las para o ponto do backup mais recente.

Para restaurar contas

  1. Vá para a página Preferências>Backups. Os arquivos de backup disponíveis estão listados na seção Restaurar a partir do backup.

  2. Marque a caixa Ativo das contas que quer restaurar.

  3. Clique em Restaurar a partir do backup.  As contas selecionadas serão restauradas.  Dependendo do tamanho do arquivo, pode levar algum tempo para restaurar a conta.  

Não feche o Zimbra Desktop durante a restauração das contas.
