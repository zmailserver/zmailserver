Criar uma agenda

Uma agenda foi criada para você. Você pode criar agendas adicionais para ter controle dos diferentes tipos de funções. Por exemplo, é possível criar uma agenda para trabalho, uma para compromissos pessoais e uma para atividades da empresa.

Se você gerenciar várias agendas, poderá agrupá-las em uma exibição hierárquica. Por exemplo, você poderia agrupar todas as Agendas compartilhadas em uma pasta Agenda chamada Compartilhados. É possível também mover a agenda

•       No painel Visão geral da agenda, selecione uma agenda e arraste-a para a agenda a ser aninhada.

•       Como segunda opção, clique com o botão direito do mouse na agenda a ser movida e selecione Mover. Na caixa de diálogo Escolher pasta, selecione a agenda a ser movida.

Adicionar uma agenda

  1. Clique na guia Agenda.

  2. Na barra de ferramentas, selecione Novo>Agenda.

  3. Na caixa de texto Nome, digite o nome da agenda.

  4. Selecione uma cor de exibição das atividades da Agenda.

  5. Para impedir que outras pessoas vejam sua agenda ao marcar compromissos, selecione Excluir esta agenda ao informar horários livres/ocupados.

  6. Para sincronizar essa agenda com uma agenda remota, selecione Sincronizar compromissos da agenda remota. Insira a URL do arquivo iCal remoto.

  7. Clique em OK. A nova agenda é exibida na lista de agendas.
