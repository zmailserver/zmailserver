Aplicar código de cores às suas agendas

Você pode aplicar código de cores às suas agendas de forma que as atividades dessa agenda sejam sempre exibidas na cor selecionada, o que é útil quando se observa atividades de várias agenda em um modo de exibição. Você pode usar uma cor mais de uma vez.

  1. Clique com o botão direito do mouse na agenda e selecione Editar propriedades.

  2. Selecione uma cor no menu suspenso.
