Importar sua agenda iCal

Você pode importar iCalendars salvos no formato .ics. iCalendars são o padrão de troca de dados de agenda na Internet e .ics é o formato padrão das informações de iCalendar.

  1. Vá para Preferências>página Importar/Exportar.

  2. Navegue até o arquivo .ics no seu computador.

  3. Selecione a pasta para o arquivo ou crie uma nova pasta.

  4. Clique em Importar.

Dependendo do tamanho do arquivo .ics, a importação pode levar alguns minutos.
