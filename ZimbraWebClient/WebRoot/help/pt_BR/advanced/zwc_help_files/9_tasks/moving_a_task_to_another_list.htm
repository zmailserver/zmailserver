Movendo uma tarefa para outra lista

  1. Clique com o botão direito do mouse em uma tarefa e selecione Mover.

  2. Selecione a lista de destino para onde deseja mover a tarefa.

    3. •    Forneça o nome da lista de tarefas ou use as setas do teclado para navegar até a lista de destino.

    •    Clique em Novo para criar uma nova lista de tarefas.

  3. Clique em OK.
