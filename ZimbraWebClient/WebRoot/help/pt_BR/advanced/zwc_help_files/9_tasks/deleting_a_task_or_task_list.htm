Excluindo uma tarefa ou lista de tarefas

Excluir uma tarefa

  1. Vá para a página Tarefas.

  2. No painel Visão geral, selecione a lista de tarefas que contém as tarefas que deseja excluir.

  3. No painel Conteúdo, selecione as tarefas a serem excluídas. É possível selecionar várias tarefas marcando a caixa de seleção ao lado da tarefa.

  4. Clique em Excluir na barra de tarefas.

Você poderá imediatamente desfazer essa ação clicando no link Desfazer  que aparece na parte superior da página do navegador. A tarefa é movida de volta para o seu local original.

Excluir uma lista de tarefas

Ao excluir uma lista de tarefas, você poderá excluir somente as listas de tarefas que criou.

  1. Vá para a página Tarefas.

  2. No painel Visão geral, selecione a lista de tarefas a ser excluída.

  3. No menu suspenso, selecione Excluir.

Você poderá imediatamente desfazer essa ação clicando no link Desfazer  que aparece na parte superior da página do navegador. A lista de tarefas é movida de volta para o seu local original.
