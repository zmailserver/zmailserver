Solicitar uma confirmação de leitura

As mensagens que você envia para outros usuários do @ClientNameShort@podem ser marcadas para solicitar uma confirmação de leitura quando o destinatário abre a sua mensagem. No entanto, os destinatários precisam ter o recurso Confirmação de leitura ativado em suas contas.

Para definir isto como o padrão

•       Vá até a página Preferências>E-mail, e na seção Escrevendo mensagens, selecione Sempre solicitar confirmação de leitura.

Para definir esta solicitação por mensagem

•       Antes de enviar a mensagem, na lista suspensa Opções, selecione Solicitar confirmação de leitura.
