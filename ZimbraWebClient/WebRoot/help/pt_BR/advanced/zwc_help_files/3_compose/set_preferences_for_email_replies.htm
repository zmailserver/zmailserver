Definir as preferências para respostas de e-mails

Para responder à mensagem no mesmo formato em que foi recebida, selecione Responder/encaminhar usando o formato da mensagem original. Isto assegura que o destinatário receba a mensagem no formato mais adequado para a leitura.

  1. Vá até a página Preferências>E-mail, seção Escrevendo mensagens.

  2. Para responder à mensagem no mesmo formato em que foi recebida, selecione Responder/encaminhar usando o formato da mensagem original.

  3. Selecione se deseja incluir o texto original na sua mensagem de resposta. Se for incluir a mensagem original:

    4. •   Selecione se as respostas anteriores devem ser marcadas com um caractere ou se os detalhes originais devem ser incluídos no cabeçalho (Os campos De, Para e Assunto). Você também pode incluir a mensagem original como um anexo.

    •    Se você deseja incluir as mensagens anteriores como um prefixo, selecione o colchete angular (>) ou barra vertical (|).

  4. Clique em Salvar.
