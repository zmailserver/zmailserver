Definir as preferências para e-mails encaminhados

Quando você encaminha um e-mail, é possível escolher como administrar o texto original ao encaminhar uma mensagem de e-mail.

  1. Vá até a página Preferências>E-mail, seção Escrevendo mensagens.

  2. Se você deseja incluir a mensagem original como um prefixo, selecione selecione o colchete angular (>) ou barra vertical (|)

Observação:   Se você selecionou um prefixo para respostas de mensagens, é preciso usar o mesmo prefixo para respostas e mensagens encaminhadas.
