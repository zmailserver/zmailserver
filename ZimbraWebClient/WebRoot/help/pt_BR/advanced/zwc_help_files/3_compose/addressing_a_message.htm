Endereçando uma mensagem

O recurso de preenchimento automático sugere nomes à medida que você digita nos campos Para, Cc, e Cco em uma nova mensagem. Uma lista de possíveis endereços de seus contatos cujo nome, sobrenome ou endereço de e-mail começam com o texto que você inseriu é exibida.

À medida que continua a digitar, a lista de endereços correspondentes diminui, já que menos resultados estão disponíveis. A lista desaparece se não houver nenhuma correspondência.

Por exemplo, para enviar um e-mail para Daniel Brito, cujo endereço de e-mail é dbrito@acme.com, quando você digita D, uma lista é exibida mostrando todas as correspondências possíveis, com as porções correspondentes realçadas. Nesse caso, poderia ser exibido:

•      Darla Eustáquio <deust@acme.com>

•      Daniel Brito <dbrito@acme.com>

•      Erica Dorneles <erica878@xyz.com>

Quando você adiciona um “a” (Da) no campo Para, a lista irá exibir:

•      Darla Eustáquio <deust@acme.com>

•      Daniel Brito <dbrito@acme.com>

Usar preenchimento automático de endereços

Para selecionar uma correspondência da lista, é possível clicar no resultado com o mouse ou usar uma das teclas a seguir para executar uma ação:

•      As teclas vírgula, ponto-e-vírgula, Enter e Tab, todas completam a primeira correspondência da lista e a substituem no campo.

•      ESC oculta a lista.

•      As setas para cima e para baixo alteram a seleção na lista. Mover o cursor do mouse sobre a lista também altera a seleção.

Para selecionar um endereço ao escrever um e-mail

  1. Digite caracteres suficientes até que o contato desejado seja a primeira correspondência.

  2. Pressione uma tecla de atalho, como Enter ou ponto-e-vírgula. O endereço é exibido no campo do destinatário.
