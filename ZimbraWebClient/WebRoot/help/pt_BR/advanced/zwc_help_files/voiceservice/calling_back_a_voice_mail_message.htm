Retornando uma chamada da mensagem de correio de voz

Observação:  Este recurso é exclusivo do ZCS Network Edition e está disponível somente no Cliente Web Avançado.

Responda a uma mensagem do correio de voz ligando para o chamador por meio do recurso Clicar para chamar.

  1. Selecione a guia Voz.

  2. Na lista de histórico do contato, selecione a mensagem do correio de voz a que você deseja responder.

  3. Clique em Retornar chamada. O número associado com o chamador será discado.
