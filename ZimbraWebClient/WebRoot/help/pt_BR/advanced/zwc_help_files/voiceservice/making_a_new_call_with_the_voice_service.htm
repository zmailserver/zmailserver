Fazendo uma nova chamada com o serviço de voz

Observação:  Este recurso é exclusivo do ZCS Network Edition e está disponível somente no Cliente Web Avançado.

É possível fazer chamadas de todos os aparelhos telefônicos registrados, como clientes de softphone Cisco ou Mitel.

  1. Na página Voz, clique em Nova chamada. Uma nova caixa de diálogo de chamada será exibida.

  2. Selecione o número De no menu suspenso. Os números listados são aqueles registrados para você.

  3. Insira ou selecione o número Para para o qual você deseja fazer a chamada. O número poderá ser obtido na sua lista de endereços ou cartões de contato, ou você poderá inserir um número manualmente.

  4. Clique em OK para fazer a chamada. Use o aparelho telefônico associado com o número de telefone que você está usando para fazer a chamada

Encerrando uma chamada

Para encerrar uma chamada, desligue o telefone ou clique na caixa de diálogo para desligar.
