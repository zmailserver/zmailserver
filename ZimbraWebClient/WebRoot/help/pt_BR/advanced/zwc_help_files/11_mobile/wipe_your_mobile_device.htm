Limpar seu dispositivo móvel

Caso seu dispositivo móvel seja perdido ou roubado, você pode remover todo o conteúdo dele através do link Limpar dispositivo na página Dispositivos móveis de Preferências. Este comando exclui todos os dados do dispositivo e restaura-o para as configurações de fábrica.

  1. Vá para Preferências>página Dispositivos móveis.

  2. Selecione o dispositivo para limpar e clique em Limpar todos os dados do dispositivo.

O status Limpeza pendente indica que o servidor está aguardando o dispositivo se conectar. Quando o dispositivo estiver limpo, o status mudará para Limpeza concluída.

Se você localizar o dispositivo enquanto o status for Limpeza pendente, você poderá cancelar a solicitação de limpeza.

Não execute este comando para realizar a limpeza de um dispositivo que esteja em sua posse.
