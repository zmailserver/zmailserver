Imprimir uma conversação (Todas as mensagens)

  1. Clique com o botão direito na conversão a ser impressa e selecione Imprimir. A conversação é aberta em uma nova janela. Uma caixa de diálogo de impressão padrão é exibida.

  2. Selecione uma impressora.

  3. Clique em OK. A conversação é enviada para a impressora selecionada.

  4. Feche a janela de conversação.

Alternativamente, você pode selecionar Imprimir no links Ações da barra de ferramentas.

Imprimir uma mensagem em uma conversação

  1. Abra a conversação e selecione a mensagem para impressão.

  2. No final da mensagem, clique no ícone da engrenagem e selecione a impressão. A mensagem é aberta em uma nova janela. Uma caixa de diálogo de impressão padrão é exibida.

  3. Selecione uma impressora.

  4. Clique em OK. A mensagem é enviada para a impressora selecionada.

  5. Feche a janela de mensagem.
