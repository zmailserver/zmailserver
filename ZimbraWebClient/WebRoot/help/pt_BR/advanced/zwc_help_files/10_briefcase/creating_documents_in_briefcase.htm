Criando documento no porta-arquivos

Você pode criar documentos no Porta-arquivos e criar o layout do conteúdo usando as ferramentas de texto com formatação disponíveis no Porta-arquivos.

  1. Vá para a guia Porta-arquivos e selecione a pasta Porta-arquivos onde pretende criar um novo documento.

  2. Clique em Novo documento. É exibida uma página em branco:

  3. Redija o conteúdo da página usando o editor de texto com formatação. Você pode adicionar estilo, cor e tabelas à página usando os links da barra de ferramentas.

  4. Digite o nome do novo documento no campo Salvar.

  5. Clique em Salvar.
