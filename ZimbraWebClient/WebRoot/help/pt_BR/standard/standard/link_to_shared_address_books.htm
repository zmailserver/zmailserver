Link para as listas de endereços compartilhadas

Se você receber uma notificação por e-mail dizendo que recebeu acesso para compartilhar a lista de endereços de outra pessoa, você terá de criar um link para essa lista. Você terá de saber o endereço de e-mail completo do proprietário e o nome exato da lista de endereços a ser compartilhada.

Para criar um link para uma lista de endereços compartilhada:

  1. No cabeçalho Lista de endereços no painel Visão geral, clique em editar.

  2. Na barra de ferramentas, selecione Link para a lista de endereços.

  3. Insira um nome para a lista de endereços compartilhada. Esse é o nome que será exibido na sua relação de listas de endereços.

  4. Insira o endereço de e-mail e o nome da agenda do proprietário.

  5. Selecione a cor dessa lista de endereços compartilhada.

  6. Clique em Criar lista de endereços.
