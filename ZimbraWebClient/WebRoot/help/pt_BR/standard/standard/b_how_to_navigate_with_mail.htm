Guia Navegando no e-mail

Quando se abre a aba E-mail, suas mensagens de e-mail são listadas. A partir dessa visualização, você pode abrir e ler suas mensagens.

  1. As opções incluem:

  • Clique em Atualizar para recuperar novas mensagens de e-mail.

  • Clique em Escrever para iniciar novas mensagens de e-mail. Ela é aberta no painel Conteúdo.

  • Clique em Imprimir para imprimir a mensagem ou a conversa.

  • Clique em Excluir para excluir uma mensagem. A mensagem é enviada para a pasta Lixeira.

  • Use a barra de ação Mover para para mover mensagens para a pasta selecionada.

  • Use a barra de ação Ações para marcar mensagens selecionadas como lidas ou não lidas, sinalizadas ou não sinalizadas, marcadas com um marcador já criado ou sem marcação.

  • Selecione Organizado por: para visualizar suas mensagens por Data ou Assunto.

  1. Barra de cabeçalho para o Painel de conteúdo. Se você marcar a caixa de seleção no Cabeçalho, todas as mensagens de e-mail naquela página também serão marcadas.

  2. No painel conteúdo, suas mensagens são exibidas como Tópicos de conversação ou como mensagens individuais. Use a caixa de seleção para selecionar a mensagem ou tópico de conversação quando você deseja realizar uma tarefa Mover para ou Ações.
