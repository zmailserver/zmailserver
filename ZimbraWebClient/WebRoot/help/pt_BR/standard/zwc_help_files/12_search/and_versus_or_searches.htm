Buscas AND x OR.

 

Buscar mensagens que contêm tanto X quanto Y é chamada uma busca AND  (e), porque a mensagem deve atender a todos os critérios especificados para que seja considerada uma correspondência. A busca por mensagens que correspondam a qualquer um dos critérios especificados é chamada uma busca OR  (ou), porque se a mensagem contiver X ou Y, é considerada uma correspondência. Consulte

  • O trabalho com um painel de busca Básico permite criar um critério de busca AND. Se você abrir várias instâncias dois painéis de busca Básico, criará um critério de busca OR entre os dois.

  •  A seleção de várias caixas nos painéis Tags ou Status permitirá a criação de uma busca AND. Se você abrir várias instâncias do painel Tags ou Status e marcar caixas de seleção em todos eles, criará uma busca OR entre os painéis.

  • Nos painéis de busca Pastas, Zimlets e Hora, várias seleções em um único painel criam uma busca OR para esses itens.   

Observação:   Apenas "OR" aparece em uma consulta. Se você selecionou mostrar a consulta da busca na barra de Busca, a caixa de texto Busca é atualizada para mostrar a consulta resultante. Com o tipo de busca AND, a palavra AND não é exibida.
