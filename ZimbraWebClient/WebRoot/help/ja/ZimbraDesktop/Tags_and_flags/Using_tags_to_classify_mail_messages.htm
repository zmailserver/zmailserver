タグを使用してメッセージや連絡先を分類する

タグは、メールメッセージ、連絡先、タスク、および予約に適用できる個人分類システムです。 タグはこれらの項目を検索する際に便利です。たとえば、「至急調達」や「優先度－中」などのタグを付けることができます。タグの付いたメールメッセージは異なるフォルダに入っていることがあります。タグを検索すると、メッセージがどのフォルダに入っているかに関係なく、そのタグが付いているすべてのメッセージが表示されます。

  • タグは、必要に応じていくつでも作成できます。

  • 複数のタグを1つのメールメッセージまたは連絡先に適用できます。

新しいタグの定義を作成する

タグの新規作成機能は、以下のいずれかの方法で使用できます。

  • ツールバーにある[新規]の横の下矢印をクリックし、[タグ]を選択します。

  • ツールバーの[タグ]を選択します。

  • リストビュー内の項目を右クリックし、[スレッドにタグを付ける]を選択します。

  • 概要ペインの[タグ]またはタグを右クリックし、[新しいタグ]を選択します。

[新しいタグを作成]ダイアログが表示されます。名前を入力し、[OK]をクリックします。タグ名には、以下の文字を除き、任意の文字を使用できます。

 :（コロン）

/（スラッシュ）

"（二重引用符）

新しいタグは、概要ペインの[タグ]領域に一覧され、メールメッセージや連絡先に適用できるようになります。
