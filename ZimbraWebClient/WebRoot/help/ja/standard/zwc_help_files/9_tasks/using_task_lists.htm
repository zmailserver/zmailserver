タスクリストを使用する

タスク機能を使用すると、To-Doリストを作成したり、タスクを開始から完了まで管理したりできます。タスクは、デフォルトのタスクリストに追加できます。また、他のタスクリストを作成して、より具体的に（会社のプロジェクトや個人のプロジェクトなどに分けて）To-Doリストを整理することができます。

タスクを進めていくと、情報を追加したり、ステータスや完了パーセンテージを変更したり、タスクに完了のマークを付けるという作業が発生することがあります。

タスクを並べ替え

タスクリストは、件名、ステータス、完了度（％）、または期限を基準にして並べ替えることができます。

  • タスクビューで、タスクリストを選択し、[内容]ペインのツールバーで並べ替えの基準となる見出しをクリックします。

タスクを作成

  1. ツールバーの[新しいタスク]をクリックします。

  2. [詳細設定]領域で、タスク情報を入力します。タスクに名前を付けるには、[件名]のみが必須です。

  3. [日付]領域で、開始日と期限を設定します。

  4. [保存]をクリックします。
