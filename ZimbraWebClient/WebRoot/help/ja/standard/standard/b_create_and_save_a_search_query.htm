検索クエリを作成して保存する

同じ検索を頻繁に実行する場合は、検索クエリを作成し、それを[検索]フォルダに保存しておくと便利です。その検索を再び実行したいときには、保存した検索フォルダをクリックするだけで検索が実行され、その結果が内容ペインに表示されます。

  1. 概要ペイン > [検索]見出しにある[編集]をクリックします。  

  2. ツールバーの[新しい検索]を選択します。

  3. [名前]テキストボックスに、検索の名前を入力します。

  4. [親フォルダ]テキストボックスで、この検索の保存場所を選択します。このフォルダは、概要ペインの[検索]見出しの下に保存されます。検索を別のフォルダに保存する場合は、ドロップダウンリストからそのフォルダを選択します。検索フォルダには虫眼鏡のアイコンが表示されます。

  5. [検索クエリ]テキストボックスで、検索クエリを作成します。使用可能な検索構文の詳細については、「クエリ言語の記述」を参照してください。

  6. [検索]をクリックします。

保存した検索を実行するには、概要ペインでその検索フォルダをクリックします。内容ペインに検索結果が表示されます。
