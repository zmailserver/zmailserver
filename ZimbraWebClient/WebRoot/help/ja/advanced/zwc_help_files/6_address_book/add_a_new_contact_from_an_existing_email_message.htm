既存のメールメッセージから新しい連絡先を追加

[送信者]、 [宛先]、[Cc]、および[Bcc]欄からメールアドレスを直接アドレスブックに追加できます。

  1. メールメッセージ内で追加する名前を右クリックし、[連絡先に追加]を選択します。

    2. 連絡先フォームが開きます。このとき、メールヘッダー内の情報はすでにフォームに記入されています。

  2. その他の情報を追加した後、あらかじめ記入されている情報を確認します。

  3. 連絡先を保存する方法と場所を選択します。

    4. •    [保存するファイルの名前]で、名前の保存形式を選択します。デフォルトでは、「姓、名」の形式となっています。

    •    [場所]で、連絡先を保存するアドレスブックを選択します。

  4.  [保存]をクリックします。
