メール返信のプリファレンスを設定

受信したメッセージと同じ形式で返信するには、[元のメッセージの形式を使用して返信/転送]を選択します。これにより、受信者は最も読みやすい形式のメッセージを受信できます。

  1. [プリファレンス]>[メール]ページの[メッセージの作成]領域へアクセスします。

  2. 受信したメッセージと同じ形式で返信するには、[元のメッセージの形式を使用して返信/転送]を選択します。

  3. 返信メッセージに元のテキストを含めるかどうかを選択します。元のメッセージを含める場合：

    4. •   前の返信に文字でマークを付けるか、ヘッダーに元の詳細を含めるかを選択します（[宛先]、[差出人]および[件名]の各欄）。元のメッセージを添付ファイルとして含めることもできます。

    •    前のメッセージにプレフィックスを付けたい場合は、山括弧（>）か縦線（|）を選択します。

  4. [保存]をクリックします。
