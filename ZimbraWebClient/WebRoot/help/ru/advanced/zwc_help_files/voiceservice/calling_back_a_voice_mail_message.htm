Вызов в ответ на голосовое сообщение

Примечание.  Эта функция доступна только в выпуске ZCS Network Edition на расширенной версии веб-клиента.

Позвоните отправителю с помощью функции «Звонок щелчком» в ответ на голосовое сообщение.

  1. Перейдите на вкладку «Голосовая почта».

  2. В списке журнала вызовов выберите необходимое голосовое сообщение.

  3. Выберите Обратный вызов. Будет набран номер, связанный с соответствующим абонентом.
