Настройка параметров безопасности для использования S/MIME

Примечание.  Эта функция доступна только в выпуске ZCS Network Edition на расширенной версии веб-клиента.

Если для вашей учетной записи включена функция S/MIME, в разделе «Параметры» будет доступна страница «Безопасность».

Параметр «Безопасность» по умолчанию настроен на автоматическое запоминание последнего параметра безопасности, который был выбран при составлении сообщения электронной почты. 

Параметр по умолчанию можно изменить, выбрав отправку только подписанных сообщений, только подписанных и зашифрованных или неподписанных и незашифрованных.  Этот параметр можно изменить на странице создания сообщения на вкладке «Безопасность», но изменение будет применено только к данному сообщению.

  1. Нажмите кнопку Настройки.

  2. В области обзора выберите Безопасность.

  3. Измените настройки.

    • Параметр Автоматически (запоминается последняя настройка) установлен по умолчанию. Запоминается последний выбранный параметр безопасности.

    • Выберите Не использовать подпись или не выполнять шифрование, если функция S/MIME вообще не используется или используется крайне редко.

    • Выберите Только использовать подпись, если требуется всегда отправлять сообщения электронной почты с цифровой подписью.

    • Выберите Использовать подпись и выполнить шифрование, если требуется всегда отправлять зашифрованные сообщения электронной почты с цифровой подписью.

  1. Нажмите кнопку Сохранить.
