Составление и отправка сообщения с цифровой подписью

Примечание. Эта функция доступна только в выпуске ZCS Network Edition на расширенной версии веб-клиента.

Сообщения с цифровой подписью могут быть отправлены адресатам, которые отправили вам открытый ключ и сертификат. Сертификат адресата сохраняется в разделе адресатов на странице контактов в адресной книге. 

  1. Выберите соответствующий элемент на панели инструментов.

  2. В поле Кому: введите адрес пользователя или выберите ссылку Кому, чтобы искать адрес в своих адресных книгах.

  3. Если для параметра безопасности не установлена по умолчанию отправка сообщений с цифровой подписью, щелкните вариант Безопасность и выберите Только использовать подпись.

  4. Введите тему сообщения.

  5. Составьте сообщение.

  6. Чтобы добавить вложение, выберите команду Вложить файл и перейдите к файлу или перетащите файл из папки на компьютере в заголовок сообщения электронной почты.

  7. Нажмите кнопку Отправить.
