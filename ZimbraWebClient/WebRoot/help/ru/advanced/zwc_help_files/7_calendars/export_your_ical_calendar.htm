Экспорт ежедневника iCal

Можно экспортировать ежедневники в формате .ics.

  1. Перейдите на страницу Настройки>Импорт и экспорт.

  2. В меню Экспорт выберите тип Ежедневник. Можно экспортировать встречи в стандартном формате .ics.

    3. Чтобы импортировать их в другую программу-ежедневник, см. справку по импорту в документации соответствующей программы.

  3. В разделе Источник выберите ежедневник, который необходимо экспортировать.

  4. Нажмите кнопку Экспорт. Выберите команду Сохранить на диске и нажмите кнопку OK.

  5. Выберите расположение для сохранения файла и введите имя файла.

  6. Нажмите кнопку Сохранить.
