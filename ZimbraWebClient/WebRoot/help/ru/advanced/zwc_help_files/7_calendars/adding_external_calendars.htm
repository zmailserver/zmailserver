Добавление внешних ежедневников

В список ежедневников можно добавлять внешние ежедневники Google или Yahoo CalDAV, а также подписки на ежедневники iCal. Изменения во внешних ежедневниках синхронизируются с добавленным ежедневником CalDAV.

Примечание.   Только ежедневники CalDAV с включенным протоколом SSL, для которых используется порт 443, могут синхронизировать события ежедневника с учетной записью.

  1. В области обзора ежедневника щелкните значок с шестеренкой и выберите пункт Добавить внешний ежедневник.

  2. Выберите, какой тип ежедневника необходимо добавить, и нажмите кнопку Далее.

  3. Выберите тип ежедневника, который необходимо добавить: Учетная запись CalDAV или Подписка iCal.

  4. Введите адрес электронной почты и пароль для данной учетной записи.

  5. Нажмите кнопку Далее.

  6. В текстовом поле Имя введите имя для этого ежедневника и выберите цвет.

  7. Чтобы исключить этот ежедневник из отчета о свободном и занятом времени, выберите команду Исключить этот ежедневник при составлении отчета о свободном/занятом времени.

  8. Нажмите кнопку ОК.

Ежедневник добавлен, и встречи из внешнего ежедневника синхронизируются с добавленным ежедневником.
