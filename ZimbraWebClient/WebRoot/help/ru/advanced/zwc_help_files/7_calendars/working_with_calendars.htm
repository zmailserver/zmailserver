Работа с календарями

С помощью ежедневника можно отслеживать и планировать встречи, собрания и события. Можно вести несколько ежедневников и перемещать события из одного ежедневника в другой. Пользователи также могут предоставлять общий доступ к своим ежедневникам другим пользователям.

Чтобы открыть ежедневник, перейдите на вкладку Ежедневник. Вид «Рабочая неделя» является видом по умолчанию. Рабочие дни и рабочее время отображаются белым цветом. Нерабочее время отображается серым цветом.

Пользователи могут изменять настройки своих ежедневников на странице «Настройки>Ежедневник».

Настройка вида ежедневника по умолчанию

Вид «Рабочая неделя» отображается по умолчанию. Можно изменить вид, который отображается при открытии ежедневника.

  1. Перейдите на страницу Настройки>Ежедневник.

  2. Выберите вид из раскрывающегося меню Вид по умолчанию.

    3. Параметр

    Описание

    День

    Отображаются мероприятия на день.

    Рабочая неделя

    Отображаются мероприятия с понедельника по пятницу.

    7-дневная неделя 

    Отображаются мероприятия на семь дней. В раскрывающемся меню Начинать неделю с дня можно выбрать день, с которого должна начинаться неделя.

    Месяц

    Отображаются мероприятия на месяц. При просмотре ежедневника в виде «Месяц» можно дважды щелкнуть дату, чтобы встречи дня, отображаемые по часам, были показаны в виде «рыбий глаз».

    Вид списка

    Отображается список встреч в рамках указанного диапазона дат для ежедневников, выбранных в области обзора.

  3. Нажмите кнопку Сохранить.
