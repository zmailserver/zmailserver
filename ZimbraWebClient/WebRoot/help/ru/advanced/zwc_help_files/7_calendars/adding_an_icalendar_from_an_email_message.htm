Добавление файла iCalendar из сообщения электронной почты

При получении сообщения электронной почты, в котором содержится ссылка на файл .ics, можно добавить ежедневник в список ежедневников в области обзора.

  1. В сообщении щелкните правой кнопкой мыши ссылку на ежедневник .ics.

  2. Выберите пункт Создать новый ежедневник.

  3. Введите имя ежедневника и выберите для него цвет.

  4. Чтобы исключить этот ежедневник из отчета о свободном и занятом времени, выберите команду Исключить этот ежедневник при составлении отчета о свободном/занятом времени.

  5. Нажмите кнопку ОК.
