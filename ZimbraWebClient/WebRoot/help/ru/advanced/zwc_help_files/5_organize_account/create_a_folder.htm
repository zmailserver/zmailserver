Создание папки

Можно создавать отдельные определенные пользователем папки или папки внутри других папок, включая системные.

Нельзя создавать две папки с одинаковыми именами в одной родительской папке. Например, в папке «Входящие» не может быть двух папок Покупки, однако одна папка Покупки может находиться в папке «Входящие», а другая — в папке «Отправленные».

  1. Перейдите на страницу Почта>Папки и щелкните значок с шестеренкой «Папки».

  2. Выберите пункт Новая папка.

  3. Введите имя новой папки и выберите цвет папки.

    4. Имена папок могут содержать любые символы, кроме двоеточия (:), косой черты
    (/) и кавычки (").

    Можно подписаться на ленту RSS/ATOM.

  4. В дереве Папки выберите расположение новой папки.

    5. •    Щелкните Папки, чтобы новая папка отображалась вверху списка.

    •    Щелкните существующую папку, чтобы разместить в ней новую папку.

  5. Нажмите кнопку ОК.
