Получение подтверждений прочтения

Сообщения, получаемые от других пользователей @ClientNameShort@, могут быть помечены для запроса подтверждения получения, которое будет отправлено при открытии сообщения. Можно указать, как обрабатывать запросы подтверждения получения.

  1. Перейдите на страницу Настройки>Почта, в раздел Получение сообщений.

  2. В разделе Подтверждение прочтения выберите параметры обработки запросов подтверждения прочтения.

Параметр

Описание

Никогда не отправлять подтверждение прочтения

При получении сообщения с запросом подтверждения прочтения этот запрос игнорируется. Пользователю не будет сообщено о запросе. Это настройка по умолчанию.

Всегда отправлять подтверждение прочтения

Сообщение о подтверждении прочтения будет отправляться при открытии сообщения электронной почты, помеченного флагом для запроса подтверждения получения.

Спрашивать меня

Вам будет предоставлена возможность отправить подтверждение получения при открытии сообщения электронной почты, помеченного флагом для запроса подтверждения получения.

  1. Нажмите кнопку Сохранить.
