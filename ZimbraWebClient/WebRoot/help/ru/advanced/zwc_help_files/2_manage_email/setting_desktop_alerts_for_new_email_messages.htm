Настройка уведомлений о новых сообщениях электронной почты

Можно настроить уведомления о поступлении новых почтовых сообщений. Для получения уведомлений почтовый ящик должен быть открыт.

  1. Перейдите на страницу Настройки>Пота.

  2. В разделе Получение сообщений>Доставка сообщения выберите настройки уведомления о новых сообщениях. Можно выбрать любые из следующих параметров.

Параметр

Описание

Воспроизведение звука

Уведомлением будет служить звуковой сигнал.  Если звук отключен, вы не услышите сигнал. Для работы этой функции необходимо установить QuickTime или Windows MediaCenter.

Подсветка вкладки «Почта»

Если вы не работаете на вкладке «Почта», она будет выделена.

Мерцание заголовка браузера

Строка заголовка браузера будет мигать, пока не будет открыта вкладка «Почта».

Показать всплывающее уведомление

Будет отображена информация о сообщении. Для работы этой функции должно быть установлено приложение Yahoo!BrowserPlus. Чтобы получить копию, перейдите на веб-сайт http://browserplus.yahoo.com/.
