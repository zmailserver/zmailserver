Настройка исходного вида почтового ящика

На странице «Настройки почты» параметр Поиск в сообщениях по умолчанию задает первоначальный вид, который отображается после входа в почтовый ящик. По умолчанию поиск выполняется в папке «Входящие». Следовательно, после входа отображаются результаты из папки «Входящие».

Для отображения при первом открытии веб-клиента можно выбрать другую папку, тег или сохраненный профиль поиска.

  1. Перейдите на страницу Настройки>Почта.

  2. В текстовом поле Отображение сообщений>Поиск в сообщениях по умолчанию укажите, какие сообщения будут отображаться при открытии почтового ящика. Введите ключевое слово поиска, а затем добавьте : (двоеточие) и папку, метку или имя сохраненного профиля поиска.

    3. В приведенной ниже таблице содержатся примеры заполнения этого текстового поля.

    :непрочитанное

    Отображение только непрочитанных сообщений.

    tag:<имя_тега>

    Отображение сообщений с определенными тегами. Например, чтобы отображались только сообщения с тегом «Задачи», введите «tag:Задачи».

    in:<имя_папки>

    Отображение сообщений, отправленных в определенную папку с помощью фильтра почты.

    Примечание.  Если тег или имя папки содержит более одного слова, необходимо использовать кавычки. Например, «in:”Непрочитанные сообщения”».

  3. Нажмите кнопку Сохранить.
