Использование средства планирования для просмотра доступности места

Средство планирования ежедневника может помочь в поиске ближайших доступных времени и мест в пределах организации. Можно задать предпочтительные время собрания, здание, место и размер помещения.

При вводе имен участников средство планирования начинает отображать в области обзора предлагаемые время и места. У вас также есть возможность поиска места проведения собрания до добавления участников. Если указаны предпочитаемые места, отображаться будут только они.

Использование средства планирования для установки настроек местоположения

Если требуется всегда резервировать помещения с определенным местонахождением, можно воспользоваться инструментом предложения мест в ежедневнике, который при создании встречи будет отображать доступность только этих помещений.

  1. На странице Новая встреча щелкните Предложить расположение.

  2. В открывшейся области щелкните значок с шестеренкой.

  3. В диалоговом окне Настройки предложений в разделе «Настройки местоположения» введите данные местоположения. Можно ввести только имя местоположения или указать более конкретную информацию.

  4. Нажмите кнопку ОК.

В столбце справа отображаются места, соответствующие вашим требованиям. Отображается расположение помещений, если оно было настроено в каталоге компании.

После выполнения данных настроек для встречи, эти данные сохраняются и используются при создании других встреч.
