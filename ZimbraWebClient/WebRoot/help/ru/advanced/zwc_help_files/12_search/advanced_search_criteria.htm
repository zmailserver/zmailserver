Доступные фильтры поиска

В таблице ниже содержатся сведения о детализации поиска.

На вкладке «Поиск» можно выбрать дополнительные фильтры для создания более сложных запросов.

Дополнительные фильтры

Описание

Получено от

Поиск определенных доменов в заголовке сообщения (поля «От», «Кому» и «Копия»). Домены, которые можно искать, предопределены.

Отправлено (кому)

Поиск определенных доменов в заголовке сообщения (поля «От», «Кому» и «Копия»). Доступные для поиска домены предопределены.

Дата отправки

Поиск сообщений, полученных на выбранную дату, до или после нее. Чтобы выполнить поиск по диапазону дат, выберите Дата отправки предшествует, а затем выберите Дата отправки следует после и введите диапазон дат.  

Вложения

Поиск сообщения электронной почты по наличию вложения. При поиске вложения определенного типа выберите нужный тип в отображаемом списке форматов.

Размер

Поиск сообщений по размеру (в байтах, КБ или МБ).

Cтатус

Поиск сообщений или контактов по их состоянию (например, прочитанные, непрочитанные, пересланные сообщения или сообщения, на которые был дан ответ).

Метка

Поиск сообщений или контактов с определенной меткой.

Папка

Ограничение поиска только выбранными папками.
