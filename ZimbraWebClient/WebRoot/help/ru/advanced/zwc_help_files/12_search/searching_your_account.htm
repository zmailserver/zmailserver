Поиск учетной записи

Поиск позволяет находить сообщения, контакты, встречи, задачи и файлы. Поиск можно выполнять по определенным словам, датам, времени, URL-адресу, размеру, метке, по состоянию сообщения («прочитано» или «непрочитано»), по наличию или отсутствию вложений, по определенному типу вложений и по другим параметрам.

С помощью функции «Поиск» можно выполнить любой текущий запрос поиска, отображаемый в текстовом поле поиска. В поле поиска можно ввести одно из ключевых слов поиска, после чего автоматически отобразится список возможных аргументов поиска. Кроме того, с помощью стрелки раскрывающегося списка слева от текстового поля поиска можно выбрать тип элементов для поиска. Поиск можно выполнять в сообщениях, контактах, в том числе в списках контактов компании, встречах, задачах и файлах.

Можно выполнить более сложный поиск, если воспользоваться специальными параметрами запроса поиска, такими как фильтры и условия. Запросы поиска можно сохранять для повторного использования в будущем.

Быстрый поиск элемента

  1. Выберите папку, в которой необходимо выполнить поиск, или в раскрывающемся меню поиска укажите ту часть учетной записи, в которой необходимо искать.

  2. В текстовом поле Поиск введите текст поиска. С параметрами поиска можно ознакомиться в разделе Синтаксис поиска.

  3. Нажмите клавишу Enter. Откроется вкладка «Поиск» с результатами поиска.

Поиск выполняется на странице поиска.

  1. В поле «Поиск» щелкните лупу. Отобразится страница поиска.

  2. На панели навигации перейдите по одной из ссылок раскрывающегося списка, чтобы создать запрос поиска. Синтаксис поиска отображается в выноске в поле поиска, расположенной в верхней части страницы.
