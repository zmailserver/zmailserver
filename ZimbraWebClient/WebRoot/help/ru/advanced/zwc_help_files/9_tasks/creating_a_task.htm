Создание задачи

Новую задачу можно создать, введя имя задачи в списке задач или открыв страницу новой задачи и добавив дополнительные данные о задаче, включая заметки и вложения.

Быстрое добавление задачи

  1. Перейдите на страницу Задачи и в области обзора выберите список задач, в который нужно добавить задачу.

  2. В текстовом поле Щелкните здесь, чтобы добавить новую задачу в области содержимого введите имя новой задачи.

  3. Нажмите клавишу Enter. Таким образом добавляется тема задачи в список задач, но не вводятся подробные сведения. Для добавления подробных сведений отредактируйте задачу.

Примечание.   Если вы по ошибке создали задачу не в том списке, откройте задачу и выберите нужный список в раскрывающемся списке задач или перетащите задачу в нужный список в области содержимого.
