Фильтрование задач по состоянию

Чтобы быстро просмотреть состояние выполнения задач, можно отфильтровать их.

•       В раскрывающемся меню Фильтровать по на панели инструментов Задача выберите тип задач для просмотра.

Параметр

Описание

Не начато

Отображение задач, выполнение которых не начато.

Завершено

Отображение завершенных задач.

Выполняется

Отображение выполняемых в настоящий момент задач.

Ожидание других

Отображение задач, ожидающих участия других лиц.

Отложенные

Отображение отложенных задач.

Все

Отображение всех задач независимо от их состояния.

Список дел

Отображение задач с состоянием «Выполняется», «Не начато» или (для просроченных задач) «Ожидание других».
