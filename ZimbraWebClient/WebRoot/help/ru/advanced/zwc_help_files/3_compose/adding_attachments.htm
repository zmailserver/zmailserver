Добавление вложений 

В сообщение можно вложить любой файл, который можно найти с помощью кнопки «Обзор» в файловой системе.

Любой вложенный файл можно открыть прямо в почтовом ящике, если установлено соответствующее приложение и открытие файлов этого типа не заблокировано администратором.

Если тип файла поддерживается установленным на компьютере программным обеспечением, обычно можно дважды щелкнуть этот файл, и компьютер автоматически запустит нужное приложение для его чтения.

Компьютерные вирусы часто распространяются посредством вложенных файлов. Поэтому системные администраторы могут заблокировать входящую почту, содержащую вложения определенных типов, часто с расширением .EXE или .ZIP в имени файла. При отправке сообщения пользователю, система электронной почты которого настроена на блокировку вложений определенных типов, отправитель может не получить извещение, если сообщение было заблокировано.

Присоединение вложения к сообщению

К сообщению электронной почты можно присоединять документы, электронные таблицы, изображения, презентации и другие типы файлов.

  1. Создание сообщения электронной почты.

  2. Под полем Тема нажмите кнопку Вложить и выберите пункт Мой компьютер.

  3. Выберите файлы и нажмите кнопку Открыть. Имена файлов отображаются под текстовым полем «Тема».

  4. Чтобы отправить сообщение и вложения, нажмите кнопку Отправить.

Примечание.   Чтобы открыть файл, получатели должны иметь соответствующее программное обеспечение. Файлы распространенных форматов, например текстовые файлы, файлы HTML и изображения (GIF или JPEG) можно открывать с помощью различных программ.

Добавление вложений при помощи перетаскивания

Чтобы добавить вложение к сообщению электронной почты, можно просто перетащить файл из папки в сообщение.

  1. Выберите один или несколько файлов, наведите на них курсор, а затем нажмите и удерживайте левую кнопку мыши.

  2. Перетащите файлы в область заголовка сообщения и отпустите кнопку. В заголовке отобразятся имена файлов.

Удаление вложения

Чтобы удалить вложение, щелкните значок x в выноске вложения.
