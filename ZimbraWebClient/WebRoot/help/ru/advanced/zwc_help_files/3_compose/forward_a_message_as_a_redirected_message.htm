Пересылка сообщения в качестве перенаправляемого

Функция перенаправления сообщений позволяет отправлять полученные сообщения электронной почты так, словно их отправляет автор исходного сообщения. В случае перенаправления сообщения вместо пересылки получателю будет легче ответить исходному отправителю.

  1. Щелкните правой кнопкой мыши сообщение и выберите пункт Перенаправить.

  2. В текстовом поле Перенаправить укажите адреса получателей.

  3. Нажмите кнопку ОК.
