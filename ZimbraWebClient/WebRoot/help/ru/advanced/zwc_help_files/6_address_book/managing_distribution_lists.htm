Управление списками рассылки

При отправке сообщений одной группе адресов на вкладке «Адресная книга» можно создать списки рассылки. Список рассылки — это группа почтовых адресов, которые входят в состав списка обычных почтовых адресов. При отправке на адрес списка рассылки сообщение электронной почты отправляется на все адреса, включенные в этот список.  Для адресата имя списка рассылки отображается в строке «Кому» такого сообщения, и при использовании вида выноски пользователь может щелкнуть значок «+», чтобы посмотреть участников списка рассылки.

Списки рассылки хранятся в адресной книге «Списки рассылки».  Если щелкнуть список рассылки в адресной книге, все его участники будут отображены в поле «Содержимое».

Можно создать список рассылки как список участников и управлять участием в нем.  Настройка дополнительных параметров открывает следующие возможности.

  • Можно пометить список как личный, чтобы он  не отображался в глобальном списке адресов.

  • Можно ограничить возможность отправки сообщений для пользователей из списка рассылки.

  • Можно предоставить пользователям из списка право подписываться и отменять подписку.

  • Можно задать нескольких владельцев для облегчения управления списком.

Быстрое создание списка рассылки

  1. Перейдите к раскрывающемуся меню Адресная книга>Новый контакт и выберите вариант Список рассылки.

  2. Введите имя списка рассылки в заголовке страницы. Это адрес электронной почты списка рассылки.

  3. Введите Короткое имя для списка рассылки. Оно может отличаться от адреса электронной почты.

  4. Введите описание списка рассылки (необязательно).

  5. Откройте вкладку Участники и выберите адреса, которые необходимо добавить в список.  Можно выполнять поиск адресов в списках контактов или в глобальном списке адресов, а также можно добавлять адреса, которые не включены в эти списки.

  6. Нажмите кнопку Сохранить.  Список рассылки добавляется в адресную книгу списков рассылки.
