Создание фильтра из сообщения электронной почты

Можно создать фильтр на основе сообщения электронной почты в папке «Входящие». Сведения о фильтре извлекаются из сообщения, и существует возможность изменения условий и действий.

  1. Щелкните сообщение электронной почты правой кнопкой мыши и выберите пункт Новый фильтр.

    2. Откроется диалоговое окно «Добавить фильтр», содержащее сведения из сообщения электронной почты.

  2. В текстовом поле Имя фильтра введите уникальное имя фильтра.

  3. При необходимости измените условия и действия с помощью раскрывающихся меню.

  4. Нажмите кнопку ОК.
