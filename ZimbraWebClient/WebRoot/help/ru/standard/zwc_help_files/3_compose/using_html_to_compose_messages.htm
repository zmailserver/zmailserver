Использование HTML для составления сообщений

Сообщения, созданные с помощью HTML-редактора, могут содержать форматированный текст, маркированные и нумерованные списки, цветной фон, таблицы и ссылки.  Однако некоторые почтовые клиенты могут не принимать сообщения с HTML-форматированием. Редактор можно менять для конкретного сообщения.

Примечание.   Для использования HTML-редактора необязательно знать HTML.

Чтобы изменить формат отправляемого сообщения, в разделе «Параметры» выберите параметр Формат HTML или Формат обычного текста.
