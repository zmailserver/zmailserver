Использование списков задач

Функция Задачи позволяет создавать список задач и управлять их выполнением. Задачи можно добавлять в список задач по умолчанию, можно создавать другие списки задач для систематизации списков задач по более конкретным видам деятельности, например, по работе или личным проектам.

При выполнении задачи может потребоваться добавить информацию, изменить состояние и процент выполнения или отметить задачу как завершенную.

Сортировка задач

Список задач можно сортировать по теме, статусу, проценту выполнения и дате выполнения.

  • В виде «Задача» выберите список задач и на панели инструментов области просмотра щелкните заголовок для сортировки.

Создание задачи

  1. На панели инструментов нажмите кнопку «Новая задача».

  2. В разделе Подробности введите информацию о задаче. Чтобы присвоить задаче имя, необходима только тема.

  3. В разделе Дата введите дату начала и дату окончания.

  4. Нажмите кнопку Сохранить.
