Поиск с условиями «И» и «ИЛИ».

 

Поиск сообщений, которые содержат X и Y, называется поиском с условием И, поскольку сообщение должно соответствовать всем указанным критериям, чтобы считаться соспадением. Поиск сообщений, соответствующих любым указанным критериям, называется поиском с условием ИЛИ, поскольку если сообщение содержит X или Y, оно считается совпадением. См.

  • Работа с одной областью простого поиска позволяет создать запрос поиска с условием «И». Если открыть несколько экземпляров областей простого поиска, можно создать запрос поиска с условием «ИЛИ» между этими областями.

  •  Установив несколько флажков в области «Метки» или «Статус», можно создать запрос поиска с условием «И». Если открыть несколько экземпляров области «Метки» или «Состояние» и установить разные флажки в каждой из них, можно создать запрос поиска с условием «ИЛИ» между этими областями.

  • При установке нескольких флажков в одной области поиска, такой как «Папки», «Модули Zimlet» и «Время», можно создать запрос поиска с условием ИЛИ для этих элементов.   

Примечание.   В запросе появится только условие «ИЛИ». Если задан параметр отображения запроса поиска на панели поиска, текстовое поле поиска будет обновляться и отображать результирующий запрос. В профиле поиска типа «И» слово «И» не отображается.
