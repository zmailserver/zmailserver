Правила синтаксиса поиска

Можно искать любой элемент: слово, имя, фамилию, номер телефона или имя домена. При поиске содержимого необходимо помнить о некоторых правилах. В процессе поиска содержимого поиск выполняется в теле сообщения, а также во всех его вложениях, которые читаются системой.

Параметр

Описание

Точное текстовое соответствие

Для поиска по фразам необходимо дословное соответствие. Варианты орфографии недопустимы. Например, если необходимо найти слово бананы, то сообщение со словом банан не будет соответствием.

Без учета регистра

При поиске не учитывается регистр: слова Юг, юг и ЮГ равнозначны.

Специальные знаки

В тексте поиска нельзя использовать следующие специальные знаки. ~ ' ! @ # $ % ^ & * () _- + ? / { }[ ] ; : "

Символы авторского права или товарного знака (

Специальные символы (например, товарные знаки, являющиеся частью слова), удаляются, когда слово индексируется для поиска. Эти специальные символы игнорируются в случае использования их как части целого слова в поисковом запросе. Например, поиск по слову Zimbra™ возвращает результаты со словом Zimbra с символом товарного знака и без него.

Подстановочные знаки

Поддерживается использование звездочки* в качестве подстановочного знака после префикса. Например, при выполнении поиска по слову ты* возвращаются элементы со словом тыква, тысяча и т. д.

Примечание. Двухбайтовый символ звездочки в поисковом запросе не рассматривается как подстановочный знак и игнорируется при поиске.

Вложения

Можно выполнить поиск читаемых системой вложений файлов. Читаемый системой файл — это файл, который можно конвертировать в формат HTML, например, документы и текстовые файлы Microsoft Office.

Контакты в адресных книгах

Необходимо найти целую строку со словами или цифрами. Например, для поиска контактов по номеру телефона, необходимо ввести полный номер телефона, как он указан в записи контакта. Выполнять поиск только по коду города нельзя.

Примечание.   Поиск содержимого в @ClientNameLong@ несколько отличается от поиска по текстовому соответствию и таких функций обработки текста, как «Найти».
