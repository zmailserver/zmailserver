Назначение меток сообщениям и контактам

Теги — это ваша личная система систематизации почтовых сообщений, контактов и встреч. Теги помогают найти эти элементы. Например, вы назначили один тег для элемента «Немедленный оборот» и другой тег для элемента «Средний приоритет». Помеченные тегом почтовые сообщения могут находиться в разных папках. Теги позволяют показывать все сообщения с определенными тегами где бы они не находились.

Можно создать и применить несколько тегов к одному элементу, чтобы систематизировать его сразу по нескольким категориям. После создания тега его можно применить к разговорам, сообщениям или контактам.

Создание новой метки

  1. В заголовке «Тег» области просмотра нажмите кнопку Изменить.

  2. В области содержимого щелкните Новый тег.

  3. Введите имя тега и выберите для него цвет.

  4. Выберите пункт Создать новый тег. Новая метка появится в области обзора в разделе меток.

Одновременное применение или удаление метки для нескольких элементов

  1. Установите флажки для элементов, которые нужно снабдить метками.

  2. В раскрывающемся меню Действия выберите метку, которую следует применить.

  3. Нажмите кнопку Перейти.

Удаление метки из элементов

  1. Установите флажки для элементов, для которых нужно удалить определенную метку.

  2. В раскрывающемся меню Действия в разделе «Удалить метку» выберите метку для удаления. Чтобы удалить несколько меток, выберите ВСЕ.

  3. Нажмите кнопку Перейти.
