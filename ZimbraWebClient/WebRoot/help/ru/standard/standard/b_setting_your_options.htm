Установка настроек пользователя

Настройки пользователя по умолчанию устанавливаются при создании учетной записи, они определяют работу почты, адресных книг и создания сообщений.

Эти настройки можно изменить на вкладке «Настройки».  Если описанная здесь настройка не отображается, то она не доступна для вашей учетной записи.

Общие

  • Пользователь входит в стандартный веб-клиент. Чтобы использовать этот тип клиента по умолчанию при входе, выберите параметр Стандартный (HTML). Стандартный клиент рекомендуется для медленных подключений к Интернету с использованием старых версий браузеров или для упрощенного доступа. Расширенный веб-клиент предлагает полный набор функций групповой работы в Интернете. Веб-клиент работает лучше всего с более новыми версиями браузеров и быстрым подключением к Интернету.

  • В разделе Тема выберите цвет фона для интерфейса почтового ящика.

  • Выберите часовой пояс, чтобы установить дату и время отправляемых и получаемых сообщений и верной ссылки на встречи ежедневника.  Этот параметр может отличаться от настроек часового пояса на вашем компьютере.

  • Если отображается параметр Изменить пароль, можно изменить пароль. Если при смене пароля возникают проблемы, обратитесь к системному администратору, чтобы узнать правила в отношении паролей.

  • В поиск папок могут быть автоматически включены папки «Спам» и «Корзина». По умолчанию поиск в этих папках не выполняется.

  • Для параметра Язык поиска установите флажок Всегда показывать строку поиска для отображения подробных данных строки поиска в текстовом окне поиска.

Почта

  • Выберите способ отображения элементов (сообщений или разговоров) на странице. По умолчанию на странице отображается 25 элементов, но можно выбрать 10, 50 или 100 элементов на страницу.

  • Выберите способ группирования почты по разговорам или сообщениям. Разговоры используются по умолчанию.

  • Укажите способ просмотра почты в виде HTML для сообщений, которые имеют формат HTML. Отключение этого параметра приводит к отображению почты в виде простого текста с отображением собственно тегов HTML вместо их применения для разметки текста.

  • Область чтения можно использовать для просмотра сообщений из папки «Входящие» и других папок. Если включен этот параметр, сообщения отображаются в области просмотра. Область чтения может отображаться либо под списком сообщений, либо справа в виде с тремя областями.

  • Если включен параметр Просмотр сообщений, рядом с темой в столбце «Тема» отображаются первые несколько слов текста сообщения.

  • Профиль поиска, заданный в поле Поиск в сообщениях по умолчанию, определяет поиск, выполняемый при входе в почтовый ящик. По умолчанию выполняется поиск в папке Входящие. После входа отображаются результаты из папки «Входящие». В этом поле можно выбрать для отображения при первом открытии веб-клиента другую папку, метку или сохраненный профиль поиска.

  • Включите и укажите электронный адрес для пересылки почты. Можно установить флажок, чтобы исходные сообщения удалялись из почтового ящика.

  • Включите и укажите адрес для отправки уведомлений о получении новых сообщений.

  • Включите параметр Отправить автоответ для ответа на входящую почту сообщением о том, что вас нет на рабочем месте. Введите текст этого сообщения в текстовом поле и установите начальную и конечную даты использования сообщения.

  • Настройте Сообщения от меня... для обработки отправляемых вами сообщений, которые должны включать ваш адрес, как одного из получателей сообщения, или адрес списка рассылки, участником которого вы являетесь.

  • Если используется клиент POP3, можно указать параметры  доступа по протоколу POP во время загрузки сообщений.

Написание письма

  • В окне Написать письмо выберите формат для написания сообщений: HTML или обычный текст. Формат HTML позволяет при написании сообщений использовать такие возможности форматирования текста, как шрифт, размер и цвет.

  • В пункте Ответить/Ответить всем укажите, включать ли исходный текст в тело письма вашего сообщения.

  • В пункте Переслать укажите, как включать текст исходного сообщения в пересылаемое сообщение: в тело сообщения, в тело сообщения с префиксом или в виде вложения в сообщение.

  • Знак префикса — устанавливает префикс, используемый для ответов и пересылаемых сообщений.

  • Отправленные сообщения — выберите параметр Сохранить копию в папке "Отправленные" для сохранения копии каждого отправляемого сообщения в папке «Отправленные».

Подписи

  • Можно создавать подписи для сообщений. Подпись может включать имя и дополнительный текст длиной не более максимального количества знаков, допустимого для вашей учетной записи. Если вы создаете несколько личных данных, можно создать разные подписи и назначить их определенным адресам.

  • Выберите расположение подписи. Здесь будет отображаться подпись в сообщениях-ответах. Подпись можно располагать над сообщениями, включенными в ответ, или в конце сообщения.

Адресная книга

  • Выберите количество контактов для отображения на странице. По умолчанию отображается 25 контактов.

  • Включите функцию для автоматического добавления адресов в адресную книгу во время отправки сообщений. Если этот параметр включен, адреса сохраняются в адресной книге отправленных по электронной почте контактов.

  • Ссылка Управление адресными книгами открывает страницу редактирование контактов адресной книги, на которой можно создать новые адресные книги и подключиться к общедоступным адресным книгам, экспортировать и импортировать списки контактов и удалить контакты из адресных книг.

Ежедневник

  • Выберите вид ежедневника. По умолчанию отображается рабочая неделя.

  • Укажите день недели, который должен быть первым днем недели ежедневника.

  • В разделе День/Неделя можно установить время начала и окончания дня ежедневника.

  • При поездках в разных часовых поясах можно установить флажок Показать список часовых поясов в представлении встреч.

  • Ссылка Управление ежедневниками открывает страницу редактирования ежедневника, на которой можно изменить существующий ежедневник, добавить новые ежедневники, подписаться на ежедневники или подключиться к общедоступному ежедневнику.

Учетные записи

При отправке нового сообщения, ответе или пересылке сообщения, с электронным адресом всегда связывается одна запись личных данных.  Это принятые по умолчанию личные данные, созданные как имя учетной записи. Можно изменить «описательное» имя, отображаемое в поле «От» ваших сообщений, и можно выбрать подпись для автоматического добавления в сообщения.

Фильтры почты

Фильтрация заключается в применении набора соответствующих правил к входящей почте и выполнении определенного действия. Эта вкладка используется для определения правил фильтрации сообщений, как описано в разделе Фильтрация сообщений.

Горячие клавиши

Быстрые клавиши дают возможность пользоваться приложением без помощи мыши. Список содержит все существующие комбинации быстрых клавиш.
