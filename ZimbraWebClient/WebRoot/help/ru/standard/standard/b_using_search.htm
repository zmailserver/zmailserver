Использование поиска

Функция поиска @CompanyName@ является мощным инструментом. Хотя вы можете научиться создавать сложные текстовые запросы, как описано в разделе Описание языка запросов, проще воспользоваться быстрой клавишей и ввести имя или слово без ключевых слов. Например, чтобы найти все сообщения, содержащие слово «Петр » в любом месте строки темы, полях «Кому», from: cc: или «Скрытая копия», теле сообщения или вложенном файле, можно просто ввести слово «Петр» в текстовом поле Найти и нажать кнопку Поиск.

Выберите место, в котором следует выполнить поиск:  почтовые сообщения, контакты, встречи, задачи, или файлы в портфеле.  Также можно выполнять поиск информации во всех папках.

Для поиска контактов из адресных книг в запрос необходимо вводить целое слово. Например, для поиска контактов по номеру телефона, необходимо ввести полный номер телефона, как он указан в записи контакта. Выполнять поиск только по коду города нельзя.

  1. В строке поиска в раскрывающемся меню выберите пункт Личные контакты. Поиск будет выполнен во всех адресных книгах.

  2. Введите полное имя или другие данные контакта в текстовом поле поиска.

  3. Нажмите кнопку Поиск.

Можно ввести любое значение: слово, имя, фамилию, номер телефона или домен.   Для дополнительных вариантов можно использовать звездочку (*) в качестве подстановочного знака.  

Использование * (звездочки) в качестве подстановочного знака в профиле поиска

Звездочку (*) можно использовать в качестве подстановочного знака при поиске содержимого со словами с похожим написанием.  

Используйте * (звездочку) как подстановочный знак после префикса. Например, при выполнении поиска по слову ты* возвращаются элементы со словом «тыква», «тысяча» и т. д.

  1. В строке поиска в раскрывающемся меню выберите элемент или тип элемента для поиска.

  2. Введите в поле поиска слово, имя или другие данные, имеющие отношение к сообщению.

  3. Нажмите кнопку Поиск.
