Автоматическое добавление подписи

Можно создавать подписи для сообщений. Подпись может включать имя и дополнительный текст длиной не более максимального количества знаков, допустимого для вашей учетной записи.

Например, подпись может быть такой:

Иван Сидоров
Заместитель директора по технической части
Отдел устройств
ООО «Восток»
303-555-1212 x111
ivan.sidorov@example.com

Если вы создаете несколько личных данных, можно создать несколько подписей и назначить их разным адресам.

  1. Откройте Настройки и выберите вкладку Подписи.

  2. Нажмите кнопку Добавить подпись.

  3. В текстовом поле Имя подписи введите подпись и понятное имя. Это не подпись, а имя для обозначения типа подписи. Например, можно назвать подпись «Официальная», чтобы было понятно, что она включает ваше имя, должность, название компании и т. д.

  4. В текстовом окне Подпись введите данные подписи точно так, как вы хотите их видеть в сообщениях. В подпись можно добавить эмблему или ссылку.

  5. Нажмите кнопку Сохранить.

  6. Перейдите на вкладку Учетные записи и выберите подпись, которая будет по умолчанию отображаться в сообщениях. В разделе Настройки основной учетной записи, в поле «Подпись» выберите имя подписи, созданной при выполнении шага 3.

  7. Нажмите кнопку Сохранить.
