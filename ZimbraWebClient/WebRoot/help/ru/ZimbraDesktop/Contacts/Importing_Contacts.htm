Импорт контактов

Можно импортировать списки контактов и адресные книги, сохраненные в файле с разделенным запятыми текстом (.csv). Импорт доступен в настройках Импорт и экспорт.

Импорт списка контактов

  1. Выберите Настройки, а затем Импорт и экспорт.

  2. В поле «Импорт файла» укажите файл .csv для импорта.

  3. В поле Тип щелкните Автоопределение для выбора типа импортируемого файла контактов:

      • Контакты учетной записи

      • Контакты Yahoo!

      • Контакты Outlook 2000

      • Контакты Outlook 2003

      • Контакты Outlook

      • Контакты Thunderbird

В списках контактов разных учетных записей данные контактов отображаются по-разному. Выбор правильного типа импортируемого файла контактов позволяет определить порядок полей для корректного отображения данных в адресной книге.

  1. Нажмите кнопку Импорт. В диалоговом окне Выбрать адресную книгу выберите существующую адресную книгу для добавления контактов или создайте новую адресную книгу, в которую будет импортирован файл контактов.

  2. Нажмите кнопку ОК. Содержимое файла контактов будет импортировано.
