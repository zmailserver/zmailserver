Установка настроек Zimbra Desktop

Вкладка «Настройки» содержит страницы функций, которые позволяют настроить работу почтового ящика, адресной книги и ежедневников для каждой учетной записи, созданной в Zimbra Desktop. При создании учетной записи в Zimbra Desktop используются настройки по умолчанию.

Изменения, вносимые в папке Общие настройки, влияют на базовую структуру Zimbra Desktop для всех учетных записей.

Глобальные

Изменения, внесенные на этой вкладке, влияют на все учетные записи в Zimbra Desktop.

  • Тема отображает список различных обложек UI. Цвет фона почтового ящика можно изменить в любое время.

  • Язык. Язык, используемый для отображения текста в Zimbra Desktop. (Настройка доступна в веб-клиенте Zimbra)

  • Если на этой вкладке отображается неверный часовой пояс, измените параметр Часовой пояс по умолчанию. Он будет определять дату и время отправки и получения сообщений и время, отображаемое для встреч ежедневника.  Этот параметр может отличаться от настроек часового пояса на вашем компьютере.

  • В меню Настройки поиска можно указать, чтобы папки «Спам» и «Корзина» автоматически включались в каждый поиск. По умолчанию поиск в этих папках не выполняется.

  • В разделе Язык поиска установите флажок Всегда показывать язык расширенного поиска на панели инструментов поиска для отображения деталей строки поиска в текстовом окне поиска.

  • Включите параметр Отображать флажки для быстрого выбора элементов в списках, чтобы в области содержимого отображался флажок для каждого элемента. Флажок используется для выбора одного или нескольких элементов, над которыми нужно выполнить одно и то же действие, например удалить, переместить, пометить как прочитанные или непрочитанные.

  • По умолчанию системы. Если этот параметр включен, Zimbra Desktop используется как почтовая программа по умолчанию. Если вы отправляете сообщение или щелкаете ссылку адреса электронной почты на веб-странице, открывается окно создания сообщения Zimbra Desktop.

Почта

  • Отображать. Выберите способ отображения элементов (сообщений или разговоров) на странице. По умолчанию на странице отображается 25 элементов, но можно выбрать 10, 50 или 100 элементов на страницу.

  • Показать почту. Укажите способ просмотра почты в виде HTML для сообщений, которые имеют формат HTML. Отключение этого параметра приводит к отображению почты в виде простого текста с отображением собственно тегов HTML вместо их применения для разметки текста.

  • Просмотр сообщений.

    • Показывать фрагменты сообщений. Если этот параметр включен, при наведении курсора на тему сообщения в списке сообщений отображается фрагмент сообщения.

    • Если включен параметр Дважды щелкните, чтобы открыть сообщение в новом окне, сообщение будет открываться по двойному щелчку в новом окне.

  • Изображения. Включите параметр Изображения, чтобы загружать изображения автоматически в сообщения в формате HTML. Если этот параметр включен, щелкать изображение в сообщении для просмотра нет необходимости.

  • Параметр В режиме чтения сообщения в области чтения указывает порядок пометки сообщений, просматриваемых в области чтения, как прочитанных или непрочитанных. Сообщения могут быть помечены как прочитанные немедленно, спустя определенное количество секунд просмотра сообщения в области чтения или всегда оставаться непрочитанными при просмотре в области чтения.

  • Принятый по умолчанию профиль поиска. Эта настройка определяет поиск, выполняемый при входе пользователя в учетную запись. По умолчанию выполняется поиск в папке Входящие. При открытии клиента Desktop отображается содержимое папки «Входящие». Для отображения при первом открытии веб-клиента другой папки, тега или сохраненного профиля поиска можно изменить это поле. Синтаксис см. в разделе Описание языка запросов.

  • Получение сообщений.

При поступлении сообщения.

    • Воспроизведение звука. При поступлении нового сообщения в папку «Входящие» раздается звуковой сигнал.  Если звук отключен, вы не услышите сигнал.

    • Подсветка вкладки «Почта». При поступлении в папку «Входящие» нового сообщения, если вы не работаете на вкладке «Почта», вкладка почты будет выделена.

    • Мерцание заголовка браузера. При поступлении в папку «Входящие» нового сообщения строка заголовка браузера будет мигать, пока не будет открыта вкладка почты.

    • Показать всплывающее уведомление. При поступлении сообщений открывается миниатюрное диалоговое окно с данными о сообщении.

Следующие действия доступны только для учетной записи Zimbra Web Client.

  • Укажите электронный адрес для пересылки. Исходные сообщения могут удаляться из почтового ящика Zimbra.

  • Укажите, нужно ли отправлять автоответ на входящие сообщения, и укажите дату начала и дату окончания.  Эта функция часто используется в качестве сообщения об отсутствии. Она позволяет извещать отправителя о том, что вас нет на работе, и сообщать ему другие дополнительные сведения. Также можно указать дату начала и дату окончания для использования этого сообщения.

  • Сообщения от меня. Настройте раздел Сообщения от меня для обработки отправляемых вами сообщений, которые содержат ваш адрес в списке получателей или которые включены в адрес, например список рассылки, участником которого вы являетесь.

Написание письма

  • Установите текстовый редактор по умолчанию: HTML или обычный текст.

  • Установите настройки шрифта по умолчанию, используемые с кодом HTML для создания в почте, документах и ежедневнике.  Можно определить стиль, размер и цвет шрифта, используемые по умолчанию. По умолчанию используется шрифт Arial размером 12 пунктов, черный.

  • Если включен параметр Всегда создавать сообщение в новом окне, для создания сообщения будет открываться новое окно, отдельное от почтового ящика. При открытом окне, создание сообщения можно просматривать почтовый ящик и перемещаться в нем.

  • Создаваемые сообщения автоматически сохраняются как черновики каждые пять минут. Если черновики сохранять автоматически не нужно, эту функцию можно выключить.

  • Укажите, нужно ли включать исходное сообщение в тело ответа. Можно включать или не включать текст исходного сообщения, включать его как вложение, включать его в тело письма с выбранным префиксом, включать только самое последнее сообщение.  (Другими словами, будет процитирован только текст, написанный отправителем сообщения, но не предыдущий текст).

  • Укажите нужно ли ставить префикс > или | перед каждой цитируемой строкой предыдущих сообщений, которые пересылаются или на которые дается ответ.

  • По умолчанию включен параметр Сохранять копии сообщений в папке «Отправленные». Копии отправленных сообщений сохраняются в папке «Отправленные».

  • Укажите как включать текст исходного сообщения при пересылке сообщения: в теле письма, в теле письма с префиксом или как вложение.

Подписи

Можно создавать подписи для сообщений. Подпись может включать имя и дополнительный текст длиной не более максимального количества знаков, допустимого для вашей учетной записи. Если вы создаете несколько личных данных, можно создать разные подписи и назначить их определенным адресам.

Контакты

  • Выберите вид контактов: в виде списка или визитных карточек.

  • Выберите количество контактов для отображения на странице. По умолчанию отображается 25 контактов.

  • Включите функцию автоматического добавления адресов в папку отправленных по электронной почте контактов во время отправки сообщений.

    Адресная книга Отправленные по электронной почте контакты заполняется при отправлении электронной почты на новый адрес, который не содержится в других ваших адресных книгах. Эту функцию можно отключить на вкладке «Адресная книга» в меню «Настройки». Снимите в настройках флажок Добавлять новые контакты к отправленным по электронной почте контактам.

Образы

Вкладка учетных записей используется для создания и управления образами. Образ — это личные данные почты, используемые в поле «От кого» отправляемых сообщений.

Стиль по умолчанию определен в основных личных данных. При создании новых личных данных, можно использовать те же самые настройки, которые используются в личных данных по умолчанию, можно так же установить другие настройки.

Фильтры почты

На вкладке «Фильтры почты» можно определить правила фильтрации сообщений. Чтобы узнать как установить правила почтовых фильтров, см. раздел Как работают фильтры.

Ежедневник

  • Выберите вид ежедневника Zimbra. По умолчанию отображается рабочая неделя.

  • Укажите день недели, который должен быть первым днем недели ежедневника.

  • В ежедневнике отображается мини-календарь.  Календарик можно сделать видимым в любом виде.

  • Можно включить параметр автоматического удаления приглашения после ответа на него. Если этот параметр выключен, приглашение остается в папке «Входящие».

  • Установка напоминания для собраний. Можно указать время напоминания о собрании: Никогда или в диапазоне времени до 60 минут до начала собрания. Можно выбрать тип напоминания: воспроизведение звука, мигание заголовка браузера или всплывающее уведомление.

  • Если вы часто назначаете встречи без приглашения участников, установите флажок Использовать диалог "Быстрое добавление".

  • При поездках в разных часовых поясах можно указать, чтобы в виде встречи отображался список часовых поясов. Тогда при планировании встреч в случае необходимости можно будет изменить часовой пояс.

Горячие клавиши

Быстрые клавиши — это клавиши или комбинации клавиш, которые позволяют быстро выбирать действие при вводе последовательности действий клавиш.  

Перейдите в папку Настройки горячих клавиш, чтобы просмотреть полный список горячих клавиш, или нажмите сочетание клавиш Ctrl+Q в любом виде для отображения всплывающего списка горячих клавиш.
