Восстановление учетных записей из архивных файлов

Если архивация учетных записей выполнялась регулярно, восстановление учетных записей до точки последней архивации не составит труда.

Восстановление учетных записей

  1. Перейдите на страницу Настройки > Архивация. Доступные файлы перечислены в разделе Восстановить из архива.

  2. Установите флажок Активно для учетных записей, которые нужно восстановить.

  3. Нажмите кнопку Восстановить из архива.  Выбранные учетные записи будут восстановлены. В зависимости от размера файла восстановление учетной записи может занять некоторое время.  

Не закрывайте Zimbra Desktop во время восстановления учетных записей.
