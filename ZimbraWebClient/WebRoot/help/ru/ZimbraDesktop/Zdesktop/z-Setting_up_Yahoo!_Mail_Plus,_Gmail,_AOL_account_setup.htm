Настройка учетных записей Yahoo! Mail и Gmail

Прежде чем вы сможете использовать клиент Zimbra Desktop, необходимо ввести настройки ваших существующих учетных записей. При первом открытии Zimbra Desktop вам будет предложено настроить учетную запись. Дополнительные учетные записи можно настроить в любое время.

  • Yahoo! Mail. Можно настроить учетные записи Yahoo! Mail, Yahoo! Mail Plus, Ymail и Rocketmail.  К вашим сообщениям, папкам почты и адресной книге можно обращаться из Zimbra Desktop. Также можно включить синхронизацию ежедневников и контактов.

  • Gmail. В учетной записи Gmail должен быть включен IMAP-доступ. При выборе Gmail, возможно, для включения IMAP понадобится войти в учетную запись Gmail. К вашим сообщениям электронной почты можно обращаться из Zimbra Desktop. Также можно включить синхронизацию ежедневников и контактов.

Как настроить учетную запись

Прежде чем начать, убедитесь в том, что есть подключение к интернету.

  1. Откройте Zimbra Desktop и в правом верхнем углу нажмите кнопку Установка.  

  1. Нажмите кнопку ДОБАВИТЬ НОВУЮ УЧЕТНУЮ ЗАПИСЬ.  В списке «Тип учетной записи» выберите необходимый тип. Откроется страница «Установка учетной записи Zimbra». Введите следующие данные.

Имя учетной записи. Это имя учетной записи, отображаемое в Zimbra Desktop. При настройке нескольких учетных записей старайтесь использовать легко различимые имена для каждой учетной записи. Например, для учетной записи Yahoo! можно использовать имя «Yahoo».

Ваше полное имя. Введите ваше имя. (Необязательно).

Адрес электронной почты. Введите ваш электронный адрес для данной учетной записи.

Пароль. Введите пароль для входа в эту учетную запись.

  1. В разделе Настройки синхронизации выберите регулярность, с которой клиент Zimbra Desktop должен проверять наличие сообщений в вашей учетной записи. По умолчанию синхронизация выполняется через каждые 15 минут.

- При выборе параметра «вручную» необходимо нажать кнопку «Отправить/Получить» на панели инструментов, чтобы синхронизировать учетную запись с сервером Zimbra.

- Выбор слишком малого периода времени, например одной минуты, может снизить производительность вашего компьютера.

- Этот параметр можно изменить в любое время, выбрав пункт «Установка учетной записи» и изменив соответствующее значение для выбранной учетной записи.

Выберите папки для синхронизации в вашем почтовом ящике.

* Синхронизировать все ежедневники.

* Синхронизировать все контакты и группы.

  1. По окончании нажмите кнопку Подтвердить и сохранить.

Клиент Zimbra Desktop проверит данные вашей учетной записи. В случае успешной проверки учетной записи откроется диалоговое окно «Служба создана». Если проверка окажется неудачной, снова отобразится диалоговое окно «Установка», в котором можно просмотреть и откорректировать данные.

После этого начнется синхронизация вашей учетной записи с Zimbra Desktop.

  1. Нажмите кнопку «Запустить Desktop», чтобы открыть клиент Desktop. Во время синхронизации учетной записи можно продолжать работу в Zimbra Desktop.
