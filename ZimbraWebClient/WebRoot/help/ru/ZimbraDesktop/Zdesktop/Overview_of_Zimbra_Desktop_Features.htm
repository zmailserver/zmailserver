По общим вопросам и диагностике неисправностей обращайтесь к справке по Zimbra Desktop

Обзор функций Zimbra Desktop

Электронная почта

  • Создание, редактирование и удаление сообщений, создание ответов и черновиков

  • Отслеживание изменений в почте с помощью функции «Разговор» или дополнительного переключения в режим обычного просмотра сообщений

  • Присоединение вложений в сообщение

  • Пересылка сообщений одному или нескольким получателям

  • Поиск сообщений и вложений по конкретным характеристикам или указанному тексту

  • Создание собственных папок для систематизации почты

  • Создание меток для систематизации почты

  • Создание фильтров для направления входящей почты в указанные папки

  • Установка разных учетных данных и адресов для управления разными ролями на работе и в личной жизни

  • Работа даже в автономном режиме — отправленные вами сообщения сохраняются в папке Исходящие и отсылаются, когда вы подключитесь к Интернету.

  • Сохранение сообщений в локальной папке для увеличения доступного объема хранилища почты

Адресные книги

  • Хранение всех контактов изо всех учетных записей в одной программе

  • Создание групп и меток для упорядочения контактов

  • Дополнение контактов фотографиями

  • Автозаполнение электронных адресов при создании сообщений

  • Импорт новых контактов из других приложений в формате .csv или экспорт контактов в файлы .csv для архивирования

Ежедневник

  • Создание нескольких ежедневников и управление ими с помощью цветового кодирования

  • Создание встреч, собраний и событий

  • Импорт общедоступных веб-ежедневников (в формате iCal)

  • Приглашение других пользователей на собрания и просмотр сведений о занятости

Задачи, портфель

  • Отслеживание выполнения заданий, включая даты начала и конца, процент выполнения

  • Редактирование документов, в том числе добавление изображений, таблиц и электронных таблиц; рассылка их по электронной почте

  • Сохранение вложений в портфеле, а не в сообщениях

Управление учетными записями

  • Архивация папок на компьютере и их удаление с сервера

  • Резервное копирование почтового ящика или его отдельных папок для последующего восстановления в случае утраты информации
