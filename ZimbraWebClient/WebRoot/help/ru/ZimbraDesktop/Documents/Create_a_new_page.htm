Создание нового документа

Можно создать документ в портфеле и оформить содержимое с помощью инструментов форматирования. На странице можно использовать проверку орфографии и ключевые слова для простого создания ссылок на другие страницы.

  1. Щелкните папку портфеля, в которой необходимо работать. В области Список документов отображаются файлы, уже содержащиеся в портфеле.

  2. В списке Создать на панели инструментов выберите Документ. Откроется чистая страница документов Zimbra.

  3. На панели инструментов введите имя документа.

  4. Создайте содержимое страницы. По умолчанию можно использовать редактор форматированного текста, чтобы добавить на страницу стили, цвет и таблицы.

  1. Нажмите кнопку Сохранить, чтобы сохранить файл и продолжить работу Чтобы закрыть документ, не сохраняя его, щелкните x на вкладке браузера.

После сохранения документа меняется версия документа.

Примечание. Если документ был создан не в том портфеле, можно выделить документ и перетащить его в правильную папку.
