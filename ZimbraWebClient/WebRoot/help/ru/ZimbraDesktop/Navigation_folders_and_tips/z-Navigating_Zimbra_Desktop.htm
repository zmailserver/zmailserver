Перемещение между элементами программы Zimbra Desktop

При открытии программы Zimbra Desktop отображается полноэкранное изображение, подобное показанному ниже.

Главная страница Zimbra Desktop содержит следующие области.

  1. Панель поиска. В этой области отображаются функции поиска и расширенного поиска. Справа расположены ссылки «Настройка учетной записи» и «Справка».

  2. Вкладки приложений. Приложения, к которым можно получить доступ отображаются на вкладках вверху.

  3. Панель инструментов. Здесь отображаются действия, которые можно выполнять в используемом в данный момент приложении. В данном примере отображается панель инструментов приложения Почта.

  4. Область содержимого. Содержимое в этой области меняется в зависимости от используемого приложения. Вид «Входящие» отображает все сообщения и разговоры в папке «Входящие». На экране с этой страницей отображается область чтения — область, которая находится под сообщениями в папке «Входящие». Если щелкнуть сообщение, оно отобразится в области чтения под списком сообщений.  Чтобы изменить размер области чтения, щелкните и перетащите разделительную полосу вверх или вниз. По умолчанию режим просмотра сообщений в области чтения включен.  Его можно отключить, выбрав ссылку «Вид» на панели инструментов.

  5. Область обзора. Здесь отображаются различные учетные записи. Для каждой учетной записи отображается ее имя. В данном примере используется компоновка «Все почтовые ящики».  В этом разделе объединены все сообщения в папках «Входящие», «Отправленные» и «Черновики» всех ваших учетных записей. Такой вариант отображения можно настроить на странице «Настройки->Общие».

    Для каждой учетной записи отображаются следующие папки.

    • Папки, в том числе системные папки Входящие, Отправленные, Черновики, Спам и Корзина, а также все папки, созданные пользователем.

    • Сохраненные условия поиска.

    • Созданные пользователем теги.

    • Ссылки на модули Zimlet, созданные для интеграции приложений сторонних производителей с почтовым ящиком.

  6. В области обзора может также отображаться календарик. Отображение календарика необязательно, его можно включить на странице Настройки->Ежедневник.

Запуск программы Zimbra Desktop

При условии доступа к интернету, если для вашей учетной записи пользователя настроена автоматическая синхронизация, новые сообщения электронной почты автоматически загружаются в Zimbra Desktop. Для этого программу даже не надо открывать.  На панели задач Windows или в строке прикрепления на Macintosh появляется значок конверта с указанием общего количества непрочитанных сообщений во всех ваших почтовых ящиках с момента последней синхронизации Zimbra Desktop. Щелкните этот значок, чтобы открыть программу.

 

Уведомление о почте и встречах

Можно настроить автоматическое появление уведомлений о новой почте и встречах. Возможны несколько режимов уведомления.

  • В Windows уведомления отображаются в виде выноски из значка-конверта в системной области панели задач. На Macintosh для получения уведомлений надо установить приложение Growl.  Кроме того, необходимо включить параметр Показать всплывающее уведомление на вкладках Настройки->Почта и ->Ежедневник. Примечание. В операционной системе Linux эта функция недоступна.

 

  • На вкладке Настройки->Почта и ->Ежедневник также можно задать другие уведомительные действия, в том числе воспроизведение звука, выделение вкладки «Почта» или «Ежедневник» или мигание заголовка окна браузера. Всплывающие уведомления можно настроить отдельно для каждой учетной записи. В некоторых учетных записях их можно отключить.
