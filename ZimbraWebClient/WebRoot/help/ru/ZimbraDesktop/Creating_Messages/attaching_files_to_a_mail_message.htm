Вложение файлов в сообщение

При составлении электронного сообщения можно вложить файлы с компьютера, Портфеля, контакты с адресной книги или электронные сообщения.

Примечание. У получателей сообщения должно быть установлено соответствующее программное обеспечение для открытия и чтения файла. Пользователи, использующие обычный настольный компьютер, смогут открыть файлы с помощью таких распространенных программ, как Microsoft Office. Файлы распространенных форматов, например текстовые файлы, HTML-файлы и изображения (.GIF или .JPG), можно открывать с помощью различных программ. Однако, все зависит от типа используемой системы.

Добавление вложений с помощью ссылки «Вложить файл»

  1. Создайте сообщение, как указано в разделе Создание нового сообщения.

  2. Нажмите кнопку . В диалоговом окне «Вложить файл(ы)» отображаются вкладки «Мой компьютер», «Портфель», «Почта» и «Контакты». Выберите вкладку согласно типу вложения.

    • Мой компьютер. Перейдите в любую папку на компьютере и выберите файл. Выберите файл и нажмите кнопку Открыть. Также можно перетащить файлы из папки в заголовок создаваемого сообщения.

    • Портфель. Выберите папку портфеля. Справа отобразятся файлы из этого портфеля. Отметьте файлы, которые необходимо вложить.

    • Почта. Если сообщение находится не в папке «Входящие», выберите папку. В выбранной папке можно выполнить поиск сообщения. Сообщения отображаются справа. Выберите сообщение, которое необходимо вложить.

    • Контакты. Выберите адресную книгу. Имена в адресной книге отображаются справа. В списке контактов можно выполнить поиск имени. Отметьте имена, которые необходимо вложить. Контакты вкладываются в сообщение как файлы vcf.

Примечание. Чтобы добавить несколько вложений за один раз, щелкайте их мышью, удерживая нажатой клавишу Ctrl или Shift. Несколько вложений добавляются в одну строку в диалоговом окне «Присоединить файлы».

  1. После добавления всех вложений на этой вкладке нажмите кнопку Вложить.

  2. Чтобы отправить сообщение и вложения, нажмите кнопку Отправить.

Добавление вложений с компьютера с помощью перетаскивания

  1. Выберите один или несколько файлов, наведите на них курсор, а затем нажмите и удерживайте левую кнопку мыши.

  2. Перетащите файлы в область заголовка сообщения и отпустите кнопку. В заголовке отобразятся имена файлов.

 

 

 

Удаление вложения

Для удаления вложения снимите флажок рядом с вложением перед отправкой сообщения электронной почты.
