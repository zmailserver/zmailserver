View Multiple Calendars

Meetings display in the selected Calendar view in the calendar color selected. If meetings are scheduled that are not in the current time view, an arrow displays on the calendarâ€™s page indicating that there are meetings scheduled that are not in the current view.

  1. In the Calendar list in the Overview pane, check the boxes for the calendars you want to view.

  2. Select the view.

Day view shows the calendars side-by-side. The calendar name is at the bottom of the page.

The Work Week, Week, and Month views show appointments on the calendar by the color associated with the calendar.
