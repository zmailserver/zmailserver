Setting Your Search Preferences

By default you spam, trash and shared items folders are not searched.  You can enable these preferences. To view the search query syntax in the search field, you can enable the Search Language preference.

  1. Go to Preferences>General page.

  2. Select Search Settings>Include Spam Folder in Searches, Include Trash Folder in Searches, and Include Shared Items in Searches to include these folders in any search you perform.

  3. Select Search Language>Show advanced search language in search toolbar to show search string details in the search text box at the top of the page. By default, this option is not selected.

  4. Click Save.
