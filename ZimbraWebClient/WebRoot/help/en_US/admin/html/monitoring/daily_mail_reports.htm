Daily Mail Report

When the Zimbra Logger package is installed, a daily mail report is automatically scheduled in the cronntab. This report is sent daily to the administrator's mailbox.

The Zimbra daily mail report includes the following information:

  • Total number of messages that moved through the Zimbra MTA

  • Average delay in seconds for message delivery

  • Errors generated from the Zimbra MT Postfix logs

  • Most active sender accounts and number of messages

  • Most active recipient accounts and number of messages

  • Total number of bounced deliveries
