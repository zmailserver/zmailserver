MAPI Logon

After the accounts are provisioned, the wizard prompts you to select a MAPI profile that can be used to connect to the Exchange server.

  • Important: You must create a temporary MAPI profile for the Migration Wizard to use, to conduct the migration. To create a new profile, refer to the Microsoft Office documentation.

You must have full administrator privileges on the mailboxes to be imported.

The account under which you log into Windows must have permission to open the specified MAPI profile, and the account must be a member of the Windows domain administrators group.

Identify a MAPI profile to use. Zimbra recommends that you create a temporary profile for the migration wizard to use to conduct the migration. The selected MAPI profile specifies a single account that will be used to open and import all mailboxes.

  1. Click Launch Profile Chooser and choose the temporary MAPI profile created for the migration.

  2. Click Next.
