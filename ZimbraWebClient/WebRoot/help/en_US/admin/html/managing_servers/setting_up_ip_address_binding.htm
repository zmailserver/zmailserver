Setting Up IP Address Binding

If the server has multiple IP addresses, IP Address binding allows you to specify which interfaces (IP addresses) you want a particular service to bind to.  For some services,  the default is to bind to all the IP addresses configured for the server.

 

  1. Go to Configure > Servers and select the server to edit.

  2. On the IP Address Bindings page, set the Web Client configuration.  Enter as xx.xx.xx.xx

 

Option

Description

Web Client Serer IP Address

Interface address on which the HTTP server listens.  

Web Client Server SSL IP Address

Interface address on which the HTTPS server listens

Web Client Server SSL Client Cert IP Address

Interface address on which HTTPS server accepting the client certificates listen.

 

  1. Set the administrator console HTTPS configuration in the Admin Console Server IP Address text box.

  2. Click Save.

  3. You must restart the server for the changes to take effect.
