Global Settings to Block Email Attachments

The Attachments page can be configured with global rules for handling attachments to an email message. You can also set rules by COS and for individual accounts. When attachment settings are configured in Global Settings, the global rule takes precedence over COS and Account settings.

Procedure

  1. Go to Configure>Global Settings.

  2. Click Attachments in the Navigation pane.

  3. Select the attachment setting to be configured..

 

Option

Description

Attachments cannot be viewed regardless of COS

Users cannot view any attachments. This global setting can be set to prevent a virus outbreak from attachments, as no mail attachment can be opened.

Attachments are viewed according to COS

 This global setting states the COS sets the rules for how email attachments are viewed.

 

  1. In the Common Extensions list, select the extensions to block.

  2. Click Add Selected or Add All.

  3. If an extension that should be blocked is not listed, in the New extension text box, type the extension and click Add.

  4. If you do not want to send a notification to the recipient when messages are blocked, disable this option.

  5. Click Save.
