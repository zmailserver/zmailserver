Override COS Settings

In most cases, the Class of Service associated with an account determines the enabled features for accounts, mailbox quota, password constraints, email message lifetime.

In Accounts, you can override the COS settings for individual accounts. When you change an account's setting, the feature is highlighted and a Reset to COS value link displays. You can see what features in the account are different from the COS assigned to the account and you can reset the feature to the COS value.

When you move an account from one COS to another, if preferences have been changed at the account level, the new COS settings do not override these changes.

  •       Disabling attachment viewing, which can be set in Global Settings, COS, and Accounts, is an exception to the usual COS override rule. If attachment viewing is disabled in Global Settings, the global settings override both the COS and Account settings.
