Creating Dynamic Distribution Lists

Dynamic distribution lists can be created.  When a dynamic distribution list is created, a member URL is created that is used to identify members of the list. When you create accounts, you can add the distribution list to the accounts Member of page and the account is added to the DL. When an account is deleted it is automatically removed from the DL.

You can create a dynamic distribution list and assign someone else as the owner.  They can manage the distribution list from their account.

A dynamic distribution list cannot include other distribution lists.

Procedure

A distribution list can include email addresses listed in your company directory and you can add other addresses to the list, but email addresses you type are not verified.

  1. Go to Manage>Distribution Lists.

  2. In the gear icon, click New.

  1. In the List Name text box, type the distribution list name to be used as the mail address. The list name cannot have any spaces.

  2. In the Display name text box, type the common name for the list.

  3. In the Description field, describe the type of list or usage.

  4. Select Dynamic Group.

  5. On the right, in the Add Members to this list section, either search for names from the company address book, or in the Or enter addresses below section, type a complete mail address.

To search for names in the company address book, enter a partial name or a complete name in the Find field and click Search, or leave the Find field blank and click Search. Names display, ten at a time, in the box below.

If the Find field is blank when you click Search, the search results return the all mail addresses in the address book. Use Back and Forward at the bottom of this box to scroll through the address list.

To use the Or enter addresses below section, type a complete mail address. Either separate addresses with a comma or press  Enter, to add the next address on a new line.

  1. If you do not want the distribution list to be viewed in the GAL, check Hide in GAL.

  2. If you want the distribution list saved on a specific mail server, select it from the drop down menu.

  3. Click Next to move to the next pages.

  4. Use the Notes page to write notes about the distribution list (optional).

  5. Open  the Aliases page to add a distribution list alias name (optional).

  6. Open the Owners page to add someone as the owner. (optional)

  7. If replies to the distribution list should be sent to a specific address, Open the Preferences page and enable the reply to field and add the reply to name and address.

  8. Click Save when all names have been added to the list. The distribution list is enabled and the URL is created.
