Advanced Search

In the search options section of the Search>Navigation pane you can create a more specific search. You can create a complex query to search for addresses by domain or server. The results of the search display in the Content pane. The following search options open as individual search panes to let you select the criteria for the search.

Basic Attributes. This pane sets the specific details about the address.

  • Name is the account user, resource name, or distribution list name before the @ sign in the email address.

  • Last Name is the last name of the user.

  • Display Name is the name configured as the display name, often this is the last name

  • ID is the complete mailbox ID must be entered. Use this to find out which domain or server an account is on.

  • For administrators or for delegated administrators. You can easily find all administrator and delegated administrator accounts.

Status.  You can select which status to search - active, maintenance, locked, closed, and locked out.

Last Login Time. You can search for accounts by the last log in time or for accounts that have never been logged in..

External Email Address.  You can enter a specific email address to look for.

Domains: Specify a specific Zimbra domain. If no domain is specified, all domains are searched. If you want to search more than one domain, but not all, open a domain pane for each specific domain you want to search.

Server. All servers are listed. The default is to search all servers. Select the servers to search.

Class of Service. Specify a COS name. When you select the COS from the list, the search field shows the COS ID

As you enter criteria, the query syntax displays in the search field.

To use Advanced search and Save the Search Query

  1. Click Search in the Home > Navigation pane.

  2. In the Search Option section, select the type of search to perform.

  3. To save the search query, click Save Search and name the search.

  4. Click OK.
