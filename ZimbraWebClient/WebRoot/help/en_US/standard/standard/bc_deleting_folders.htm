Deleting Folders

You can delete custom folders. You cannot delete system folders.

When you delete a folder, its contents are moved to the Trash. You can recover contents from the Trash by dragging and dropping them to another folder.

Note: The Trash folder is purged on a regular basis, so items in the Trash do not stay there forever.

Delete a folder:

  1. Select the folder to delete

  2. In the Folders heading in the Overview pane, click edit.

  3. Select Delete this folder.

  4. Click Delete Folder.

If there are messages in the folder that you want to save, move them to another folder before deleting.

Delete the contents of a folder

You can delete everything in one of your folders, but not the folder itself.

  1. Select the folder  that you want to delete its contents.

  2. In the Folder heading in the Overview pane, click edit.

  3. Check Permanently hard delete all items in this folder. This operation cannot be undone.

  4. Click Delete All Items. The contents of the folder are immediately deleted.
