Marking Email as Unread

Messages that you opened are marked as read and are changed from bold to regular text.

  1. Click the check box to select the messages to be marked as unread.

  2. In the Actions drop-down menu, select Mark as unread.  The selected messages are made bold and the checkmark is removed.

Note: To mark all messages in a conversation as unread, select the conversation. To mark an individual message in a conversation as unread, open the conversation and select the message.

You can use the same method to mark unread mail as read.
