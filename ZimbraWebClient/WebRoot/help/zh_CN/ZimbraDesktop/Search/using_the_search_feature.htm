使用搜索功能

搜索功能用于查找邮件、联系人、约会、“文档”页面和文件。 您可通过特定的字词、按日期、时间、URL、大小、标记、邮件是否为已读、邮件是否含附件、是否有特定文件类型的附件等等进行搜索。

Zimbra Collaboration Suite 提供了两种搜索工具：

  • 搜索。这是一种快速搜索方式，执行当前显示在搜索框中的任何搜索条件。在搜索字段中输入一个搜索关键字后，就会自动显示一个列表，其中列出了可能的搜索参数。此外，搜索框左侧的下拉箭头用于选择所要搜索的内容的类型。您可选择在邮件、您的联系人（其中包括公司联系人列表）、任务、约会、页面和文件中进行搜索。这种方式的目标就是快速搜索，详情请见快速搜索设置中的说明。

  • 高级搜索可打开一个新窗格，更适合执行较复杂的搜索。 您可将高级搜索条件保存起来，以便以后再次使用。

搜索邮件

如果您熟悉文本匹配搜索或文字处理功能，如 Microsoft Word 中的查找/替换功能，那么请注意，Zimbra Web Client 的内容搜索功能与执行字面字符串匹配还是有一定区别的。

ZCS 搜索的语法原理如下：

  • 您可以搜索短语，但是短语中每个单词只与整词匹配。不允许拼写变体。例如，如果您搜索 bananas，则包含 banana 的邮件就不是匹配项。您可按域名进行搜索，其中可包括英文句号(.)。

  • 搜索功能不区分大小写，South、south 和 SOUTH 被视为同一个词。

  • 以下特殊字符不能在搜索文字中使用。 ~ ' ! # $ % ^ & * () _- + ? / { }[ ] ; : "

  • 可使用星号 * 作为通配符。如要搜索 do*，返回的条目可能是 dog、door 等。

  • 搜索内容意味着搜索邮件的正文，以及可能有的任何（系统可读的）附件。系统可读文件附件，是一种可以转换成 HTML 格式的文件类型。其中包括 Microsoft Office 文档（Word、PowerPoint 或 Excel），以及文本文件，但是不包括图像或音频文件。
