使用富文本编辑器

您可使用富文本编辑器格式化页面内容来在公文包中编写您的电子邮件和文档。

使用富文本编辑器进行编辑时，您可通过工具栏轻松格式化页面布局。选择字号、字体、文本样式、字体颜色和背景颜色。这些格式化工具的用法与许多文字处理工具相同。另外，您可通过单击的方式将表格、附件和图像添加到页面中。

添加段落样式

工具栏上的 图标显示可用的段落样式。利用这些段落样式，您可轻而易举地在文档中格式化文本并保持一致的风格。这些样式包括：用于大多数段落的普通项目，表示章节标题的六级标题，可使文本变为斜体的字型以及可将文本显示为 Courier 字体时使用的预定的格式。这些样式不能修改，但可使用编辑选项更改字体、字号、颜色等等。

在页面上添加表格

添加表格到页面时，您须分三步进行。即添加表格，配置表格属性和向表格添加内容。若需添加表格，单击工具栏上的表格图标，选择“插入表格”，然后选择表格的行数和列数。

在页面上添加附件和图像

您可上传文档文件所链接的文件。单击别针图标， 或图像， 图标并浏览以查找要上传的文件。如果您的公文包中已经包含要上传的文件，您将收到通知。

链接到网页

您可以链接到一个外部网络 URL，并为该链接赋予一个描述性的标题，而不是显示 URL。单击 图标，然后输入要链接到的页面或网络 URL。
