创建新的行事历

只为您创建了一个行事历，您可以创建更多的行事历以便跟踪不同类型的功能。例如，您可以为工作、个人、公司的各项活动创建分开的行事历，也可订用互联网上的行事历，还可以链接到与您共享的行事历。

创建行事历

  1. 选择“行事历”选项卡。

  2. 在行事历概述窗格中，单击编辑。

  3. 在工具栏中，单击新行事历。

  4. 键入行事历的名称。

  5. 选择一种显示行事历活动的颜色。此操作可使您为邮箱中的行事历设定颜色，从而使该行事历的活动总是以选定的颜色显示。

  6. 如果不希望其他人看到行事历中的活动，请选中在报告忙/闲时排除这个行事历。

  7. 默认情况下，已在 UI 中检查处于启用状态。打开“行事历”视图时，将显示此行事历中的活动。如果不希望随同别的行事历一起查看此行事历，请取消选中此框。在这种情况下，要查看此行事历的活动，须选择该行事历并将其打开。

  8. 单击创建行事历。

订阅其他行事历

  1. 选择行事历选项卡。

  2. 在行事历概述窗格中，单击编辑。默认的行事历信息随即显示。  

  3. 在工具栏中，单击订阅行事历。  

  4. 输入该行事历的名称，远程行事历的有效 iCal 文件 URL，选择一种显示该行事历活动的颜色。

  5. 如果不希望其他人看到行事历中的活动，请选中在报告忙/闲时排除这个行事历。

  6. 默认情况下，已在 UI 中检查处于启用状态。打开“行事历”视图时，将显示此行事历中的活动。如果不希望随同别的行事历一起查看此行事历，请取消选中此框。在这种情况下，要查看此行事历的活动，须选择该行事历并将其打开。

  7. 单击创建行事历。
