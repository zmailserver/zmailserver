自动使用签名

可在电子邮件末尾添加自动签名。签名可包括您的姓名及任何其他文本。如果使用的是 HTML 编辑器，可为您的签名设置格式、添加 URL 链接以及添加图形。

可创建多个签名。例如，可为发送到客户的电子邮件创建正式签名，为发送到朋友的电子邮件创建非正式签名。

如果创建多个电子邮件身份（也称为身份），可创建不同的签名，并分别指派给特定的地址。

  1. 转到首选项 > 签名页面。

  2. 在名称文本框中，键入用于标识签名的描述性名称。由于您可以创建多个签名，因此在此处使用可识别名称会很有帮助。

  3. （可选）要设置纯文本格式，请单击 HTML 格式，然后选择纯文本格式。

  4. 在文本框中，按照您希望的样子键入签名文本。如果使用的是 HTML 格式，可添加图像并创建指向签名文本的链接。

  5. 在使用签名部分，从下拉菜单中选择要用于从您的不同帐户发送的邮件的默认签名。

  6. 选择签名在邮件中的位置。

    7. •    选择邮件正文上面将签名添加在您回复的末尾、所包含邮件的前面。

    •    选择邮件正文下面将签名添加在邮件的末尾。

  7. 单击保存。
