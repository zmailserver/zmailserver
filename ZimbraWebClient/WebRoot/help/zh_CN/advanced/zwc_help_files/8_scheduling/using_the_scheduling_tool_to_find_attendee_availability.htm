使用安排工具查找参会者空闲时间

安排会议时，可以查看参会者的忙碌或空闲时间，以便可以找到适合所有参会者的会议时间。当您将参会者添加到“参会者”字段时，“概述”窗格中的建议面板会显示参会者在 30 分钟时段内的空闲状态。

默认时间首选项将提供您的工作时间和参会者工作时间内的建议时间。

•      仅包括我的工作时间。这是您在行事历首选项中设置的每周的工作日和时间。

•      仅包括其他参与者的工作时间。搜索地点时，需要考虑每个人的工作时间。

您可以更改此信息。

  1. 在新约会页面中，单击建议一个时间。

  2. 在打开的建议时间窗格中，单击齿轮图标。

  3. 取消选中您不想用于计算参会者可用性的框。

  4. 如果为重复会议，选择参会者应参加的重复会议的实例数。

  5. 单击确定。

在小行事历中查看可用性

小行事历通过颜色代码显示可参与会议的参会者。

•      绿色表示在行事历中的这些天内，所有参会者都可以参加会议，至少有一个地点可用。

•      红色表示超过一半的参会者不能在那几天参加会议。
