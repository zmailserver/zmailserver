使用电子邮件设置会议

您可快速创建会议请求，方法是将邮件、会话拖放到小行事历上的一个日期上。拖放邮件或对话时，邮件中的信息将用于预先填充“约会”页面上的许多字段。

  1. 单击某个特定邮件或对话，然后将其拖到小行事历上的某一日期上。

  2. “添加参会者”对话框随即打开。同时，“约会”页面打开，其中的许多字段填充有邮件中的信息。

    3. •    约会的“主题”就是邮件的主题。

    •    参会者是邮件或对话中最近邮件的“收件人:”和“抄送:”字段中的所有电子邮件地址所对应的人员。当您将一位联系人从联系人列表拖到小行事历中时，该联系人的第一个电子邮件地址即会添加到参会者字段中。

    •    邮件的文本成为邀请的文本。邮件的附件不会附加到约会请求中。

  3. 在“约会”页面中，检查会议请求详细资料并进行任何必要的更改。

  4. 设置会议的时间，如果是重复会议，设置重复模式。

  5. 单击发送。
