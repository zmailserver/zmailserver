Enregistrement des requêtes

Si vous créez une recherche et pensez que vous l'utiliserez ultérieurement, vous pouvez l'enregistrer.

  1. Créez l'interrogation de recherche en saisissant le texte dans la barre de Recherche ou en créant une requête avec la recherche Avancée

  2. Pour enregistrer la recherche, cliquez sur dans la barre de recherche. La boîte de dialogue Enregistrer la recherche s'affiche.

  3. Affectez un nom à la recherche enregistrée. Les recherches sont enregistrées par défaut dans un dossier de recherche dans la section Recherches du panneau d'aperçu. Vous pouvez, si vous le souhaitez, choisir un autre dossier.

  4. Cliquez sur OK. Votre recherche est enregistrée.

Pour utiliser une recherche enregistrée, cliquez sur le dossier de recherches. Les résultats de la recherche s'affichent immédiatement dans le panneau de contenu.
