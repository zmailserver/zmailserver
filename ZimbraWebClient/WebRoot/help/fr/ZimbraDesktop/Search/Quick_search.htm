Recherche rapide

Un raccourci de recherche consiste à simplement taper un nom ou un mot sans mot-clé. Par exemple, pour rechercher tous les mails contenant le mot " tim" n'importe où dans une ligne d'objet, dans les champs À : De : Cc : ou Cci : dans le corps du mail, ou dans les pièces jointes, vous pouvez simplement saisir tim dans la zone de recherche, puis cliquez sur Recherche.

Vous pouvez saisir n'importe quelle valeur, telle qu'un mot, un prénom, un nom de famille, un numéro de téléphone ou un nom de domaine. Pour rechercher des contacts dans vos carnets d'adresses, vous devez saisir un mot complet dans le champ. Par exemple, pour rechercher des contacts par numéro de téléphone, vous devez saisir le numéro de téléphone complet tel qu'il apparaît dans les données du contact ; vous ne pouvez pas rechercher simplement par indicatif.

Pour pouvoir utiliser la fonction de recherche rapide, assurez-vous que vous avez sélectionné le type d'élément que vous voulez rechercher. La flèche déroulante située à gauche de la zone de recherche vous permet de choisir le type d'éléments à rechercher. Vous pouvez rechercher des messages, des contacts personnels, , les pages ou tous les types. Lorsque vous recherchez plusieurs types d'éléments, l'icône affichée dans la liste vous permet de connaître le type des éléments inclus.

Dans Zimbra Desktop , la recherche est effectuée uniquement dans le compte dans lequel vous travaillez.  

Pour rechercher rapidement des contacts dans votre carnet d'adresses :

  1. Cliquez sur la flèche du menu Rechercher, , choisissez Contacts personnels.

  2. Saisissez un nom ou d'autres éléments de données du contact dans la zone Rechercher.

  3. Cliquez sur Recherche, pour lancer la recherche.

Pour effectuer une recherche rapide dans les mails :

  1. Cliquez sur la flèche du menu Rechercher, , choisissez Mails.

  2. Saisissez un mot, un nom ou toute autre donnée relative au mail dans la zone Rechercher.

  3. Cliquez sur Recherche, pour lancer la recherche.

Pour effectuer une recherche rapide dans les mails reçus un jour donné, cliquez avec le bouton droit de la souris sur le mini-calendrier et sélectionnez Mail. Le panneau de contenu affiche tous les mails reçus à la date spécifiée.
