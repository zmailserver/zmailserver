Recherche de messages envoyés à des destinataires spécifiques

Vous pouvez rechercher les messages par destinataire à qui les messages ont été envoyés de l'une des manières suivantes :

  • À partir des dossiers Envoyés ou Brouillons

  • En utilisant le champ À/Cc de la Recherche avancée ou de la Recherche simple

Quand vous recherchez dans un dossier, seuls les résultats des messages se trouvant dans ce dossier s'affichent. Quand vous utilisez l'option Recherche avancée, vous pouvez sélectionner dans quels dossiers rechercher. Les résultats incluent les messages à partir des dossiers sélectionnés pour la recherche.

Recherche dans les dossiers Envoyés et Brouillons

  1. Dans les dossiers Envoyés et Brouillons, sélectionnez un message avec le nom du destinataire que vous souhaitez trouver.

  2. Cliquez sur À dans l'en-tête. La boîte de dialogue Sélectionner une adresse s'affiche.

  3. Cliquez sur OK. La recherche est exécutée et les messages envoyés à ce destinataire s'affichent dans le Panneau de contenu.

Utilisation de la recherche avancée

  1. Cliquez sur Avancé pour ouvrir la zone de recherche avancée.

  2. Dans le champ À/Cc de la boîte de dialogue Recherche simple, saisissez le nom à rechercher. Plus le nom est spécifique, meilleurs sont les résultats. La recherche s'effectue dans la Boîte de réception par défaut.

  3. Pour rechercher dans d'autres dossiers, cliquez sur Dossier, puis sélectionnez les dossiers dans lesquels rechercher.

Dès que vous saisissez un critère dans un panneau de recherche, ou dès que vous avez effectué tout changement de sélection dans tout autre panneau de recherche, la recherche s'exécute.

Les éléments correspondant aux critères de la recherche s'affichent dans la partie inférieure droite. Si la fonction de conversation est activée, les conversations contenant des mails qui correspondent aux critères de la recherche s'affichent également.
