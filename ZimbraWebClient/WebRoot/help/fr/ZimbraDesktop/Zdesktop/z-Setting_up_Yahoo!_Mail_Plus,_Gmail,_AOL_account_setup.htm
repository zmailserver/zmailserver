Configuration de comptes Yahoo! Mail et Gmail

Avant d'utiliser Zimbra Desktop, vous devez spécifier les paramètres du ou des comptes existants.  La première fois que vous activez Zimbra Desktop, un message vous demande de créer un compte.  Vous pouvez créer d'autres comptes à tout moment.

  • Compte Yahoo! Mail. Vous pouvez créer un compte Yahoo! Mail, Comptes Yahoo! Mail Plus, Ymail ou Rocketmail.  Vos mails, dossiers de mails et carnets d’adresses sont disponibles sur Zimbra Desktop. Vous pouvez choisir de synchroniser les calendriers et les contacts également.

  • Compte Gmail. Votre compte Gmail doit être configuré de manière à autoriser les accès IMAP. Lorsque vous sélectionnez Gmail, un message peut vous demander de vous connecter à votre compte Gmail au préalable pour activer le support IMAP. Vos mails sont disponibles sur Zimbra Desktop. Vous pouvez choisir de synchroniser les calendriers et les contacts également.

Configurer votre compte

Avant de commencer, vérifiez que vous êtes correctement connecté à Internet.

  1. Ouvrez Zimbra Desktop et en haut à droite, cliquez sur Configuration.  

  1. Cliquez sur AJOUTER UN NOUVEAU COMPTE. Dans la liste Type de compte, sélectionnez le type de compte. La page de configuration de compte Zimbra apparaît. Entrez les informations ci-dessous.

Nom du compte. Nom qui apparaît comme nom de compte dans Zimbra Desktop. Si vous configurez plusieurs comptes, efforcez-vous de choisir des noms qui faciliteront leur identification. Par exemple, pour un compte Yahoo!, vous pourriez utiliser Yahoo.

Votre nom complet. Entrez votre nom (facultatif).

Adresses mail. Entrez l'adresse mail que vous utiliserez avec ce compte.

Mot de passe. Entrez le mot de passe que vous spécifiez pour vous connecter à votre compte.

  1. Dans Paramètres de synchronisation, sélectionnez la fréquence à laquelle Zimbra Desktop doit vérifier les messages de  votre compte. La valeur par défaut est 15 minutes.

-Si vous sélectionnez la valeur manuellement, vous devez cliquer sur le bouton Envoyer/Recevoir de la barre d'outils pour synchroniser votre compte avec le serveur Zimbra.

-Si vous sélectionnez une fréquence trop rapide (par exemple une minute), les performances de votre ordinateur risquent d'être affectées.

-Vous pouvez changer cette valeur à tout moment en cliquant sur Config compte, sélectionnant le compte et modifiant le paramètre.

Dans votre boîte aux lettres, sélectionnez le ou les dossiers qui doivent être synchronisés.

* Synchroniser tous les calendriers

* Synchroniser tous les contacts et les groupes

  1. Lorsque vous avez terminé, cliquez sur Valider & Enregistrer.

Zimbra Desktop valide les informations de votre compte. Si cette validation de compte s'exécute sans problème, la boîte de dialogue Mes comptes apparaît. Si cette validation de compte ne peut pas être exécutée, la boîte de dialogue initiale est réaffichée pour vous permettre de revoir vos entrées et de corriger les erreurs.

La synchronisation du compte avec Zimbra Desktop démarre automatiquement.

  1. Cliquez sur Lancer Desktop pour activer votre Desktop. Pendant la synchronisation du compte, vous pouvez continuer à travailler dans Zimbra Desktop.
