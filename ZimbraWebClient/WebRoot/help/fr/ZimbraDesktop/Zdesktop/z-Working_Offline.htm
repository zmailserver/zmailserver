Travailler hors ligne

Les mails que vous téléchargez de vos comptes restent toujours accessibles dans Zimbra Desktop même si vous n'êtes pas connecté à Internet Travailler dans Zimbra Desktop lorsque vous n'êtes pas connecté à Internet s'appelle "travailler hors ligne".

Lorsque vous travaillez hors ligne, Zimbra Desktop ne peut pas accéder à vos comptes, mais vous pouvez continuer de travailler dans Zimbra Desktop comme si vous étiez connecté à Internet : Vous pouvez consulter vos mails, écrire des mails, nettoyer les dossiers et effectuer d'autres modifications qui sont synchronisées avec votre compte lorsque vous accéder de nouveau à Internet.

Lorsque vous travaillez hors ligne, le carré affiché à côté du nom de votre compte est rouge.

 

Remarque : il est possible que vous soyez hors ligne avant que l’icône devienne rouge lors de la mise à jour de l'état du réseau lorsque Zimbra Desktop essaie de se synchroniser avec le serveur. Si cette tentative échoue, l'icône devient rouge.

 

Vous pouvez forcer tous les comptes à passer hors ligne en cliquant sur la petite icône à côté de Comptes dans le Panneau d'aperçu.

Cliquez à nouveau sur l'icône pour revenir en ligne.  

Actions possibles en mode hors ligne

  • Lire des mails, les déplacer d'un dossier à un autre, marquer d'un drapeau ceux qui requièrent votre attention.

  • Écrire la réponse à un mail et l'envoyer, faire suivre un mail, créer un mail. Les messsages sont enregistrés dans le dossier Courrier sortant ; ils sont envoyés automatiquement lorsque vous vous reconnectez à Internet.

  • Vous pouvez spprimer des mails Ils sont placés dans le dossier Corbeille.  Pour le compte Zimbra, les messages sont transférés vers le dossier Corbeille de votre compte lorsque vous vous reconnectez à Internet. Si vous videz votre dossier Corbeille, les mails sont supprimés.

  • Exécuter les fonctions Recherche et Recherche avancée pour rechercher des mails et des pièces jointes présents dans votre compte Zimbra Desktop.

  • Vous pouvez créer une recherche et l'enregistrer.

  • Vous pouvez modifier vos préférences Zimbra Desktop . Ces options affectent la manière dont vous travaillez dans Zimbra Desktop. Les modifications de préférences effectuées dans Zimbra Web Client Desktop sont automatiquement répercutées dans les de votre compte Zimbra Web Client. Les modifications apportées dans les préférence des autres compte de messagerie affectent uniquement les paramètres Zimbra Desktop.  Les paramètres du compte ne changent pas.
