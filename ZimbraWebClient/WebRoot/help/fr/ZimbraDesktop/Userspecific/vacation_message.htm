Création de messages d'absence

Remarque : Disponible pour les comptes de la société Zimbra Web Client uniquement.

Cette fonction est s'appelle également Réponse automatique d'absence. Vous pouvez configurer un message d'absence qui répond automatiquement aux personnes qui vous envoient des mails lorsque vous êtes absent pour une durée prolongée. Ce message est envoyé une seule fois à chaque destinataire, quel que soit le nombre de mails que la personne vous envoie pendant votre d'absence.

Pour configurer un message d'absence :

  1. Dans l'onglet Préférences, sélectionnez Mail.

  2. Dans la section Réception des mails, cochez Envoyer une réponse automatique par mail.

  3. Dans la zone de texte, saisissez le message à envoyer, tel que "Je suis actuellement absent. Je consulte ma messagerie vocale, mais pas ma boîte aux lettres. Je serai de retour le 1er juin 2010."

  4. Définissez les dates de début et de fin pour l'utilisation de ce message.

  5. Cliquez sur Enregistrer. La fonction de message d'absence est immédiatement activée.
