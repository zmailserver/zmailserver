Utilisation de l'éditeur RTF

Vous pouvez créer vos messages et documents dans le porte-documents en utilisant l'éditeur RTF pour formater le contenu de la page.

Lorsque vous éditez en utilisant l'éditeur RTF, la barre d'outils permet de formater facilement la présentation de votre page. Choisissez la taille et le type de polices, les styles de texte, les couleur depolices et les couleurs d'arrière-plan. Ces outils de formatage fonctionnent de la même manière que de nombreux outils de traitement de texte. En outre, vous pouvez cliquer pour ajouter des tableaux, des pièces jointes et des images à votre page.

Ajouter des styles aux paragraphes

L'icône    dans la barre d'outils affiche les styles de paragraphe disponibles. Ces styles de paragraphes facilitent le formatage du texte et permettent de conserver un style cohérent dans le document. Les styles disponibles sont : Normal pour la plupart des paragraphes, six niveaux de Titres pour identifier les titres de sections, Adresse pour mettre le texte en italique et Préformaté si vous voulez afficher le texte dans la police Courier. Ces styles ne peuvent pas être modifiés, mais vous pouvez utiliser les options d'édition pour modifier la police, la taille, la couleur, etc.

Ajouter des tableaux à votre page

L'ajout de tableaux à votre page est une opération en trois étapes. Vous ajoutez un tableau, configurez les propriétés du tableau, puis ajoutez un contenu au tableau. Pour ajouter un tableau, cliquez sur l'icône Tableau dans la barre d'outils, choisissez Insérer un tableau, puis sélectionnez le nombre de lignes et de colonnes du tableau.

Ajout de pièces jointes et d'images à votre page

Vous pouvez uploader des fichiers liés au fichier de document. Cliquez sur le trombone ou sur des images, l'icône et sur Parcourir pour rechercher le fichier à uploader. Vous recevez un message si vous uploadez un fichier qui existe déjà dans votre porte-documents.

Lien vers une  page Web

Vous pouvez établir un lien vers une URL Web externe et  donner au lien un titre descriptif au lieu d'afficher  l'URL. Cliquez sur l'icône et entrez la page ou l'URL Web à lier.
