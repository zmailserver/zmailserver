Types de fichiers de pièce jointe pris en charge

Vous pouvez ouvrir toutes les pièces jointes du mail directement depuis Zimbra Web Client, à condition que votre poste de travail dispose de l'application appropriée et que l'extension ne se soit pas bloquée par votre administrateur système.  Par exemple, pour ouvrir un document créé avec Microsoft Word, vous devez lancer l'application Microsoft Word sur votre ordinateur.

Sur la plupart des ordinateurs, le type de fichier est indiqué par son extension, généralement représentée par un point suivi d'un code de trois lettres. Des fichiers expenses.xls ou New-Items.doc sont des exemples de fichiers. Si le type du fichier est pris en charge par le logiciel de votre ordinateur, vous pouvez généralement cliquer deux fois sur le fichier et l'ordinateur lance automatiquement l'application adaptée à la lecture de ce fichier.

De nombreux types de fichiers peuvent être ouverts directement à partir de votre navigateur avec divers plug-ins. En général, si vous cliquez sur le lien d'un fichier dans une page Web, votre navigateur détecte automatiquement si vous disposez du plug-in adéquat et vous invite à le télécharger si vous n'en êtes pas doté. Remarque : il est possible que les politiques d'administration de votre système n'autorisent pas les utilisateurs à télécharger et installer des logiciels.

Remarques :
Sur les ordinateurs Windows, l'ordinateur présume que le type de fichier dans le nom du fichier correspond au format concerné. Même si le fichier est correctement formaté en interne, l'ordinateur ne peut pas le reconnaître et ouvrir l'application correspondante et s'il ne porte pas l'extension de type de fichier correcte.

Un nombre croissant de virus informatiques visent à se propager via des fichiers joints. Par conséquent, dans certains environnements, les administrateurs systèmes ont décidé de bloquer les mails entrants contenant certains types de pièces jointes, notamment les fichiers ayant l'extension . ou dont le nom de fichier contient EXE ou .ZIP. Si vous envoyez un mail à un utilisateur dont le système de messagerie a été configuré pour bloquer certains types de pièces jointes, il est possible que vous receviez une notification indiquant que le mail a été bloqué.

Voici la liste des extensions de fichier les plus couramment utilisées.

  • .TXT - indique un fichier texte, c'est-à-dire un fichier contenant uniquement des informations textuelles. Les fichiers textes peuvent être ouverts et lus avec de nombreux éditeurs de texte, notamment Bloc-notes, WordPad, Word, des éditeurs de ligne de commande, des navigateurs, et même des tableurs.

  • .HTM - Indique un fichier HTML, un type de fichier texte spécifique généralement ouvert avec un navigateur Web.

  • .ZIP - Indique un fichier qui a été compressé à l'aide d'une compression ZIP. Vous devrez disposer du programme WinZip ou d'un autre programme de compression pour pouvoir lire et extraire le fichier ZIP.

  • .EXE - Indique un fichier exécutable ou un fichier de programme. Vous double-cliquez sur ce type de fichier pour lancer l'application.

  • .DOC - Indique un fichier de document Microsoft Word.

  • .JPG - Indique une image graphique au format JPEG. Vous pouvez ouvrir des fichiers JPEG dans un navigateur ou en utilisant divers programmes graphiques, tels que Paint ou Adobe Photoshop.

  • .GIF - Indique une image graphique de format GIF. Vous pouvez ouvrir des fichiers GIF dans un navigateur ou à l'aide d'un programme graphique.

  • .MPG - Indique un clip vidéo ou une animation de format MPEG. Ce type de fichier peut souvent être lu avec un plug-in de navigateur ou un lecteur tel que Windows Media Player.

  • .WMV - Indique un clip vidéo de format WMV, qui est un format exclusif à Windows.

  • .PDF - Indique un document de format de document portable. Vous pouvez généralement l'ouvrir dans votre navigateur à l'aide d'un plug-in.
