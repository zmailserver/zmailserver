Demander un accusé de réception

Remarque : Dans Zimbra Desktop, l'option Accusé de lecture est uniquement disponible avec le Zimbra Web Client .

Vous pouvez marquer un mail que vous rédige afin de recevoir un message d'accusé de réception lorsque le destinataire ouvre le mail.  Lorsque vous rédigez un mail, choisissez Accusé de lecture dans le menu Options de la barre d'outils de création. Le destinataire a alors le choix d'envoyer ou pas l'accusé de lecture.

Dans la section> Accusé de lecture de Préférences>Mail, les utilisateurs peuvent définir leurs préférences pour les mails marqués d'un drapeau indiquant une demande d'accusé de réception. Les options suivantes peuvent être définies :

  • Ne jamais envoyer un accusé de lecture. Cette option est définie par défaut. Lorsque vous recevez un mail avec une demande d'accusé de lecture, la requête est ignorée. Le destinataire n'est pas averti de la demande.

  • Toujours envoyer un accusé de lecture. Un accusé de lecture est envoyé automatiquement lorsque le mail marqué d'un drapeau pour un accusé de lecture est ouvert.

  • Me demander.  Il est demandé aux destinataires s'ils souhaitent envoyer l'accusé de réception.

Aucun mail d'accusé de réception de lecture n'est envoyé si un destinataire choisit plusieurs mails, clique avec le bouton droit et choisit Marquer le mail lu.

 

Remarque : Le drapeau d'accusé de lecture ne fonctionne pas pour les mails envoyés aux utilisateurs de systèmes autres que ZWC
