Menus contextuels (clic droit)

De nombreuses parties de Zimbra Desktop affichent des menus contextuels d'action quand vous placez le curseur sur un élément, tel qu'un dossier, puis cliquez avec le bouton droit de la souris. Les options du menu qui s'affichent varient suivant le type d'élément.

Cliquez avec le bouton droit sur les éléments suivants pour afficher des menus contextuels :

  • Conversations. Vous pouvez marquer des conversations comme lues ou non lues, appliquer des tags aux conversations, supprimer des conversations, ou déplacer des conversations dans des dossiers.

  • Mails. Vous pouvez répondre au mail ou le faire suivre, le marquer comme lu ou non lu, lui appliquer des tags, supprimer le mail, le transférer vers un autre dossier, l'imprimer, afficher les données de l'en-tête d'origine et créer un filtre. L'option de modification du message comme nouveau message permet de renvoyer un message sans avoir à le faire suivre.

  • Noms des en-têtes de mails. Vous pouvez effectuer des recherches sur le nom, rédiger un mail à envoyer à la personne ou ajouter la personne à votre liste de contacts personnels. Si le nom correspond à un contact existant, vous pouvez modifier les données du contact.

  • Dossiers. Vous pouvez créer un dossier, marquer tous les éléments du dossier comme lus, supprimer ou renommer le dossier, déplacer le dossier, extraire tous les dossiers ou effectuer une recherche. Vous ne pouvez pas supprimer les dossiers système (Boîte de réception, Envoyés, Corbeille, Spams ou Brouillons).

  • Barre d'outils de l'onglet Mail. Vous pouvez sélectionner les colonnes à afficher dans le panneau de contenu du mail.
