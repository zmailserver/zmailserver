Affichage des mails

Il est possible d'afficher les messages par conversation ou par message. La vue de conversation affiche vos mails regroupés par objet. L'affichage du message affiche chaque message individuel que vous recevez et les réponses à un mail apparaissent comme des messages séparés dans votre boîte de réception.

Par défaut, les messages sont affichés dans la vue Conversation, mais vous pouvez modifier l'affichage depuis le lien Affichage dans la barre d'outils Mail.

Vous pouvez également indiquer que vous voulez afficher les messages dans le Panneau de lecture dans le bas ou à droite dans le panneau de contenu ou désactiver le ^panneau de lecture.
