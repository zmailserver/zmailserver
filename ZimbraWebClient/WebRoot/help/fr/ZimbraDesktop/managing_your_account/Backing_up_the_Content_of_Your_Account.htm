Transférer les données d'un compte vers un autre

Vous pouvez exporter toutes les données de votre compte (mails, pièces jointes, contacts, calendriers, tâches, dossiers Porte-documents) ou limiter l'exportation à certains éléments.  Vous pouvez alors enregistrer les données sur votre ordinateur ou dans un autre emplacement pour disposer d'une sauvegarde. Les données exportées sont enregistrées sous la forme d'un fichier-archive (tar+gzip, tgz) que vous pouvez importer pour restaurer votre compte.

Vous pouvez exporter tout ou partie des données d'un compte (mails, pièces jointes, contacts, calendriers, tâches, carnets ou/et dossiers du Porte-documents) à l'aide de la fonction Import/Export de Zimbra Desktop, puis importer ces données vers un autre compte dans Zimbra Desktop.  Les données exportées sont enregistrées sous la forme d'un fichier-archive via tar+gzip (tgz) que vous pouvez importer vers un autre compte dans Desktop.  Vous pouvez également utiliser la fonction Export/Import pour enregistrer tout ou partie des données de votre compte sur votre ordinateur ou dans un autre emplacement comme sauvegarde. Les données sont enregistrées sous la forme d'un fichier-archive via tar+gzip (tgz) que vous pouvez importer rapidement pour restaurer votre compte.

 Remarque : Les dossiers de contact exportés sont enregistrés dans des fichiers .csv. Les dossiers du calendrier exportés sont enregistrés dans des fichiers .ics.

Les données sont copiées sans être supprimées de votre compte.

Le fichier est exporté à partir de Préférences> Importer/Exporter.

Pour exporter des données

  1. Dans l'onglet Préférences du compte à exporter, cliquez sur le dossier Importer/Exporter.

  2. Vous pouvez exporter toutes les données du compte ou sélectionner le type de dossier à exporter.

    • Pour exporter toutes les données du compte, sélectionnez Compte. Remarque : Les dossiers Corbeille et Spam ne peuvent pas être exportés.

    • Pour exporter une liste de contacts ou un calendrier, sélectionnez l'option correspondante Lorsque vous sélectionnez un calendrier ou un contact pour l'exporter, sélectionner le format du fichier d'exportation. Voir Exportation de carnets d'adresses et Exportation de votre calendrier iCal.

    • Pour sélectionner différents types d'éléments à exporter simultanément, cochez Paramètres Avancés pour développer la vue. Cochez la case du ou des types d'éléments à exporter Tous les éléments correspondant au type choisi dans tous les dossiers sous le dossier sélectionné sont inclus. Pour exporter des données spécifiques, vous pouvez utiliser le champ de filtrage de recherche pour définir le type de donnés à exporter. Voir également la description du langage de requête et les exemples de recherche.  

  1. Cliquez sur Exporter et sélectionnez l'emplacement d'enregistrement du fichier.

  2. Cliquez sur OK.

Remarque : Le fichier exporté peut être consulté à l'aide d'un programme d'archivage tel que WinRAR.

Pour importer les données de compte enregistrées

Pour restaurer un compte, il suffit d'utiliser le fichier que vous avez exporté.  

  1. Dans l'onglet Préférences, cliquez sur le dossier Import/Export.

  2. Dans la section Importer, recherchez le fichier à importer.  Si vous importez un calendrier ou des contacts, vous devez spécifier le format des données importées Voir Importer des contacts et Importer votre calendrier iCal.  Lorsque  vous sélectionnez Compte, le seul fichier tgz que vous avez exporté peut être importé vers votre compte.  

Remarque : Le type Compte correspond au compte que vous utilisez actuellement dans (Zimbra, Yahoo!, Gmail, etc.).

  1. Pour rechercher le fichier à importer, cliquez sur Parcourir.

  2. Sélectionnez la destination du fichier.

  3. Si vous importez les données d'un compte, cochez la case appropriée dans les champs Éliminer les doublons.

    • Ignorer. Les fichiers en double ne sont pas résolus. Tous les fichiers sont ajoutés et la version existante d'un objet est conservée.

    • Remplacer. Les éléments qui ont changé depuis l'exportation des données du compte reviennent à la version du fichier exporté. Les anciens éléments sont réimportés. Les nouveaux éléments ajoutés à votre compte depuis la création du fichier exporté sont conservés.

    • Réinitialiser. Tous les éléments du type sélectionné sous le dossier sélectionné sont supprimés avant l'importation des nouveaux éléments.

  1. Si vous préférez importer des types de données spécifiques à partir du fichier tgz, cochez la case Paramètres avancés pour développer la vue et sélectionner les types d'éléments à restaurer.

Dans la section Types de données, sélectionnez les types d'éléments du fichier tgz enregistré à importer. Lorsque vous cliquez sur Importer, seuls ces types d'éléments sont importés.

  1. Cliquez sur Importer.
