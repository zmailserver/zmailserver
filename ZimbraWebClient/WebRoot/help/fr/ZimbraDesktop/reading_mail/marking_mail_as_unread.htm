Marquer des mails comme non lus

Les messages que vous avez ouverts sont marqués comme étant lus. Les conversations contenant des messages non lus sont affichées en gras.

Vous pouvez définir les préférences pour marquer ou non un message comme étant lu quand vous lisez le message dans le panneau de lecture. Vous pouvez définir votre préférence pour marquer un message que vous consultez dans le panneau de lecture comme étant lu ou seulement marquer un message comme étant lu si vous l'ouvrez dans le panneau de contenu (ou dans une fenêtre séparée). Il est possible de définir cela dans Préférences mail.

Vous pouvez rétablir l'état d'un mail (de lu à non lu) en cliquant avec le bouton droit de la souris sur le mail, puis en choisissant Marquer comme non lu dans le menu qui s'affiche. Vous pouvez également marquer une conversation comme étant non lue.

  • Lorsque vous marquez une conversation comme n'ayant pas été lue, tous les messages qu'elle contient sont marqués comme n'ayant pas été lus.

  • Lorsque vous marquez un seul message comme n'ayant pas été lu, la conversation qui contient ce message est aussi marquée comme tel.

Vous pouvez utiliser le même procédure pour marquer des mails non lus comme étant lus.
