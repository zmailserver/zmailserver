Exportation de votre calendrier iCal

Vous pouvez exporter vos calendriers et enregistrer le fichier au format iCalendars (.ics). iCalendars est le standard pour les échanges des données de calendrier via Internet et .ics est le format standard des informations iCalendar.  

La fonction d'exportation est disponible depuis vos préférences d'importation/exportation.

Pour exporter un calendrier iCal :

  1. Sélectionnez Préférences, puis cliquez sur Importer/Exporter

  2. Dans la section "Export", sélectionnez le type Calendrier

  3. Dans le champ Source, cliquez pour sélectionner le calendrier à exporter.

  4. Cliquez sur Exporter.

  5. Sélectionnez l'emplacement dans lequel vous voulez enregistrer le fichier .ics, puis cliquez sur OK.
