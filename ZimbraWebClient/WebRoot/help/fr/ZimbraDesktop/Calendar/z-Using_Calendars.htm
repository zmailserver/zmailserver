Utiliser les calendriers dans Zimbra Desktop

Vous pouvez conserver un calendrier Zimbra Desktop pour chacun de vos comptes.

Les calendriers d'un compte Zimbrasont synchronisés avec Zimbra Desktop. Lorsque vous ajoutez un nouvel événement, modifiez ou supprimez un événement dans Zimbra Desktop, ces modifications sont synchronisées avec votre calendrier du compte Zimbra.

Vous pouvez conserver un calendrier dans Zimbra Desktop pour chacun de vos autres comptes, mais les événements du calendrier ne sont pas synchronisés avec vos calendriers de compte pour ces comptes.

Si vous utilisez un autre programme de calendrier, vous pouvez l'enregistrer en tant que fichier iCalendar (.ics) et l'importer dans votre calendrier. Si vous effectuez des modifications pour le calendrier dans Zimbra Desktop, ces modifications ne sont pas synchronisées avec le calendrier. Si vous rechargez le calendrier .ical, le calendrier de votre application Zimbra Desktop est écrasé et tous les événements que vous avez ajoutés seront perdus.
