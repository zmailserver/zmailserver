Affichage de la disponibilité des participants et de l'emplacement dans le panneau de suggestion

Lorsque vous organisez des réunions, vous pouvez vérifier les horaires pendant lesquels les participants sont occupés ou libres pour vous permettre de trouver un horaire approprié pour tous les participants. Lorsque vous ajoutez des participants au champ Participants, un panneau de suggestion dans le panneau d'aperçu indique les disponibilités par créneaux de 30 minutes pour les participants et pour au moins une salle (emplacement). Le nombre à droite est le nombre d'emplacements disponibles à l'heure définie. Vous pouvez cliquer sur ce nombre pour afficher la liste des emplacements suggérés et sélectionner un emplacement. L'emplacement sélectionné est ajouté au champ Emplacement dans la page du rendez-vous.

Remarque : Les emplacements des salles sont affichés s'ils ont été configurés dans l'annuaire de l'entreprise.

Le mini-calendrier vous suggère la disponibilité des participants par un code de couleur.

  • Vert signifie que tous les participants et au moins un emplacement sont disponibles ces jours-là.

  • Orange signifie que tout le monde ne peut pas être présent, mais que plus de la moitié le peut et qu'au moins un emplacement est disponible. Vous pouvez placer le curseur sur la date pour afficher jusqu'à 10 noms de personne indisponibles.

  • Rouge signifie que plus de la moitié des participants ne peut pas être présente à la réunion ces jours-là.

Dans l'exemple ci-dessous, tous les participants sont disponibles tout le mois sauf le 18 janvier. Moins de la moitié est disponible à cette date.

Afficher les emplacements pour l'organisateur

Pour trouver les emplacements disponibles en tant qu'organisateur, ouvrez une page de rendez-vous et avant d'ajouter un invité, dans la zone Horaires suggérés du Panneau d'aperçu, cliquez sur Afficher suggestions pour.(date).

Gestion de la production des suggestions

Vous pouvez configurer l'affichage des informations dans le panneau de suggestion. La configuration par défaut est l'affichage des disponibilités des participants et de l'emplacement dès que vous saisissez le nom du premier participant dans le champ Participant. Vous pouvez modifier le paramètre par défaut et configurez d'autres préférences à partir de la boîte de dialogue Préférences de suggestion.

Cliquez sur le titre Horaires suggérés dans le panneau d'aperçu pour ouvrir la boîte de dialogue Préférences de suggestion.

Configurez les options suivantes :

  • Suggestions. Les options sont configurées sur Automatique (par défaut)  ou Manuel.

    • Pour afficher manuellement l'heure des participants et les emplacements, cliquez sur Manuel. Lorsque vous souhaitez afficher  les informations sur les participants / emplacements, cliquez sur Afficher suggestions pour.(date). dans la zone Suggérer horaires.

  • Préférences d'emplacement.  Dans cette section, vous pouvez définir le type des emplacement à inclure dans votre recherche.  Laissez cette section vierge si vous n'avez pas de préférences d'emplacement.

    • Dans le champ Nom, vous pouvez saisir un nom d'emplacement partiel ; les  emplacements dont le nom comporte cette chaîne de caractères seront inclus dans la recherche.  Par exemple, le nom de l'emplacement des salles du bâtiment 3 comporte les caractères  "Bldg3".  Si vous ajoutez "Bldg3" dans le champ Nom, seuls les emplacements dont le nom comporte les caractères "Bldg3" seront suggérés.

    • Si vous ne souhaitez pas que les emplacements soient suggérés avec les disponibilités des participants, désélectionnez la case Suggérer les salles disponibles. 

  • Préférences horaires. Vous spécifiez les heures qui devraient être incluses dans la recherche de disponibilité des participants.

    • Mes heures de travail uniquement. Recherche uniquement les horaires qui correspondent à vos heures de travail configurées. Ce sont la semaine ouvrée et les heures que vous avez défini dans vos préférences de calendrier.

    • Toutes les heures de travail des participants.Lors de la recherche d'un emplacement, les heures de travail de toutes les personnes sont prises en considération. Remarque :  Si vous incluez des participants se trouvant dans des fuseaux horaires différents, notamment ceux séparés par de nombreux fuseaux horaires, vous ne pourrez jamais trouver un emplacement permettant de rassembler tous les participants.  Dans ce cas, le panneau de suggestions n'affiche aucune ressource d'emplacement.

    • Heures chômées. Cette recherche fixe les heures qui ne sont pas marquées comme ouvrées, telles que les soirées et les week-ends.
