Modification des informations d'un contact

Pour rechercher et ouvrir une fenêtre de contact :

  1. Activez l'onglet Carnet d'adresses. La case de recherche affiche dans:contacts.

  2. Dans le champ de Recherche, saisissez les critères de recherche, par exemple un nom ou un prénom ou le nom du groupe.

Pour effectuer une recherche par adresse mail, vous devez saisir l'adresse complète. Le prénom ou le nom de famille doivent correspondre à un mot complet. Par exemple, vous pouvez saisir Dupont ou dupont@acme.com.

  1. Dans la liste déroulante en regard du champ de recherche, sélectionnez Contacts pour rechercher dans vos carnets d'adresses

  2. Dans la fenêtre Modifier le contact, double-cliquez sur le nom pour ouvrir la fenêtre de contact et apporter des modifications.

  3. Effectuez les modifications, puis cliquez sur Enregistrer pour valider vos changements.

Pour ouvrir une fenêtre de contact à modifier :

  1. Dans un carnet d'adresses, cliquez avec le bouton droit sur le contact à modifier, puis choisissez Modifier le contact.

  2. Effectuez les modifications, puis cliquez sur Enregistrer pour valider vos changements.

Pour transférer un contact vers un autre carnet d'adresses, procédez comme suit :

Vous pouvez transférer un contact de l'une des manières suivantes :

Depuis la fiche du Contact

  1. Sélectionnez le contact, puis cliquez sur Modifier pour ouvrir la fiche du contact.

  2. Dans le menu déroulant du Carnet d'adresse, sélectionnez le carnet d'adresse à partir duquel vous souhaitez transférer le contact.

  3. Cliquez sur Enregistrer.

À partir du nom du contact

  1. Cliquez avec le bouton droit sur le contact à déplacer.

  2. Sélectionnez Déplacer. Dans la boîte de dialogue Déplacer contact, sélectionnez le carnet d'adresses.

  3. Cliquez sur OK.
