Configuration de règles de filtre

Le filtrage des mails permet de définir des règles pour gérer vos mails entrants et sortants.  Les filtres trient automatiquement vos mails en fonction des règles que vous avez définies.

Vous pouvez trier vos mails entrants pour :

  • Trier les messages entrants dans les dossiers

  • Marquer automatiquement les mails

  • Transférer les mails

  • Marquer le mail comme lu ou l'affecter d'un drapeau

  • Supprimer les mails

Par exemple, vous pouvez définir une règle de filtre pour identifier les mails provenant de votre supérieur hiérarchique direct et les transférer vers le dossier "De mon chef" ou transférer automatiquement les mails provenant d'une adresse spécifique vers le dossier Corbeille.

Vous pouvez trier vos mails sortants pour :

  • Trier les messages qui sont enregistrés dans le dossier Éléments envoyés dans d'autres dossiers

  • Marquer automatiquement les mails

  • Transférer les mails

  • Marquer le mail comme lu ou l'affecter d'un drapeau

  • Supprimer les mails

Conditions de filtre

Chaque filtre est une règle avec une ou plusieurs conditions et une destination. Chaque règle de filtre contient plusieurs conditions. Par exemple, si votre directeur vous envoie des mails provenant de plusieurs adresses telles que jane.smith@company.com et de jane-smith23@yahoo.com, vous pouvez créer le filtre "Directeur" ayant deux conditions, une pour chacune des adresses mails.

Les conditions incluent :

  • Des adresses spécifiques dans les champs De, À, Cc dans l'en-tête de mail

  • Mots ou caractères dans le champ d'objet ou dans le contenu d'un mail

  • La présence ou pas de pièces jointes

Toutes les conditions vous permettent de spécifier "n'est pas" comme condition négative. Par exemple, vous pouvez spécifier les mails qui ne contiennent pas un mot particulier.

Vous pouvez combiner des conditions pour rechercher un mail ayant des caractéristiques plus complexes.

Aucun vs Tous

Les conditions d'une règle de filtre peuvent être groupées à l'aide des éléments Indifférent ou Tous . L'utilisation de ces termes est identique aux recherches de type "ET" vs "OU" décrites dans la fonction Rechercher, avec Indifférent correspondant à OU et Tous correspondant à ET.

Si vous choisissez Indifférent lors de la configuration des conditions d'une règle de filtre, tout mail correspondant à l'une de ces conditions est considéré comme une correspondance. Néanmoins, si vous choisissez Tous, toutes les conditions spécifiées dans cette règle de filtre doivent être satisfaites pour que le filtre s'applique au mail.

Actions de règles de filtre

Chaque règle de filtre peut spécifier une ou plusieurs actions. Ces actions comprennent :

  • Conserver le mail dans la boîte de réception (aucune action)

  • Transférer le mail vers un dossier spécifié

  • Marquer le mail

  • Marquer le mail comme lu ou l'affecter d'un drapeau

  • Supprimer le mail. Cette action supprime le mail silencieusement. Remarque : cette action n'est pas la même que l'action Effacer qui vous est proposée dans votre menu. Lorsque vous effacez un objet, il est placé dans le dossier Corbeille. L'action Supprimer empêche le mail d'entrer dans votre boîte de réception

  • Transférer le mail vers une autre adresse

Remarque : dans chaque règle de filtre, Ne pas appliquer d'autres filtres  doit toujours être la dernière action. Elle empêche l'application d'autres règles de filtre aux mails entrants qui satisfont la règle actuelle.  Si cette action n'est pas spécifiée, les règles des filtres sont évaluées aux mails, même si le filtre actuel a une correspondance. Vous ne voulez probablement pas qu'un même message corresponde à plusieurs règles de filtres et exécuter plusieurs actions éventuellement contradictoires.

Moment d'application des filtres

Chaque mail entrant est testé par rapport aux règles de filtre dès l'instant où le mail est distribué et les actions du filtre sont alors appliquées aux mails correspondants.

Vous pouvez également appliquer des règles de filtre à un ensemble de mails déjà stockés dans votre compte. Dans le sous-onglet Filtres de mails de l'onglet Préférences, le bouton  Appliquer le filtre permet de sélectionner le filtre à appliquer aux dossiers de mails sélectionnés.

Ordre de filtre

Les filtres sont appliqués dans l'ordre dans lequel ils apparaissent dans les préférences de filtrede mail. Vous pouvez modifier cet ordre.

Dans chaque filtre, les conditions sont utilisées pour tester chaque mail. Si le mail satisfait collectivement aux conditions (la désignation Indifférent ou Tous comme convenu dans la règle de filtre), il est considéré comme une correspondance. L'ordre des conditions dans chaque filtre n'est pas important.

Si le mail correspond aux conditions, toutes les actions associées à la règle de filtre sont appliquées dans l'ordre dans lequel elles apparaissent dans le filtre.

Filtres actifs/ inactifs

L'ensemble des filtres définis actuellement sont répertoriés dans l'onglet Options, sur la page Filtres de mails. La case Active permet d'activer/désactiver les filtres sans avoir à supprimer les règles de filtre.
