Supprimer des dossiers

Vous pouvez supprimer des dossiers personnalisés, mais pas de dossiers système.

Lorsque vous supprimez un dossier, son contenu est déplacé vers le dossier Corbeille. Vous pouvez récupérer ce contenu dans le dossier Corbeille en le sélectionnant et en le faisant glisser vers un autre dossier.

Remarque: Le dossier Corbeille est régulièrement vidé, par conséquent les objets qu'il contient risquent d'être définitivement supprimés.

Supprimer un dossier :

  1. Sélectionnez le dossier à supprimer

  2. Dans le titre Dossiers du panneau d'aperçu, cliquez sur modifier.

  3. Sélectionnez Supprimer ce dossier.

  4. Cliquez sur Supprimer dossier.

Si vous souhaitez sauvegarder certains mails du dossier, déplacez-les vers un autre dossier avant de le supprimer.

Supprimer le contenu d'un dossier

Vous pouvez supprimer tout le contenu de l’un de vos dossiers, mais pas le dossier lui-même.

  1. Sélectionnez le dossier dont vous voulez supprimer le contenu.

  2. Dans le titre Dossier du panneau d'aperçu, cliquez sur modifier.

  3. Cochez Supprimer définitivement tous les objets de ce dossier. Cette action est irréversible.

  4. Cliquez sur Supprimer tous les objets. Le contenu du dossier est immédiatement supprimé.
