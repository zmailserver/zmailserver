Marquer un mail avec un drapeau

L'icône de drapeau de la liste de mails est un indicateur oui/non qui indique si le mail a été marqué d'un drapeau. Cette icône peut être utilisée pour indiquer un élément exigeant une action, ou pour distinguer le mail ou la conversation parmi un grand nombre d'autres éléments de moindre importance.

Pour activer le drapeau, choisissez le mail ou la conversation, puis dans le menu déroulant Actions, sélectionnez Ajouter drapeau.
