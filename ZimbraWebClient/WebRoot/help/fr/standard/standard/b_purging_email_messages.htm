Purger des mails

Pour supprimer des mails contenus dans la Corbeille, ouvrez le dossier Corbeille, puis, dans le panneau de contenu, cliquez sur Supprimer. Tous les mails sont définitivement supprimés.

En outre, votre compte peut être configuré pour vider automatiquement les anciens messages de tout dossier après une période spécifiée. L’administrateur système établit les règles concernant la durée pendant laquelle les mails sont conservés dans vos dossiers.  

Une fois purgés, les mails sont définitivement supprimés.
