Création de calendriers

Un calendrier a été créé pour vous ; vous pouvez en créer de nouveaux pour effectuer un suivi des différents types de fonctions. Par exemple, vous pouvez créer un calendrier pour le travail, un calendrier personnel, un calendrier pour les activités de la société, vous pouvez vous abonner à un calendrier accessible sur Internet et vous pouvez établir un lien vers un calendrier que vous partagez.

Créer un calendrier

  1. Activez l'onglet Calendrier.

  2. Dans le panneau Aperçu du calendrier, cliquez sur modifier.

  3. Dans la barre d'outils, cliquez sur Nouveau calendrier.

  4. Saisissez le nom du calendrier.

  5. Sélectionnez la couleur des activités du calendrier. Vous pouvez insérer un code couleur aux calendriers dans votre boîte aux lettres afin que les activités du calendrier s'affichent toujours dans la couleur choisie.

  6. Si vous ne voulez pas que d'autres utilisateurs aient accès aux activités du calendrier, cochez Exclure ce calendrier au sujet de vos disponibilités.

  7. Par défaut, l'interface enregistrée est activée. Les activités du calendrier s'affichent lorsque vous ouvrez l'affichage de calendrier. Si vous ne voulez pas afficher le calendrier en même temps que l'autre calendrier, désélectionnez la case. Dans ce cas, vous devez sélectionner le calendrier et l'ouvrir afin de consulter ses activités.

  8. Cliquez sur Créer calendrier.

Synchroniser un autre calendrier

  1. Sélectionnez l'onglet Calendrier.

  2. Dans le panneau Aperçu du calendrier, cliquez sur modifier. Les informations par défaut du calendrier s'affichent.  

  3. Dans la barre d'outils, cliquez sur S'inscrire au calendrier.  

  4. Saisissez le nom du calendrier, entrez l'URL d'un fichier iCal correct pour le calendrier distant, puis choisissez une couleur pour l'affichage des activités du calendrier.

  5. Si vous ne voulez pas que d'autres utilisateurs aient accès aux activités du calendrier, cochez Exclure ce calendrier au sujet de vos disponibilités.

  6. Par défaut, l'interface enregistrée est activée. Les activités de ce calendrier s’affichent lorsque vous ouvrez l'affichage du Calendrier. Si vous ne voulez pas afficher le calendrier en même temps que l'autre calendrier, désélectionnez la case. Dans ce cas, vous devez sélectionner le calendrier et l'ouvrir afin de consulter ses activités.

  7. Cliquez sur Créer calendrier.
