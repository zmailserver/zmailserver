Récupération d'objets supprimés

Le dossier Corbeille est disponible pour chaque application dans votre compte. Lorsque vous ouvrez le dossier Corbeille d'une application, seuls les éléments de cette application sont affichés. Mais quand vous sélectionnez Vider la corbeille à partir d'une application quelconque, les éléments de toutes les applications sont supprimés.

Vous pouvez récupérer des fichiers dans la corbeille en ouvrant le dossier Corbeille, en sélectionnant les mails à récupérer, puis en les transférer vers un autre dossier.

Attention : il est possible que la Corbeille soit vidée de manière automatique, ce qui implique que les mails qu'elle contient seront définitivement supprimés.
