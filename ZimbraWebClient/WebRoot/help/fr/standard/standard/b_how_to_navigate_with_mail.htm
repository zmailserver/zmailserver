Naviguer dans l'onglet Mail

Lorsque vous ouvrez l'onglet Mail, tous vos mails sont répertoriés. Dans cet affichage, vous pouvez ouvrir et lire vos mails.

  1. Les options incluent :

  • Cliquez sur Actualiser pour récupérer les nouveaux mails.

  • Cliquez sur Écrire pour rédiger un mail. Il s'affiche dans le panneau Contenu.

  • Cliquez sur Imprimer pour imprimer le message ou la conversation.

  • Cliquez sur Supprimer pour supprimer un message. Le message est placé dans le dossier Corbeille.

  • Utilisez la barre d'action Déplacer pour transférer des mails vers le dossier sélectionné.

  • Utiliser la barre d'action Actions pour marquer les messages sélectionnés comme lus ou non lus, balisés ou non balisés, étiquetés avec une étiquette déjà créé, ou non étiquettés.

  • Sélectionnez Agencé par: pour afficher vos messages en fonction de la date ou par l'objet.

  1. Barre d'en-tête pour le panneau Contenu. Si vous cochez la case dans l'en-tête, tous les mails présents dans la page sont également cochés.

  2. Dans le panneau Contenu, vos mails s'affichent sous forme de conversations ou de simples messages. Cochez pour sélectionner le mail ou la conversation pour lesquels vous voulez exécuter la tâche Déplacer ou Autres actions.
