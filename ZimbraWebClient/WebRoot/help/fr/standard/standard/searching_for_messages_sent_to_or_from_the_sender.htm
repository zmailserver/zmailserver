Recherche de messages envoyés à l'expéditeur ou reçus de celui-ci

Vous pouvez rechercher les messages envoyés à l'expéditeur d'un message ou reçus de celui-ci de l'une des manières suivantes :

  • En cliquant avec le bouton droit sur un message envoyé à l'expéditeur ou reçu de celui-ci

  • En utilisant le champ À/Cc du panneau Recherche avancée ou Recherche simple

Quand vous recherchez dans un dossier, seuls les résultats des messages se trouvant dans ce dossier s'affichent. Quand vous utilisez l'option Recherche avancée, vous pouvez sélectionner dans quels dossiers rechercher. Les résultats incluent les messages à partir des dossiers sélectionnés pour la recherche.

Rechercher les messages que vous avez reçus

Remarque : Sélectionnez un message et non une conversation pour effectuer cette recherche.

  1. Dans votre Boîte de réception ou dans d'autres dossiers personnels, sélectionnez un message reçu de l'expéditeur que vous voulez trouver.

  2. Cliquez avec le bouton droit sur le message et sélectionnez Rechercher les mails.

  3. Choisissez Reçu de l'expéditeur ou Envoyé à l'expéditeur. La recherche est exécutée et les messages qui correspondent à votre requête s'affichent dans le Panneau de contenu.

Utilisation de la recherche avancée

  1. Cliquez sur Avancé pour ouvrir la zone de recherche avancée.

  2. Dans le panneau Recherche simple, dans le champ De, saisissez le nom de l'expéditeur du message à rechercher. Plus le nom est spécifique, meilleurs sont les résultats. La recherche s'effectue dans la Boîte de réception par défaut.

  3. Pour rechercher dans d'autres dossiers, cliquez sur Dossier et sélectionnez les dossiers dans lesquels rechercher.

Dès que vous saisissez un critère dans un panneau de recherche, ou dès que vous avez effectué tout changement de sélection dans tout autre panneau de recherche, la recherche s'exécute.

Les éléments correspondant aux critères de la recherche s'affichent dans la partie inférieure droite. Si la fonction de conversation est activée, les conversations contenant des mails qui correspondent aux critères de la recherche s'affichent également.
