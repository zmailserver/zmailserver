Nettoyer votre calendrier

Vous pouvez retirer de votre calendrier les rendez-vous passés et futurs.

  1. Dans le titre Calendriers du panneau d'aperçu, cliquez sur modifier.

  2. Sélectionnez le calendrier à nettoyer.

  3.  Cochez la case Supprimer définitivement tous les rendez-vous de ce calendrier. Cette action ne peutpas être défaite.

  4. Cliquez sur Supprimer tous les rendez-vous.
