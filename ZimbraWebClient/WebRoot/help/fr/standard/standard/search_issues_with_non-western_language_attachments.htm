Problèmes de recherche avec les pièces jointes dans des langues autres que les langues occidentales

Si vous recherchez une pièce jointe comportant du texte dans une langue autre qu'une langue occidentale et si la pièce jointe n'est pas incluse dans les résultats de la recherche, cela peut être dû au fait qu'il n'est pas possible d'effectuer une recherche sur la pièce jointe.

Les documents de langues non occidentales qui n'ont pas un paramètre jeu de caractères (charset) identifié dans l'en-tête type de contenu de la pièce jointe pourrait ne pas être indexés correctement. Le paramètre jeu de caractères identifie le type de codage de caractères, ce qui est de savoir comment @Nom de la société@ identifie comment indexer le contenu. Les pièces jointes doivent être indexées correctement pour être inclues dans les recherches. Ce problème concerne généralement les fichiers ayant des pièces jointes en texte brut ou des pièces jointes csv.
