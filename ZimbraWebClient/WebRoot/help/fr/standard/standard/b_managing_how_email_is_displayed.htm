Choisir comment les mails sont affichés

Vous pouvez afficher vos mails sous forme de messages ou de conversations.

Pour modifier le mode de groupement des mails, accédez à l'onglet Préférences>Mail puis modifiez Grouper les mails par.

Vous pouvez également consulter les messages dans l'affichage à trois panneaux avec le panneau de lecture à gauche.

Mails

Les mails regroupés comme messages s'affichent dans vos dossier mails en commençant par le mail le plus récent. Les mails non lus sont affichés en gras. Lorsque vous examinez la ligne du mail, vous disposez des informations suivantes : expéditeur, existence d'une pièce jointe, objet du mail, nom du dossier dans lequel le mail est sauvegardé, taille et date à laquelle le mail a été reçu. Si les mails comportent des drapeaux ou des tags, ces informations sont également affichées.

Conversations

Si vous regroupez les mails par conversation, vos messages sont regroupés par objet. Une conversation est un groupe de messages contenant le même objet. Si l'objet est modifié, le message n'est pas inclus dans la conversation.

L'objet ne s'affiche qu'une fois dans la fenêtre Conversation. Le nombre de messages dans la conversation s'affiche entre parenthèses après l'objet. Lors de la réception d'un mail, la conversation est déplacée en haut de vos mails. La liste des expéditeurs contient le nom des huit derniers participants à la conversation. Les conversations contenant des messages non lus sont affichées en gras.

Fenêtre de conversation

Double-cliquez sur la conversation pour afficher les messages dans la conversation. Cliquez sur un message pour afficher son contenu dans le panneau de lecture au-dessous.

Dans la barre d'outils de mail, vous pouvez cliquer sur l'icône pour afficher le message dans une nouvelle fenêtre.

Tous les messages associés à la conversation sont affichés, même s'ils sont conservés dans d'autres dossiers. La colonne Dossier affiche le dossier dans lequel le message est conservé. Les conversations ne se poursuivent pas indéfiniment. Lorsque le dernier message de la conversation date d'au moins 30 jours, les messages envoyés avec le même objet entament une nouvelle conversation.
