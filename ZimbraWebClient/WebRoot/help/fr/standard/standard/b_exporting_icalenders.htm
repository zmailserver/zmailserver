Exportation de iCalendars

Vous pouvez exporter les calendriers enregistrés au format .ics. iCalendars est le standard pour les échanges des données de calendrier via Internet, et .ics est le format standard des informations iCalendar.

  1. Activez l'onglet Calendrier.

  2. Dans le panneau Aperçu du Calendrier, cliquez sur Modifier.

  3. Sélectionnez le calendrier à exporter.

  4.  Dans le champ Exporter .ICS, cliquez sur le lien du calendrier.

  5. Dans la boîte de dialogue, sélectionnez Enregistrer le fichier.

  6. Cliquez ensuite sur OK.
