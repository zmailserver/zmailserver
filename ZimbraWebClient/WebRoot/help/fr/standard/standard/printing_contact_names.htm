Impression des noms de contacts

Vous pouvez imprimer un carnet d'adresses dans sa totalité ou imprimer uniquement les données de contacts sélectionnés.

  • Pour imprimer les contacts sélectionnés dans le Carnet d'adresses, cochez la case située près des contacts et cliquez sur Imprimer.

  • Pour imprimer tous les contacts dans le carnet d'adresses, sélectionner la case à cocher dans la ligne Nom , qui sélectionne tous les contacts dans le carnet d'adresses et cliquer sur Imprimer.
