Personnalisation de l'affichage initial de votre boîte aux lettres

Sur la page Préférences mails, la fonction Recherche Mail par défaut permet de choisir la vue initiale qui s'affiche lorsque vous vous connectez à votre boîte aux lettres. La recherche par défaut s'effectue dans votre boîte de réception. Lorsque vous vous connectez, les résultats de votre boîte de réception s'affichent.

Vous pouvez modifier cette option pour qu'un autre dossier, un autre tag ou une recherche enregistrée s'affiche lorsque vous ouvrez le client Web pour la première fois.

  1. Aller à la page Préférences>messagerie.

  2. Dans la zone de texte affichant les messages>de recherche de messagerie par défaut des messagesspécifier quels messages sont affichés lorsque vous ouvrez votre boîte aux lettres électronique. ESaisissez le mot clé de recherche, suivi d'un : (deux points) et le dossier, balise, ou un nom de recherche sauvegardé.

    3. Le tableau suivant propose une liste d'exemples à saisir dans la zone de texte.

    est:non lu

    Pour afficher seulement les messages non lus

    balise :<nom de balise>

    Pour afficher les mails affectés de tags spécifiques. Par exemple, pour afficher seulement les messages marqués avec le tag ToDo, tapez tag:ToDo

    dans :<nom de dossier>

    Pour afficher les mails envoyés vers un dossier spécifique à l'aide d'un filtre de mails

    Remarque : utilisez des guillemets si le nom du dossier ou du tag comporte plusieurs mots. Par exemple, dans:”Mail non lu”

  3. Cliquez sur Enregistrer.
