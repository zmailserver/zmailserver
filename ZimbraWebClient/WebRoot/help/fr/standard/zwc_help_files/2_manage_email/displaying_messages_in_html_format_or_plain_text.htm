Affichage des mails au format HTML ou texte simple

Vous pouvez afficher les mails dans deux formats : HTML et texte simple.

•      Les mails HTML peuvent comporter un formatage, une numérotation, des puces, un fond coloré et des liens qui peuvent faciliter la lecture des mails.

•      Les mails au format texte simple peuvent toujours être lus par d'autres clients de messagerie. Le texte simple ne prend pas en charge les formatages, les styles, les liens et les images. Vous pouvez consulter des mails au format texte simple, même s'ils contiennent des balises HTML. Les balises HTML sont affichées dans le corps du mail au format texte simple.

Définir le format d'affichage du mail

  1. Ouvrez la page Préférences>Mail.

  2. Dans la rubrique Affichage des mails>Afficher le mail, sélectionnez Format HTML (si possible) ou Format Texte.

  3. Cliquez sur Enregistrer.
