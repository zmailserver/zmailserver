Enregistrement d'un mail en tant que brouillon

Vous pouvez enregistrer un mail pour le terminer et l'envoyer plus tard.

•       Cliquez sur Enregistrer le brouillon. Le mail est enregistré dans le dossier Brouillons.

Pour récupérer un brouillon, ouvrez le dossier Brouillons et cliquez sur le mail. Une fois que le mail est envoyé, il est retiré du dossier Brouillons.
