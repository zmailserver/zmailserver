Et par rapport Ou recherches

La recherche de messages qui contiennent à la fois X et Y est appelée une recherche ET, parce que le message doit répondre à tous les critères spécifiés pour être considéré comme une correspondance. La recherche de messages qui correspondent à aucun des critères spécifiés est appelée une recherche OU, parce que si le message contient soit X ou Y, alors il est considéré comme une correspondance. Voir

  • Travailler avec un panneau de recherche de base vous permet de construire une requête de recherche ET. Si vous ouvrez plusieurs instances des panneaux de recherche de base, vous créez une requête de recherche OU entre les panneaux.

  • La sélection de plusieurs boîtes dans les panneaux balises ou état vous permet de construire une requête de recherche ET. Si vous ouvrez plusieurs instances des panneaux Balises ou état et vous sélectionnez les différentes cases à cocher dans chacun, vous créez une requête de recherche OU entre les panneaux.

  • Dans les dossiers, Zimlets et panneaux de recherche temporelle, faire des sélections multiples au sein d'un seul panneau crée une recherche OUpour ces éléments.

Remarque : Seulement "OU" apparaît dans une requête. Si vous sélectionnez l'option pour afficher la requête de recherche dans la barre de recherche, la recherche de ka zone de texte se met à jour pour afficher la requête résultante. Avec le type de recherche ET, le mot ET n'apparaît pas.
