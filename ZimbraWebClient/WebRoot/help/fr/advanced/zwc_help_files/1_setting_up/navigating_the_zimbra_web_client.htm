Navigation dans @ClientNameLong@

Lorsque vous vous connectez, un écran similaire à celui ci-dessous s'affiche. Cette vue est la vue Mail. La vue change lorsque vous utilisez une autre application, telle que le carnet d'adresses ou le calendrier.

Option

Description

En-tête fenêtre

L'en-tête de la fenêtre affiche:

•Champ de recherche

•Utilisateur sous lequel vous êtes connecté. À partir du menu déroulant de votre nom d'utilisateur, vous pouvez accéder aux fonctions aide du produit, nouvelles fonctionnalités, à propos, modifier le mot de passe et déconnexion.

Onglets d'application

Les applications accessibles sont répertoriées dans les onglets de cette ligne, telles que les applications Mail ou Calendrier.

Panneau d'aperçu

Le panneau d'aperçu affiche:

•Dossiers : y compris les dossiers système (Boîte de réception, Envoyés, Brouillons,

•Spams (et Corbeille) ainsi que les dossiers personnalisés que vous avez créés ;

•Recherches : demandes de recherche que vous avez créées et enregistrées en vue d'une utilisation future ;

•Tags – tous les tags que vous avez créés. Cliquez sur un tag pour afficher rapidement tous les mails marqués avec celui-ci.

•Les liens Zimlet créés pour intégrer les applications tierces à votre boîte aux lettres.

•Lorsque vous utilisez l'application Calendrier, la fenêtre d'aperçu affiche votre liste de calendriers et les Zimlets.

Mini-calendrier

Le mini-calendrier est facultatif. Vous pouvez désactiver le mini-calendrier en utilisant la page Préférences>de calendrier.

Barre d'outils

La barre d'outils affiche les actions disponibles pour l'application que vous utilisez actuellement.

Panneau de contenu

Le panneau de contenu change selon l'application utilisée. Dans l'affichage Boîte de réception, le panneau affiche tous les mails ou conversations de votre boîte de réception.

Panneau de lecture

Les messages électroniques sont affichés dans le panneau de lecture si vous avez cette fonction activée. Activer cette fonction à partir du menu déroulant Afficher sur ​​la barre d'outils de messagerie.

Barre de recherche

Les options de recherche sont affichées dans cette zone. Vous pouvez également sauvegarder des recherches.

Obtenir de l'aide

Lorsque vous placez le curseur de la souris sur des éléments, tels que des boutons de commande, des conversations ou des adresses mails dans les en-têtes de mails, une petite zone de texte apparaît; elle contient la description de l'élément.

Informations de dossier

Si vous déplacez le curseur sur un dossier, le nombre de mails présents dans le dossier et la taille de ce dossier s'affichent.
