Modifier votre mot de passe

Vous pouvez modifier votre mot de passe. Si vous avez des difficultés pour modifier votre mot de passe, si vous l'avez oublié ou si l'option Mot de passe n'est pas affichée, contactez votre administrateur.

  1. Ouvrez la page Préférences>Général.

  2. Dans le menu Options de connexion>Mot de passe, cliquez sur Changer le mot de passe.

  3. Saisissez votre ancien mot de passe, votre nouveau mot de passe, et Confirmer.

  4. Cliquez sur Changer le mot de passe.

  5. Cliquez sur Enregistrer.
