Utilisation des clients avancés et standard

Deux versions de @ClientNameLong@ sont disponibles.

Le client Web avancé offre un ensemble complet de fonctions de collaboration Web. Il fonctionne avec les navigateurs les plus courants et les connexions Internet haut débit.

Si vous avez une connexion Internet lente ou si vous préférez une messagerie HTML, vous pouvez utiliser la version client Web standard. Elle contient globalement les mêmes fonctions que la version client Web avancé mais vous pouvez y accéder différemment.

Outils de navigation

•Le bouton Retour du navigateur vous permet de revenir à la page précédente que vous consultez.。 Vous pouvez également utiliser le bouton Suivant.

•N’utilisez pas le bouton Recharger (Actualiser) du navigateur. Cela redémarrera votre session.

Changer de version client

Vous pouvez définir la version du client par défaut pour vous connecter à. Si vous changez souvent de version client, vous pouvez la modifier.

  1. Aller à la page Préférences>générales.

  2. Dans la section Options de connexion>Connectez-vous à l'aide , sélectionner la façon dont vous souhaitez vous connecter.

    3. •Avancé. Offre un ensemble complet de fonctions de collaboration Web. Ce client Web exige un navigateur récent et une connexion Internet rapide.

    •Standard. Recommandé pour les connexions Internet lentes, pour des navigateurs moins récents et pour une accessibilité plus facile.

  3. Cliquez sur Enregistrer.

Lors de votre prochaine connexion, votre sélection sera définie par défaut.
