Fonctionnement des bulles d'adresse

Les noms dans le champ d'adresse s'affichent dans des « bulles » et non sous forme de texte. Les bulles sont plus interactives que les adresses mail affichées sous forme de texte. Lorsque vous utilisez des bulles pour les adresses mail, les actions suivantes sont disponibles :

•      Lorsque vous adressez un mail, vous pouvez cliquer avec le bouton droit sur une bulle pour afficher un menu d'options, notamment Supprimer, Modifier, Modifier des contacts.

•      Lorsque vous lisez un mail, vous pouvez cliquer avec le bouton droit sur une bulle pour afficher un menu d'options, notamment Ajouter aux contacts, Nouveau mail, Recherche avancée.

•      Vous pouvez cliquer avec le bouton gauche sur une seule bulle et la faire glisser vers un autre champ d'adresse ou sélectionner plusieurs bulles et faire glisser les adresses en utilisant l'outil de sélection « élastique » pour déplacer les bulles. La sélection « élastique » consiste à cliquer sur la partie blanche du champ et à faire glisser la zone de sélection au-dessus des bulles à sélectionner. Les bulles sélectionnées sont surlignées en bleu.  Vous pouvez maintenant les glisser-déposer dans un autre champ.

•      Vous pouvez utiliser CTRL+C pour copier et CTRL+V pour coller l'adresse.

•      Vous pouvez utiliser la touche de tabulation ou les flèches de navigation vers la gauche ou la droite pour vous déplacer entre les bulles.

•      Lorsque la saisie automatique est utilisée pour sélectionner une liste de distribution, la bulle comporte un signe + (plus) afin d'identifier l'adresse comme étant une liste de distribution. Vous pouvez cliquer sur le « + » pour développer la liste de distribution et afficher ses membres. Vous pouvez également cliquer sur une adresse de la liste pour envoyer un mail individuel à l'adresse sélectionnée.

Les bulles sont activées par défaut. Vous pouvez activer ou désactiver les bulles en vous rendant sur la page Préférences>Général, section Autre.
