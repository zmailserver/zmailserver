Envoi d'accusés de lecture

Les mails provenant d'autres utilisateurs @ClientNameShort@ peuvent être marqués afin que ces derniers reçoivent un mail d'accusé de réception lorsque vous ouvrez le mail. Vous pouvez spécifier la manière de gérer les demandes d'accusés de réception:

  1. Aller à la page Préférences>messagerie, section Réception des mails

  2. Dans la section Accusé de lecture, sélectionnez la manière dont vous souhaitez gérer les demandes d'accusés de lecture.

Option

Description

Ne jamais envoyer d'accusé de lecture.

Lorsque vous recevez un mail avec une demande d'accusé de lecture, la requête est ignorée. Vous n'êtes pas informé de la requête. Cette option est définie par défaut.

Toujours envoyer un accusé de lecture

Un mail d'accusé de lecture est envoyé lorsque vous ouvrez un mail affecté d'un drapeau indiquant une demande d'accusé de réception.

Me demander

Vous avez la possibilité d'envoyer un accusé de réception lorsque vous ouvrez un mail affecté d'un drapeau indiquant une demande d'accusé de réception.

  1. Cliquez sur Enregistrer.
