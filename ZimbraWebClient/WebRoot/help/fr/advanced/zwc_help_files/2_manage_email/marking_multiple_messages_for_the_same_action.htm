Marquer plusieurs mails pour la même action

Sur la page de contenu, utilisez les cases à cocher pour sélectionner un ou plusieurs mails ou conversations pour lesquel(le)s vous souhaitez exécuter la même action.

Remarque :   si les cases à cocher ne s'affichent pas, ouvrez la page Préférences>Général et sélectionnez Paramètres dans la section Autre.

Le fait de sélectionner la case située dans l'en-tête permet de cocher automatiquement tous les mails de la page.

Supprimer plusieurs mails à la fois

  1. Dans le panneau Contenu du mail, cochez les cases correspondant aux mails que vous souhaitez supprimer.

  2. Cliquez avec le bouton droit de la souris et sélectionnez Supprimer.

Marquer plusieurs mails à la fois

  1. Dans le panneau Contenu du mail, cochez les cases correspondant aux mails auxquels vous souhaitez appliquer le même tag.

  2. Cliquez avec le bouton droit de la souris et sélectionnez Libeller les Conversations.

  3. Sélectionnez le tag dans votre liste ou créez un tag.

Imprimer plusieurs mails à la fois

  1. Dans le panneau Contenu du mail, cochez les cases correspondant aux mails que vous souhaitez imprimer.

  2. Cliquez avec le bouton droit de la souris et sélectionnez Imprimer.

  3. Dans la boîte de dialogue Impression, cliquez sur OK. Tous les mails sont imprimés comme un seul fichier continu.

Faire suivre plusieurs mails à la fois

  1. Dans le panneau Contenu du mail, cochez les cases correspondant aux mails que vous souhaitez transférer.

  2. Cliquez avec le bouton droit de la souris et sélectionnez Faire suivre. Une page de création de mail s'ouvre avec les mails à transférer en pièce jointe.

  3. Rédigez le mail et envoyez.
