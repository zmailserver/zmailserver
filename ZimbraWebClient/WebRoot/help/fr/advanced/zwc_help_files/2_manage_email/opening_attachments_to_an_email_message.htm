Ouverture des pièces jointes d'un mail

Les mails peuvent comporter des pièces jointes, telles que des documents de traitement de texte, des feuilles de calcul, des fichiers texte, des fichiers ZIP, des images, des applications exécutables ou tout autre type de fichier résidant dans le réseau informatique de l'expéditeur. Un trombone s'affiche à côté de l'objet des mails contenant des pièces jointes.

Vous pouvez ouvrir toutes les pièces jointes directement à partir de votre compte à condition de disposer de l'application adéquate et que l'extension ne soit pas bloquée par votre administrateur système.

Si le type de fichier est supporté par un logiciel installé sur votre ordinateur, vous pouvez généralement double-cliquer sur le fichier et votre ordinateur lance automatiquement l'application adaptée à la lecture de ce fichier.

Les virus informatiques se propagent souvent via des pièces jointes. Ainsi, les administrateurs système peuvent bloquer un mail entrant contenant certains types de pièces jointes, souvent ceux dont le nom de fichier contient l'extension .EXE ou .ZIP .

Les virus informatiques peuvent se propager via des pièces jointes de mails. Il est courant que les administrateurs système appliquent un filtre antivirus pour les mails entrants ainsi qu'une analyse antivirus des ordinateurs. Néanmoins, il est toujours possible qu'un virus passe au travers.

Dans le cas des virus propagés par mails, la majorité d'entre eux sont activés lors de l'ouverture ou de la visualisation de la pièce jointe contenant le virus. Un grand nombre de ces virus est connu pour se propager à l'aide des entrées dans les carnets d'adresses des utilisateurs. Par conséquent, nous vous recommandons de rester vigilant lorsque vous ouvrez des pièces jointes, même si elles proviennent d'expéditeurs que vous connaissez.

Visualiser les pièces jointes

Pour visualiser une pièce jointe, ouvrez le message :

•    Cliquez sur le nom de la pièce jointe pour l'ouvrir (si vous disposez de l'application correcte).

•    Cliquez sur Télécharger pour choisir s'il faut ouvrir le fichier ou l'enregistrer dans votre ordinateur. Si vous sélectionnez Enregistrer sur disque, une boîte de dialogue de navigation apparaît pour que vous puissiez sélectionner l'emplacement où vous voulez enregistrer le fichier.

Le fichier reste attaché au mail jusqu'à ce que vous le supprimiez.

Aperçu des pièces jointes

Si vous disposez de l'option Aperçu, vous pouvez rapidement visualiser le contenu de la pièce jointe sans avoir à la télécharger ou à l'ouvrir.

•       Pour obtenir un aperçu d'une pièce jointe, cliquez sur Aperçu pour ouvrir la pièce jointe, même si l'application n'est pas installée sur votre réseau.

Supprimer des pièces jointes

Les pièces jointes attachées à vos mails ont une incidence sur le quota de votre compte. Il est possible que vous souhaitiez supprimer les pièces jointes volumineuses d'un mail. La suppression d'une pièce jointe supprime le fichier de votre boîte aux lettres. Si vous avez téléchargé le fichier, vous en possédez toujours une copie.

•       Pour supprimer une pièce jointe, ouvrez le message et cliquez sur Supprimer.
