Gérer une liste d'adresses acceptées

Vous pouvez ajouter les adresses mail de certains domaines à une liste d'adresses acceptées afin que les images insérées dans les mails provenant de ces adresses s'affichent automatiquement lorsque vous ouvrez le message.

Cette liste peut contenir jusqu'à 500 adresses.

  1. Ouvrez la page Préférences>Adresses acceptées.

  2. Saisissez l'adresse mail ou le nom de domaine dans la zone de texte.

    3. Pour inclure toutes les adresses d'un domaine spécifique, tapez * avant le nom @domain.com name.

  3. Cliquez sur Ajouter.

  4. Cliquez sur Enregistrer.
