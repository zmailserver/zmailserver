Réception d'une notification de nouveau mail sur une autre adresse mail

Vous pouvez configurer vos préférences mails afin de recevoir une notification sur une autre adresse mail lorsque vous recevez un message dans votre boîte aux lettres. Cette fonction peut s'avérer utile si vous recevez peu de mails sur ce compte mais qu'ils sont assez importants pour nécessiter une réponse rapide.

  1. Ouvrez la page Préférences>Mail.

  2. Dans la section Réception des mails>Arrivée d'un mail, sélectionnez Envoyer un mail de notification à.

  3. Saisissez l'adresse mail dans la zone de texte.

  4. Cliquez sur Enregistrer.
