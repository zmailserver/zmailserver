Vider votre dossier Corbeille

Chaque application de votre compte dispose d'un dossier Corbeille. Lorsque vous ouvrez le dossier Corbeille à partir d'une application telle que la messagerie, seuls les objets supprimés de cette application s'affichent. Toutefois, lorsque vous sélectionnez Vider la corbeille à partir d'une application, les éléments de toutes les applications placés dans la corbeille sont supprimés.

Vous pouvez vider votre dossier Corbeille à tout moment.

•       Cliquez avec le bouton droit de la souris sur le dossier Corbeille et sélectionnez Vider la corbeille. Tous les mails sont supprimés.

Les éléments supprimés de la corbeille sont retirés de votre compte et votre quota de compte est mis à jour.

Remarque :   les mails peuvent être supprimés automatiquement si des règles de purge ont été configurées pour la corbeille.
