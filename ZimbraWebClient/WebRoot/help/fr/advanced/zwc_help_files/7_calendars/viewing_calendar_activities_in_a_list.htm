Affichage des activités du calendrier dans une liste

L'affichage Liste vous permet de voir les détails ci-dessous (non disponibles dans les autres vues Calendrier). Vous pouvez modifier la façon dont les informations sont présentées.

Lors de l'utilisation de l'affichage Liste vous pouvez :

•      Afficher les rendez-vous d'une plage de dates spécifique : définissez les dates de-àdans les zones de texte Afficher les rendez-vous.

•      Cliquer sur l'en-tête de la colonne Calendrier pour gérer le rendez-vous par calendrier individuel.

•      Cliquer sur l'en-tête de la colonne État pour voir le rendez-vous en fonction de votre état.

•      Cliquer sur l'en-tête de la colonne Sujet pour voir un affichage alphabétique des sujets.

•      Cocher plusieurs cases de rendez-vous pour agir sur plusieurs mails en même temps. Cliquez avec le bouton droit de la souris et sélectionnez l'action à exécuter.
