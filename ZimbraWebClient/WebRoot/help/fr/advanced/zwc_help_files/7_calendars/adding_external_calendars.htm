Ajout de calendriers externes

Vous pouvez ajouter vos calendriers externes CalDAV Yahoo ou Google et vos abonnements iCal à votre liste de calendriers. Les modifications apportées à votre calendrier externe sont synchronisées avec le calendrier CalDAV que vous avez ajouté.

Remarque :   seuls les calendriers CalDAV avec SSL activé et utilisant le port 443 peuvent être activés pour synchroniser les événements du calendrier avec votre compte.

  1. Dans le panneau d'aperçu du Calendrier, cliquez sur l'icône d'engrenage et sélectionnez Ajouter un calendrier externe.

  2. Sélectionnez le type de calendrier à ajouter et cliquez sur Suivant.

  3. Sélectionnez le type de calendrier à ajouter, soit un Compte CalDAV, soit un Abonnement iCAL.

  4. Entrez l'adresse mail et le mot de passe que vous utiliserez pour ce compte.

  5. Cliquez sur Suivant.

  6. Dans la zone de texte Nom, saisissez le nom de ce calendrier et sélectionnez une couleur.

  7. Si vous voulez exclure ce calendrier du rapport de vos disponibilités (libre/occupé), sélectionnez Exclure ce calendrier pour le rapport des disponibilités.

  8. Cliquez sur OK.

Ce calendrier est ajouté et les réunions du calendrier externe sont synchronisées avec le calendrier que vous avez ajouté.
