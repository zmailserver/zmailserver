Affichage des rendez-vous d'une journée dans votre vue semaine ouvrée

Les événements d'une journée s'affichent dans la première ligne du calendrier sans indiquer l'horaire. Vous pouvez cacher les événements d'une journée dans votre calendrier.

•       Dans la vue semaine ouvrée, cliquez sur le trait gras de la première ligne. La première ligne est cachée.

Vous pouvez activer ou désactiver l'affichage des rendez-vous d'une journée.

•       Cliquez sur le trait gras sous la ligne d'en-tête pour afficher les rendez-vous.
