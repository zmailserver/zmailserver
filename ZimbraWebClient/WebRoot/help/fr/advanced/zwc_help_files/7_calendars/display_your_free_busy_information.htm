Affichage de vos informations de disponibilité (libre/occupé)

La fonction libre/occupé vous permet de consulter les temps de réunion programmés sur vos calendriers. La fonction libre/occupé est activée par défaut, mais vous pouvez la désactiver ou limiter les personnes ayant accès aux informations.

  1. Ouvrez la page Préférences>Calendrier.

  2. Faites défiler la page jusqu'à la section Autorisations, sélectionnez l'option Libre/Occupé que vous souhaitez définir pour ce calendrier.

    3. Si vous sélectionnez Autoriser uniquement les utilisateurs internes ci-dessous à consulter mes infos Libre/Occupé, saisissez l'adresse mail des utilisateurs concernés dans le champ.

  3. (Facultatif) Dans la section Invitations, indiquez les personnes pouvant vous inviter à une réunion.

    4. Si vous sélectionnez Autoriser uniquement les utilisateurs internes ci-dessous à consulter mes infos Libre/Occupé, saisissez les noms ou les adresses mail complètes des utilisateurs concernés dans le champ.

  4. Si vous imposez des restrictions aux personnes autorisées à vous inviter à des réunions, vous pouvez sélectionner l'option Envoyer une réponse automatique aux utilisateurs qui ne sont pas autorisés à m'inviter.

  5. Cliquez sur Enregistrer.

Remarque :   ces paramètres n'affectent pas les calendriers que vous avez configurés pour qu'ils n'affichent pas vos disponibilités.
