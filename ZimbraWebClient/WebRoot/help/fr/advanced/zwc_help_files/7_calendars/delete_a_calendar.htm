Suppression d'un calendrier

Vous pouvez supprimer n'importe quel calendrier de votre liste de calendriers, excepté votre calendrier par défaut. Lorsque vous supprimez un calendrier, les réunions créées à partir de ce calendrier ne sont pas supprimées des calendriers des participants invités.

Avant de supprimer un calendrier, vous devez annuler toutes les invitations que vous aviez envoyées.

  1. Dans le panneau d'aperçu des calendriers, cliquez avec le bouton droit de la souris sur le calendrier à supprimer.

  2. Cliquez sur Supprimer. Le calendrier est placé dans le dossier Corbeille.
