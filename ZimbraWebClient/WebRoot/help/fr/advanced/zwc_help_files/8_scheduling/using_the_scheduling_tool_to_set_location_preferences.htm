Utilisation de l'outil de planification pour afficher la disponibilité de l'emplacement

L'outil de planification du calendrier peut vous aider à trouver des créneaux et des lieux de réunion prochainement disponibles dans votre société. Vous pouvez configurer des préférences pour vos heures de réunions, vos bâtiments, votre lieu et votre taille de salle favoris.

Pendant que vous entrez les noms des participants, l'outil de planification suggère des heures et des lieux dans le panneau d'aperçu. Vous pouvez également chercher un endroit pour votre réunion avant d'ajouter des participants. Si vous configurez vos préférences de lieux, seuls ces lieux seront affichés.

Utilisation de l'outil de planification pour définir les préférences de lieux

Si vous voulez toujours réserver des salles qui sont dans un lieu spécifique, vous pouvez utiliser l'outil Lieux suggérés de l'application Calendrier pour suggérer la disponibilité pour ces salles lorsque vous créez un rendez-vous.

  1. Dans la page Nouveau rendez-vous, cliquez sur Suggérer un emplacement.

  2. Dans le panneau qui s'ouvre, cliquez sur l'icône d'engrenage.

  3. Dans la boîte de dialogue Préférences de suggestion, saisissez les informations sur l'emplacement dans la section Préférences d'emplacement. Vous pouvez entrer le nom de l'emplacement uniquement, ou vous pouvez être plus précis.

  4. Cliquez sur OK.

La colonne de droite affiche les emplacements qui répondent à vos exigences. Les emplacements des salles sont affichés s'ils ont été configurés dans le répertoire de la société.

Une fois la configuration effectuée pour un rendez-vous, ces informations sont enregistrées et disponibles lorsque vous créez d'autres rendez-vous.
