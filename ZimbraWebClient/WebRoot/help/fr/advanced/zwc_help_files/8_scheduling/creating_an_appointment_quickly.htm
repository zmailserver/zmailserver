Création rapide de rendez-vous

L'option QuickAdd/Ajout rapide facilite la création d'un rendez-vous.

Remarque :   la fonction QuickAdd/Ajout rapide est une option. Si vous préférez ouvrir la page de rendez-vous en double-cliquant ou en la sélectionnant dans le calendrier, désactivez cette fonction dans vos préférences de calendrier.

  1. Cliquez avec le bouton droit de la souris sur le mini-calendrier ou sur l’heure de début dans l’un des affichages de votre calendrier. Vous pouvez choisir de créer un Nouveau rendez-vous ou un Nouveau rendez-vous journée entière.

  2. Dans la zone de texte Sujet, ajoutez une brève description de la réunion. Le texte ajouté ici devient la description dans le calendrier.

  3. Dans la zone de texte Lieu, entrez le lieu où se déroule la réunion. Lorsque vous commencez à saisir l'emplacement alors qu'il figure dans le répertoire de la société, les lieux correspondants s'affichent automatiquement. Vous pouvez également cliquer sur Suggérer un emplacement pour voir une liste d'endroits disponibles dans votre société

  4. Dans le menu déroulant Afficher, sélectionnez l'option Libre/Occupé pour ce rendez-vous.

  5. Si ce rendez-vous doit être ajouté à un autre calendrier que le calendrier par défaut, sélectionnez le calendrier.

  6. Dans le menu déroulant Marquer comme, choisissez si le rendez-vous est privé ou public. Les détails d'événements marqués comme étant privés ne s'affichent pas dans le calendrier partagé d'un bénéficiaire. Les seules informations indiquées dans le calendrier partagé sont l’heure de début et de fin, la durée, les informations de récurrence et l’identité de l’organisateur.

  7. Définissez le jour et l'heure de début et de fin. Si le rendez-vous est récurrent, sélectionnez la fréquence à laquelle ce rendez-vous doit être programmé dans Répétition.

  8. (Facultatif) Si vous souhaitez ajouter d'autres participants, programmer une ressource ou ajouter des notes au rendez-vous, cliquez sur Plus de détails.

  9. Cliquez sur OK.
