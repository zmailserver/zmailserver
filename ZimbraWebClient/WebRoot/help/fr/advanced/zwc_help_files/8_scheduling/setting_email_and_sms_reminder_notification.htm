Configuration des notification de rappel des e-mails et des SMS

Vous pouvez configurer d'avoir des rappels de rendez-vous et tâches qui vous sont envoyés par e-mail et comme un message de texte sur votre dispositif mobile. Le rappel est envoyé en fonction de l'heure de rappel qui est configurée dans la tâche ou rendez-vous.

Remarque: Vous pouvez configurer la notification lorsque vous travaillez dans une tâche ou page de calendrier et cliquer sur Configurer. Une notification que vous configurez ou modifiez à partir d'une tâche ou page de calendrier, change l'adresse ou le numéro de téléphone dans la page Préférences>notifications. Une notification vous est alors envoyée au sujet de la tâche ou calendrier à venir. Vous ne pouvez pas configurer différentes adresses de notification ou numéro de SMS pour les différentes tâches et les éléments de calendrier.

Configurer la notification par email

  1. Aller à la page Préférences>notifications.

  2. Entrez l'adresse e-mail à laquelle doit être envoyée le rappel.

  3. Cliquez sur Enregistrer.

Configurer la notification par SMS

  1. Aller à la page Préférences>notifications.

  2. Sélectionner votre pays à partir du menu déroulant Region.

  3. Sélectionner votre opérateur de téléphonie à partir du menu déroulant Opérateur de télécommunications.

  4. Entrez votre numéro de téléphone dans le champ Numéro de téléphone.

  5. Cliquer sur Envoyer le code de vérification. Un message de texte est envoyé à votre téléphone.

  6. Entrez le code que vous avez reçu dans le message dans le champ Code de vérification.

  7. Cliquer sur Valider le code. Le champ État indique le nombre confirmé.

  8. Cliquez sur Enregistrer.
