Rappeler un message vocal

Remarque :  cette fonctionnalité est pour ZCS Network Edition uniquement et est disponible uniquement dans le client Web avancé.

Répondre à un message vocal en rappelant l'expéditeur à l'aide de la fonction cliquer pour appeler.

  1. Sélectionner l'onglet Voix.

  2. Dans la liste historique des appels, sélectionner le message vocal auquel vous souhaitez répondre.

  3. Cliquer sur Rappeler. Le nombre qui est associé à l'appelant est composé.
