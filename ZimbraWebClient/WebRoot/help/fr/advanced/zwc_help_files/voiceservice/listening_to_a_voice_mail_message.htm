Écoute d'un message vocal

Remarque :  cette fonctionnalité est pour ZCS Network Edition uniquement et est disponible uniquement dans le client Web avancé.

Lorsque vous recevez un message vocal, vous recevez un message dans votre corbeille d'arrivée avec un message joint dans un format vocal, tel que wav.

Les détails du message sont inclus dans le message, comme le numéro de téléphone, l'expéditeur du message, la durée de l'appel, la date et l'heure à laquelle l'appel a été reçu. Vous pouvez reproduire l'enregistrement du message vocal en cliquant sur la pièce jointe de messagerie vocale, ou bien vous pouvez aller sur l'onglet Voix et cliquer sur la flèche de lecture pour lire le message.

Étant donné que ce messagerie vocale est reçue comme les autres message e-mail, vous pouvez répondre ou transmettre le message, ou effectuer d'autres actions qui sont typiques d'un message e-mail telles que le marquage du message Vous pouvez télécharger le message vers un dossier, placer le message dans un porte-documents, ou supprimer le message.

Écouter un nouveau message vocal à partir de la page messagerie vocale

1. Sélectionner l'onglet Voix.

2.Dans l'historique des appels, sélectionner le message vocal que vous voulez écouter.

Écouter un message vocal à partir d'une boîte de réception de message

1.Sélectionner un message e-mail dans votre boîte de réception qui a une pièce jointe au message de messagerie vocale.

2.Cliquer sur la pièce jointe Message vocal pour écouter le message.

Marquage d'un message comme écouté ou non écouté

1.Sélectionner l'onglet Voix.

2.Dans la liste historique des appels, cliquer droit sur un message vocal.

3.Cliquer sur Marqué comme non écouté o Marqué comme écouté, selon la façon dont vous voulez afficher le message dans votre boîte de réception.
