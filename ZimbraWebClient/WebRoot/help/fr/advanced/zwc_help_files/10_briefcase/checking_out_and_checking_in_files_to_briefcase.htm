Extraction et enregistrement de fichiers dans le Porte-documents

Vous pouvez ouvrir des fichiers dans votre porte-documents chaque fois que vous vous connectez à votre compte. Vous devez disposer du logiciel approprié pour ouvrir et lire ou mettre à jour des fichiers que vous n'avez pas créés avec les outils de documents dans le porte-documents.

Si vous avez partagé un porte-documents avec d'autres personnes, vous pouvez utiliser la fonction Enregistrer ou extraire du porte-documents pour gérer la modification des fichiers. L'extraction et l'enregistrement en cours sont légèrement différents si vous avez uploadé le fichier dans votre porte-documents ou si vous l'avez créé dans le porte-documents

  1. Ouvrez l'onglet Porte-documents et sélectionnez le fichier que vous voulez extraire.

  2. Cliquez sur Actions>Extraire un fichier.

  3. Dans la boîte de dialogue qui s'affiche, choisissez d'enregistrer le fichier dans votre ordinateur. Vous avez extrait le fichier et par conséquent, il a été marqué en lecture seule et les modifications apportées ne peuvent pas être enregistrées. Vous devez enregistrer le fichier sur votre ordinateur pour le modifier.

  4. Modifiez le fichier avec l'outil approprié et enregistrez-le.

  5. Une fois que vous avez enregistré le fichier sur votre ordinateur, cliquez avec le bouton droit de la souris sur le fichier verrouillé dans la liste de fichiers du porte-documents, puis sélectionnez Enregistrer le fichier.

  6. Dans la boîte de dialogue Enregistrer le fichier dans le Porte-documents, recherchez le fichier modifié sur votre ordinateur. Si vous voulez ajouter des informations concernant les modifications, saisissez-les dans la zone de texte Remarques.

  7. Cliquez sur Enregistrer. Le fichier est uploadé. Le verrouillage est annulé et les détails du fichier, notamment le nouveau numéro de version et les remarques, sont mis à jours dans la liste de fichiers.
