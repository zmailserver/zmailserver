Envoi de fichiers à partir de votre Porte-documents

Lorsque vous travaillez dans des dossiers du Porte-documents, vous pouvez envoyer un lien vers les fichiers de votre Porte-documents, et vous pouvez envoyer le fichier spécifié sous forme de pièce jointe de mails. Les destinataires doivent disposer du logiciel adéquat pour ouvrir les fichiers.

Envoyer un mail contenant un lien vers un fichier

Vous pouvez envoyer un mail qui contient un lien vers un fichier du porte-documents. Le destinataire doit disposer de l'autorisation de partage appropriée pour travailler avec le fichier, et le logiciel approprié pour ouvrir le fichier. Toute modification apportée à un fichier par le destinataire est répercutée sur le fichier lié au dossier Porte-documents.

  1. Ouvrez l'onglet Porte-documents et sélectionnez le fichier à partager.

  2. Cliquer sur Actionset sélectionner Envoyer lien(s). Un panneau de confirmation apparaît pour vous rappeler que le destinataire doit avoir l'autorisation d'afficher le contenu du dossier. Cliquez sur Oui pour exécuter l'opération.

  3. Une nouvelle fenêtre de rédaction de mail s'ouvre avec le lien affiché dans le corps du mail. Rédigez le mail et cliquez sur Envoyer pour envoyer le mail.

Envoyer un fichier comme pièce jointe de mail

Vous pouvez envoyer un fichier comme pièce jointe de mail directement depuis Porte-documents. Les destinataires doivent disposer du logiciel adéquat pour pouvoir ouvrir le fichier.

  1. Ouvrez l'onglet Porte-documents et sélectionnez le fichier à joindre.

  2. Cliquer sur Actionset sélectionner Envoyer comme pièce(s) jointe(s). La fenêtre de création de mail apparaît avec la pièce-jointe.

  3. Ajoutez l'adresse mail du destinataire et rédigez le mail.

  4. Cliquez sur Envoyer.
