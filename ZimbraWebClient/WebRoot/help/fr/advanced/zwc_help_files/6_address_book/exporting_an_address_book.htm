Exportation d'un carnet d'adresses

Vous pouvez exporter vos carnets d'adresses à un fichier séparé par des virgules (.csv).

  1. Aller à la page Préférence>Importer/Exporter.

  2. Dans la section Type>d'exportation, sélectionner ce que vous voulez exporter.

    Option

    Description

    Comptes

    Toutes les données de compte peuvent être exportées vers un format "Tar-GZipped" (.tgz), qui peut être importé dans le système.

    Vous pouvez sélectionner les paramètres avancés.

    Calendrier

    Vous pouvez exporter vos rendez-vous au format iCalendar (.ics) standard, puis les importer dans un programme de gestion de calendrier compatible.

    Contacts

    Vous pouvez exporter vos contacts dans le format standard " de valeurs séparées par des virgules" (.csv), puis les importer dans un autre programme de gestion de contacts.

  3. Sélectionnez le dossier Source des contacts à exporter.

  4. Cliquez sur OK.

  5. Cliquez sur Exporter.
