Vérification orthographique d'un mail

Vous pouvez utiliser le vérificateur orthographique à tout moment pour vérifier votre orthographe.

  1. Dans la barre d'outils, cliquez sur le lien Abc. Les mots inconnus dans le mail sont surlignés par le vérificateur orthographique.

  2. Cliquez avec le bouton droit de la souris sur un mot mis en évidence. Un menu déroulant affiche les suggestions de correction.

  3. Choisissez le mot correct. Le mot est mis en évidence dans une autre couleur.

  4. Pour accepter vos modifications et fermer le vérificateur orthographique, cliquez sur Reprendre la modification.

Avant de fermer le vérificateur orthographique, vous pouvez modifier un mot corrigé et récupérer son orthographe d'origine. Cliquez sur le mot surligné, puis sélectionnez l’orthographe initiale en haut du menu.

Vérifier l'orthographe de tous les mails avant leur envoi

  1. Ouvrez la page Préférences>Mail>Rédiger.

  2. Sélectionnez l'option Vérification orthographique obligatoire avant tout envoi de mail.

  3. Cliquez sur Enregistrer.
