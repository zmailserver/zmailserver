Travailler avec des filtres de mails

Vous pouvez définir des filtres pour gérer les mails entrants et sortants, y compris les mails relatifs au calendrier et la configuration du filtre de flux d'activité pour gérer les messages moins importants. Un filtre est assorti d’une ou plusieurs conditions et d’une ou plusieurs actions. Une condition peut se baser sur l'objet, une personne spécifique ou les mails d'une date particulière. Si le mail satisfait les conditions, les actions indiquées sont exécutées.

Vous pouvez créer des filtres pour ce qui suit :

•      Trier les messages entrants dans les dossiers. Par exemple, tous les mails de votre responsable sont automatiquement déplacés dans votre dossier Directives d'administration à la réception.

•      Marquer les mails. Par exemple, les mails dans la liste de distribution d'événements d'entreprise sont marqués avec le tag Événements.

•      Faire suivre ou supprimer un mail. Par exemple, tous les mails reçus d'une adresse mail particulière sont déplacés dans le dossier Corbeille.

•      Trier les messages envoyés dans les dossiers relatifs à un sujet.

•      Appliquer un drapeau aux mails importants.

Les filtres sont appliqués dans l'ordre dans lequel ils sont listés. Vous pouvez modifier cet ordre à tout moment.

Remarque :   un mail est testé par chaque filtre actif sauf si vous spécifiez qu'un message est traité uniquement par le premier filtre qui correspond. Généralement, une fois qu'il y a une correspondance, d'autres filtres ne sont pas appliqués car le mail peut subir plusieurs actions éventuellement contradictoires.

Configuration de règles de filtre

Des règles de filtre sont assorties d’une ou plusieurs conditions et d’une ou plusieurs actions. Si le mail satisfait les conditions, les actions indiquées sont exécutées.

Vous pouvez également définir des règles de flux d'activité. Les règles de flux d'activité sont définies pour déplacer les messages les moins importants hors de votre boîte de réception, dans un dossier Flux d'activité que vous pouvez vérifier comme vous le souhaitez.
