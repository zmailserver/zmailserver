Création d'un filtre pour les invitations de calendrier

Vous pouvez créer un filtre pour organiser les invitations du calendrier que vous recevez. Vous pouvez filtrer les invitations que vous recevez et celles que vous envoyez. Vous pouvez définir le filtre sur lequel agir

  1. Aller à la page Préférences>filtres.

  2. Cliquez sur Filtres message entrant ou Filtres message sortant.

  3. Sélectionnez Nouveau filtre.

  4. Dans la zone de texte Nom du filtre, saisissez un nom unique pour le filtre.

  5. Pour activer un filtre, cochez la case Actif dans le coin supérieur droit.

  6. Dans le menu déroulant Sujet, sélectionnez Calendrier.

  7. Sélectionnez la condition l'invitation a obtenu une réponse/l'invitation n'a obtenu aucune réponse ou l'invitation est demandée/n'est pas demandée dans le menu déroulant.

  8. (Facultatif) Pour ajouter des conditions supplémentaires, cliquez sur le bouton + et répétez les étapes ci-dessus. L'ordre des conditions n'est pas important.

  9. Dans Exécuter les actions suivantes, choisissez une action dans le menu déroulant, telle que le déplacement d'une invitation vers un dossier spécifique.

    10. Selon l'action que vous sélectionnez, vous devrez peut-être accéder à un nom de dossier ou de tag.

  10. (Facultatif) Pour ajouter des actions supplémentaires, cliquez sur le bouton + et répétez l'étape ci-dessus. Les actions sont appliquées dans l'ordre dans lequel elles sont listées.

  11. La règle par défaut est d'arrêter la recherche de filtres correspondants après la première correspondance. Si vous voulez appliquer plusieurs filtres à un message, désélectionnez Ne pas appliquer d'autres filtres.

  12. Cliquez sur OK.
