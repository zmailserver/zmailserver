Accepter une invitation à partager des éléments

Si vous recevez une notification par mail indiquant que vous êtes autorisé à partager le dossier de messagerie d'une autre personne, vous pouvez accepter ou refuser le partage.

Si vous acceptez, le dossier partagé s'affiche dans la liste d'applications spécifiée dans le panneau d'aperçu. Cette information n'est en fait pas enregistrée dans votre boîte aux lettres.

  1. Pour accepter l'invitation, cliquez sur Accepter le partage.

  2. Dans la boîte de dialogue Accepter le partage, vous pouvez renommer le dossier et sélectionner une couleur pour le surligner.

  3. Choisir s'il faut envoyer une confirmation à la personne qui a envoyé l'information.

  4. Cliquez sur Oui. Le dossier est ajouté à votre panneau d'aperçu et le mail est placé dans votre dossier Corbeille.
