Supprimer un tag

  1. Cliquez avec le bouton droit de la souris sur un élément qui est marqué.

  2. Dans le menu déroulant, sélectionnez Tag . Par exemple, le sujet peut être une « Conversation », cela s'affiche : Libeller la conversation.

  3. Sélectionnez Supprimer le tag. Si l'élément est marqué de plusieurs tags, sélectionnez le tag que vous voulez supprimer ou sélectionnez Tous les tags pour supprimer tous les tags.

Vous pouvez supprimer un tag d'un mail lorsqu'il est ouvert dans le panneau de lecture. Le tag s'affiche dans l'en-tête du message sous Tags et vous pouvez cliquez sur le X sur le tag pour supprimer le tag.
