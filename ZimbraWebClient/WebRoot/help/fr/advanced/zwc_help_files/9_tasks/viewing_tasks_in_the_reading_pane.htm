Affichage des tâches dans le panneau de lecture

Vous pouvez activer un panneau de lecture pour afficher le contenu d'une tâche sans ouvrir la tâche. Le panneau de lecture peut s'afficher au bas ou à droite du panneau de contenu, ou vous pouvez le désactiver pour qu'il n'y en ait pas.

  1. Ouvrez la page Tâches.

  2. Pour sélectionner comment afficher le panneau de lecture, cliquez sur Affichage dans la barre d'outils Tâches.

 

Option

Description

Panneau de lecture en bas

La liste de tâches s'affiche au bas du panneau.

Panneau de lecture à droite

L'affichage de la liste de tâches ne montre pas l'état d'une tâche ; le pourcentage terminé s'affiche sous forme de barre de progression.

Panneau de lecture désactivé

Le panneau de lecture ne s'affiche pas
