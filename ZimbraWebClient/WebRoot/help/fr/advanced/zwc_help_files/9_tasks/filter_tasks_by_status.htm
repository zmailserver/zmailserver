Filtrer les tâches par état

Vous pouvez filtrer les tâches dans votre liste de tâches afin de les voir rapidement selon leur état.

•       Dans le menu déroulant Filtrer par de la barre d'outils Tâche, sélectionnez le type de tâches que vous voulez afficher.

Option

Description

Non lancé(e)

Affiche les tâches qui ne sont pas commencées.

Fini

Affiche les tâches qui sont terminées.

En cours

Affiche les tâches qui sont en cours.

En attente d'une autre personne

Affiche les tâches qui sont en attente d'une autre personne.

Reporté

Affiche les tâches qui sont reportées.

Tout

Affiche toutes les tâches, quel que soit leur état.

Liste de tâches

Affiche les tâches ayant l'état En cours, Non lancé(e) ou, si la tâche est arrivée à échéance, En attente d'une autre personne.
