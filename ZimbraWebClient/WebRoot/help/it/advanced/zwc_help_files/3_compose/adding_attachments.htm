Aggiunta di allegati 

Puoi allegare qualsiasi file del tuo sistema individuabile tramite Sfoglia.

Puoi aprire qualsiasi file allegato direttamente dalla tua casella di posta, a condizione che sia disponibile l'applicazione opportuna e che l'estensione non sia bloccata dall'amministratore.

Se il tipo di file è supportato dal software installato nel computer, in genere è sufficiente cliccare due volte sul file e il computer avvierà automaticamente l'applicazione in grado di leggerlo.

I virus informatici spesso si diffondono tramite gli allegati. Pertanto, gli amministratori di sistema possono bloccare i messaggi in entrata contenenti determinati tipi di allegati, spesso contenenti l'estensione .EXE o .ZIP nel nome file. Se invii un messaggio email a una persona che usa un sistema configurato per bloccare determinati tipi di allegati, potresti non ricevere alcuna notifica nel caso in cui il messaggio email venga bloccato.

Aggiungere un allegato a un messaggio

Puoi allegare documenti, fogli di calcolo, immagini, presentazioni e altri tipi di file a un messaggio email.

  1. Componi il messaggio email.

  2. Sotto il campo Oggetto, clicca su Allega e seleziona Il mio computer.

  3. Seleziona i file e clicca su Apri. I nomi dei file vengono visualizzati sotto il boxino Oggetto.

  4. Clicca su Invia per inviare il messaggio e gli allegati.

Nota:   i destinatari devono disporre del software appropriato per aprire il file. I formati di file comuni, come i file di testo o HTML e i file di immagini, ad esempio GIF o JPG, possono essere aperti con vari programmi.

Aggiungere allegati mediante trascinamento

Puoi aggiungere facilmente un allegato a un messaggio email trascinando il file da una cartella nel messaggio email.

  1. Seleziona uno o più file e tieni premuto il cursore sui file da allegare.

  2. Trascina i file nell'area di intestazione del messaggio e rilascia il cursore. I nomi dei file verranno visualizzati nell'intestazione.

Rimozione di un allegato

Per rimuovere un allegato, clicca sulla x nel bubble dell'allegato.
