Utilizzo dei client avanzati e standard

Sono disponibili due versioni di @ClientNameLong@.

Il client web avanzato offre la serie completa di funzioni di collaborazione web. È compatibile con i browser più recenti e le connessioni Internet più veloci.

Se la tua connessione Internet è lenta o preferisci la visualizzazione dei messaggi in formato HTML, puoi utilizzare la versione client web standard, che offre molte delle funzioni della versione client web avanzato ma alla quale si accede in un altro modo.

Strumenti browser

•      Il pulsante Indietro del browser ti porta alla pagina precedente che stavi visualizzando. Puoi usare anche il pulsante Avanti.

•      Non usare il pulsante Ricarica (Aggiorna) del browser. Questa azione infatti comporta il riavvio della sessione.

Alternare le versioni client

Puoi impostare la versione predefinita del client per accedere. Se modifichi con una certa frequenza il client utilizzato, puoi cambiarlo.

  1. Passa alla pagina Preferenze>Generali.

  2. Nella sezione Opzioni di accesso>Accedi utilizzando, seleziona la modalità di accesso.

    3. •    Avanzato. Offre la serie completa di funzioni di collaborazione web ed è particolarmente adatta se si usano browser aggiornati e connessioni Internet veloci.

    •    Standard. È l'opzione consigliata quando le connessioni Internet sono lente, si usano browser non aggiornati o se si desidera un accesso più semplice.

  3. Clicca su Salva.

All'accesso successivo, l'opzione selezionata corrisponderà all'impostazione predefinita.
