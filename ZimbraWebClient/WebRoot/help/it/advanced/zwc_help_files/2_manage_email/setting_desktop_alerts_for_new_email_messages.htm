Impostazione degli avvisi sul desktop per l'arrivo di nuove email

Puoi impostare avvisi di notifica sul desktop in occasione della ricezione di nuovi messaggi email. Per poter ricevere gli avvisi, devi aver aperto la casella di posta.

  1. Passa alla pagina Preferenze>Mail.

  2. Nella sezione Ricezione messaggi>Arrivo di messaggi, seleziona la modalità di notifica in occasione della ricezione di un nuovo messaggio. Seleziona una delle seguenti opzioni.

Opzione

Descrizione

Riproduci un suono

La notifica avviene tramite emissione di un segnale acustico.  Se il volume è azzerato non sentirai alcun suono. Affinché questa opzione funzioni, deve essere installato QuickTime o Windows Media Center.

Evidenzia la scheda Mail

La scheda Mail viene evidenziata se al momento della ricezione non stai lavorando nella scheda Mail.

Fai lampeggiare il titolo del browser

Il browser lampeggia finché non apri la scheda Mail.

Mostra una notifica pop–up

Vengono visualizzate informazioni sul messaggio. Deve essere installato Yahoo!BrowserPlus. Per ottenerne una copia, visita il sito web http://browserplus.yahoo.com/
