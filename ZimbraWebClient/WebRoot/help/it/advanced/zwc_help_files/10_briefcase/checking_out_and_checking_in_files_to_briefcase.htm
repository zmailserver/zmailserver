Checkout e checkin dei file nella valigetta

Puoi aprire i file nella tua Valigetta ogni volta che accedi al tuo account. Devi possedere l'applicazione software appropriata per aprire e leggere o aggiornare i file che non hai creato con gli strumenti per i documenti nella Valigetta.

Se hai una valigetta condivisa con altri, potresti voler usare la funzione di checkout e checkin della Valigetta per gestire la modifica dei file. Il processo di checkout e checkin è leggermente diverso a seconda che il file sia quello caricato nella tua valigetta o un documento creato nella valigetta

  1. Passa alla scheda Valigetta e seleziona il file di cui desideri eseguire il checkout.

  2. Clicca su Azioni>Checkout del file.

  3. Nella finestra di dialogo che compare, scegli di salvare il file sul tuo computer. Poiché hai eseguito il checkout del file, devi salvarlo sul tuo computer per modificarlo, perché il file è stato contrassegnato come file di sola lettura e le modifiche al file non possono essere salvate.

  4. Modifica il file con lo strumento appropriato e salvalo.

  5. Dopo aver salvato il file sul tuo computer, clicca con il tasto destro del mouse sul file bloccato dall'elenco di file della valigetta e seleziona Checkin del file.

  6. Nella finestra di dialogo Checkin del file nella Valigetta che si apre, cerca il file modificato sul tuo computer. Per aggiungere delle informazioni sulle modifiche, inseriscile nell'area di testo Note.

  7. Clicca su Checkin. Il file viene caricato. Il blocco viene rimosso e i dettagli del file, tra cui un nuovo numero di versione e le note, sono aggiornati nell'elenco dei file.
