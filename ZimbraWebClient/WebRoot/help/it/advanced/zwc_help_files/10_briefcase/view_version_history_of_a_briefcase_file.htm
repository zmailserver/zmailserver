Visualizzazione della cronologia versione di un file Valigetta

Il controllo delle versioni consente di gestire le modifiche apportate ai documenti nella Valigetta. Ogni volta che si apporta una modifica a un file e lo si salva, il numero di versione viene aggiornato. Il numero di versione più recente è indicato nell'elenco dei documenti dei file.

•       Clicca sulla freccia nel riquadro di anteprima per visualizzare la cronologia delle versioni e gli autori. Se il riquadro di anteprima è disattivato, visualizzerai una colonna con il numero di versione.

Ripristino di un documento alla versione precedente

•       Clicca con il tasto destro del mouse su un'altra versione e seleziona Ripristina come versione corrente.
