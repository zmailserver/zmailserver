Differenza tra tipi di ricerche And e Or

 

Una ricerca di messaggi contenenti sia X che Y viene definita ricerca 'AND' . Per poter essere considerato una corrispondenza, infatti, il messaggio deve soddisfare tutti i criteri specificati. La ricerca di messaggi che soddisfano uno qualsiasi dei criteri specificati viene chiamata ricerca OR , in quanto il messaggio viene considerato una corrispondenza se contiene o X o Y. V.

  • Il riquadro di ricerca di base consente di creare una query di ricerca con operatore AND. Se si aprono due riquadri di ricerca di base, è possibile creare una query di ricerca con operatore OR tra i due.

  •  Selezionare più boxini nei riquadri Tag o Status vi consente ci creare una query AND. Se si aprono due riquadri di Tag o Status, è possibile creare una query di ricerca con operatore OR tra i due.

  • Nei riquadri di ricerca Cartelle, Zimlet, e Ora, una selezione multipla all'interno di un singolo riquadro crea una ricerca OR per quegli elementi.   

Nota:   Solo"OR" appare in una query. Se hai scelto di visualizzare la query nella barra di ricerca, il boxino di ricerca si aggiorna per mostrare la query risultante. Nelle ricerche di tipo 'AND', la parola 'AND' non appare.
