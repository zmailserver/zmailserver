Creazione di un nuovo documento

Puoi creare un documento nella Valigetta, progettare la disposizione del contenuto con strumenti di formattazione di tipo Rich Text, controllare l'ortografia e usare parole chiave per creare con facilità link ad altre pagine.

  1. Clicca sulla cartella della Valigetta da utilizzare. Nel riquadro Elenco documenti verranno visualizzati i file esistenti

  2. Nell'elenco Nuovo sulla barra degli strumenti, seleziona Documento. Si apre una pagina vuota di Zimbra Docs.

  3. Nella barra degli strumenti, immetti il nome del documento.

  4. Crea il contenuto della pagina. Puoi utilizzare l'editor Rich Text per aggiungere stili, colore e tabelle alla pagina.

  1. Clicca su Salva per salvare il file e procedere. Per chiudere il documento senza salvare clicca sulla x nella scheda del browser.

Quando salvi un documento, la relativa versione viene modificata.

Nota: Se il documento è stato creato nella Valigetta errata, è possibile spostare il file selezionandolo e trascinandolo nella cartella desiderata.
