在Zimbra Desktop中使用記事簿

你可在 Zimbra Desktop 中為你的每一個帳戶維持一個記事簿。

Zimbra 帳戶記事簿與 Zimbra Desktop 同步。當你加入一個新事件，修改一個事件或從 Zimbra Desktop 中刪除一個事件時，所做的改動會反向同步到你的 Zimbra 帳戶中。

你可在 Zimbra Desktop 中為你的其他每個帳戶維持一個記事簿，但記事簿事件不會反向同步到那些帳戶中的記事簿。

如果你還使用另一個記事簿程式，你可將記事簿儲存為 iCalendar (.ics)檔案，然後匯入你的記事簿中。如果你對 Zimbra Desktop 中的記事簿做了任何改動，則不會反向同步到記事簿。如果重新載入 .ical 記事簿，則你的 Zimbra Desktop 中的記事簿即被覆寫，所加入的任何事件均被丟棄。
