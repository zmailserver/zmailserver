加入事項

你只需在事項清單中輸入事項名稱即可建立一項新事項，也可以開啟事項頁面並加入有關該事項的詳細資訊，其中包括註釋和附件。 打開事項頁面時，「事項」索引標籤顯示於「應用程式」工具列，且「事項」視窗隨即出現在「內容」窗格中。你可打開多個「事項」索引標籤，且可在「應用程式」工具列上的不同索引標籤之間移動。

若需快速加入事項

  1. 從窗格中選擇事項清單

  2. 在「內容」窗格中，按下第一行，其上顯示的是按下此處加入新事項。    

  3. 屏幕上隨即顯示一個欄位。輸入事項的名稱。

  4. 按 Enter 鍵。此事項即被加入你的事項清單中。

備註：若誤將事項放入了另一清單，可開啟事項並在「事項清單」下拉選單中選擇正確的清單，或將「內容」窗格中的如事項拖曳至正確的清單中。

若需加入有詳細資料的事項

你可使用事項清單建立事項並追蹤其進度。另外，你可在事項和附件中撰寫註釋，以方便事項的處理。

透過輸入開始日期、結束日期和設定優先級，你可估算出專案的時間長度。啟動事項時，你可選擇其狀態，如未開始和進行中，並可選擇完成的百分比。

  1. 從「概況」窗格中選擇事項清單資料夾。

  2. 在「事項」工具列中，按下新建。 一個無標題事項頁面開啟，一個「事項」索引標籤顯示於工具列上。

  3. 在「詳細資料」區，

    • 在「主題」欄位輸入事項名稱。如相關，在「地點」欄位填寫地點。

    • 選擇優先級 - 正常、低或高

    • 如果在選擇所需事項清單之前按下了「新建」，則可從事項清單選單中選擇所需的下拉事項清單。

  4. 在「進展」區，選擇開始日期、結束日期。如已啟動事項，你可將未開始改為清單中的一個選項，並更新完成百分比。

  5. 設定提醒日期和時間，如已在偏好設定 > 提醒頁面設定電郵地址或短訊提醒，則剔選相應方格，電郵或短訊。如果你沒有設定提醒，按下設定前往「通知偏好設定」頁面作出設定。

  6. 你可在文字欄位中加入註釋，若需加入附件，可按下工具列上的加入附件。

  7. 按下儲存便可儲存該事項。該項新事項隨即被加入事項清單中。
