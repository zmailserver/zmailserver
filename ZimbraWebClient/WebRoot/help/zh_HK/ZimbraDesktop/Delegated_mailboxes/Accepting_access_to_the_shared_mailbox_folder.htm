接受他人的分享郵箱邀請

備註：僅 Zimbra 帳戶可使用。

如果你收到他人的郵件，通知你可使用對方的分享郵件夾，你可以接受，也可以謝絕。

如果接受，分享郵件夾將在窗格中你的「資料夾」清單中顯示。當在 Zimbra Desktop 中進行操作，以查看分享資料夾的內容時，須連接至互聯網。分享資料夾的內容未下載至你的電腦。

如果接受

  1. 按下郵件中的接受分享。「接受分享」對話方格隨即開啟，並告知授予你的角色。

  2. 接受前，你可以更改郵件夾名稱然後選擇一種顏色以凸顯資料夾。

  3. （選項）可選擇是否發回確認資訊。

  4. 按下是。該郵件夾即被加入你的資料夾清單中。通知郵件隨即移至垃圾桶。
