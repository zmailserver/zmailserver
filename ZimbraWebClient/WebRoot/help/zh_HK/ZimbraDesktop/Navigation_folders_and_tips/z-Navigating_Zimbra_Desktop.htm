關於常見的問題以及故障排除方法，請參見「Zimbra Desktop 常見問題」

瀏覽 Zimbra Desktop

如下圖所示，打開 Zimbra Desktop 後，你會看到一個全屏視窗。

Zimbra Desktop 主頁面由以下區域組成：

  1. 搜尋欄行。搜尋與進階搜尋顯示在此區域。帳戶設定和常見問題連結在右側。

  2. 應用程式索引標籤。可使用的應用程式列在頂端的各個索引標籤中。

  3. 工具列。顯示目前使用的應用程式的可用操作項。在此例中顯示的是郵件應用程式工具列。

  4. 內容窗格。此區域中的內容取決於所使用的應用程式。在「收件箱」視窗中，內容窗格顯示的是收件箱中所有的郵件和對話。在該頁面的螢幕中，閱讀窗格是「收件箱」中郵件下方的那塊區域。按一封郵件時，該郵件將顯示在郵件清單下邊的閱讀窗格中。 若需調整閱讀窗格的尺寸，可按下並上下拖曳分隔條。系統的預設值是在讀取窗格中查看郵件。 你可將閱讀窗格移至左側，或在工具列的視窗連結中關閉閱讀窗格。

  5. 概況窗格。顯示你的各個帳戶。各帳戶名稱將顯示。此範例顯示的是「所有郵箱」版面。 你的各個帳戶的收件箱、寄件箱和草稿箱內的所有郵件將在該部分合併顯示。此偏好設定可在偏好設定 > 一般頁面設定。

    每個帳戶都包括下列資料夾：

    • 資料夾，包括系統資料夾收件箱、寄件箱、草稿箱、雜件箱以及垃圾桶，還包括所有自己建立的自訂資料夾

    • 可能已儲存的搜尋項

    • 你所建立的標籤

    • 在郵箱內部建立的 Zimlet 連結，以便整合第三方應用程式。

  6. 在窗格下可顯示一個小日曆。小日曆是一選項，可在偏好設定 > 記事簿頁面啟用。

開啟 Zimbra Desktop

無論何時接達網路，如果已設定帳戶自動同步，則新的郵件都將自動下載到你的 Zimbra Desktop 中。此功能並不需要你打開 Desktop。 Windows Tray（或 Mac Dock）中有一信封圖示，它會顯示自上次 Zimbra Desktop 同步起你所有帳戶的新郵件總數。按此信封圖示打開此桌面應用程式。

 

郵件和預約提醒

可設定自動的郵件和預約提醒。許多不同的提醒選項都可選用。

  • 在 Windows 中，提醒將以系統欄的氣球信封圖示的方式顯示。對於 Mac，若要使用郵件和預約提醒，你必須先安裝 Growl。 你亦將需在偏好設定 > 郵件和 > 記事簿中啟用顯示彈出的通知。備註：此功能在 Linux 下不起作用。

 

  • 在偏好設定 > 郵件和 > 記事簿中，還可設定一些其他的提醒操作，包括播放聲音、凸顯郵件或記事簿索引標籤、或讓瀏覽器的標題閃爍。這種彈出的通知須每個帳戶單獨設定。可在一些帳戶中開啟此功能，而在另一些帳戶中關閉此功能。
