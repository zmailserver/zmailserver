利用郵件設立會議

你可快速建立會議請求，方法是：將郵件、對話拖曳至小日曆的一日期上。拖曳郵件或對話時，郵件中的資料將用於填充「預約」頁面上的許多欄位。

  1. 按一下指定的郵件或對話，然後將其拖曳至小日曆中的某一日期上。

  2. 加入參會者對話方格開啟該預約頁面，該頁面載入了許多從該郵件的訊息預置的欄位。

    3. •    預約的主題就是郵件的主題。

    •    參會者為該郵件或對話中最近郵件的收件人：和副本：欄位中所有郵件地址的相應人士。當將某聯絡人從聯絡人清單拖曳至小日曆上時，該聯絡人的第一個電郵地址將加入參會者欄位中。

    •    郵件內容變成邀請內容。附件不會附加到預約請求中。

  3. 在預約頁面，檢查會議請求的詳情並進行必要的修改。

  4. 設定會議的時間，如果是定期重複的會議，設定重複模式。

  5. 按傳送。
