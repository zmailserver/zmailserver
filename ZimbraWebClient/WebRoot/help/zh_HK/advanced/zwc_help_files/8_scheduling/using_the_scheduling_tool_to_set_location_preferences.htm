利用日程安排工具檢視位置偏好設定

記事簿日程安排工具可助你尋找下一可用的空閒時間和你公司所有的會議地點。你可為首選的會議時間、建築、地點和房間大小進行設定。

在你輸入姓名的同時，日程安排工具開始在概況窗格建議時間及地點。你也可選擇在加入參會者前為你的會議尋找地點。如果設定你的地點喜好，只有這些地點會顯示出來。

利用日程安排工具設定位置偏好設定

如果你經常要預留在某特定地點的房間，當你建立一個預約時，你可使用在記事簿應用程式中的建議地點工具建議那些房間是否可用。

  1. 在新預約頁面，按建議位置。

  2. 在開啟的窗格中，按一下齒輪圖示。

  3. 在建議偏好設定對話方格中的位置偏好設定部份，輸入有關該地點的資料。你可以只輸入該地點的名稱，或可以更具體。

  4. 按確定。

有邊的欄顯示符合你要求的地點。房間地點如已在公司目錄中設定，則會顯示出來。

一旦你為某預約進行了設定，此資料即被儲存，並在你進行預約時即可使用。
