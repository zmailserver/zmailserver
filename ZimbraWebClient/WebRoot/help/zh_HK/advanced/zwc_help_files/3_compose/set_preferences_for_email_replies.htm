設定電郵回覆的偏好設定

若要以與收到的郵件相同的格式回覆，選擇回覆/轉寄時使用原始郵件的格式。此功能確保收件人得到具有最方便閱讀格式的郵件。

  1. 前往偏好設定>郵件頁面，撰寫郵件部份。

  2. 若要以與收到的郵件相同的格式回覆，選擇回覆/轉寄時使用原始郵件的格式。

  3. 選擇是否在你的回覆郵件中包含原始郵件文字。如果是包含原始郵件：

    4. •   選擇是否用在之前的回應加符號標記，或在郵件標題上包括原始詳細資料（收件人、寄件人及主題欄位）。你也可將原始郵件作為附件包括。

    •    如果你要在之前的回應加上字首，選擇尖括弧 (>) 或直槓 (|)。

  4. 按下儲存。
