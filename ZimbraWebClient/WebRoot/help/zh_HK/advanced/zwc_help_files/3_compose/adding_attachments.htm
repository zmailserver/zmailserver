加入附件 

用瀏覽按鈕在檔案系統中找到的任何檔案，都可作為附件加入。

你可從你的郵箱中直接開啟任何附件，條件是你須有合適的應用程式，並且副檔名沒有被系統管理員封鎖。

如果檔案類型是你電腦上安裝的軟件所支援的類型，通常可以按兩下檔案，你的電腦就自動啟動相應的應用程式讀取該檔案。

電腦病毒通常透過檔案附件傳播。故此，系統管理員可能阻止有特定類型附件的郵件進入系統，通常是以 .EXE 或 .ZIP 副檔名作為部份的檔案名稱。如果將郵件發到這樣的系統，即被設定為阻止某些特定類型的附件之系統，則你有可能不會收到該郵件被阻止的通知。

加入附件

你可附加文件、試算表、圖片、幻燈片以及其他類型的檔案在郵件。

  1. 撰寫郵件。

  2. 在主題欄位下面，按加入附件並選擇我的電腦。

  3. 選取檔案並按下開啟。檔案名稱顯示在主題字框下面。

  4. 按下傳送，傳送郵件和附件。

備註：   收件人必須有適當的軟件才能開啟該檔案。文字檔、HTML 檔案等常用檔案格式和 GIF 或 JPEG 檔案等圖片檔，可用各種程式開啟。

用拖曳檔案的方式加入附件

你可透過從一個資料夾拖曳檔案到你的郵件，輕易地在郵件內加入附件。

  1. 選取一個或多個檔案，將滑鼠指標放在要附加的檔案上，並一直按住。

  2. 將檔案拖曳至郵件標題區，然後鬆開滑鼠指標。檔案名稱顯示在標題中。

移除附件

若要移除附件，按附件氣泡上的 x。
