自訂撰寫郵件選項

從你的偏好設定裏，你可以選擇建立簡單的純文字郵件或 HTML 格式郵件，你也可以設定套用在你要傳送郵件的署名和身份。

除撰寫郵件的選項外，其他你可設定的選項包括讀信回條請求、是否在新視窗撰寫郵件，以及要求在你的郵件傳送前對郵件進行拼字檢查。

  1. 前往偏好設定>郵件頁面，撰寫郵件部份。

  2. 選擇你希望如何傳送你的郵件：以 HTML 格式或以文字格式。

  3. 如果你選擇以 HTML 格式，在下拉選單中選擇字體樣式、大小及顏色。

  4. 在設定中，選擇你希望如何管理郵件撰寫的選項。

    5. 選項

    描述

    寫信時自動儲存郵件草稿

    當你正在撰寫你的郵件時，副本會定期儲存在你的草稿箱。

    永遠要求讀取回條

    所有你傳送的郵件要求收件人在打開你的郵件時返回讀信回條訊息。

    當你設定此為預設，你可從寫信工具列上的選項下拉選單中為個別郵件更改此設定。

    回覆/轉寄時使用原始郵件的格式

    即便你的預設文字編輯器不同於收到的郵件，剔選此方格後即可以與收到的郵件相同的格式回覆。這在確保收件人得到的郵件具有他們所習慣的格式方面很有用。

    總在新視窗中寫信

    另一寫信視窗開啟。在寫信視窗開啟的情況下，你可在你的郵箱內查看和瀏覽。

    傳送郵件前，強行進行拼字檢查

    傳送前所有郵件會自動進行拼字檢查。

    儲存副本到寄件備份箱

    此項預設為已啟用。你可以取消選擇此選項。

  5. （可選）按帳戶頁面前往帳戶偏好設定頁面設定你的身份。

  6. （可選）按管理你的署名…前往署名偏好設定頁面加入署名，在傳送郵件時可用。

  7. 按下儲存。
