使用 S/MIME 設定安全偏好設定

備註： 這是 ZCS Network Edition 的專門功能，並只可用於 Advanced Web Client。

如果你的帳戶已啟用 S/MIME 功能，在你的偏好設定裏可使用安全頁面。

安全偏好設定的預設值是用來自動記住你上次撰寫郵件時選擇的安全選項。 

你可以更改預設偏好設定至始終透過加密、已簽署及加密、或不簽署也不加密來傳送你的郵件。  你可以透過撰寫郵件頁面中的安全索引標籤更改該選項，但這項變更只限於該郵件。

  1. 按下偏好設定。

  2. 在概況窗格中，按下安全。

  3. 改變設定。

    • 自動（記住上次設定）為預設值。你上一次選擇的安全選項已被記住

    • 如果你完全或甚少使用 S/MIME 功能，選擇不要簽署或加密

    • 如果你經常使用電子署名傳送郵件，請選擇只限簽署

    • 如果你經常使用已加密電子署名傳送郵件，請選擇簽署及加密

  1. 按下儲存。
