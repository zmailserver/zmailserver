搜尋你可使用的過濾器

請參閱下表，了解如何建立更詳細的搜尋。

在「搜尋」索引標籤中，你可選擇進階篩選器建立更複雜的查詢。

進階篩選器

描述

郵件來源

搜尋在郵件標題（寄件人 / 收件人 / 副本）中包含的特定網域。你可搜尋的網域是預先定義的。

傳送至

搜尋在郵件標題 （寄件人 / 收件人 / 副本) 中包含的特定網域。你可搜尋的網域是預先定義的。

傳送日期

搜尋在某個選定日期之後、之前或當天接收到的郵件。若要搜尋兩個日期之間的郵件，請依次選擇傳送日期早於和傳送日期晚於，然後輸入日期範圍。  

附件

根據是否有附件搜尋郵件。如果你正搜尋特定附件類型，請從顯示的清單格式中選擇。

大小

按大小（以位元組、KB 或 MB）搜尋郵件。

狀態

按照郵件的狀態（如已讀、未讀、已轉寄或已回覆）搜尋郵件。

標記

搜尋特定標記的郵件或聯絡人。

資料夾

將搜尋範圍限制在選定的資料夾。
