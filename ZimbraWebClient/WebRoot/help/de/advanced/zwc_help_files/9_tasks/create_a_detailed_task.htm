Detaillierte Aufgabe erstellen

In Ihren Aufgabenlisten legen Sie Aufgaben an und verfolgen deren Fortschritt. Außerdem können Sie Notizen verfassen und Dateien anhängen, auf die Sie über die Aufgabe schnell zugreifen können. Es ist nützlich, einer Aufgabe alle relevanten Informationen beizufügen, wenn Sie Ihre Aufgabenlisten für andere Nutzer freigeben.

Sie können die Länge des Projekts schätzen, indem Sie das Anfangsdatum und das Fertigstellungsdatum eingeben und die Priorität festlegen. Wenn Sie die Aufgabe anlegen, können Sie den Status auswählen, z. B. Nicht begonnen oder In Bearbeitung. Außerdem können Sie den Prozentsatz der Fertigstellung auswählen.

  1. Öffnen Sie die Seite Aufgabe und wählen Sie Neue Aufgabe aus.

  2. Geben Sie im Abschnitt Details die Aufgabeninformationen ein. Nur Betreff ist ein Pflichtfeld, damit die Aufgabe einen Namen erhält.

    Option

    Beschreibung

    Betreff

    (Erforderlich) Geben Sie eine kurze Beschreibung der Aufgabe ein. Der hier eingegebene Text wird als Beschreibung in der Aufgabenliste angezeigt.

    Ort

    Geben Sie den Ort ein, an dem die Aufgabe ausgeführt wird.

    Priorität

    Geben Sie die Priorität der Aufgabe an.

    Aufgabenliste

    Wählen Sie die Aufgabenliste aus, in der die Aufgabe angezeigt werden soll.

  3. Geben Sie im Abschnitt Fortschritt die Informationen zum Verfolgen des Fortschritts der Aufgabe ein. Diese Angabe ist für Ihre Aufgabe nicht erforderlich.

    Option

    Beschreibung

    Status

    Geben Sie den Status der Aufgabe an. Sie können die Aufgabe jederzeit erneut öffnen und den Status ändern.

    Prozent

    Wählen Sie einen Prozentwert für den Status der Aufgabe aus. Sie können die Aufgabe jederzeit erneut öffnen und den Prozentwert ändern.

    Anfangsdatum

    Legen Sie das Anfangsdatum für die Aufgabe fest.

    Fälligkeitstag

    Legen Sie den Fälligkeitstag für die Aufgabe fest.

    Erinnerung

    Legen Sie ein Datum und eine Uhrzeit für die Erinnerung fest.

    Benachrichtigung

    Wenn Sie auf der Seite Einstellungen > Benachrichtigungen eine Mail-Adresse oder eine SMS-Benachrichtigung konfiguriert haben, aktivieren Sie das entsprechende Kontrollkästchen für die Erinnerung.

    •Wenn Sie keine Benachrichtigung konfiguriert haben, klicken Sie auf Konfigurieren, um zur Seite Einstellungen > Benachrichtigungen zu navigieren und eine solche einzurichten.

    Anhänge hinzufügen

    Klicken Sie in der Symbolleiste auf Anhänge hinzufügen, um im Textfeld Notizen einzugeben und Anhänge hinzuzufügen.

  4. Klicken Sie auf Speichern.
