Aufgaben im Lesefenster anzeigen

Sie können ein Lesefenster aktivieren, in dem der Inhalt einer Aufgabe dargestellt wird, ohne dass die Aufgabe dafür geöffnet werden muss. Das Lesefenster kann unten oder rechts im Inhaltsfenster angezeigt werden. Sie können das Lesefenster auch deaktivieren, damit es nicht angezeigt wird.

  1. Öffnen Sie die Seite Aufgaben.

  2. Klicken Sie in der Symbolleiste "Aufgabe" auf Ansicht, um auszuwählen, wie das Lesefenster dargestellt werden soll.

 

Option

Beschreibung

Lesefenster unten

Die Aufgabenlistenansicht wird unten im Fenster angezeigt.

Lesefenster rechts

Der Status einer Aufgabe wird in der Aufgabenlistenansicht nicht angezeigt, und der Prozentwert der Fertigstellung wird als Fortschrittsbalken angezeigt.

Lesefenster ausgeschaltet

Das Lesefenster wird nicht angezeigt.
