Aufgabenliste erstellen

Wenn Sie eine neue Aufgabe erstellen, wählen Sie einen Aufgabenordner aus, dessen Aufgaben überwacht werden sollen. Aufgabenordner werden auch als Aufgabenliste bezeichnet. In der Liste Aufgaben werden standardmäßig alle Aufgaben in Ihrem Aufgabenordner angezeigt. Neben der Standardaufgabenliste können Sie verschiedene weitere Aufgabenlisten erstellen. Außerdem können Sie die Listen mit den Aufgaben von einer Remote-iCal-Aufgabenliste synchronisieren.

  1. Öffnen Sie die Seite Aufgaben.

  2. Klicken Sie in der Übersicht auf das Zahnradsymbol für die Aufgaben und wählen Sie Neue Aufgabenliste aus.

  3. Geben Sie einen Namen ein und wählen Sie eine Farbe für die neue Aufgabenliste aus.

  4. Aktivieren Sie das entsprechende Kontrollkästchen und geben Sie die URL ein, um Aufgaben von einem Remote-Dienst zu synchronisieren.

  5. Klicken Sie auf OK. Die Aufgabenliste wird in der Aufgabenliste angezeigt.
