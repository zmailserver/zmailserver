Aufgabenlisten nutzen

Mit der Funktion Aufgaben können Sie Listen mit zu erledigenden Tätigkeiten anlegen und diese Aufgaben bis zur Fertigstellung der Aufgaben verwalten. Sie können Aufgaben in die Standardaufgabenliste aufnehmen. Sie können auch andere Aufgabenlisten anlegen, um Ihre eigenen Listen nach speziellen Aktivitäten zu organisieren, beispielsweise berufliche oder private Projekte. Sie können eine neue Aufgabenliste erstellen und die Aufgaben einer Remote-To Do-Liste von iCal synchronisieren.

Während Sie eine Aufgabe bearbeiten, können Sie Informationen hinzufügen, den Status sowie den Prozentwert der Fertigstellung ändern und die Aufgabe als abgeschlossen markieren.
