Aufgaben in einer Aufgabenliste sortieren

Sie können eine Aufgabenliste sortieren, um die Aufgaben schnell nach ihrem Status anzuzeigen.

  1. Öffnen Sie die Seite Aufgaben.

  2. Wählen Sie im Übersichtsfenster die Aufgabenliste aus, die Sie sortieren möchten.

  3. Klicken Sie in der Kopfzeile des Inhaltsfensters mit der rechten Maustaste auf das Dropdown-Menü Sortiert nach.

  4. Klicken Sie auf die gewünschte Sortierung der Aufgabenliste.
