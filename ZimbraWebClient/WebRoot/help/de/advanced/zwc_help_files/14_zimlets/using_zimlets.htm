Zimlets nutzen

Zimlets sind zusätzliche Anwendungen, die die Funktionalität Ihres Clients erweitern. Wenn Sie @ClientNameShort@ ein Zimlet hinzufügen, können Sie in Ihren Mails Informationen anzeigen und mit den Anwendungen von Dritten interagieren.

Die Installation von @ClientNameShort@ enthält bereits eine Reihe vorkonfigurierter Zimlets. Sie können weitere Zimlets auf der Seite "Zimlets" aktivieren, Zimlets aus der Galerie von Zimbra hinzufügen oder benutzerdefinierte Zimlets erstellen, um mit den Ressourcen Ihres Unternehmens oder anderen Anwendungen zu interagieren, die in @ClientNameLong@ definiert wurden.
