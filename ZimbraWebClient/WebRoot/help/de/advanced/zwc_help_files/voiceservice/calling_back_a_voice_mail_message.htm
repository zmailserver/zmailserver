Sprachnachricht beantworten

Hinweis: Diese Funktion ist nur für ZCS Network Edition vorgesehen und nur im erweiterten Webclient verfügbar.

Sie können eine Sprachnachricht beantworten, indem Sie den Absender mithilfe der Click-to-Call-Funktion zurückrufen.

  1. Wählen Sie die Registerkarte "Sprachnachrichten" aus.

  2. Wählen Sie in der Anrufsverlaufsliste die Sprachnachricht aus, die Sie beantworten möchten.

  3. Klicken Sie auf Rückruf. Die Rufnummer des Anrufers wird gewählt.
