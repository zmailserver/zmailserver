Lesebestätigung anfordern

Mail, die Sie an andere @ClientNameShort@-Nutzer senden, können markiert werden, damit eine Lesebestätigung angefordert wird, wenn der Empfänger die Mail öffnet. Beim Empfänger muss jedoch die Funktion Lesebestätigung für den Account aktiviert sein.

So legen Sie dies als Standardeinstellung fest

•       Öffnen Sie die Seite Einstellungen > Mail und wählen Sie im Abschnitt Mails verfassen die Option Immer Lesebestätigung anfordern aus.

So legen Sie diese Anforderung für eine Mail fest

•       Wählen Sie vor dem Senden der Mail in der Dropdown-Liste Optionen die Option Lesebestätigung anfordern aus.
