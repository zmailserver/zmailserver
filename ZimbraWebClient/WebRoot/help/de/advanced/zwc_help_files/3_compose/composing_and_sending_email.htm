Mail verfassen und senden

Sie können Mails verfassen und sofort senden, einen Entwurf verfassen und später fertigstellen und senden oder die Mail verfassen und einen Zeitpunkt angeben, zu dem die Mail später gesendet werden soll.

Sie können die Optionen zum Verfassen von Mails auf der Seite Einstellungen > Mail anpassen.

Neue Mail verfassen

Abhängig von den Maileinstellungen wird beim Klicken auf "Neue Mail" die Registerkarte "Verfassen" in der Symbolleiste "Anwendung" angezeigt oder die Seite zum Verfassen wird in einem eigenen Fenster geöffnet.

  1. Klicken Sie in der Registerkarte "Mail" auf Neue Mail.

    2. Je nach Ihren Maileinstellungen wird die leere Seite zum Verfassen auf einer neuen Registerkarte oder als neues Fenster geöffnet.

  2. Geben Sie im Textfeld An die Mail-Adressen der Personen ein, an die Sie die Mail senden möchten. Geben Sie die Adresse ein oder klicken Sie auf An, um in den Kontakten oder der globalen Adressliste zu suchen.

    3. Hinweis: Sie können die Blase mit Adressen im Textfeld mit der Maus in das Textfeld "CC" oder "BCC" ziehen und Sie können die Blase mit Adressen kopieren, indem Sie CTRL+C drücken.

  3. Wenn Sie mehrere Mail-Identitäten konfiguriert haben (auch als Persönlichkeiten bezeichnet), wird Von über dem Feld An angezeigt. Wählen Sie die Identität aus, die für diese Mail verwendet werden soll.

  4. (Optional) Klicken Sie unter Optionen auf Signaturen und wählen Sie die Signatur aus, die der Mail hinzugefügt werden soll.

  5. (Optional) Um eine Kopie der Mail an andere Empfänger zu senden (CC) oder ohne Wissen des Empfängers eine Blindkopie (BCC) zu senden, geben Sie im Feld "CC" bzw. "BCC" Mail-Adressen ein.

    6. Hinweis: Wenn das Feld "CC" oder "BCC" eine Adresse enthält, können Sie die Mail ohne Adresse im Feld "An" senden.

  6. Geben Sie im Feld Betreff das Thema der Mail ein.

  7. Klicken Sie auf Senden, um die Mail zu senden.

Eine Kopie der gesendeten Mail wird im Ordner "Gesendet" gespeichert.

Wenn Sie die Mail nicht sofort versenden wollen, klicken Sie statt auf "Senden" auf den Pfeil neben "Senden". Wählen Sie dann die Option Später senden aus. Geben Sie im Dialogfeld Sendezeit für die Mail angeben das Datum und die Uhrzeit ein, an dem bzw. zu der die Mail gesendet werden soll. Die Mail wird im Ordner "Entwürfe" gespeichert und automatisch zum Datum und zur jeweiligen Uhrzeit versendet.
