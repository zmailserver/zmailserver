Kalendereinstellungen einrichten

Unter "Einstellungen > Kalender" können Sie die folgenden Optionen einrichten. Alle Änderungen an den Einstellungen müssen Sie abschließend mit Speichern bestätigen.

  • Standardansicht. Geben Sie an, wie Ihr @ClientNameShort@-Kalender angezeigt werden soll. Standardmäßig wird die Arbeitswoche angezeigt. Folgende Ansichtsoptionen sind verfügbar: Tag, Arbeitswoche, Wochenansicht, Monat, Liste und Planung.

  • Woche beginnen am. Geben Sie an, welcher Wochentag in Ihrem Kalender der erste Tag der Woche sein soll.

  • Voreingestellte Sichtbarkeit von Terminen. Hier können Sie "Öffentlich" oder "Privat" einstellen. "Öffentlich" ist die Standardeinstellung. Alle eingehenden Kalendereinladungen werden als "Öffentlich" markiert, solange sie vom Ersteller nicht als "Privat" gekennzeichnet werden. Wenn diese Einstellung auf "Privat" festgelegt ist, werden alle eingehenden Kalendereinladungen als "Privat" markiert. Details zu Ereignissen, die als privat markiert sind, werden nicht angezeigt.

  • Mini-Kalender immer anzeigen. Hier geben Sie an, ob der Minikalender am unteren Rand des Übersichtsfensters angezeigt werden soll.

  • Kalender mit Wochennummern anzeigen. Wenn diese Option aktiviert ist, werden die Wochennummern auf dem Mini-Kalender angezeigt. Damit die richtigen Wochennummern angezeigt werden, muss die Arbeitswoche an einem Montag beginnen.

  • Empfangene Termine automatisch in Kalender einfügen. Wenn diese Option aktiviert ist, werden per Mail in Ihrem Posteingang empfangene Einladungen automatisch in Ihren Kalender eingefügt. Sie können die Einladung entweder im Posteingang oder im Kalender annehmen bzw. ablehnen. Falls diese Option nicht aktiviert ist, wird der Termin in Ihren Kalender eingefügt, wenn Sie auf "Annehmen" oder "Vorläufig" klicken.

  • Delegation für Apple iCal CalDAV Client freigeben. Apple iCal kann so konfiguriert werden, dass Sie mit dem CalDAV-Protokoll auf Ihre Kalender zugreifen können. Wenn diese Option aktiviert ist, werden freigegebene Kalender auf der Registerkarte "iCal-Accounts-Bevollmächtigung" des Apple Mac-Computers angezeigt. Wenn diese Option deaktiviert ist, werden alle freigegebenen Kalender in der Kalenderliste von @ClientNameShort@ angezeigt. Diese Funktion kann ab iCal 4.0 verwendet werden.

Hinweis: Die folgenden Funktionen sind nicht verfügbar, wenn Sie über die Registerkarte "iCal-Accounts-Bevollmächtigung" auf freigegebene @ClientNameShort@-Kalender zugreifen.

    • Verwenden der Links + oder "-" zum Hinzufügen oder Löschen von freigegebenen Kalendern von der Registerkarte "Bevollmächtigung".

    • Anzeigen des speziellen Rollentyps, der dem freigegebenen Kalender zugewiesen ist. Die Registerkarte "Bevollmächtigung" zeigt nur an, ob die Rolle nur über Lesezugriff oder über Lese-/Schreibzugriff verfügt. Sie können nicht feststellen, ob Sie über Verwaltungs- oder Administratorenrechte verfügen.

    • Freigabe Ihrer Kalender über diese Registerkarte.

  • Eine Einladung nach ihrer Beantwortung löschen. Wenn diese Option aktiviert ist, werden Einladungen nach ihrer Beantwortung automatisch in Ihren Papierkorb verschoben.

  • Meine Einladungen weiterleiten an. Sie können Mails, die Sie empfangen, an einen oder mehrere Nutzer weiterleiten, für die Sie Ihren Kalender freigegeben haben.

  • Erinnerungen anzeigen. Legen Sie fest, wie viele Minuten vor einem Termin Sie daran erinnert werden möchten. Standardmäßig werden Sie fünf Minuten vorher erinnert. Außerdem können Sie festlegen, welche Art von Benachrichtigung Sie erhalten möchten.

  • Wöchentliche Arbeitszeiten. Wählen Sie die Tage und Arbeitsstunden Ihrer Arbeitswoche aus. Wählen Sie Benutzerdefiniert aus, um Ihre Arbeitsstunden nach Tag anzupassen. Diese Informationen werden in Ihrer Ansicht "Frei/Belegt" angezeigt.

  • Schnelleingabe von Terminen verwenden. Geben Sie an, ob neue Termine über das Dialogfeld für die Schnelleingabe erstellt werden sollen. Dies ist häufig nützlich, wenn Sie viele Termine ohne Teilnehmer anlegen.

  • Liste der Zeitzonen in Terminansicht. Aktivieren Sie diese Option, wenn Sie häufig Besprechungen mit Teilnehmern in verschiedenen Zeitzonen planen.

  • Berechtigungen. Geben Sie an, welche Nutzertypen Ihre Frei/Belegt-Informationen anzeigen und Sie zu Besprechungen einladen können. Standardmäßig können alle Nutzer Ihre Frei-/Belegt-Zeiten sehen und Sie zu Besprechungen einladen.
