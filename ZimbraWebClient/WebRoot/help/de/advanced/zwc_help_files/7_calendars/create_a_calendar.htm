Kalender erstellen

Ein Kalender wird für Sie erstellt. Darüber hinaus können Sie weitere Kalender für verschiedene Zwecke erstellen. Beispielsweise können Sie einen Kalender für berufliche, einen für private und einen für andere Aktivitäten erstellen.

Wenn Sie mehrere Kalender verwalten, können Sie diese in einer hierarchischen Ansicht gruppieren. So können Sie etwa alle freigegebenen Kalender in einem Ordner mit dem Namen "Freigegeben" zusammenfassen. Kalender können auch verschoben werden.

•       Wählen Sie in der Kalenderübersicht einen Kalender aus und ziehen Sie ihn auf den Kalender, unter dem er angeordnet werden soll.

•       Sie können auch mit der rechten Maustaste auf den Kalender klicken, der verschoben werden soll, und dann Verschieben auswählen. Im Dialogfeld Ordner auswählen können Sie den Ordner auswählen, in den der Kalender verschoben werden soll.

Kalender hinzufügen

  1. Klicken Sie auf die Registerkarte Kalender.

  2. Wählen Sie in Symbolleiste Neu > Kalender aus.

  3. Geben Sie im Textfeld Name den Namen des Kalenders ein.

  4. Wählen Sie eine Farbe zum Anzeigen der Kalenderaktivitäten aus.

  5. Wählen Sie Diesen Kalender bei Frei-/Belegt-Zeiten ignorieren aus, wenn Sie verhindern möchten, dass andere Personen Ihren Kalender bei der Terminplanung anzeigen können.

  6. Wählen Sie Termine von externem Kalender synchronisieren aus, um Ihren Kalender mit einem externen Kalender zu synchronisieren. Geben Sie die URL für die externe iCal-Datei ein.

  7. Klicken Sie auf OK. Der neue Kalender wird in der Kalenderliste angezeigt.
