Frei/Belegt-Informationen anzeigen

Mit der Funktion "Frei/Belegt" können andere Personen die belegten Besprechungszeiträume in Ihren Kalendern anzeigen. Die Funktion "Frei/Belegt" ist standardmäßig aktiviert. Sie können sie jedoch ein- und ausschalten, um zu kontrollieren, welche Personen diese Informationen anzeigen können.

  1. Öffnen Sie die Seite Einstellungen > Kalender.

  2. Blättern Sie zum Abschnitt Berechtigungen, und wählen Sie die gewünschte Frei/Belegt-Option für diesen Kalender aus.

    3. Wenn Sie Nur die folgenden internen Nutzer dürfen meine Frei-/Belegt-Informationen sehen auswählen, können Sie im Textfeld die Mail-Adressen der internen Nutzer angeben.

  3. (Optional) Im Bereich Einladungen können Sie angeben, welche Personen Sie zu Besprechungen einladen können.

    4. Wenn Sie Nur die folgenden internen Nutzer dürfen meine Frei-/Belegt-Informationen sehen auswählen, können Sie im Textfeld die internen Nutzernamen oder die vollständige Mail-Adresse der Nutzer eingeben.

  4. Wenn Sie den Personenkreis einschränken, der Sie zu Besprechungen einladen kann, können Sie auch die Option Automatische Antwort an Nutzer senden, die mich nicht einladen dürfen verwenden.

  5. Klicken Sie auf Speichern.

Hinweis:   Diese Einstellungen wirken sich nicht auf Kalender aus, in denen keine Frei/Belegt-Informationen angezeigt werden.
