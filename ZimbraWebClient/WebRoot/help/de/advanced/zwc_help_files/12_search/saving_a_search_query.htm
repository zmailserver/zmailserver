Suchabfrage speichern

Wenn Sie eine Suche erstellen, die Sie später noch einmal durchführen möchten, können Sie sie speichern.

  1. Erstellen Sie die Suchabfrage.

  2. Wählen Sie im Dropdown-Menü der Registerkarte "Suchen" die Option Als Suchordner speichern aus.

  3. Geben Sie einen Namen für die Suchabfrage ein.

  4. Wählen Sie eine Farbe für die Suchabfrage aus.

  5. Geben Sie an, wo die Suchabfrage gespeichert werden soll.

  6. Klicken Sie auf OK.

Suchabfragen werden im Navigationsfenster gespeichert.

Um eine gespeicherte Suche zu verwenden, klicken Sie im Suchordner darauf. Die Suchergebnisse werden sofort im Inhaltsfenster angezeigt.
