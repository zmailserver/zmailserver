Suchfilter, die Sie verwenden können

In der folgenden Tabelle finden Sie Informationen zum Erstellen von detaillierteren Suchabfragen.

Auf der Registerkarte "Suchen" können Sie erweiterte Filter auswählen, um komplexere Abfragen zu erstellen.

Erweiterte Filter

Beschreibung

Empfangen von

Hiermit können Sie nach bestimmten Domains in der Kopfzeile der Mail (Von, An und Cc) suchen. Die Domains, nach denen Sie suchen können, sind vorgegeben.

Gesendet an

Hiermit können Sie nach bestimmten Domains in der Kopfzeile der Mail ("Von", "An" und "CC") suchen. Die Domains, nach denen Sie suchen können, sind vorgegeben.

Sendedatum

Hiermit können Sie nach Mails suchen, die nach, vor oder an einem bestimmten Datum empfangen wurden. Wenn Sie innerhalb eines bestimmten Datumsbereichs suchen möchten, wählen Sie Sendedatum ist vor und dann Sendedatum ist nach aus und geben Sie den Datumsbereich ein.

Anhänge

Hiermit können Sie nach Mails mit einem Anhang suchen. Wenn Sie nach einem bestimmten Anhangtyp suchen, wählen Sie den Typ aus der angezeigten Liste der Formate aus.

Größe

Mit dieser Option können Sie Mails anhand der Größe (Bytes, KB oder MB) suchen.

Status

Mit dieser Option können Sie anhand des Status nach Mails suchen, z. B. gelesen, ungelesen, weitergeleitet oder beantwortet.

Tag

Hiermit können Sie nach Mails oder Kontakten mit einem bestimmten Tag suchen.

Ordner

Mit dieser Option begrenzen Sie die Suche auf bestimmte Ordner.
