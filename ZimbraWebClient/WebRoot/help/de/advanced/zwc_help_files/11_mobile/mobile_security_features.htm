Sicherheitsfunktionen bei Mobilgeräten

Ihr Administrator hat möglicherweise eine Sicherheitsrichtlinie für Mobilgeräte eingerichtet, gemäß der Sie eine PIN (Personal Identification Number, Persönliche Kennnummer) auf Ihrem Mobilgerät erstellen müssen. Sollten Sie Ihre PIN vergessen haben, wenden Sie sich an den Systemverwalter.

Wenn Sie über eine PIN verfügen, ist die Richtlinie möglicherweise mit einer Funktion zum Aussperren von Nutzern eingerichtet, wenn diese eine falsche PIN eingeben.  Die Einstellungen für die PIN-Sperre werden vom Systemadministrator konfiguriert. Wenn eine andere Person versucht, sich mit einer falschen PIN bei einem Gerät anzumelden und dabei die Anzahl der zulässigen Versuche überschritten wird, wird auf dem Gerät lokal ein Löschvorgang ausgelöst. Dabei werden alle Daten auf dem Gerät gelöscht. Diese Richtlinieneinstellung kann nur vom Systemadministrator konfiguriert werden. Nutzer können sie nicht ändern.  Weitere Informationen erhalten Sie von Ihrem Systemadministrator.
