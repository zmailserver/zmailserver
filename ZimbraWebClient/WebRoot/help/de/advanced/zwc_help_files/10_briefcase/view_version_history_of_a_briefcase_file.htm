Versionsverlauf einer Datei im Ordner "Dateien" anzeigen

Mithilfe der Versionskontrolle können Sie die Änderungen an Ihren Dokumenten im Ordner "Dateien" verwalten. Jedes Mal, wenn Sie eine Änderung an einer Datei vornehmen und diese Datei speichern, wird die Versionsnummer aktualisiert. Die neueste Versionsnummer wird in der Dateiliste angezeigt.

•       Klicken Sie auf den Pfeil im Vorschaubereich, um den Versionsverlauf und die Autoren anzuzeigen. Wenn der Vorschaubereich deaktiviert ist, wird eine Spalte mit der Versionsnummer angezeigt.

Ein Dokument in einer früheren Version wiederherstellen

•       Klicken Sie mit der rechten Maustaste auf eine andere Version und wählen Sie Als aktuelle Version wiederherstellen aus.
