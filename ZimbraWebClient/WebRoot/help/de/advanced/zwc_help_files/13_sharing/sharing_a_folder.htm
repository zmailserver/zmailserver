Ordner freigeben

Sie können Mail-Ordner, Adressbücher, Kalender und Aufgabenlistenordner freigeben. Wenn Sie einem Nutzer eine Freigabe erteilen, kann diese von der anderen Person angenommen oder abgelehnt werden.

Hinweis:   Der Papierkorb und der Ordner "Spam" sind von der Freigabe von Mail-Ordnern ausgeschlossen.

  1. Klicken Sie mit der rechten Maustaste auf den Ordner, den Sie freigeben möchten.

  2. Klicken Sie auf Ordner freigeben.

  3. Geben Sie im Dialogfeld Eigenschaften der Freigabe an, ob die Freigabe für interne, externe oder öffentliche Nutzer erfolgen soll.

  4. Führen Sie abhängig von der Art der Freigabe folgende Aktionen aus:

    5. •    Geben Sie für Interne Nutzer oder Gruppen die Mailadressen der Nutzer ein. Legen Sie im Bereich Rolle die Zugangsberechtigungen fest.

    •    Geben Sie fürExterne Nutzer die Mailadressen der Nutzer ein. Externe Nutzer erhalten eine Mail mit einem Link zum freigegebenen Objekt. Wenn sie zum ersten Mal auf das freigegebene Objekt zugreifen, werden ein Anzeigename sowie ein Passwort erstellt.

    •    Geben Sie unter Öffentlich die URL ein, die verwendet werden soll.

  5. Wählen Sie im Bereich Mail den Typ von Nachricht aus, die an Nutzer gesendet werden soll, denen Sie Zugriff gewähren. Diese Mail kann sowohl an interne als auch an externe Nutzer gesendet werden. Hinweis: Wenn Sie Öffentlich ausgewählt haben, müssen Sie die URL öffentlich bekanntgeben.

  6. Klicken Sie auf OK.

Wenn ein Nutzer die Freigabe für einen Ordner annimmt, wird eine Kopie des Ordners in seinem Account erstellt.
