Mehrere Nachrichten für eine Aktion kennzeichnen

Mit den Kontrollkästchen auf der Inhaltsseite können Sie eine oder mehrere Mails oder Unterhaltungen auswählen, für die die gleiche Aktion ausgeführt werden soll.

Hinweis:   Wenn in der Mailansicht keine Kontrollkästchen angezeigt werden, wählen Sie auf der Seite "Einstellungen" > "Allgemein" im Abschnitt "Sonstiges" die Option "Einstellungen" aus.

Wenn Sie das Kontrollkästchen in der Kopfzeile aktivieren, werden automatisch die Kontrollkästchen für alle Mails auf der Seite aktiviert.

Mehrere Mails gleichzeitig löschen

  1. Aktivieren Sie im Mail-Inhaltsfenster die Kontrollkästchen für die zu löschenden Mails.

  2. Klicken Sie mit der rechten Maustaste und wählen Sie Löschen.

Mehrere Mails gleichzeitig mit Tags markieren

  1. Aktivieren Sie im Mail-Inhaltsfenster die Kontrollkästchen für die Mails, die mit dem gleichen Tag markiert werden sollen.

  2. Klicken Sie mit der rechten Maustaste und wählen Sie Unterhaltungen mit Tag versehen aus.

  3. Wählen Sie einen Tag aus der Liste aus oder erstellen Sie einen neuen Tag.

Mehrere Mails gleichzeitig drucken

  1. Aktivieren Sie im Mail-Inhaltsfenster die Kontrollkästchen für die zu druckenden Mails.

  2. Klicken Sie mit der rechten Maustaste und wählen Sie Drucken.

  3. Klicken Sie im Dialogfeld "Drucken" auf OK. Alle Mails werden als eine zusammenhängende Datei gedruckt.

Mehrere Mails gleichzeitig weiterleiten

  1. Aktivieren Sie im Mail-Inhaltsfenster die Kontrollkästchen für die Mails, die weitergeleitet werden sollen.

  2. Klicken Sie mit der rechten Maustaste und wählen Sie Weiterleiten. Die Seite zum Verfassen einer Mail wird geöffnet und die Mails werden als Anhänge angefügt.

  3. Vervollständigen Sie die Mail und senden Sie sie.
