Optionen für Spam-Mail zum Blockieren oder Zulassen bestimmter Adressen festlegen

Sie können die Maileinstellungen so einrichten, dass bestimmte Adressen in Ihrem Posteingang blockiert oder nicht blockiert werden. Sie erstellen dabei eine Liste mit Adressen, die in Ihrem Posteingang nicht zugelassen sind, und anderen Adressen, die eigentlich in Ihrem Posteingang zugelassen sein sollten, jedoch als Spam erkannt werden könnten. Der Spam-Filter stellt anhand dieser Liste sicher, dass Sie Mails wie gewünscht erhalten.

Wenn Sie bemerken, dass bestimmte Mail-Adressen immer fälschlicherweise in Ihren Posteingang oder Ihren Ordner "Spam" zugestellt werden, können Sie festlegen, dass diese Adressen blockiert bzw. nicht blockiert werden. 

  1. Öffnen Sie die Seite Einstellungen > Mail.

  2. Blättern Sie zum Abschnitt Optionen für Spam-Mail.

  3. Geben Sie im Textfeld Mails blockieren von Mail-Adressen ein, die immer als Spam behandelt werden sollen. Sie können bis zu 100 Adressen angeben.

  4. Geben Sie im Textfeld Mails zulassen von vertrauenswürdige Mail-Adressen ein, die in Ihrem Ordner "Spam" abgelegt werden, z. B. für abonnierte Newsletter. Mails von diesen Mail-Adressen werden niemals blockiert. Sie können bis zu 100 Adressen angeben.

  1. Klicken Sie auf Speichern.
