Unterhaltung (alle Mails) drucken

  1. Klicken Sie mit der rechten Maustaste auf die zu druckende Unterhaltung und wählen Sie Drucken aus. Die Unterhaltung wird in einem neuen Fenster geöffnet. Es wird ein Standarddialogfeld zum Drucken angezeigt.

  2. Wählen Sie einen Drucker aus.

  3. Klicken Sie auf OK. Die Unterhaltung wird an den ausgewählten Drucker übertragen.

  4. Schließen Sie das Unterhaltungsfenster.

Alternativ können Sie "Drucken" über den Link "Aktionen" in der Symbolleiste auswählen.

Eine Mail in einer Unterhaltung drucken

  1. Öffnen Sie die Unterhaltung und wählen Sie die zu druckende Mail aus.

  2. Klicken Sie unten in der Mail auf das Zahnradsymbol und wählen Sie die Option "Drucken" aus. Die Mail wird in einem neuen Fenster geöffnet. Es wird ein Standarddialogfeld zum Drucken angezeigt.

  3. Wählen Sie einen Drucker aus.

  4. Klicken Sie auf OK. Die Mail wird an den ausgewählten Drucker übertragen.

  5. Schließen Sie das Mailfenster.
