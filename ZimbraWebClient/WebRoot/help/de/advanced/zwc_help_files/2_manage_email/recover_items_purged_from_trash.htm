Gelöschte Elemente aus dem Papierkorb wiederherstellen

Wenn Gelöschte Objekte wiederherstellen angezeigt wird, nachdem Sie mit der rechten Maustaste auf den Papierkorb geklickt haben, können Sie gelöschte Mails aus dem Papierkorb wiederherstellen.

  1. Klicken Sie mit der rechten Maustaste auf den Mail-Papierkorb und wählen Sie die Option Gelöschte Objekte wiederherstellen aus.

  2. Wählen Sie eine oder mehrere Mails aus, die wiederhergestellt werden sollen.

    3. Suchen Sie mit dem Textfeld Suchen nach einem bestimmten Objekt, das wiederhergestellt werden soll.

  3. Klicken Sie auf das Symbol Ordner unter dem Textfeld "Suchen" und wählen Sie den Zielordner aus oder erstellen Sie einen neuen Ordner.

  4. Klicken Sie auf OK.
