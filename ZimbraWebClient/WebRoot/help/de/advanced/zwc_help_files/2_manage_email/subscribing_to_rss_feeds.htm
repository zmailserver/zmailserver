RSS-Feeds abonnieren

Sie können Websites abonnieren, die von Ihnen bevorzugte RSS-Feeds (Really Simple Syndication) und Podcast-Feeds bieten und aktuelle Informationen direkt an Ihre Mailbox senden. Sie müssen die URL des Feeds kennen, um das Abonnement einzurichten.

  1. Klicken Sie in der Kopfzeile Ordner auf das Zahnradsymbol und wählen Sie Neuer Ordner aus.

    2. Hinweis:  Sie müssen für jeden RSS-Feed einen eigenen Ordner anlegen.

  2. Geben Sie im Textfeld Name den Namen für den RSS-Feed-Ordner ein.

  3. Aktivieren Sie RSS-/ATOM-Feed abonnieren.

  4. Geben Sie im Feld URL die URL für den Feed ein.

  5. Wählen Sie im Ordnermenü aus, wo der Ordner für den RSS-Feed angelegt werden soll. Wenn Sie den Stammordner auswählen, wird der neue Ordner in Ihrer Ordneransicht auf der obersten Ebene angezeigt, d. h. auf der gleichen Ebene wie Ihr Posteingangsordner.

  6. Klicken Sie auf OK. Der neue Ordner wird im Übersichtsfenster mit dem Symbol für RSS-Feeds angezeigt.

Ihr RSS-Datenfeed wird automatisch alle 12 Stunden aktualisiert. Sie können jederzeit in der Symbolleiste auf Feed laden klicken, um den Inhalt zu aktualisieren.

Hinweis:   RSS-Feeds, die Sie speichern, werden gegen Ihr Accountkontingent gewertet.
