Termine, Besprechungen und Ereignisse planen

Sie können einzelne oder wiederkehrende Termine, Besprechungen und Ereignisse erstellen. Während Sie einen Termin erstellen, können Sie vor dem Einladen der Teilnehmer sehen, ob diese verfügbar sind, einen Raum sowie Ausrüstung reservieren und eine Nachricht verfassen, die zusammen mit der Einladung gesendet werden soll.

Sie können folgende Aktivitäten planen:

  • Termine: Termine sind Aktivitäten, bei denen keine anderen Personen oder Ressourcen beteiligt sind. Wenn Sie einen Termin planen, wird keine Mail zur Bestätigung des Termins gesendet. Sie können wiederkehrende Termine einrichten. Über die Schnelleingabe können Sie rasch einen Termin erstellen.

  • Ereignisse: Ein Ereignis ist eine Aktivität, die einen ganzen Tag in Anspruch nimmt. Ereignisse werden im Kalender nicht mithilfe von Uhrzeiten angezeigt; sie erscheinen als Banner im oberen Bereich des Kalenderplans.

  • Besprechungen: Besprechungen sind Termine, bei denen andere Personen beteiligt sind. Wenn Sie eine Besprechung anlegen, wählen Sie Teilnehmer und Ressourcen aus und senden diesen per Mail eine Einladung. Sie können Besprechungen verlegen und wiederkehrende Besprechungen einrichten.
