Mail- und SMS-Erinnerungsbenachrichtigungen einrichten

Sie können festlegen, dass Termin- und Aufgabenerinnerungen per Mail an Sie und als SMS an Ihr Mobilgerät gesendet werden. Die Erinnerung wird zu der Erinnerungsuhrzeit gesendet, die in der Aufgabe oder im Termin konfiguriert ist.

Hinweis: Sie können Benachrichtigungen einrichten, wenn Sie auf einer Aufgaben- oder Kalenderseite arbeiten und auf Konfigurieren klicken. Wenn Sie auf einer Aufgaben- oder Kalenderseite eine Benachrichtigung einrichten oder ändern, wird die Adresse oder Telefonnummer auf der Seite "Einstellungen > Benachrichtigungen" geändert. Anschließend wird an Sie eine Benachrichtigung über die bevorstehende Aufgabe oder das bevorstehende Kalenderereignis gesendet. Sie können nicht für unterschiedliche Aufgaben oder Kalenderobjekte unterschiedliche Benachrichtigungsadressen oder SMS-Nummern einrichten.

Mail-Benachrichtigung einrichten

  1. Öffnen Sie Einstellungen > Benachrichtigungen.

  2. Geben Sie die Mail-Adresse ein, an die die Erinnerung gesendet werden soll.

  3. Klicken Sie auf Speichern.

SMS-Benachrichtigung einrichten

  1. Öffnen Sie Einstellungen > Benachrichtigungen.

  2. Wählen Sie im Dropdown-Menü Region Ihr Land aus.

  3. Wählen Sie im Dropdown-Menü Netzbetreiber Ihren Telefonnetzbetreiber aus.

  4. Geben Sie im Feld Telefonnummer Ihre Telefonnummer ein.

  5. Klicken Sie auf Prüfcode senden. An Ihr Telefon wird eine SMS gesendet.

  6. Geben Sie im Feld Prüfcode den Code aus der SMS ein, die Sie empfangen haben.

  7. Klicken Sie auf Code prüfen. Im Feld "Status" wird die bestätigte Nummer angezeigt.

  8. Klicken Sie auf Speichern.
