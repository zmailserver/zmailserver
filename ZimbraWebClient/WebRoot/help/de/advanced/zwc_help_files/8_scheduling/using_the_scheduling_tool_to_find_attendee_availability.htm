Teilnehmerverfügbarkeit mit dem Planungswerkzeug überprüfen

Beim Planen von Besprechungen können Sie überprüfen, ob die Teilnehmer belegt oder frei sind. So können Sie einen günstigen Termin finden, an dem alle Teilnehmer an der Besprechung teilnehmen können. Während Sie Teilnehmer im Feld "Teilnehmer" hinzufügen, werden im Übersichtsfenster Vorschläge zu ihrer Verfügbarkeit in 30-Minuten-Zeitabschnitten angezeigt.

Standardmäßig werden Uhrzeiten vorgeschlagen, die mit ihren und den Arbeitszeiten der anderen Teilnehmer übereinstimmen.

•      Nur meine Arbeitszeiten einschließen. Das sind die wöchentlichen Arbeitszeiten, die Sie in Ihren Kalendereinstellungen festgelegt haben.

•      Nur die Arbeitszeiten anderer Teilnehmer einschließen. Bei der Suche nach einem Ort werden die Arbeitszeiten aller Teilnehmer berücksichtigt.

Sie können dies ändern.

  1. Klicken Sie auf der Seite Neuer Termin auf Zeit vorschlagen.

  2. Klicken Sie im Fenster Zeiten vorschlagen, das geöffnet wird, auf das Zahnradsymbol.

  3. Deaktivieren Sie die Kontrollkästchen, die nicht zum Berechnen der Teilnehmerverfügbarkeit verwendet werden sollen.

  4. Geben Sie bei wiederkehrenden Besprechungen an, bei wie vielen Instanzen der wiederkehrenden Besprechung ein Teilnehmer verfügbar sein sollte.

  5. Klicken Sie auf OK.

Verfügbarkeit im Mini-Kalender anzeigen

Im Mini-Kalender wird die Verfügbarkeit der Teilnehmer farblich codiert angezeigt.

•      Tage, an denen alle Teilnehmer und mindestens ein Ort verfügbar sind, werden grün angezeigt.

•      Tage, an denen mindestens die Hälfte der Teilnehmer nicht verfügbar ist, werden rot angezeigt.
