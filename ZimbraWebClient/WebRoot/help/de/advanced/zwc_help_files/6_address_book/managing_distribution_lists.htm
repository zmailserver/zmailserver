Verteiler verwalten

Wenn Sie Mails an eine Gruppe von Adressen senden, können Sie auf der Registerkarte "Adressbuch" Verteiler erstellen. Ein Verteiler ist eine Gruppe von Mail-Adressen in einer Liste mit einer gemeinsamen Mail-Adresse. Wenn eine Mail an einen Verteiler gesendet wird, wird sie an jede Person gesendet, deren Adresse im Verteiler enthalten ist. Für den Empfänger wird der Verteilername in der Zeile "An" der Mail angezeigt. In der Blasenansicht kann der Empfänger auf "+" klicken, um die Mitglieder des Verteilers anzuzeigen.

Verteiler werden im Adressbuch "Verteiler" gespeichert. Wenn Sie im Adressbuch auf einen Verteiler klicken, werden im Feld "Inhalt" alle Mitglieder des Verteilers angezeigt.

Ein Verteiler kann als Liste von Mitgliedern erstellt werden und Sie können die Zugehörigkeit der Mitglieder zur Liste verwalten. Sie können folgende zusätzliche Optionen konfigurieren:

  • Den Verteiler als privat markieren, damit er in der globalen Adressliste nicht angezeigt wird

  • Beschränken, wer Mail an den Verteiler senden darf

  • Benutzer den Verteiler abonnieren und das Verteilerabonnement kündigen lassen

  • Mehrere Besitzer einrichten, die den Verteiler verwalten

Einen Schnellverteiler einrichten

  1. Öffnen Sie das Dropdown-Menü Adressbuch>Neuer Kontakt und wählen Sie Verteiler aus.

  2. Geben Sie in der Seitenüberschrift einen Namen für den Verteiler ein. Dies ist die Mail-Adresse des Verteilers.

  3. Geben Sie in Anzeigename einen Anzeigenamen für den Verteiler ein. Dieser muss nicht mit der Mail-Adresse übereinstimmen.

  4. Geben Sie die Beschreibung des Verteilers ein (optional).

  5. Klicken Sie auf die Registerkarte Mitglieder und wählen Sie Adressen aus, die dem Verteiler hinzugefügt werden sollen. Sie können in den Kontaktlisten oder in der globalen Adressliste Adressen suchen und Sie können Adressen hinzufügen, die nicht in diesen Listen enthalten sind.

  6. Klicken Sie auf Speichern. Der Verteiler wird dem Adressbuch "Verteiler" hinzugefügt.
