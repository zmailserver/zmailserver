Neuen Kontakt aus einer Mail übernehmen

Sie können Mail-Adressen direkt aus den Feldern Gesendet von, An, CC und BCC in Ihr Adressbuch aufnehmen.

  1. Klicken Sie in der Mail mit der rechten Maustaste auf den Namen, der hinzugefügt werden soll, und wählen Sie dann Zu Kontakten hinzufügen aus.

    2. Ein neues Kontaktformular wird geöffnet. Die Informationen aus der Kopfzeile der Mail sind bereits eingetragen.

  2. Überprüfen Sie diese Informationen und tragen Sie ggf. weiteren Daten ein.

  3. Wählen Sie aus, wie und wo der Kontakt gespeichert werden soll.

    4. •    Wählen Sie unter Speichern als aus, wie der Name angezeigt werden soll. Standardmäßig wird zuerst der Nach- und dann der Vorname angegeben.

    •    Wählen Sie unter Ort aus, in welchem Adressbuch der Kontakt gespeichert werden soll.

  4. Klicken Sie auf Speichern.
