Mitglieder zu einer Kontaktgruppenliste hinzufügen

  1. Öffnen Sie die Seite Adressbuch und wählen Sie die Kontaktgruppe aus, der Sie Mitglieder hinzufügen möchten.

  2. Klicken Sie mit der rechten Maustaste auf die Kontaktgruppe und wählen Sie Gruppe bearbeiten aus.

  3. Sie können ein neues Gruppenmitglied finden, indem Sie einen Namen in das Textfeld Suchen eingeben.

  4. Wählen Sie im Dropdown-Menü das Adressbuch aus, das Sie durchsuchen möchten.

    5. Namen, die Ihrem Eintrag entsprechen, werden im Feld darunter angezeigt.

  5. Wählen Sie die Namen aus, die Sie eingeben möchten, und klicken Sie auf Hinzufügen oder auf Alle hinzufügen. Die Namen werden der Liste Mitglieder hinzugefügt.

    6. Wenn sich die gesuchte Adresse nicht in der Liste befindet, können Sie die Mail-Adresse im Feld Oder geben Sie die Adressen unten ein eingeben. Trennen Sie einzelne Adressen durch ein Komma (,) oder Semikolon (;) oder drücken Sie die Eingabetaste.

  6. Klicken Sie auf Speichern.
