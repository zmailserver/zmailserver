Tag entfernen

  1. Klicken Sie mit der rechten Maustaste auf ein mit einem Tag versehenes Objekt.

  2. Wählen Sie im Dropdown-Menü mit Tag versehen aus. Das Thema kann z. B. eine Unterhaltung sein, dies wird als Unterhaltung mit Tag versehen angezeigt.

  3. Wählen Sie Tag entfernen aus. Wenn für das Objekt mehrere Tags vorhanden sind, wählen Sie das zu entfernende Tag aus oder wählen Sie Alle Tags, wenn Sie alle Tags entfernen möchten.

Sie können einen Tag aus einer Mail entfernen, wenn die Mail in einem Lesefenster geöffnet ist. Das Tag wird in der Mail-Kopfzeile unter Tags angezeigt. Sie können auf das X im Tag klicken, um das Tag zu löschen.
