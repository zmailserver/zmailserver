Filter aus einer Mail erstellen

Sie können einen Filter auf Grundlage einer Mail in Ihrem Posteingang erstellen. Die Filterinformationen werden aus der Mail extrahiert und Sie können dann die Bedingungen und Aktionen ändern.

  1. Klicken Sie mit der rechten Maustaste auf die Mail und wählen Sie Neuer Filter aus.

    2. Das Dialogfeld "Filter hinzufügen" wird angezeigt und enthält die Informationen aus der Mail.

  2. Geben Sie im Textfeld Filtername einen eindeutigen Namen für den Filter ein.

  3. Ändern Sie die Bedingungen und Aktionen in den Dropdown-Menüs nach Bedarf.

  4. Klicken Sie auf OK.
