Filter für Kalendereinladungen erstellen

Sie können einen Filter zum Organisieren der Kalendereinladungen erstellen, die Sie erhalten. Sie können von Ihnen empfangene und gesendete Einladungen filtern. Sie können den Anwendungsbereich dieses Filters konfigurieren.

  1. Öffnen Sie die Seite Einstellungen > Filter.

  2. Klicken Sie auf Filter für eingehende Mails oder Filter für ausgehende Mails.

  3. Wählen Sie Neuer Filter aus.

  4. Geben Sie im Textfeld Filtername einen eindeutigen Namen für den Filter ein.

  5. Um den Filter zu aktivieren, aktivieren Sie das Feld Aktiv oben rechts.

  6. Wählen Sie im Dropdown-Menü Betreff die Option Kalender aus.

  7. Wählen Sie im Dropdown-Menü eine Bedingung für Einladung wird beantwortet/Einladung wird nicht beantwortet oder Einladung wird angefordert/Einladung wird nicht angefordert aus.

  8. (Optional) Wenn Sie weitere Bedingungen hinzufügen möchten, klicken Sie auf die Schaltfläche + und wiederholen Sie die oben genannten Schritte. Die Reihenfolge der Bedingungen spielt keine Rolle.

  9. Wählen Sie unter Folgende Aktionen ausführen im Dropdown-Menü eine Aktion aus, z. B. das Verschieben einer Einladung in einen bestimmten Ordner.

    10. Je nach der von Ihnen ausgewählten Aktion müssen Sie eventuell zu einem Ordner- oder Tagnamen navigieren.

  10. (Optional) Wenn Sie weitere Bedingungen hinzufügen möchten, klicken Sie auf die Schaltfläche + und wiederholen Sie den oben genannten Schritt. Die Aktionen werden in der Reihenfolge ausgeführt, in der sie angegeben wurden.

  11. Standardmäßig wird nach der ersten Übereinstimmung nicht nach weiteren Übereinstimmungen gesucht. Wenn mehrere Filter auf die Mail angewendet werden sollen, deaktivieren Sie Keine weiteren Filter anwenden.

  12. Klicken Sie auf OK.
