Mit Platzhaltern filtern

Platzhalter können in Vergleichen mit dem Vergleichsoperator "Entspricht dem Muster" verwendet werden. Die beiden Wildcards sind * und ?.

Sternchen (*)

Das Sternchen * ist ein Wildcard für null oder mehr beliebige Zeichen.

Beispiel für eine Suchzeichenkette in der Betreffzeile: Bananen*split

Entsprechungen: Betreffzeilen wie "Bananensplit", "Bananen", "Bananen-Erdbeer-Split", jedoch nicht "Super-Bananensplit"

Beispiel für eine Suchzeichenkette in der Betreffzeile: w*Buch

Entsprechungen: Betreffzeilen wie "weißes Buch" oder "wessen Buch", aber nicht "wessen Buch ist das?"

Fragezeichen (?)

Das Fragezeichen ist ein Wildcard für genau ein Zeichen.

Beispiel für eine Suchzeichenkette in der Betreffzeile: Bananen?split

Entsprechungen: Betreffzeilen wie "Bananen-Split", "Bananen!split" oder "Bananen Split"

Rückstrich als Escape-Zeichen: \

Es kann sein, dass Sie eine genaue Entsprechung einer Zeichenkette angeben möchten, die Zeichen enthält, die als Platzhalter gelten. Angenommen, Sie möchten eine Entsprechung für eine Betreffzeile definieren, in der ein Ausdruck auf beiden Seiten von jeweils drei Sternchen umgeben ist, z. B.

***MEHR GELD!!!*** oder

***GRATISGESCHENK***

Um das Zeichen eines Wildcards als das Zeichen selbst und nicht als Wildcard für andere Zeichen anzugeben, geben Sie direkt vor dem Wildcard einen Rückstrich (\) ein. Die Vergleichszeichenkette "\*\*\**\*\*\*" gibt z. B. einen Betreff mit drei Sternchen vor und nach einer beliebigen Zeichenkette in der Mitte (einschließlich Leerzeichen) an.
