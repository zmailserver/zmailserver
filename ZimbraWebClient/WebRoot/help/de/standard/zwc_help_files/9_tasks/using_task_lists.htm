Aufgabenlisten nutzen

Mit der Funktion Aufgaben können Sie Listen mit zu erledigenden Tätigkeiten anlegen und diese Aufgaben bis zur Fertigstellung der Aufgaben verwalten. Sie können Aufgaben in die Standardaufgabenliste aufnehmen. Sie können auch andere Aufgabenlisten anlegen, um Ihre eigenen Listen nach speziellen Aktivitäten zu organisieren, beispielsweise berufliche oder private Projekte.

Während Sie eine Aufgabe bearbeiten, können Sie Informationen hinzufügen, den Status sowie den Prozentwert der Fertigstellung ändern und die Aufgabe als abgeschlossen markieren.

Aufgaben sortieren

Sie können die Aufgabenliste nach Betreff, Status, Prozentsatz der Fertigstellung und Fälligkeitsdatum sortieren.

  • Wählen Sie in der Aufgabenansicht eine Aufgabenliste aus und klicken Sie in der Symbolleiste im Inhaltsfenster auf die Überschrift, nach der Sie sortieren möchten.

Aufgabe erstellen

  1. Klicken Sie in der Symbolleiste auf "Neue Aufgabe".

  2. Geben Sie im Abschnitt Details die Aufgabeninformationen ein. Nur Betreff ist ein Pflichtfeld, damit die Aufgabe einen Namen erhält.

  3. Geben Sie im Abschnitt Datum das Startdatum und das Fälligkeitsdatum ein.

  4. Klicken Sie auf Speichern.
