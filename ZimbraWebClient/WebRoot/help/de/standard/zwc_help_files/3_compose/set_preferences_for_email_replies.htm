Einstellungen für Mail-Antworten einrichten

Wenn Sie eine Mail im gleichen Format beantworten möchten, in dem sie empfangen wurde, wählen Sie Im Format der ursprünglichen Mails antworten/weiterleiten aus. Hiermit wird sichergestellt, dass der Empfänger die Mail im am besten lesbaren Format erhält.

  1. Öffnen Sie auf der Seite Einstellungen>Mail den Abschnitt Mails verfassen.

  2. Wenn Sie eine Mail im gleichen Format beantworten möchten, in dem sie empfangen wurde, wählen Sie Im Format der ursprünglichen Mails antworten/weiterleiten aus.

  3. Wählen Sie aus, ob der ursprüngliche Text in Ihrer Antwortmail enthalten sein soll. Wenn der ursprüngliche Text eingeschlossen wird:

    4. •   Wählen Sie aus, ob die bisherigen Antworten mit einem Zeichen markiert werden sollen oder die ursprünglichen Details in der Kopfzeile (die Felder "An", "Von" und "Betreff") eingeschlossen werden sollen. Sie können die ursprüngliche Mail auch als Anhang anfügen.

    •    Wenn Sie den bisherigen Antworten ein Präfix voranstellen möchten, wählen Sie als Zeichen eine eckige Klammer (>) oder einen senkrechten Strich (|) aus.

  4. Klicken Sie auf Speichern.
