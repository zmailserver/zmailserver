Mails im HTML-Format verfassen

Im HTML-Editor erstellte Mails können formatierten Text, Nummerierungen, Aufzählungszeichen, farbigen Hintergrund, Tabellen und Links enthalten.  Einige Mail-Clients können jedoch keine HTML-Mails anzeigen. Sie können den Editor für eine einzelne Mail wechseln.

Hinweis:   Sie müssen nicht mit HTML vertraut sein, um den HTML-Editor zu verwenden.

Wenn Sie das Format der Mail ändern möchten, die Sie gerade senden, wählen Sie unter "Optionen" entweder Als HTML formatieren oder Als Plain-Text formatieren aus.
