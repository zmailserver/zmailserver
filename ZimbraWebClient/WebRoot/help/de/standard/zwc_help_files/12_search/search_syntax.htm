Regeln für die Suchsyntax

Sie können nach einem beliebigen Objekt suchen, z. B. nach einem Wort, Vornamen oder Nachnamen, nach einer Telefonnummer oder einem Domainnamen. Wenn Sie nach Inhalten suchen, müssen Sie jedoch einige Regeln beachten. Bei der Suche nach Inhalten werden der Text der Mail sowie alle Dateianhänge durchsucht, die vom System gelesen werden können.

Option

Beschreibung

Exakte Übereinstimmung

Wenn Sie nach Ausdrücken aus mehreren Wörtern suchen, muss jedes Wort im Ausdruck eine exakte Übereinstimmung sein. Rechtschreibvarianten sind nicht zulässig. Angenommen, Sie suchen nach Bananen. In diesem Fall werden Mails mit Banane nicht als Übereinstimmung betrachtet.

Keine Berücksichtigung von Groß-/Kleinschreibung

Die Groß- und Kleinschreibung spielt bei der Suche keine Rolle. Die Schreibweisen Südlich, südlich und SÜDLICH werden genau gleich behandelt.

Sonderzeichen

Die folgenden Sonderzeichen können nicht für Suchvorgänge verwendet werden. ~ ' ! @ # $ % ^ & * () _- + ? / { }[ ] ; : "

Copyright- und Warenzeichensymbole

Sonderzeichen, z. B. Warenzeichensymbole, die Teil eines Wortes sind, werden entfernt, wenn das Wort für die Suche indiziert wird. Diese Sonderzeichen werden ignoriert, wenn Sie bei der Suche nach einem ganzen Wort als Teil des Wortes verwendet werden. Beispielsweise werden bei der Suche nach "Zimbra™" alle Vorkommen von Zimbra, einschließlich der Vorkommmen ohne das Warenzeichensymbol, gefunden.

Wildcards

Das Sternchen * kann als Wildcard im Anschluss an eine bestimmte Zeichenfolge verwendet werden. Beispielsweise gibt eine Suche nach Ha* Objekte mit dem Wort Hand, Haus usw. zurück.

Hinweis: In Suchabfragen gilt ein Doppel-Byte-Sternchen nicht als Platzhalterzeichen und wird während der Suche ignoriert.

Anhänge

Dateianhänge, die vom System gelesen werden können, werden durchsucht. Damit eine Datei vom System gelesen werden kann, muss sie in HMTL konvertiert werden können. Dies trifft beispielsweise auf Microsoft Office-Dokumente und auf Textdateien zu.

Kontakte in Adressbüchern

Sie müssen nach einem ganzen Wort oder nach einer Ziffernfolge suchen. Wenn Sie beispielsweise Kontakte anhand ihrer Telefonnummer suchen möchten, müssen Sie die vollständige Telefonnummer so eingeben, wie Sie im Eintrag für den Kontakt angezeigt wird. Eine Suche nur nach der Vorwahl ist nicht möglich.

Hinweis: Die Suche nach Inhalten in @ClientNameLong@ unterscheidet sich geringfügig von der Suche nach Textübereinstimmungen oder der Suche in Textverarbeitungsprogrammen.
