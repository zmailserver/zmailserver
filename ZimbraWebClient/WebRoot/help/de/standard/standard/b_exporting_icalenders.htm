iCalendars exportieren

Sie können Kalender, die im Format .ics gespeichert wurden, exportieren. iCalendar ist das Standardformat für den Austausch von Kalenderdaten über das Internet, das Dateiformat lautet .ics.

  1. Öffnen Sie die Registerkarte "Kalender".

  2. Klicken Sie im Übersichtsfenster des Kalenders auf Bearbeiten.

  3. Wählen Sie den zu exportierenden Kalender aus.

  4. Klicken Sie im Feld In .ICS-Datei exportieren auf den Kalenderlink.

  5. Wählen Sie im Dialogfeld In Datei speichern aus.

  6. Klicken Sie auf OK.
