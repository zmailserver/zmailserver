Mails suchen, die vom Absender gesendet oder empfangen wurden

Sie können mit einer der folgenden Methoden Mails suchen, die an den Absender einer Mail gesendet oder von diesem empfangen wurden:

  • Durch Rechtsklick auf eine Mail, die vom Absender gesendet oder empfangen wurde

  • Mit dem Feld "An/CC" in "Erweiterte Suche" bzw. "Einfache Suche"

Wenn Sie einen Ordner durchsuchen, werden nur Ergebnisse für Mails innerhalb dieses Ordners angezeigt. Wenn Sie die Option "Erweiterte Suche" nutzen, können Sie die zu durchsuchenden Ordner auswählen. Als Ergebnis werden die Mails aus den für die Suche ausgewählten Ordnern ausgegeben.

Empfangene Mails durchsuchen

Hinweis: Wählen Sie für diese Suche eine Mail und keine Unterhaltung aus.

  1. Wählen Sie im Posteingang oder in anderen persönlichen Ordnern eine Mail von dem Absender aus, den Sie suchen.

  2. Klicken Sie mit der rechten Maustaste auf die Mail und wählen Sie Mails suchen aus.

  3. Wählen Sie entweder Empfangen von Absender oder An Absender gesendet aus. Die Suche wird durchgeführt und Mails, die der Auswahl entsprechen, werden im Inhaltsfenster angezeigt.

Erweiterte Suche durchführen

  1. Klicken Sie auf "Erweitert", um den Bereich für die erweiterte Suche zu öffnen.

  2. Geben Sie im Feld Von des Fensters Einfache Suche den zu suchenden Namen des Absenders der Mail ein. Je genauer der Name eingegeben wird, desto besser sind die Ergebnisse. Standardmäßig wird der Posteingang durchsucht.

  3. Zum Durchsuchen anderer Ordner klicken Sie auf Ordner und wählen Sie dann die Ordner aus, die durchsucht werden sollen.

Sobald Sie in einem Suchfenster Kriterien eingeben oder in einem anderen Suchfenster eine Auswahl geändert haben, wird die Suche ausgeführt.

Elemente, die den Suchkriterien entsprechen, werden unten rechts aufgeführt. Wenn die Unterhaltungsfunktion aktiviert ist, werden Unterhaltungen, die Mails enthalten, welche den Suchkriterien entsprechen, aufgelistet.
