Ordner löschen

Benutzerdefinierte Ordner können Sie löschen. Systemordner können Sie nicht löschen.

Wenn Sie einen Ordner löschen, wird sein Inhalt in den Papierkorb verschoben. Sie können Inhalte aus dem Papierkorb wiederherstellen, indem Sie diese mit der Maus in einen anderen Ordner ziehen.

Hinweis: Der Papierkorb wird in regelmäßigen Abständen geleert, d. h., darin enthaltene Objekte bleiben nicht für unbeschränkte Zeit verfügbar.

Ordner löschen:

  1. Wählen Sie den zu löschenden Ordner aus.

  2. Klicken Sie im Übersichtsfenster unter der Überschrift "Ordner" auf Bearbeiten.

  3. Aktivieren Sie Diesen Ordner löschen.

  4. Klicken Sie auf Ordner löschen.

Wenn der Ordner Mails enthält, die Sie speichern möchten, verschieben Sie sie vor dem Löschen in einen anderen Ordner.

Inhalt eines Ordners löschen

Sie können den gesamten Inhalt eines Ordners, jedoch nicht den Ordner selbst löschen.

  1. Wählen Sie den Ordner aus, dessen Inhalt Sie löschen möchten.

  2. Klicken Sie im Übersichtsfenster unter der Überschrift "Ordner" auf Bearbeiten.

  3. Aktivieren Sie Alle Objekte in diesem Ordner dauerhaft löschen. Dieser Vorgang kann nicht rückgängig gemacht werden.

  4. Klicken Sie auf Alle Objekte löschen. Der Inhalt des Ordners wird sofort gelöscht.
