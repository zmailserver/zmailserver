Mails und Kontakte mit Tags klassifizieren

Tags sind Ihr persönliches Klassifizierungssystem für Mails, Kontakte, Aufgaben und Termine. Tags helfen Ihnen, diese Objekte zu finden. Sie können beispielsweise einen Tag Sofort erledigen einrichten und einen anderen Mittlere Priorität. Die mit Tags markierten Mails können sich in unterschiedlichen Ordnern befinden. Sie können nach einem Tag suchen, und alle Mails mit diesem Tag werden angezeigt, ganz gleich, in welchem Ordner sie sich befinden.

  • Sie können eine beliebige Anzahl Tags erstellen.

  • Sie können einer Mail oder einem Kontakt mehrere Tags zuordnen.

Neue Tags erstellen

Die Funktion Neuer Tag rufen Sie auf eine der folgenden Arten auf:

  • Klicken Sie in der Symbolleiste auf den Pfeil neben Neu und wählen Sie Tag aus.

  • Klicken Sie in der Symbolleiste auf Tag.

  • Klicken Sie in einer Listenansicht mit der rechten Maustaste auf ein Objekt und wählen Sie Unterhaltung mit Tag versehen .

  • Klicken Sie mit der rechten Maustaste auf Tags oder auf einen Tag im Übersichtsfenster und wählen Sie Neuer Tag.

Daraufhin wird das Dialogfeld Neuer Tag angezeigt. Geben Sie einen Namen ein und klicken Sie auf OK. Für Tagnamen sind alle Zeichen zulässig, außer:

 : (Doppelpunkt)

/ (Schrägstrich)

" (Anführungszeichen)

Der neue Tag wird im Übersichtsfenster unter Tags aufgeführt und kann beliebigen Mails oder Kontakten zugewiesen werden.
