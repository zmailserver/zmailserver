RSS-/Atom-Feeds abonnieren

Sie können Websites abonnieren, die von Ihnen bevorzugte RSS-Feeds (Really Simple Syndication) und Podcast-Feeds bieten und aktuelle Informationen direkt an Ihre Mailbox senden.

Für jeden RSS-Feed legen Sie einen Ordner an. Halten Sie vor dem Fortfahren die URL für den RSS bereit.

  1. Klicken Sie im Übersichtsfenster in der Überschrift Ordner auf das Ordnersymbol.

  2. Geben Sie im Feld Name den Namen für den RSS-Feed-Ordner ein.

  3. Aktivieren Sie RSS-/ATOM-Feed abonnieren.

  4. Geben Sie im Feld URL die URL des RSS-Feeds ein.

  5. Wählen Sie im Ordnermenü aus, wo der Ordner für den RSS-Feed angelegt werden soll. Wenn Sie den Stammordner auswählen, wird ein Ordner angelegt, der in Ihrer Ordneransicht auf der obersten Ebene angezeigt wird, d. h. auf der gleichen Ebene wie Ihr Posteingangsordner.

  6. Klicken Sie auf OK.

Das Symbol für RSS-Feeds wird in Ihrem Navigationsfenster als Ordner angezeigt. Öffnen Sie den RSS-Ordner, um den Link anzuzeigen.

Ihr RSS-Datenfeed wird automatisch alle 12 Stunden aktualisiert. Sie können den Inhalt jederzeit aktualisieren. Dazu öffnen Sie den RSS-Ordner und klicken auf Feed laden.

Hinweis:  RSS-Feeds, die Sie speichern, werden gegen Ihr Accountkontingent gewertet.
