Ansichten benutzerspezifisch einrichten

Sie können festlegen, wie Mails, Kalender und Aufgaben im Inhaltsfenster angezeigt werden.

Mailansicht ändern

Standardmäßig wird beim Öffnen Ihrer Mailbox der Posteingang angezeigt. Dies können Sie auf eine der folgenden Arten benutzerspezifisch einrichten:

  • Sie können den Ordner ändern, der beim ersten Anmelden angezeigt wird.  Standardmäßig wird der Posteingang mit allen darin enthaltenen Mails angezeigt.  Sie können jeden beliebigen Mail-Ordner für die erste Ansicht nach dem Anmelden auswählen, Sie können auch bestimmte Suchkriterien festlegen, z. B. ungelesene Mails.  Dann werden nach dem Anmelden nur die ungelesenen Mails angezeigt.  Weitere Informationen finden Sie unter  Mail-Startseite benutzerspezifisch einrichten.

  • Wählen Sie aus, ob das Lesefenster angezeigt werden soll oder nicht, und wenn ja, ob das Lesefenster unten oder rechts dargestellt werden soll. Diese Änderungen nehmen Sie über den Link "Ansicht" in der Mail-Symbolleiste vor.

  • Wenn Ihr Lesefenster ausgeschaltet ist oder im unteren Bereich angezeigt wird, können Sie die Spalten auswählen, die in der Mailliste im Inhaltsfenster dargestellt werden sollen.  Unter Umständen wollen Sie im Inhaltsfenster nicht alle verfügbaren Details zu einer Mail anzeigen. Klicken Sie mit der rechten Maustaste auf den Spaltenkopf, um ein Menü mit den Spaltenüberschriften aufzurufen. Um eine Spalte aus dem Inhaltsfenster zu entfernen, klicken Sie auf den Spaltennamen. Dieser Befehl dient zum Umschalten, wenn Sie also die Spalte wieder anzeigen möchten, klicken Sie erneut auf den Spaltennamen.

Kalenderansicht ändern

Sie können festlegen, welche Ansicht beim Öffnen des Kalenders angezeigt wird: Tag, Arbeitswoche, 7-Tage-Woche, Monat oder Liste. Sie können außerdem festlegen, mit welchem Wochentag Ihre Kalenderwoche beginnen soll. Siehe Kalenderansichten.

Aufgabenansicht ändern

Sie können auswählen, welche Spalten im Inhaltsfenster dargestellt werden, wenn Sie Ihre Aufgaben anzeigen. Klicken Sie mit der rechten Maustaste auf den Spaltenkopf, um ein Menü mit den Spaltenüberschriften aufzurufen. Um eine Spalte aus dem Inhaltsfenster zu entfernen, klicken Sie auf die Spalte.  Dieser Befehl dient zum Umschalten, wenn Sie also die Spalte wieder anzeigen möchten, klicken Sie erneut auf den Spaltennamen.

Wählen Sie aus, ob das Vorschau-Fenster angezeigt werden soll oder nicht, und wenn ja, ob das Vorschau-Fenster unten oder rechts dargestellt werden soll. Diese Änderungen nehmen Sie über den Link „Ansicht“ in der Symbolleiste vor.

Ansicht "Dateien" ändern

Sie können auswählen, welche Spalten im Inhaltsfenster dargestellt werden, wenn Sie Ihre "Dateien" anzeigen. Klicken Sie mit der rechten Maustaste auf den Spaltenkopf des Ordners "Dateien", um ein Menü mit den Spaltenüberschriften aufzurufen. Um eine Spalte aus dem Inhaltsfenster zu entfernen, klicken Sie auf die Spalte.  Dieser Befehl dient zum Umschalten, wenn Sie also die Spalte wieder anzeigen möchten, klicken Sie erneut auf den Spaltennamen.

Wählen Sie aus, ob das Vorschau-Fenster angezeigt werden soll oder nicht, und wenn ja, ob das Vorschau-Fenster unten oder rechts dargestellt werden soll. Diese Änderungen nehmen Sie über den Link "Ansicht" in der Symbolleiste vor.
