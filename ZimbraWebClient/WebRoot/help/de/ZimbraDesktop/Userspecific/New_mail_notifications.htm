Mailbenachrichtigung an eine andere Adresse

Hinweis: Nur für Zimbra-Accounts verfügbar.

Sie können Ihre Mailoptionen so einstellen, dass Sie an einer anderen Adresse informiert werden, wenn Sie neue Mails in Ihrer ZWC-Mailbox empfangen.  Diese Funktion ist besonders nützlich, wenn Sie nur wenige Mails in Ihrem Zimbra-Account empfangen, auf die Sie dann jedoch zeitnah reagieren möchten.

So richten Sie die Benachrichtigung ein

  1. Klicken Sie auf die Registerkarte „Einstellungen“.

  2. Klicken Sie auf Mail.

  3. Aktivieren Sie unter „Mails empfangen“ die Option (Eine Mail-)Benachrichtigung senden an und geben Sie die gewünschte Mail-Adresse ein.

  4. Klicken Sie auf Speichern. Die Benachrichtigungsfunktion wird sofort aktiviert.
