Signatur automatisch einfügen

Sie können verschiedene Signaturen erstellen und sie den jeweiligen Adressen, die Sie angelegt haben, zuweisen. Siehe Mail-Identitäten.

Wenn Sie Ihre Signatur in HTML formatieren, können Sie Ihren Firmennamen mit der Webseite verknüpfen und Ihrer Signatur Bilder, wie beispielsweise ein Logo, hinzufügen.

So legen Sie eine Signatur an

  1. Öffnen Sie die Registerkarte Einstellungen und dann Signaturen.

  2. Klicken Sie auf Signatur hinzufügen und geben Sie den Namen für diese Signatur in das Feld Signaturname ein.

  3. Klicken Sie auf Als HTML formatieren, um Ihre Signatur anzupassen, wenn Sie einen Signaturstil erstellen wollen.

  4. Geben Sie im Textfeld Signatur Ihre Signatur genau so ein, wie sie in Ihren Mails angezeigt werden soll. In HTML können Sie beim Text unter Schriftarten, -größen und -farben wählen und Ihrer Signatur Verknüpfungen und Bilder hinzufügen. Bilder können Sie entweder herunterladen oder Sie können eine Verknüpfung zu einer Bild-URL herstellen. 

  5. Wählen Sie unter Signaturen verwenden aus, an welcher Stelle die Signatur in Ihre Mails eingefügt werden soll. Wählen Sie Über beigefügten Mails, um Ihre Signatur am Ende des von Ihnen verfassten, beantworteten oder weitergeleiteten Texts einzufügen. Wählen Sie Unter beigefügten Mails, um die Signatur ganz am Ende der Mail einzufügen.

  6. Klicken Sie auf Speichern.

  7. Um diese Signatur Ihrem Accountnamen zuzuweisen, gehen Sie zum Ordner Persönlichkeiten . Wählen Sie dort den Namen der Account-Persönlichkeit im Feld Signatur aus. Wählen Sie anschließend den Signaturnamen aus der Dropdown-Liste aus.
