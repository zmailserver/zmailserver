Kontaktgruppen anlegen

Mit der Kontaktgruppen-Funktion können Kontaktlisten mit mehreren Mail-Adressen angelegt werden. Wenn Sie beim Verfassen einer Mail eine Kontaktgruppe auswählen, werden alle Kontakte, deren Adressen sich in der Liste der Gruppe befinden, automatisch ins Adressfeld der Mail eingefügt.

So legen Sie Kontaktgruppen an

Sie können Adressen im Firmenverzeichnis (globale Adressliste, GAL) und in Ihren Adressbüchern auswählen, und Sie können Adressen von Hand eingeben. Hinweis: Von Hand eingegebene Mail-Adressen werden nicht überprüft.  

Hinweis: Außer für Zimbra-Accounts werden Gruppenlisten, die Ihren Zimbra Desktop-Kontaktlisten hinzugefügt werden, nicht wieder mit Ihren Mailbox-Accounts abgeglichen.

  1. Klicken Sie in der Symbolleiste auf Neu und wählen Sie Neue Kontaktgruppe. Daraufhin wird das Formular Neue Kontaktgruppe angezeigt.

Wenn Sie die Kontaktgruppe in einem anderen Adressbuch als "Kontakte" anlegen möchten, wählen Sie zunächst das gewünschte Adressbuch aus und klicken dann auf Neu > Verteiler.

  1. Hinweis: Nachdem Sie eine Kontaktgruppe angelegt haben, können Sie sie in eine andere Liste verschieben.

  1. Geben Sie im Feld Gruppenname einen Namen für diese Gruppenliste ein.  

  2. Um Mitglieder zu dieser Gruppe hinzuzufügen, geben Sie im Feld Suchen einen Namen ein. Wählen Sie dann aus, ob in der globalen Adressliste oder in Ihren persönlichen Kontakten gesucht werden soll. Klicken Sie auf Suchen.

  3. Namen, die Ihrem Eintrag entsprechen, werden im Feld darunter angezeigt. Wählen Sie die gewünschten Namen aus und klicken Sie auf Hinzufügen. Die Namen werden in die Liste der Gruppenmitglieder aufgenommen.

Sie können die Mail-Adressen auch direkt in das Listenfeld Gruppenmitglieder eingeben. Eingegebene Adressen trennen Sie durch ein Komma (,), Semikolon (;) oder mithilfe der Eingabetaste, um eine neue Zeile zu beginnen.

  1. Klicken Sie in der Symbolleiste auf Speichern. Die neue Kontaktgruppe wird zu Ihrem Adressbuch hinzugefügt.

Das Symbol zeigt, dass es sich bei dem Kontaktnamen um eine Kontaktgruppe handelt.
