Neue Kontakte hinzufügen

Sie können neue Kontakte hinzufügen, indem Sie in der Symbolleiste auf den Pfeil von klicken oder indem Sie mit der rechten Maustaste auf einen Namen in der Kopfzeile einer Mail klicken und Kontakte auswählen.

Zur Erstellung eines Kontakts können Sie einfach einen Namen oder aber detaillierte Informationen zu Ihren Kontakten hinzufügen, einschließlich mehrerer Mail-Adressen, Telefonnummern, Mail-Adressen sowie einem Bild.

So fügen Sie einen neuen Kontakt hinzu

  1. Klicken Sie in der Symbolleiste auf den Pfeil neben Neu und wählen Sie Kontakt aus. Das Kontaktformular wird geöffnet.

  2. Geben Sie den Kontaktnamen ein. Klicken Sie auf , um weitere Namen hinzuzufügen, wie beispielsweise einen zweiten Vornamen oder den Abteilungsnamen.  

  3. Geben Sie weitere Kontaktdaten ein und laden Sie ein Foto dieses Kontakts hoch. Die Bilddatei muss von Ihrem Computer aus aufrufbar sein.

    • Das Zeichen kennzeichnet Felder, die mehrere Einträge haben können, wobei jeder Eintrag durch eine spezielle Kennzeichnung identifiziert werden kann. Die Telefonnummer kann beispielsweise als Mobiltelefon, Geschäftlich, Privat oder Fax gekennzeichnet sein.

    • Über das Feld Sonstiges können Sie benutzerdefinierte Felder eingeben. Es sind zwei benutzerdefinierte Felder identifiziert, „Geburtstag“ und „Jahrestag“. Sie können weitere benutzerdefinierte Felder anlegen.

      • Wenn "Geburtstag" angezeigt wird, klicken Sie in das Menü. Wählen Sie Benutzerdefiniert, um ein benutzerdefiniertes Feld hinzuzufügen.

      • Ändern Sie "Benutzerdefiniert" in einen aussagekräftigen Titel für dieses Feld, und geben Sie die Informationen in das Feld Text eingeben ein.

  1. Wählen Sie unter Speichern als (rechts) aus, wie der Name in Ihrem Adressbuch angezeigt werden soll. Standardmäßig wird der Kontakt nach dem Nachnamen oder Vornamen gespeichert, Sie können ihn jedoch nach Vornamen, Nachnamen oder Unternehmen speichern.

  2. Wählen Sie unter Ort (rechts) das Adressbuch aus, in dem der Name gespeichert werden soll.

  3. Klicken Sie auf Speichern.

Hinweis: Nur die Zimbra-Accountnamen, die Ihren Zimbra Desktop-Kontaktlisten hinzugefügt werden, werden wieder mit Ihrem Account synchronisiert.

 

So übernehmen Sie einen neuen Kontakt aus einer Mail

  1. Öffnen Sie die Mail. Die Kopfzeile wird am oberen Rand der Maildetails grau angezeigt. Sie können Mail-Adressen aus den Feldern „Von:“, „An:“, „CC:“ und „BCC:“ übernehmen.

  2. Klicken Sie mit der rechten Maustaste auf den Namen, den Sie in Ihr Adressbuch übernehmen möchten, und wählen Sie Zu Kontakten hinzufügen.

  3. Daraufhin wird das Kontaktformular geöffnet. Die aus dem Mailkopf übernommenen Daten werden bereits angezeigt. Ergänzen Sie ggf. weitere Informationen und prüfen Sie auch die übernommenen Felder auf ihre Richtigkeit.

  4. Wählen Sie unter Speichern als aus, wie der Name angezeigt werden soll. Standardmäßig wird der Kontakt in der Form "Nachname, Vorname" angezeigt.

  5. Wählen Sie unter Ort das Adressbuch aus, in dem der Name gespeichert werden soll.

  6. Klicken Sie auf Speichern.

 

Sie können auch Kontaktgruppen anlegen.
