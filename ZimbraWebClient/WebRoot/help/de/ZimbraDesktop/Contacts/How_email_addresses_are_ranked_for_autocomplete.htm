Rangfolge von Mail-Adressen für die Autovervollständigung

Wenn Sie die Autovervollständigung für die Suche von Mail-Adressen verwenden, werden passende Namen in einer "Rangfolge" angezeigt.  Kontakte, an die Sie Mails gesendet haben, haben eine höhere Priorität als Kontakte, denen Sie keine oder weniger häufig Mails gesendet haben.

Das bedeutet, dass die am häufigsten angeschriebenen Kontakte ganz oben in der Liste aufgeführt werden.

Wenn Sie nicht möchten, dass der Kontaktname immer ganz oben angezeigt wird, können Sie auf Vergessen klicken. Der Kontakt wird an einer niedrigeren Position eingeordnet.
