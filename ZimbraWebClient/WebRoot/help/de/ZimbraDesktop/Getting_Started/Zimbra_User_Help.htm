Wissenswertes zu Zimbra Desktop

Zimbra Desktop ist ein Offline-Client, mit dem Sie Ihre verschiedenen Mail-Accounts in einem Desktop-Tool verwalten können. Zimbra Mail, Yahoo! Mail, Gmail, Outlook und andere geschäftliche oder private Mail-Accounts, die POP oder IMAP verwenden, lassen sich problemlos in Zimbra Desktop einrichten.  Jeder Account wird getrennt gepflegt.

Nachdem Sie Ihre Accounts eingerichtet haben, werden Ihre Mails auf Ihren Computer geladen und sind unabhängig davon, ob eine Verbindung zum Internet besteht oder nicht, jederzeit abrufbar. Siehe hierzu Offline arbeiten. Die Mails all Ihrer Accounts können in Zimbra Desktop gelesen, beantwortet, weitergeleitet und gelöscht werden. Der Mailspeicher richtet sich nach der Festplattengröße Ihres Computers und nicht nach einer festgelegten Account-Kapazität.

Wenn Sie über einen Ihrer Accounts neue Mails erhalten und eine Internetverbindung besteht, werden die Mails automatisch in Zimbra Desktop geladen. Zu diesem Zweck braucht Zimbra Desktop nicht geöffnet zu sein. Im Infobereich unter Windows oder im Dock unter Mac wird ein Briefumschlagsymbol und die Gesamtzahl neuer Mails in allen Ihren Accounts angezeigt. Klicken Sie auf das Symbol, um die Desktop-Anwendung zu öffnen.

Wesentliche Funktionen in Zimbra Desktop

In Zimbra Desktop haben Sie nicht nur zentralen Zugriff auf Ihre verschiedenen Mail-Accounts, sondern können auch Ihre Kontakte verwalten und einen Kalender oder Aufgabenlisten nutzen sowie Ihre Anhänge im Ordner "Dateien" speichern.

Eine Liste der wesentlichen Funktionen finden Sie unter Übersicht über die Funktionen von Zimbra Desktop.

Arten von Accounts, die Sie einrichten und verwalten können

Folgende Mail-Accounts können eingerichtet werden.

  • Zimbra-Account. Mit Zimbra Desktop können Sie auf Ihren Mail-Account zugreifen, wenn dieser unter Zimbra ausgeführt wird. Sie haben Zugriff auf Mails, Ordner, Adressbücher, Kalender, Aufgabenlisten und die Dokumente im Ordner "Dateien". Greifen Sie über Zimbra Web Client über einen Browser auf einen Zimbra-Account zu. Einstellungen, die Sie in ZWC für Ihren Zimbra-Account festlegen, sind über den Zimbra-Desktop verfügbar. Hinweis: Um alle Zimbra-Funktionen über Zimbra Desktop aufrufen zu können, sollte es sich bei Zimbra um die Version ZCS ab 5.0 handeln. Ihr Zimbra-Systemadministrator kann Ihnen die Zimbra-Versionsnummer mitteilen.

  • Yahoo! Mail. Sie können Yahoo! Mail-, Yahoo! Mail Plus-, Yahoo! Small Business-, Ymail- oder Rocketmail-Accounts einrichten. Ihre Mails, Ihr Mail-Ordner und Ihr Adressbuch stehen in Zimbra Desktop zur Verfügung.

  • Gmail. Ihr Gmail-Account muss so eingerichtet sein, dass der IMAP-Zugriff möglich ist. Möglicherweise müssen Sie sich bei Ihrem Gmail-Account anmelden, um IMAP zu aktivieren. Ihre Mails und Kontakte stehen in Zimbra Desktop zur Verfügung.

  • Microsoft Exchange-IMAP-Account. Die Mails Ihres Exchange-IMAP-Accounts stehen in Zimbra Desktop zur Verfügung.

Wenn Sie einen anderen als die genannten Accounts nutzen, können Sie diesen trotzdem in Zimbra Desktop einrichten, sofern es sich um einen POP3- oder einen IMAP4-Account handelt. Sie benötigen dann die vollständigen Einstellungsdaten für Ihren Account einschließlich der Angaben zu den Servern für eingehende bzw. ausgehende Mails. Sie erhalten diese Informationen bei Ihrem Systemadministrator oder Internetanbieter.
