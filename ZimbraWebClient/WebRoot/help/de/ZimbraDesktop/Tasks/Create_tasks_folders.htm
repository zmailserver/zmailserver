Aufgabenlisten anlegen

Sie können verschiedene Aufgabenlisten anlegen und konfigurieren, um sie mit Aufgabenlisten von Remote-Diensten, wie der To Do-Liste von iCal, zu synchronisieren.

So legen Sie eine Aufgabenliste an:

  1. Klicken Sie auf die Registerkarte Aufgaben, um die Aufgaben-Anwendung zu öffnen.

  2. Klicken Sie im Übersichtsfenster in die Kopfzeile „Aufgaben“ und wählen Sie Neue Aufgabenliste aus.

  3. Geben Sie unter Neue Aufgabenliste erstellen einen Namen für die Aufgabenliste ein und wählen Sie eine Farbe zum Anzeigen der Aufgaben aus.

  4. Aktivieren Sie Aufgaben vom Remote-Dienst synchronisieren, um die neue Liste mit einer To Do-Liste von iCal zu synchronisieren. Geben Sie die iCal-URL ein.

  5. Klicken Sie auf OK. Die Liste wird im Übersichtsfenster unter "Aufgaben" hinzugefügt.
