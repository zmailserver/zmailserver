Ihre Accounts sichern

Sie können Ihre Zimbra Desktop- Accounts auf der Seite Einstellungen>Sicherungen im Bereich "Alle Accounts" sichern.  Standardmäßig werden nur die lokalen Ordner gesichert, da diese  Elemente enthalten, die auf anderen Mail-Servern nicht gesichert werden.  

So sichern Sie Ihre Accounts

  1. Wechseln Sie zur Registerkarte Einstellungen>Sicherungen.

  2. Wählen Sie im Bereich Sicherungen konfigurieren die Accounts aus, die gesichert werden sollen. Denken Sie daran, dass Elemente in Ihren lokalen Ordnern und in Ihren POP-Account-Ordnern auf Ihrem Computer gespeichert werden. Es ist ratsam, diese Accounts zu sichern. Sie können all die Accounts, die Sie in Zimbra Desktop erstellt haben, sichern. Beachten Sie jedoch, dass die Dateien ziemlich groß sein können. Um herauszufinden, wie viel Festplattenspeicher Sie benötigen, bewegen Sie den Mauszeiger über den Account-Namen. Die Größe der Mailbox wird angezeigt.

  3. Wenn Sie die Sicherung planen möchten, wählen Sie das Zeitintervall für die Sicherung aus: Jeden Tag, Jede Woche, Jeden Monat.

  4. Geben Sie in das Feld Ort den Pfad zu dem Verzeichnis ein, in dem die Sicherung gespeichert werden soll. Beispiel: Um Ihren Ordner ZD_backups auf einem externen Datenträger E zu speichern, geben Sie E:\ZD_backups ein.

  5. Wählen Sie im Feld Retention die Anzahl an Sicherungen aus, die Sie speichern möchten. Standardmäßig wird nur eine (1) Sicherungsdatei gespeichert. Sie können wählen, bis zu fünf (5) Sicherungen zu speichern, bevor die älteste Sicherung überschrieben wird.

Wenn Sie Ihre Sicherung so planen, dass sie automatisch ausgeführt wird, erfolgt die Sicherung gemäß dem von Ihnen festgelegten Zeitplan. Zimbra Desktop muss geöffnet sein, damit Sicherungen vorgenommen werden können. Wenn eine Sicherungssitzung für einen Zeitpunkt geplant wurde, an dem Sie nicht bei Zimbra Desktop angemeldet waren, startet die Sicherung, sobald Sie sich das nächste Mal anmelden.

Wenn Sie Ihre Accounts sofort sichern möchten, oder wenn Ihr Zeitplan auf  Nie festgelegt ist, klicken Sie auf Konten jetzt sichern, um eine Sicherungssitzung auszuführen. Je nach Größe der Accounts, die gesichert werden sollen, kann dies eine Weile dauern.  Schließen Sie Zimbra Desktop  nicht, während die Sicherung ausgeführt wird.Sie können weiterhin in Zimbra Desktop arbeiten.

Hinweis: Die Entscheidung, Ihre Accounts jetzt zu sichern, hat keine Auswirkungen auf Ihren nächsten geplanten Sicherungstermin.

Informationen zur Wiederherstellung Ihres Accounts finden Sie unter Accounts aus Ihren Backup-Dateien wiederherstellen.
