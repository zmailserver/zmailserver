Automatische Vervollständigung von Adressen

Die Funktion Autovervollständigung zeigt Namensvorschläge an, sobald Sie beginnen, in die Felder An, CC oder Bcc einer neuen Mail Namen einzugeben. Es wird eine Liste möglicher Adressen aus Ihrem Adressbuch oder aus der "Globalen Adressliste" angezeigt, die mit dem Text, den Sie eingeben, beginnt.

Je mehr Text Sie in das Feld An: eingeben, desto kleiner wird die Anzahl der angezeigten Adressen, weil immer weniger Einträge mit dem von Ihnen eingegebenen Text übereinstimmen. Die Liste wird nicht mehr angezeigt, wenn es keine Entsprechungen mehr gibt.

Beispiel

Angenommen, Sie möchten eine Mail an David Brinker senden, dessen Mail-Adresse dbrinker@acme.com lautet. Sobald Sie den ersten Buchstaben "d" in das Adressfeld eingeben, wird eine Liste mit allen möglichen Entsprechungen angezeigt, wobei die übereinstimmenden Teile hervorgehoben sind. In diesem Fall wird Folgendes angezeigt:

Dieter Ehrmann < dehr@acme.com>

David Brinker < dbrinker@acme.com>

Erica Diehl <erica878@xyz.com>

Wenn Sie weitere Zeichen in das Feld An: eingeben, wird die Liste der Entsprechungen kleiner.

Übereinstimmung

Die Funktion "Autovervollständigung" vergleicht den Text, den Sie eingeben, mit den folgenden Teilen eines Kontakts:

  •  Vorname

  •  Nachname

  • Mail-Adresse

Die Suche nach Entsprechungen beginnt immer am Anfang des Felds. Der Text "r;hart" entspricht "r;hartmut müller", aber nicht "r;eckhart müller".

Autovervollständigung nutzen

Wenn der von Ihnen eingegebene Text einem oder mehreren Ihrer Kontakte entspricht, werden die Entsprechungen in einer Liste unterhalb des Felds angezeigt, in das Sie gerade Text eingeben. Standardmäßig ist die erste Entsprechung hervorgehoben.

Die Namen werden in einer "Rangordnung" angezeigt. Kontakte, denen Sie bereits Mails gesendet haben, werden vor Kontakten angezeigt, denen Sie noch keine oder nicht so häufig Mails gesendet haben. Das bedeutet, dass die am häufigsten angeschriebenen Kontakte ganz oben in der Liste aufgeführt werden. Wenn Sie nicht wollen, dass der Kontaktname immer ganz oben angezeigt wird, können Sie auf Vergessen klicken. Der Kontakt wird an einer niedrigeren Position eingeordnet.

Mit bestimmten Tasten können Sie bestimmte Funktionen auslösen. Auch mit der Maus können Sie eine Entsprechung auswählen.

  • Durch Eingabe von einem Komma oder Semikolon bzw. durch Drücken der Eingabetaste oder der Tab-Taste wird der Name vervollständigt. Der von Ihnen eingegebene Text wird durch die hervorgehobene Entsprechung ersetzt.

  • Über die ESC-Taste blenden Sie die Liste aus.

  • Mit den Pfeiltasten auf/ab ändern Sie die Auswahl in der Liste. Auch mit dem Mauszeiger können Sie die Auswahl in der Liste ändern.

  • Wenn Sie mit der Maus auf eine Entsprechung klicken, wird dieser Name in das Eingabefeld übernommen.

Dadurch, dass eine einzige Taste (z. B. ein Semikolon) die Vervollständigung eines Namens bewirkt, können Sie Adressen schnell eingeben. Wenn Sie einige Zeichen kennen, durch die ein bestimmter Kontakt an die erste Stelle gelangt, können Sie diese Zeichen und ein Semikolon eingeben und schon ist die Adresse des Kontakts eingegeben. Wenn Sie z. B. wissen, dass die Eingabe von "r; dbb" dazu führt, dass David Brinker an erster Stelle angezeigt wird,  geben Sie "r; dbb;" ein und die vollständige Adresse "r;David Brinker < dbrinker@acme.com>" erscheint im Adressfeld.

Hinweis: Sie können einstellen, wie die Funktion "Autovervollständigung" auf der Seite Einstellungen, Adressbuch ausgeführt wird.

Die Autovervollständigung ist auch in anderen Anwendungen verfügbar (z. B. beim Hinzufügen von Teilnehmern zu einem Termin im Kalender), wo sie auf die gleiche Weise funktioniert.
