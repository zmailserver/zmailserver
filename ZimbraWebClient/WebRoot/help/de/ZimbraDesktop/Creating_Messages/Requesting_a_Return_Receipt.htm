Lesebestätigung anfordern

Hinweis: In Zimbra Desktop ist die Option "Lesebestätigung anfordern" nur verfügbar, wenn mit dem Zimbra Web Client gearbeitet wird.

Sie können eine von Ihnen geschriebene Mail kennzeichnen, damit Ihnen eine Lesebestätigung gesendet wird, wenn der Empfänger Ihre Mail öffnet.  Beim Verfassen einer Mail wählen Sie Lesebestätigung anfordern im Menü Optionen der Symbolleiste "Verfassen" aus. Der Empfänger kann auswählen, ob die Lesebestätigung gesendet wird oder nicht.

Im Bereich "Einstellungen>Lesebestätigung der Mail" können die Benutzer ihre Einstellungen für Mails festlegen, die mit der Option "Lesebestätigung anfordern" gekennzeichnet sind. Folgende Optionen sind möglich:

  • Nie Lesebestätigungen senden. Dies ist die Standardeinstellung. Wenn eine Mail empfangen wird, für die eine Lesebestätigung gesendet werden soll, wird diese Anforderung ignoriert. Der Empfänger wird nicht über die Anforderung informiert.

  • Immer Lesebestätigungen senden. Beim Öffnen einer Mail, für die eine Lesebestätigung gesendet werden soll, wird die Bestätigung automatisch gesendet.

  • Nachfragen.  Die Empfänger werden gefragt, ob die Lesebestätigung gesendet werden soll.

Es wird keine Lesebestätigung gesendet, wenn ein Empfänger mehrere Mails auswählt, mit der rechten Maustaste klickt und Als gelesen markieren auswählt.

 

Hinweis: Die Anforderung der Lesebestätigung funktioniert nicht bei Mails, die an Nutzer gesendet werden, die statt mit ZWC mit anderen Systemen arbeiten.
