Unzustellbare Mail

Gelegentlich ist eine Mail "unzustellbar", d. h. sie kann dem Empfänger aus einem bestimmten Grund nicht zugestellt werden. Folgende Gründe sind möglich:

  • Die Adresse des Empfängers ist ungültig oder existiert nicht. Prüfen Sie, ob Sie die richtige Mail-Adresse eingegeben haben.

  • Das Mail-System des Empfängers weist die Mail aus einem bestimmten Grund zurück.

  • Die Mailbox des Empfängers ist voll.

Wenn eine Mail abgewiesen wird, erhalten Sie eine Mailbenachrichtigung über die Unzustellbarkeit der Mail. Diese Benachrichtigung enthält zumindest einen Teil des Inhalts Ihrer ursprünglichen Mail sowie einige vom System generierte Fehlermeldungen mit weiteren Angaben, weshalb die Mail nicht zustellbar ist. Versuchen Sie, das Problem zu beheben, und senden Sie die Mail erneut.

Wenn die Mail wegen einer angeblich fehlerhaften Mail-Adresse nicht zugestellt wird, Sie sich jedoch sicher sind, dass die Adresse gültig ist, müssen Sie sich auf andere Art an den Empfänger wenden, damit dieser das Problem mithilfe des Systemadministrators beheben kann.

Wenden Sie sich in folgenden Fällen an Ihren Systemadministrator:

  • Keine Ihrer Mails, die Sie senden, kann zugestellt werden.

  • Mails, die Sie an Adressen aus Ihrem Firmenadressbuch senden, kommen mit Fehlermeldungen wie "Unbekannter Nutzer" oder "Fehlerhafte Adresse" zurück.

  • Sie erhalten Unzustellbarkeitsmeldungen für Mails, die Sie nie gesendet haben.
