Selecting Options to Migrate
When you migrate account files, only email messages, calendar and contacts, tasks, rules, and out of office settings are imported. You can also select to migrate your sent, deleted (trash), and junk (spam) folders. Other types of files, such as meeting requests, notes/journal, alerts, personal distribution lists, are not imported.
The Options dialog is used to set rules about what files should be imported, and setting filters such as migrating after a specified date, maximum message size, and if you want to skip folders.
1.
In Options, select the user account data to migrate by choosing the areas of to migrate.
•
Items: Select to migrate email messages and attachments, calendar data, contact lists, task lists, rules and out of office settings.
•
Additional Folders: Select whether to migrate mail currently in the Sent, Deleted Items (Trash), and Junk (Spam) folders.
•
Filters: Select how to filter the information you are migrating, including migration dates, maximum message size, and which folders in the account to skip. The default maximum message size is 0.
•
Verbose Logging. Select to track changes and settings.This allows extra data to be created if the migration fails. If a message is not migrated and this option is enabled, you can inspect the log to determine the cause of failure.
