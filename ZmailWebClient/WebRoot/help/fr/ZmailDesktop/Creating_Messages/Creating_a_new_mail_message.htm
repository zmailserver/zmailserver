À propos de la création de mails

Vous pouvez rédiger et envoyer vos mails juste après les avoir écrits, ou vous pouvez rédiger un brouillon et y revenir plus tard pour le terminer et l'envoyer.

La première étape pour rédiger un mail consiste à cliquer sur dans la barre d'outils pour ouvrir une nouvelle page. Vous pouvez également cliquer avec le bouton droit de la soruis sur le nom dans la section De d'un mail, puis sélectionner Nouveau mail pour ouvrir une nouvelle page.

Selon vos Préférences de messagerie, l'onglet Écrire apparaît dans la barre d'outils de l'Application et la nouvelle page s'ouvre ou une nouvelle page peut s'ouvrir. Vous pouvez disposer de plusieurs onglets Écrire ou naviguer entre les différents onglets dans la barre d'outils.

Sur la nouvelle page, saisissez l'adresse mail de la personne ou des personnes à qui vous envoyez le mail dans le champ De : .

  • Vous pouvez rechercher l'adresse mail d'une personne en cliquant sur À afin d'effectuer la recherche dans vos contacts ou dans la liste d'adresses globale, le cas échéant.

  • Pour ajouter des adresses Cci, cliquez sur le champ Afficher Cci près de Cc : dans l'en-tête du message.

Si vous avez configuré plusieurs avatars, le champ De s'affiche dans la nouvelle page. Sélectionnez l'avatar à utiliser en tant qu'adresse d'expéditeur.

Vous pouvez rédiger votre message dans le format HTML ou le format texte simple. Le format par défaut est configuré dans vos Préférences >générales. Pour modifier rapidement le format de ce message, cliquez surOptionsdans la barre d'outils de nouveau mail, puis choisissez le format HTML ou texte simple.

  • Format HTML vous permet de formater votre message avec différents styles des caractères, de créer des tableaux, d'ajouter des couleurs et d'insérer des liens hypertextes.

  • Le Format texte simple génère un texte sans style ni formatage. Ce type de message est lisible sur n'importe quel ordinateur.

Lorsque vous travaillez dans vos comptes Zimbra, vous pouvez demander un accusé de réception. Cliquez sur Options dans la barre d’outils de rédaction, puis sélectionnez Demander un accusé de réception.

Vous pouvez joindre des fichiers, y compris des images, des documents, des feuilles de calcul et des fichiers audio à votre mail.
