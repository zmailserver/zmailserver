Enregistrement de mails sous forme de brouillons

Vous pouvez enregistrer votre message sous la forme d'un brouillon à terminer et envoyer ultérieurement.  Les messages sont enregistrés dans votre dossier Brouillons jusqu'à ce que vous décidiez de les envoyer ou de les supprimer de ce dossier.

Vous pouvez activer une préférence pour enregistrer automatiquement votre mail come brouillon pendant que vous le rédigez. Vous pouvez également cliquer sur Enregistrer le brouillon en cours de création pour enregistrer le mail comme brouillon.

Configuration des préférences pour enregistrer automatiquement les brouillons

Allez dans l'onglet Préférences>de messagerie, puis activez la fonction Enregistrer automatiquement le brouillon des mails en cours de création. Les mails que vous rédigez sont automatiquement enregistrés dans des brouillons après quelques minutes.

Pour enregistrer manuellement un mail :

  1. Rédigez votre mail comme décrit dans la section Création d'un mail.

  2. Cliquez sur Enregistrer le brouillon.

Le brouillon du mail est enregistré dans votre dossier Brouillons.

Pour récupérer un brouillon

  • Ouvrez le dossier Brouillons, cliquez sur le message, ou sélectionnez sur le message, puis cliquez sur Modifier dans la barre d'outils.

Vous pouvez rédiger et envoyer le message terminé de la même manière qu'un mail normal. Après son envoi, le message est supprimé du dossier Brouillons.
