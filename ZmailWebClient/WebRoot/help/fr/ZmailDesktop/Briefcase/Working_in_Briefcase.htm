Travailler dans un porte-documents

Le porte-documents peut être utilisé pour partager et gérer les documents que vous créez dans votre porte-documents, ainsi que les documents et les images que vous y placez. Le porte-documents vous permet de sauvegarder des fichiers dans votre compte pour que vous puissiez accéder à ces fichiers lorsque que vous vous connectez à votre compte depuis n'importe quel ordinateur.

Vous pouvez uploader des documents, des feuilles de calculs, des présentations, des images et des fichiers PDF à partir de votre ordinateur ou de votre réseau et sauvegarder dans votre porte-documents les pièces jointes envoyées avec vos mails.  Vous pouvez créer plusieurs dossiers de porte-documents afin d'organiser les fichiers. .

Les fichiers que vous uploadez vers le porte-documents sont des copies des fichiers d'origine stockés sur votre ordinateur.  Les modifications apportées aux fichiers dans votre porte-documents ne modifient pas les fichiers d'origine.

Création de dossier de porte-documents

  1. Activez l'onglet Porte-documents.

  2. Sélectionnez Porte-documents dans la Nouvelle liste  de la barre d'outils.

  1. La boîte de dialogue Créer un porte-documents s'affiche. Saisissez le nom du nouveau dossier Porte-documents.

Ce nom est doit être unique dans la hiérarchie des dossiers de votre boîte aux lettres. Les dossiers Porte-documents ne peuvent pas porter le même nom qu'un dossier de niveau supérieur dans votre mail Zimbra, calendrier ou carnet d'adresse. Par exemple, si vous disposez d'un calendrier intitulé Vacances, vous ne pouvez pas nommer un porte-documents de niveau supérieur Vacances. Vacances peut être le nom d'un porte-documents dans un autre porte-documents.

  1. Sélectionnez la  hiérarchie des dossiers. Vous pouvez créer un porte-documents de niveau supérieur, ou vous pouvez l'insérer dans un porte-documents existant.

  2. Cliquez sur OK. Le nouveau porte-documents s'affiche dans le panneau d'aperçu.

Uploader des fichiers

Quand vous uploadez un fichier, vous transférez un fichier de votre ordinateur vers votre compte. Cela permet de rendre le fichier disponible à chaque fois que vous vous connectez à votre compte. Le quota de votre compte détermine le nombre de fichiers pouvant être uploadés vers votre porte-documents.

Pour uploader des fichiers

  1. Ouvrez Porte-documents, puis cliquez sur Uploader fichier dans la barre d'outils

  2. Dans la boîte de dialogue Uploader un nouveau fichier dans le porte-documents, cliquez sur Parcourir pour trouver les fichiers à uploader. Vous pouvez sélectionner plusieurs fichiers à uploader une fois que les fichiers sont dans le même dossier. Pour ajouter d'autres fichiers, cliquez sur Ajouter et recherchez le fichier.

Lorsque vous uploadez un fichier portant le même nom qu'un fichier existant, une boîte de dialogue s'affiche."

    • Si vous cliquez sur « Non », le fichier ne sera pas uploadé. .

  1. Lorsque tous les fichiers sont uploadés, cliquez sur OK. Les fichiers s'affichent dans le porte-documents.  

Afficher le contenu dans les panneaux d'aperçu

Vous pouvez sélectionner le mode d'affichage de vos fichiers de vos dossiers du porte-documents dans un panneau d'aperçu.

  • Panneau d'aperçu en bas/Panneau d'aperçu à droite. Tous les fichiers au format reconnu sont affichés dans le panneau d'aperçu. Vous pouvez désactiver le panneau d'aperçu.

Partager votre porte-documents

Les dossiers de Porte documents peuvent être partagés avec d'autres utilisateurs dans votre société et en dehors. Vous pouvez autoriser d'autres utilisateurs  à créer et à modifier le contenu d'un porte-documents ou simplement consulter les pages. Voir Partager vos dossiers Zimbra
