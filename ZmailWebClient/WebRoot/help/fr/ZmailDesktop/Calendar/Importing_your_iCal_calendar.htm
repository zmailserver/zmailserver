Importation de votre calendrier iCal

Vous pouvez importer les calendriers iCalendars (iCal) enregistrés au format .ics. iCalendars est le standard pour les échanges des données de calendrier via Internet, et .ics est le format standard des informations iCalendar. La fonction d'importation est disponible dans votre onglet Préférences Import/Export.

Vous pouvez importer votre calendrier iCal dans un calendrier existant ou créer un calendrier et importer le fichier .ics dans le nouveau calendrier.

Pour importer un calendrier iCal :

  1. Sélectionnez Préférences, puis cliquez sur le fichier Importer/Exporter

  2. Dans le champ Fichier, retrouvez le fichier .ics à importer.

  3. Dans le champ Destination, cliquez sur Calendriers et sélectionnez le dossier pour importer le calendrier ; sélectionnez un de vos calendriers existants ou cliquez sur Nouveau pour en créer un.

  4. Cliquez sur Importer.  

  5. Cliquez sur OK.

Les calendriers iCal sont mis à jour à régulièrement en fonction de la fréquence d'interrogation de calendrier à distance définie par l'administrateur système.
