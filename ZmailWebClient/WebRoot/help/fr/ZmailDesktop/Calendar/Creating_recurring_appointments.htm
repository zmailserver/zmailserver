Création de rendez-vous récurrents

Les rendez-vous récurrents sont des rendez-vous qui se répètent selon une certaine forme de planification. Vous pouvez programmer des réunions récurrentes pour qu'elles se répètent de manière quotidienne, hebdomadaire, mensuelle ou annuelle. De plus, vous pouvez personnaliser des planifications de rendez-vous récurrents.

  1. À partir de l'un des affichages (à l'exception de la vue Préférences), cliquez sur la flèche de la barre d'outils, puis sélectionnez Rendez-vous.

Pour programmer un rendez-vous d'une autre manière, cliquez ici.

  1. Le champ De s'affiche, vous avez créé plus d'un avatar pour votre compte. Sélectionnez l'identité à utiliser lors de la création de ce rendez-vous.

  2. Le champ Objet doit être renseigné. Saisir une description. Le rendez-vous s'affiche dans le calendrier comme cette description.

  3. Saisissez le nom des participants. Lorsque vous saisissez les noms, le mini-calendrier dans le panneau d'aperçu affiche les horaires suggérés auxquels tous les participants sont disponibles et les emplacements disponibles en vert.  Vous pouvez saisir le nom des participants de l'une des manières suivantes :

    • Dans le champ Participants, saisissez les adresses mail, en les séparant par un point-virgule (;). Lorsque vous commencez à saisir les noms, la fonction de saisie automatique affiche les noms correspondants dans votre liste de contacts. Le planificateur affiche toutes les informations libre/occupé disponibles.

    • Cliquez sur Participants pour rechercher les adresses à ajouter.

    • Ouvrez le Planificateur pour afficher les informations libre/occupé lorsque vous saisissez les noms de participants. Si les plannings des participants sont connus, leurs disponibilités s'afficheront dans des barres horizontales à côté de leur nom.

Participants facultatifs. Si certains participants n'ont pas l'obligation d'être présents à la réunion, mais que vous voulez les inviter, sélectionnez Afficher facultatif et saisissez les adresses mail dans le champ, ou dans le Planificateur, cliquez sur l'icône à côté du nom et sélectionnez Participant facultatif.  Lorsque l'invitation est envoyée, leurs noms sont marqués comme facultatifs.

  1. Cliquez sur Emplacement pour rechercher un emplaceemnt qui a été créé comme un compte de ressources et qui est répertorié dans l'annuaire de votre entreprise ou utilisez le champ Emplacement pour décrire où se trouve le lieu de la réunion, par exemple en donnant une adresse, un numéro de bâtiment, l'emplacement des bureaux.  Cochez l'option Permettre des localisations multiples, si vous souhaitez réserver plusieurs emplacements pour la réunion.  Cliquez sur Afficher les ressources pour sélectionner l'équipement à réserver.

  2. Cliquez sur Afficher pour indiquer comment votre disponibilité au cours de cette période doit s'afficher dans le planificateur. Sélectionnez soit libre, à confirmer, occupé, soit à l'extérieur.

  3. Si vous disposez de plusieurs calendriers, sélectionnez celui pour lequel vous voulez configurer l'événement dans le menu Calendrier.

  4. Saisissez les heures de début et de fin.

Remarque : si le fuseau horaire s'affiche, il reflète votre fuseau horaire. Il n'est généralement pas nécessaire de le modifier. Si vous organisez des réunions avec des participants se trouvant dans des fuseaux horaires différents, l'invitation sera envoyée en indiquant l'heure de la réunion correspondant à leur fuseau horaire. Par exemple, si vous créez une réunion avec des participants se trouvant en Californie et à New York, l'invitation indique l'heure du fuseau horaire du Pacifique pour les participants résidant en Californie, et l'heure du fuseau horaire de l'Est des États-Unis (trois heures plus tard) pour les participants de New York.

  1. Cliquez sur Répétition pour afficher les modèles récurrents : quotidienne, hebdomadaire, mensuelle, annuelle, ou personnalisée. La réunion sera programmée en suivant le modèle de récurrence, à partir des date et heure de début spécifiées. La répétition personnalisée permet de planifier plus de modèles spécifiques.

    • Tous les jours. Personnalisez les répétitions quotidiennes pour les configurer chaque jour ou à une autre fréquence (par exemple, tous les deux ou quatre jours).

    • Chaque semaine. Personnalisez les répétitions hebdomadaires en choisissant s'il s'agit d'un rendez-vous récurrent hebdomadaire ou d'un rendez-vous suivant une autre configuration hebdomadaire – par exemple, tous les quinze jours.  Vous pouvez organiser la réunion un jour de semaine spécifique.

    • Chaque mois. Personnalisez les répétitions mensuelles en choisissant s'il s'agit d'un rendez-vous récurrent mensuel ou d'un rendez-vous suivant une autre configuration mensuelle – par exemple, tous les deux mois.  Un calendrier s'affiche et vous pouvez sélectionner le ou les jours pour lesquels vous voulez que la réunion soit programmée.

    • Chaque année. Personnalisez les répétitions annuelles pour choisir le mois et la date.

Cliquez sur Personnaliser pour sélectionnez un modèle récurrent.

  1. Pour modifier l'heure d'envoi de la notification de rappel, cliquez sur le champ Rappel. Si vous voulez que le rappel soit envoyé à une adresse mail ou par SMS, cochez la case correspondante.

  2. Utilisez le champ Texte pour ajouter des informations supplémentaires à inclure dans le mail. Pour ajouter des pièces jointes, cliquez sur Ajouter pièces jointes dans la barre d'outils.

  3. Dans Options dans la barre d'outils, vous pouvez désactiver les options Solliciter des réponses et Envoyer un mail de notification et vous pouvez également marquer le rendez-vous comme privé.

  4. Cliquez sur Envoyer. Un mail d'invitation est envoyé aux participants.

Si vous souhaitez envoyer l'invitation plus tard, cliquez sur Enregistrer. Une copie locale du rendez-vous sera enregistrée et les invitations ne seront pas envoyées aux participants.
