Filtrer vos mails

Le filtre applique un ensemble de règles de correspondance au courrier entrant, puis exécute une action spécifiée.

Vous pouvez filtrer vos mails entrants pour les trier dans des dossiers, les marquer automatiquement, les faire suivre, ou les supprimer. Exemples de règle :

  • Tous les mails envoyés par votre directeur sont rangés dans le dossier "Directives commerciales".

  • Tous les mails contenus dans la liste de diffusion "événements de la société" sont marqués avec le tag "Événements".

Pour créer ou modifier une règle de filtre :

  1. Activez l'onglet Préférences .

  2. Activez le dossier Filtres, puis cliquez sur Nouveau filtre.

  3. La boîte de dialogue Ajouter un filtre s'affiche. Elle vous permet de définir la règle et la destination du nouveau filtre.

  4. Dans le champ Nom du filtre, tapez le nom unique de la règle de filtre.

  5. Dans la zone Si des conditions parmi les conditions suivantes sont satisfaites, choisissez une préférence de regroupement.  

    • L'une des conditions : dès qu'une des conditions spécifiées dans le filtre est satisfaite, l'action définie est déclenchée.

    • Toutes les conditions : l'action définie est déclenchée seulement si toutes les conditions spécifiées dans le filtre sont satisfaites.  

Vous pouvez répéter les étapes suivantes afin de configurer plusieurs conditions et actions dans un seul filtre. Pour configurer plusieurs conditions, cliquez sur le lien +.

    • Dans la première liste déroulante, choisissez une condition à utiliser.  

    • Choisissez une méthode de comparaison . Les options indiquées dépendent de la sélection effectuée dans la zone déroulante précédente. Par exemple, "est" ou "n'est pas" peuvent s'afficher.

    • Saisissez un mot ou un groupe de mots à comparer dans la zone du texte.

Cliquez sur pour ajouter des conditions. Vous pouvez continuer d'ajouter des conditions ou passer à l'étape suivante, à savoir ajouter une ou plusieurs actions.

  1. Dans le champ Exécuter les actions suivantes, choisissez une action dans le menu déroulant.

Spécifiez un dossier ou le nom d'un tag, si nécessaire. Ce que vous spécifiez dépend de l'action que vous choisissez.

Cliquez sur pour ajouter des actions. Vous pouvez continuer d'ajouter des actions ou cliquer sur OK pur arrêter.

  1. Dans la partie supérieure de la boîte de dialogue Ajouter un filtre, notez la case. Cochez cette case pour activer la règle de filtre.  Désélectionnez-la pour désactiver cette règle.

  2. Cochez Ne pas appliquer d'autres filtres s'il s'agit du seul filtre à exécuter sur les mails qui remplissent les conditions du filtre. Remarque : dans chaque règle de filtre, Ne pas appliquer d'autres filtres doit toujours être la dernière action. Elle empêche l'application d'autres règles de filtre aux mails entrants qui satisfont la règle actuelle.

    Remarque : Si cette action n'est pas spécifiée, les règles des filtres ultérieurs continuent à être appliquées aux mails entrants, même si la ou les conditions du filtre actuel sont satisfaites. Vous ne voulez probablement pas qu'un même message corresponde à plusieurs règles de filtres et exécuter plusieurs actions éventuellement contradictoires.

Les modifications que vous apportez aux règles des filtres sont enregistrées immédiatement. Le filtre est appliqué de manière automatique dès la réception de nouveaux mails.  

Vous pouvez également appliquer des règles de filtre aux mails déjà stockés dans votre compte. Dans Préférences de filtres, cliquez sur Appliquer le filtre et sélectionnez les dossiers principaux auxquels le filtre doit être appliqué.

Remarque : Vous pouvez également créer un filtre à partir d'un mail. Cliquez avec le bouton droit sur le mail, puis choisissez Nouveau filtre. La boîte de dialogue Modifier filtre s'affiche avec les champs De, À et Objet définis avec les données du mail.  Si les conditions du filtre sont correctes, attribuez-lui un nom et cliquez sur OK.
