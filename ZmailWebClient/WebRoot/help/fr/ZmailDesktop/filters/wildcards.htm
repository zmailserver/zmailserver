Substitution de filtre

Les substitutions peuvent être utilisées dans les comparaisons qui exécutent l'opérateur de comparaison "Modèle de correspondance". Les deux caractères de substitution sont * et ?

Astérisque (*)

L'astérisque * est un paramètre de substitution pour "aucun ou plusieurs caractères de n'importe quel type".

Exemple de recherche de caractères dans le champ d'objet : palette*bananes

avec Correspondances : Lignes d'objet telles que " palettebananes", "bananes", "palette de casserole de feuilles de bananes" mais pas "super-banane-palette"

Exemple de recherche de caractères dans le champ d'objet : w*bandana

avec Correspondances : Lignes d'objet telles que "bandana blanc" ou "dont le bandana", mais pas "à qui appartient ce bandana ?"

Point d'interrogation (?)

Le point d'interrogation est un paramètre de substitution pour "un caractère (et pas davantage !)".

Exemple de recherche de caractères dans le champ d'objet : banane?bateau

avec Correspondances : Lignes d'objet telles que " bananesbateau", "banane-bateau", "banane!bateau" ou "banane bateau"

Le caractère d'échappement est la barre oblique : \

Vous pouvez vouloer spécifier une correspondance exacte pour une chaîne dont les caractères sont normalement considérés comme des caractères de substitution. Par exemple, si vous voulez spécifier une correspondance pour une ligne d'objet dans laquelle l'en-tête principal est entouré de trois astérisques, par exemple

***MORE MONEY!!!*** ou

***FREE GIFT***

Pour spécifier un caractère de substitution en tant que tel plutôt qu'une substitution pour d'autres caractères, placez la barre oblique inverse (\) juste avant le caractère. Par exemple, le caractère de comparaison "\*\*\**\*\*\*" spécifie un objet avec trois astérisques placés avant et après une chaîne de caractères (espaces compris).
