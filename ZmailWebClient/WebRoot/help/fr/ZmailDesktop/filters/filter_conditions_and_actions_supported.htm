Conditions et actions de filtre prises en charge

Les règles de filtre ne sont pas sensibles aux majuscules, ce qui signifie que les règles ignorent les lettres en majuscules.

Conditions

Les champs de comparaison incluent les éléments suivants :

  • De. À utiliser pour spécifier un nom dans l'en-tête De d'un mail.

  • À. Identique au champ De, mais la recherche des noms spécifiés s'effectue dans l'en-tête À.

  • Cc. Identique au champ De, mais s'affiche dans l'en-tête Cc:.

  • À ou Cc. Identique au champ De, mais s'affiche dans le champ À ou Cc.

  • Objet. Recherche dans l'en-tête Objet du mail.

  • En-tête nommé. Lorsque vous sélectionnez cette option, un champ d'entrée de texte supplémentaire s'affiche devant l'opérateur de comparaison (la section "contient"). Cette option vous permet d'indiquer l'en-tête mail de votre choix. Vous pouvez non seulement spécifier les champs standard À, ou De, mais également les champs de Date, Répondre à et d'autres champs personnalisés qui peuvent être inclus dans l'en-tête du message. Vous pouvez utiliser cette option pour filtrer les mails qui ont des en-têtes "mal formatés", c'est-à-dire des en-têtes auxquels il manque certaines informations qui sont normalement considérées comme standard pour un mail. Parfois les spams (générés automatiquement), omettent des informations, telles que les champs Expéditeur et Répondre à. Utilisez la deuxième zone de texte (le champ situé immédiatement à droite de l'opérateur de comparaison) pour spécifier l'en-tête à tester.

  • Taille. Utilisez cette option pour sélectionner les mails d'une taille supérieure ou inférieure à la taille spécifiée. La taille d'un mail inclut les pièces jointes. Vous pouvez l'utiliser pour supprimer les mails trop volumineux.

  • Date. Utilisez cette option pour spécifier les mails envoyés avant ou après une date.

  • Contenu. Les options de contenu sont Contient/Ne contient pas ; elles permettent de spécifier des mots correspondants dans le corps du mail. Vous ne pouvez pas filtrer les mots des pièces jointes.

  • Est un accusé de lecture.Les accusés de lecture peuvent être filtrés.

  • Pièce jointe. Vous pouvez filtrer vos mails, qu'ils contiennent ou non des pièces jointes.

  • Adresse dans. Vérifie la présence d'une adresse dans vos contacts. Le champ suivant sert à indiquer l'adresse.

  • Invitation dans le calendrier. Vous pouvez filtrer les mails relatifs au calendrier.

Les opérateurs de comparaison incluent :

  • Correspond exactement/Ne correspond pas exactement. Recherche une correspondance exacte. Par exemple, en spécifiant les correspondances d'Objet - Banane, les correspondances seraient "Banane" et non pas "Bananes" ou "camion plein de feuilles de bananes".

  • Contient/ne contient pas. Signifie que le champ Objet doit contenir l'élément spécifié.  Par exemple, la recherche de Objet contient Banane établit une correspondance avec "Je vais manger une banane".

  • Correspond/ne correspond pas aux caractères de substitution. Le caractère de substitution * est utilisé dans le champ de comparaison du filtre de mail pour représenter un ou plusieurs caractères du filtre. Par exemple, en spécifiant les correspondances d'objet "banane*", les correspondances seraient "bananes" ou "casserole de feuilles de bananes", mais pas "ma casserole de feuilles de bananes."

Actions

  • Conserver dans la Boîte de réception. Enregistre les mails dans la boîte de réception. Si aucune règle de filtre ne correspond à un mail, cette action s'exécute par défaut.

  • Ranger dans un dossier. Transfère un mail vers un dossier défini.

  • Supprimer. Supprime le mail sans le distribuer. Le mail ne fait pas partie de vos dossiers, même pas du dossier Corbeille.

  • Faire suivre. Fait suivre le mail à l'adresse indiquée.

  • Marquer. Sélectionnez comme lu ou avec drapeau.

  • Libeller avec. Vous pouvez marquer les mails correspondants avec un tag sélectionné.

Actions multiples

Vous pouvez créer un ensemble de règles de filtre constituées de plusieurs actions. Les combinaisons d'actions dans vos règles sont interprétées comme suit :

  • Supprimer. Si elle est combinée avec d'autres actions, l'action de suppression est ignorée et les autres actions deviennent effectives.

  • Ranger dans un dossier. Plusieurs actions "Ranger dans" entraînent la création de plusieurs copies du mail stockées dans différents dossiers. Si aucun dossier n'est spécifié, le mail sera conservé dans la boîte de réception.

  • Tag/Drapeau. Ces actions s'appliquent au mail envoyé par la dernière action exécutée. Dans le cas de plusieurs actions "Ranger dans", il peut arriver que plusieurs copies du mail soient sauvergardées sans tag, et d'autres avec.

  • Conserver dans la Boîte de réception. Plusieurs actions "conserver" peuvent être définies, mais une seule copie du mail est conservée dans la boîte de réception.

  • Faire suivre à l'adresse. Le mail est transféré vers l'adresse indiquée.
