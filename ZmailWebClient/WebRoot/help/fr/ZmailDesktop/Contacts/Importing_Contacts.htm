Importer des contacts

Vous pouvez importer des listes de contacts et des carnets d'adresses enregistrés dans le format de fichier texte .csv (comma-separated). La fonction d'importation est disponible dans le dossier Import/Export de l'onglet Préférences.

Pour importer une liste de contacts :

  1. Sélectionnez Préférences, puis cliquez sur Importer/Exporter

  2. Dans le champ "Importer Fichier", accédez au fichier .csv à importer.

  3. Dans le champ Type, cliquez sur Détecter automatiquement pour sélectionner le type de fichier de contact à exporter :

      • Contacts du compte

      • Contacts Yahoo!

      • Contacts Outlook 2000

      • Contacts Outlook 2003

      • Contacts Outlook

      • Contacts Thunderbird

Les listes de contacts provenant de différents comptes risquent de ne pas afficher les informations dans le même ordre. Par conséquent, il est important de sélectionner le type (format) du ou des fichiers de contacts à importer ; en effet, cette valeur conditionne l'ordre des champs et l'affichage approprié des informations dans votre carnet d'adresses.

  1. Cliquez sur Importer. Dans la boîte de dialogue Sélectionnez un carnet d'adresses, spécifiez un carnet d'adresses existant pour ajouter les contacts ou créez un carnet d'adresses pour importer le fichier.

  2. Cliquez sur OK. Le contenu du fichier de contacts est importé.
