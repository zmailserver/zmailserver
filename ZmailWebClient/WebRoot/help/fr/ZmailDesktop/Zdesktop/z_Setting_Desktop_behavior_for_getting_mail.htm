Configuration du comportement du bureau pour actualiser

Dans la section Préférences>Mail>Affichage des mails, le fait d'activer l'une des options dans Lorsque je clique sur "Envoyer/Recevoir" définit si votre affichage de la page sur laquelle vous êtes change ou non quand vous cliquez sur Envoyer/Recevoir.

  • Si Exécuter ma recherche par défaut est sélectionné, quand vous cliquez sur Envoyer/recevoir, la recherche mail est exécutée pour les messages qui doivent être délivrés dans votre boîte de réception et votre affichage est automatiquement redirigé à partir du dossier que vous voyez dans l'affichage de votre Boîte de réception.  

  • Si Actualiser la vue actuelle est sélectionné, quand vous cliquez sur Envoyer/recevoir à partir de tout affichage de dossier, la demande d'actualisation recherche des mises à jour pour la vue actuelle de votre dossier et pour les autres dossiers. L'affichage de votre dossier actuel ne change pas. Si un nouveau mail est reçu vers d'autres dossiers, ces dossiers passent en gras et le nombre de messages non lus est mis à jour.
