Sauvegarde de vos comptes

Vous pouvez sauvegarder vos comptes Zimbra Desktop  à partir de la page Préférences>Sauvegardes dans la section Tous les comptes.  Par défaut, la sauvegarde ne s'applique qu'aux dossiers locaux, puisque  ceux-ci contiennent des éléments qui ne sont pas sauvegardés sur d'autres serveurs de messagerie.  

Pour sauvegarder vos comptes

  1. Aller à la page Préférences>Sauvegardes.

  2. Dans la section Configurer les sauvegardes, sélectionnez les comptes à sauvegarder. Notez que les éléments contenus dans vos dossiers locaux et vos dossiers de compte POP sont sauvegardés sur votre ordinateur. Ces comptes sont de bons choix de sauvegarde. Vous pouvez sauvegarder tous les comptes que vous avez créés dans Zimbra Desktop, mais sachez que les fichiers peuvent être assez volumineux. Pour avoir une estimation de l'espace disque nécessaire, placez le curseur sur le nom du compte ; la taille de la boîte aux lettres s'affiche.

  3. Si vous souhaitez planifier la sauvegarde, sélectionnez la fréquence des sauvegardes : Chaque jour, Chaque semaine, Chaque mois.

  4. Dans le champ Emplacement, saisissez le chemin du dossier où la sauvegarde doit être enregistrée. Par exemple, pour sauvegarder dans le dossier ZD_sauvegardes du disque externe E, tapez E:\ZD_sauvegardes.

  5. Dans le champ Conservation, sélectionnez le nombre de sauvegardes que vous souhaitez enregistrer. Par défaut, un seul fichier de sauvegarde sera enregistré. Vous pouvez choisir d'enregistrer jusqu'à 5 sauvegardes avant que la sauvegarde la plus ancienne ne soit remplacée.

Si vous avez planifié l'exécution automatique de la sauvegarde, celle-ci s'exécutera conformément à votre planification. Zimbra Desktop doit être ouvert pour que les sauvegardes puissent s'exécuter. Si une session de sauvegarde a été planifiée pendant une période où vous n'êtes pas connecté à Zimbra Desktop, la sauvegarde commencera lors de votre prochaine connexion.

Si vous souhaitez sauvegarder vos comptes immédiatement ou si la planification de la sauvegarde est paramétrée sur  Jamais, cliquez sur Sauvegarder les comptes maintenant pour lancer une session de sauvegarde. Suivant la taille des comptes, la sauvegarde peut prendre un certain temps.  Ne fermez pas Zimbra Desktop  pendant l'exécution de la sauvegarde. Vous pouvez continuer à travailler sur Zimbra Desktop.

Remarque : Si vous choisissez de sauvegarder vos comptes maintenant, la prochaine planification de sauvegarde ne sera pas réinitialisée.

Pour restaurer votre compte, consultez Restauration des comptes à partir de vos fichiers de sauvegarde.
