Récupération d'objets supprimés

Le dossier Corbeille est disponible pour chaque application Zimbra que vous utilisez. Seuls les éléments spécifiques à l'application sélectionnée sont affichés quand vous ouvrez le dossier Corbeille de l'application. Mais quand vous sélectionnez Vider la corbeille à partir d'une application, les éléments de toutes les applications sont supprimés.

Avant de vider le dossier Corbeille, vous pouvez récupérer les fichiers de la corbeille en les faisant glisser et en les déposant dans un autre dossier.

Les messages qui restent dans la corbeille Zimbra peuvent être automatiquement supprimés et vous ne pouvez plus les récupérer dans ce cas. Les messages qui restent dans les dossiers Corbeille des autres comptes ne sont pas supprimés automatiquement ; ils sont conservés jusqu'à ce que vous vidiez le dossier Corbeille.
