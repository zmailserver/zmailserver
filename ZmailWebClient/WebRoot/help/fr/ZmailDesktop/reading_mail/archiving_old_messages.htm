Purger des mails

Pour supprimer des mails de la corbeille, faites un clic droit sur le dossier Corbeille puis sélectionnez Vider la corbeille. Tous les messages sont supprimés. Si Restaurer objets supprimés s'affiche quand vous cliquez avec le bouton droit sur le dossier Corbeille, vous pouvez récupérer les mails que vous aviez supprimés dans la Corbeille. Il est possible de récupérer les objets supprimés de la Corbeille dans un délai de 30 jours après la suppression de la Corbeille. Passés ces 30 jours, les objets sont supprimés de manière permanente.
