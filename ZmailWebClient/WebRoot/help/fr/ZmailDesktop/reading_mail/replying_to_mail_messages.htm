Répondre à des mails

Vous pouvez répondre à l'expéditeur d'un mail ou vous pouvez répondre à l'expéditeur et à tous les destinataires du mail.

  • Répondre permet de répondre uniquement à l'expéditeur du mail d'origine. Cette option n'inclut aucune autre adresse présente dans les champs À, ou Cc :. En cliquant sur Répondre une nouvelle fenêtre pour rédiger un mail apparaît et remplit le champ À : avec le nom de la personne qui vous a envoyé le mail.

  • Répondre à tous inclut tous les destinataires du mail précédent. Vous utilisez cette option si vous voulez que toutes les personnes ayant reçu le mail précédent puissent également recevoir votre réponse.

L'utilisation de l'option Répondre permet uniquement de conserver la zone de texte. Les pièces jointes du mail d'origine ne sont pas includes dans la réponse.

Lorsque vous cliquez sur Répondre ou sur Répondre à tous ,la page de création de mail s'affiche avec le texte du mail auquel vous répondez.

Suivant la configuration de vos  préférences de messagerie, le texte des mails précédents peut contenir un caractère indiquant le nombre d'échanges précédents effectués depuis que ce texte a été rédigé. Par exemple, si la réponse fait partie d'une conversation en cours et que vous avez activé l'option Inclure le mail original et coché Utiliser un préfixe, le texte de chaque partie de la conversation est affichée avec un > ou un préfixe.

> Mail précédent

> > Mail avant le mail précédent

> > > Et ainsi de suite

Remarque : Si le message fait partie d'une conversation, la modification de la ligne Objet: fait apparaître votre réponse dans une nouvelle conversation.

Pour répondre à un mail :

    1. Ouvrez le mail, puis cliquez sur Répondre ou sur Cliquez sur la flèche pour choisir Répondre à tous. Une fenêtre de nouveau mail s'affiche. Elle contient les adresses dans l'en-tête et le mail d'origine dans le corps du texte.

    2. Rédigez votre réponse.

    Remarque : Si vous ajoutez automatiquement une signature à vos mails, elle est insérée dans les mails selon les préférences que vous avez définiesau bas de tous les mails dans la fenêtre de nouveau mail ou au bas du mail que vous rédigez, sous le texte.

    1. Si vous voulez modifier la partie du mail d'origine à inclure dans votre réponse, cliquez sur Options dans la barre d'outils du nouveau mail. Votre option par défaut est cochée, mais vous pouvez uniquement la modifier pour ce mail.

    2. Cliquez sur Envoyer.
