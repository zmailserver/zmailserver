Ajouter une tâche

Créez une nouvelle tâche en saisissant simplement son nom dans votre liste de tâches. Vous pouvez également ouvrir la page des tâches et y ajouter des informations détaillées, notamment des notes et des pièces jointes.  Quand vous ouvrez une page de tâches, un onglet Tâches s'affiche dans la barre d'outils Application et le formulaire Tâches s'ouvre dans le panneau de contenu. Vous pouvez disposer de plusieurs onglets Tâches ou naviguer entre les différents onglets dans la barre d'outils Application.

Pour ajouter rapidement une tâche

  1. Choisissez la liste de tâches dans le panneau d'aperçu

  2. Dans le panneau de contenu, cliquez sur la ligne supérieure qui affiche Cliquez ici pour ajouter une nouvelle tâche.    

  3. Un champ de texte s'affiche. Saisissez le nom de la tâche.

  4. Appuyez sur Entrée. La tâche est ajoutée à votre liste de tâches.

Remarque : Si vous avez créé une tâche dans la mauvaise liste, ouvrez la tâche puis sélectionnez la liste correcte dans le menu déroulant des listes de tâches. Vous pouvez aussi sélectionner et faites glisser la tâche du panneau de contenu vers la liste qui convient.

Pour ajouter une tâche comportant des informations détaillées

Vous pouvez utiliser vos listes de tâches pour créer et suivre l'avancement d'une tâche. Vous pouvez également écrire des notes dans votre tâche et joindre des fichiers pour un accès facile à partir de la tâche.

Vous pouvez évaluer la durée du projet en saisissant la date de début et d'échéance et établir sa priorité. Lorsque vous démarrez la tâche, vous pouvez choisir un statut similaire à Non commencée ou En cours, puis sélectionner l'état d'avancement.

  1. Choisissez la liste de tâches dans le panneau d'aperçu.

  2. Dans la barre d'outils de tâches, cliquez sur Nouvelle.  Une page de tâches sans titre s'ouvre et un onglet Tâches s'affiche dans la barre d'outils.

  3. Dans la section Détails,

    • tapez le nom de la tâche dans le champ Objet. Si cela est pertinent, indiquez un lieu dans le champ Lieu.

    • Sélectionnez la Priorité - Normale, Basse ou Haute

    • Si vous avez cliqué sur Nouvelle avant d'avoir sélectionné la liste de tâches que vous souhaitiez, vous pouvez sélectionner celle qui convient dans le menu déroulant Liste de tâches.

  4. Dans la section Progression, sélectionnez la date de début, la date d'échéance. Si la tâche a commencé, vous pouvez remplacer  Non lancé(e) par une autre option dans la liste et mettre à jour le pourcentage effectué.

  5. Choisissez une date et une heure de rappel et, si vous avez configuré une alerte mail ou SMS depuis la page Préférences>Notification, cochez la case appropriée, Mail ou SMS.  Si vous n'avez pas configuré de notification, cliquez sur Configurer pour accéder à la page des préférences de notification.

  6. Ajoutez des notes dans le champ de texte. Pour joindre des pièces, cliquez sur Ajouter pièces jointes dans la barre d'outils.

  7. Cliquez sur Enregistrerpour sauvegarder la tâche. La nouvelle tâche est ajoutée à la liste de tâches.
