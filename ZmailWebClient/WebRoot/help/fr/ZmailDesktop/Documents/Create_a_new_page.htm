Créer un document

Vous pouvez créer un document dans Porte-documents, concevoir la présentation de contenu en utilisant les outils en format RTF. Vous pouvez procéder à une vérification orthographique et utiliser des mots-clés pour créer des liens vers d'autres pages.

  1. Cliquez sur le dossier de porte-documents dans lequel vous souhaitez travailler. Le panneau Liste de documents affiche les fichiers figurant dans le porte-documents.

  2. Dans la liste Nouveau sur la barre d'outils, sélectionnez Document. Une page Zimbra Docs vierge s'affiche.

  3. Dans la barre d'outils, entrez le nom du document.

  4. Rédiger le contenu de la page. Vous pouvez utiliser l'éditeur de format RTF pour ajouter des styles, des couleurs et des tableaux à votre page.

  1. Cliquez sur Enregistrer pour sauvegarder le fichier. Pour fermer le document sans sauvegarder, cliquez sur la croix (X) dans l'onglet du navigateur.

Lorsque vous sauvegardez un document, sa version est modifiée.

Remarque : Si vous n'avez pas créé le document dans le porte-documents approprié, vous pouvez transférer le fichier vers le porte-documents correct en sélectionnant le document et en le faisant glisser vers le dossier correspondant.
