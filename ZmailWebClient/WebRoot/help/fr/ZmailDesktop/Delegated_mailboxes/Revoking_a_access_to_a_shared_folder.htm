Modification d'accès à un dossier partagé

Remarque : Disponible pour les comptes de la société Zimbra uniquement.

Vous pouvez modifier les autorisations ou retirer le partage à tout moment.

  1. Cliquez avec le bouton droit de la souris sur un dossier que vous partagiez et cliquez sur Éditer propriétés. Les utilisateurs partageant le dossier sont répertoriés dans la section Partages pour ce dossier.

  2. Cliquez sur Modifier pour modifier les autorisations ou
    sur Retirer pour supprimer les autorisations.

  3. Indiquez si un mail décrivant les modifications doit être envoyé.

  4. Cliquez sur OK.
