À propos de Zimbra Desktop

Zimbra Desktop est un client hors ligne qui vous permet de gérer vos différents comptes de messagerie dans un seul outil de bureautique. La messagerie Zimbra, Yahoo! mail, Gmail, Outlook et tout autre compte de messagerie professionnel ou personnel utilisant POP ou IMAP sont facilement configurables dans Zimbra Desktop.  Chaque compte de messagerie est conservé séparément.

Après avoir configuré vos comptes, vos mails sont téléchargés sur votre ordinateur et vous pouvez y accéder, même si vous n'êtes pas connecté à Internet. Voir la section Travailler hors ligne. Vous pouvez lire, répondre, transférer et supprimer les mails de tous vos comptes de messagerie à partir de Zimbra Desktop. L'espace de stockage des mails est défini en fonction du volume de stockage de votre ordinateur et non pas en fonction d'un quota de compte prédéfini.

Lorsque vous recevez un nouveau mail dans l'un de vos comptes de messagerie alors que vous êtes connecté à Internet, le mail est automatiquement téléchargé vers votre Zimbra Desktop. Il n'est pas nécessaire d'ouvrir Desktop pour effectuer cette opération.  Une icône d'enveloppe apparaît dans la ligne de barre d'état Windows ou Mac Dock indiquant le nombre total des mails non lus pour l'ensemble de vos comptes. Cliquez sur l'icône pour ouvrir l'application de bureautique.

Les fonctions principales sont disponibles sur Zimbra Desktop

Vous pouvez non seulement accéder à tous vos comptes de messagerie dans une même interface, Zimbra Desktop, mais également de gérer vos contacts. Cette application dispose d'un calendrier, d'une application de liste de tâches et d'un outil porte-documents pour conserver vos pièces jointes.

Voir à la section Aperçu des fonctions de Zimbra Desktop pour consulter la liste des fonctions principales.

Les types de comptes que vous pouvez configurer et gérer

Les comptes de messagerie suivants peuvent être configurés.

  • Compte Zimbra. Vous pouvez utiliser Zimbra Desktop pour accéder à votre compte de messagerie s'il utilise Zimbra. Cela comprend les mails, les dossiers, les carnets d'adresses, les calendriers, les listes des tâches et les documents de porte-documents. Pour accéder à un compte Zimbra via un navigateur, utilisez le Zimbra Web Client. Les préférences que vous définissez dans ZWC pour votre compte Zimbra sont disponibles à partir de Zimbra Desktop. Remarque : Pour pouvoir bénéficier de toutes les fonctions Zimbra sur Zimbra Desktop, la version de Zimbra ZCS doit être la version 5.0 ou une version ultérieure. Votre administrateur système Zimbra peut vous indiquer la version de Zimbra.

  • Yahoo! Mail. Vous pouvez créer un compte Yahoo! Mail, Yahoo! Mail Plus, Yahoo! Small Business, Ymail ou Rocketmail.  Vos mails, dossiers de mails et carnets d'adresses sont disponibles sur Zimbra Desktop.

  • Gmail. Votre compte Gmail doit être configuré pour autoriser l'accès IMAP. Il est possible que vous deviez vous connecter à votre compte Gmail pour activer IMAP. Vos mails et vos contacts sont disponibles sur Zimbra Desktop.

  • Compte Microsoft Exchange IMAP. Les mails de votre compte Exchange IMAP sont disponibles sur Zimbra Desktop.

Si votre compte ne fait pas partie de ceux cités précédemment, vous pouvez tout de même configurer le compte sur Zimbra Desktop s'il s'agit d'un compte POP3 ou IMAP4. Vous devrez remplir les informations de paramètres se rapportant à votre compte, y compris les informations du serveur de messagerie entrant et du serveur de messagerie sortant Si nécessaire, contactez votre administrateur système ou votre FAI.
