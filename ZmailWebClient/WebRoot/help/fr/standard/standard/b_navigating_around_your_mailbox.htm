Utilisation de votre boîte aux lettres

Lorsque vous vous connectez à votre boîte aux lettres, l'écran d'affichage est semblable à celui-ci :

 

Option

Description

En-tête de la fenêtre

Affiche la Barre de recherche

Panneau d’aperçu

Affiche les éléments suivants :

•       Nom d'utilisateur : nom d'utilisateur sous lequel vous vous êtes connecté

•       Affichage de quota : quota de votre boîte aux lettres

•       Dossiers : comprend notamment les dossiers système (Boîte de réception, Envoyé, Brouillons, Courrier indésirable et Corbeille.) ainsi que les dossiers personnalisés que vous avez créés.

•       Recherches : demandes de recherche que vous avez créées et enregistrées pour les utiliser ultérieurement

•       Tags : tags que vous savez créés. Cliquez sur un tag pour afficher rapidement tous les mails marqués avec ce tag.

Onglets d’application.

Les applications accessibles sont répertoriées dans les onglets de cette ligne, comme les applications Mail ou Calendrier. Les liens Aide et Déconnexion sont également disponibles à droite de cette barre.

Barre d'outils

Cette barre s'affiche en haut et de nouveau dans le panneau de contenu et affiche les éléments suivants :

Onglets d'outils : cliquez sur un onglet pour accéder aux différents outils disponibles pour votre compte.

Boutons d'actions : cliquez sur un bouton d'action pour effectuer une tâche disponible dans l'application que vous utilisez actuellement. Les champs Déplacer vers et Actions donnent les choix de menu disponibles dans l'application.

Panneau de contenu.

Le contenu de cette zone change selon l'application utilisée. Dans l'affichage Boîte de réception, le panneau affiche tous les mails ou conversations dans votre Boîte de réception.

Barre de recherche

Affiche la Recherche et la zone Recherche avancée. Vous pouvez également sauvegarder des recherches.

Panneau de lecture

Affiche le contenu des mails.

 

Déplacer vers...

Le menu Déplacer vers... affiche les déplacements que vous pouvez effectuer au sein l'application dans laquelle vous vous trouvez. Dans cet exemple, dans l'application de messagerie, le message provenant de Dan Pattern est sélectionné et les options pour le message sont de le déplacer vers des dossier de votre liste de dossiers.

Action…

Le menu Plus d'actions... affiche les différentes actions que vous pouvez effectuer au sein l'application dans laquelle vous vous trouvez. Cet exemple concerne l'application de messagerie.

Fonctions de navigation non disponibles

Les fonctions de navigation suivantes ne sont pas disponibles dans le ZWC standard.

  •   Menus déroulants

  • Châssis mobile

  • Info-bulles

  • Cliquer avec le bouton droit pour les fonctions du client

  • Informations sur le dossier
