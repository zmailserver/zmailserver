Joindre des fichiers à votre mail

Les mails peuvent inclure des pièces jointes. Vous pouvez joindre des documents, des feuilles de calcul, des photos, des diaporamas et d'autres types de fichiers.

  1. Rédigez le mail comme décrit dans la section Création d'un mail.

  2. Cliquez sur Ajouter pièces jointes. Une nouvelle fenêtre apparaît, vous permettant de télécharger jusqu'à 10 pièces jointes.

  3. Pour télécharger un fichier, cliquez sur Parcourir... pour localiser le fichier.

  4. Sélectionnez le fichier, puis cliquez sur Ouvrir.

  5. Téléchargez des pièces jointes supplémentaires en procédant de la même manière.

  6. Une fois toutes les pièces jointes ajoutées, cliquez sur Terminé. Les pièces jointes sont répertoriées sous la ligne d'objet.

  7. Lorsque le mail est prêt à être envoyé, cliquez sur Envoyer pour l'envoyer avec ses pièces jointes.

Remarque : Les destinataires de votre mail doivent disposer du logiciel approprié pour pouvoir ouvrir et lire le fichier. Pour les fichiers créés avec des logiciels communément utilisés, tels que Microsoft Office, les utilisateurs d'un système bureautique standard peuvent les ouvrir. Les fichiers de formats courants, tels que HTML, et les fichiers images, tels que .GIF ou .JPG, peuvent être ouverts à partir de différents programmes. Néanmoins, cela dépend du type de système de l'utilisateur.

Pour supprimer une pièce jointe, désélectionnez la case de la pièce jointe avant d'envoyer le mail.
