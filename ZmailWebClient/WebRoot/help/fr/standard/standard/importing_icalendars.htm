Importer iCalendars

Vous pouvez importer les calendriers iCalendars (iCal) enregistrés au format .ics. iCalendars est le standard pour les échanges des données de calendrier via Internet, et .ics est le format standard des informations iCalendar.

Vous pouvez importer votre calendrier iCal dans un calendrier existant ou créer un calendrier et importer le fichier .ics dans le nouveau calendrier.

Pour importer un calendrier iCal :

  1. Activez l'onglet Calendrier.

  2. Dans le panneau Aperçu du Calendrier, cliquez sur Modifier.

  3. Sélectionnez un calendrier dans la liste ou créez un calendrier devant recevoir le fichier .ics importé et cliquez sur Modifier.

  4. Dans le champ Importer de .ICS, retrouvez le fichier .ics à importer.

  5. Cliquez sur Importer.

  6. Cliquez sur Fermer.
