Filtrage de vos messages

Le filtre applique un ensemble de règles de correspondance au courrier entrant, puis exécute une action spécifiée.

Vous pouvez filtrer vos mails entrants pour les trier dans des dossiers, les marquer automatiquement, les faire suivre, ou les supprimer. Exemples de règle :

  • Tous les mails envoyés par votre directeur sont rangés dans le dossier « Directives commerciales ».

  • Tous les mails contenus dans la liste de diffusion « événements de la société » sont marqués avec le tag « Événements ».

Créer ou modifier une règle de filtre :

  1. Activez l'onglet Préférences.

  2. Activez le dossier Filtres de mails, puis cliquez sur Nouveau filtre.

  3. Entrez un nom de filtre.

  4. Dans la zone Si --- les conditions ci-dessous sont satisfaites, sélectionnez une condition.  Vous pouvez maintenant spécifier la condition.

Vous pouvez répéter les étapes suivantes afin de configurer plusieurs conditions et actions dans un seul filtre.

  • Dans la première liste déroulante, choisissez une condition à utiliser.  

  • Choisissez une méthode de comparaison. Les options indiquées dépendent de la sélection effectuée dans le menu déroulant précédent. Par exemple, il est possible que les options « est » ou « n'est pas » ne soient pas affichées.

  • Saisissez un mot ou un groupe de mots à comparer dans la zone de texte.

Cliquez sur ajouter pour ajouter des conditions. Vous pouvez continuer à ajouter des conditions ou passer à la section suivante, qui consiste à ajouter une ou plusieurs actions.

  1. Dans le champ Exécuter les actions suivantes, choisissez une action dans la liste déroulante.

  • Spécifiez un dossier ou le nom d'un tag, si nécessaire. Ce que vous spécifiez dépend de l'action que vous sélectionnez.

  • Cliquez sur ajouter pour ajouter des actions. Vous pouvez continuer d'ajouter des actions ou cliquer sur OK pur arrêter.

  1. Cochez la case actif à côté de Nom du filtre pour activer la règle de filtre.

Le filtre est appliqué de manière automatique dès la réception de nouveaux mails.
