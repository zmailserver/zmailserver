Répondre à des mails

Vous pouvez répondre à l'expéditeur d'un message ou à l'expéditeur et tous les destinataires.

Par défaut, lorsque vous répondez à un message, le message d'origine est inclus dans le corps du message, mais les pièces jointes ne sont pas incluses. Vous pouvez également choisir de mettre un préfixe devant chaque réponse dans le message pour suivre les différentes réponses.

Par exemple, chaque réponse dans le message est marquée d'un chevron (>). La réponse la plus récente a un > devant chaque ligne. La réponse précédente en a deux (>>), celle d'avant en a trois (>>>) :

> Mail précédent

> > Mail avant le mail précédent

> > > Et ainsi de suite

Répondre à un message

  1. Choisissez le mail auquel vous souhaitez répondre.

  2. Cliquez sur Répondre ou Rép. à tous dans la barre d'outils. Une fenêtre de nouveau mail s’affiche avec les adresses dans l'en-tête et le mail original dans le corps du texte.

    3. Si vous affichez les mails dans la vue Conversation, vous pouvez cliquer sur répondre en bas du message.

  3. Rédigez votre réponse.

  4. Cliquez sur Envoyer.

Remarque :  si vous ajoutez automatiquement votre signature à vos mails, elle est placée dans le mail selon vos paramètres de préférence.
