Indication du niveau de priorité d'un mail

Vous pouvez indiquer le niveau d'importance d'un message. Par exemple, si vous avez une demande qui doit être traitée le plus vite possible.

•       Avant d'envoyer le mail, dans le menu déroulant Priorité de l'en-tête du mail, choisissez l'option de priorité : Haute, Normale, ou Basse.

Le destinataire voit le drapeau de priorité du mail dans sa boîte de réception et dans le mail.
