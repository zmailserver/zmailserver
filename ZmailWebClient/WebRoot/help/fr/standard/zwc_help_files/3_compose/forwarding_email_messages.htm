Transférer des mails

Vous pouvez transférer un mail à une autre personne. Le transfert d'un mail inclut toutes les pièces jointes.

  1. Choisissez le mail à transférer.

  2. Cliquez sur Faire suivre dans la barre d'outils. Le texte du message transféré apparaît dans le corps de la page de composition, sauf si vous avez configuré vos préférences pour transférer les mails en tant que pièce jointe.

  3. Entrez les destinataires.

  4. Cliquez sur Envoyer.
