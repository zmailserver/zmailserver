Lecture d'un mail

Par défaut, les nouveaux mails sont placés dans votre boîte de réception. Si vous avez configuré des règles de filtre de mails, les nouveaux messages peuvent apparaître dans d’autres dossiers que votre boîte de réception. Le nombre entre parenthèses à côté de chaque dossier correspond au nombre de mails non lus dans ce dossier.

Lire un mail dans la fenêtre de conversation

  1. Cliquez sur la conversation contenant le mail. Le panneau de contenu change et affiche les mails individuels de la conversation. La colonne Fragment affiche la première ligne du corps du message.

  2. Pour ouvrir le mail, cliquez sur la ligne qui contient le message à lire.

Si le Panneau de lecture est activé, le corps du message s'affiche dans le panneau de lecture. Sinon, cliquez sur le message. Le message s'affiche dans le panneau de contenu.

Lire un mail dans l'affichage Message simple

  • Si la fonction Panneau de lecture est activée, cliquez sur le message, puis lisez-le dans le panneau de lecture.

  • Si la fonction Panneau de lecture est désactivée, cliquez surle message. Le message s'affiche dans le panneau de contenu.
