Exemples de recherche

  • Rechercher pour un type spécifique de fichier en pièce jointe. Un mail ayant en pièce jointe une feuille de calculs du budget de l'année prochaine. Vous ne vous rappelez plus qui est l'expéditeur ni à quelle date il a été envoyé, mais vous savez que le fichier a l'extension .XLS.

Ouvrez simplement le menu déroulant Pièces jointes et sélectionner le type d'extension de pièce jointe. Une liste de cases à cocher apparaît alors au-dessous de la case d'option. Faites défiler, puis cochez la case à côté de Microsoft Excel.

  • Rechercher un contact dans une entreprise spécifique. A contact who works at a company called "Rivendell".

Dans la barre de recherche dans le menu déroulant sélectionner Contacts et entrez le nom de l'entreprise dans la zone de texte Rechercher.

  • Rechercher un message sur tous vos dossiers. Vous savez que vous avez reçu un message, mais vous n'êtes pas sûr dans quel dossier vous l'avez sauvegardé.

Cliquer sur l'icône de la loupe dans la barre de recherche pour ouvrir la page de recherche. Dans le menu déroulant État sélectionner n'importe où. Pour inclure votre Corbeille et les dossiers indésirable dans votre recherche, vous devez l'activer dans la page>Préférences générales.

  • Rechercher par une plage de dates. Un message d'une personne prétend vous avoir envoyé "au début de Décembre."

Cliquer sur l'icône de la loupe dans la barre de recherche pour ouvrir la page de recherche. Pour définir une plage de dates, dans le menu déroulant envoyéDate sélectionner d'abord et d'ajouter une date, puis sélectionner après et ajouter une date de fin.

  • Rechercher un message auquel vous n'avez pas répondu. Vous avez reçu des e-mails auxquels vous n'avez pas encore répondu

Cliquer sur l'icône de la loupe dans la barre de recherche pour ouvrir la page de recherche. Dans le menu déroulant État sélectionner solo.
