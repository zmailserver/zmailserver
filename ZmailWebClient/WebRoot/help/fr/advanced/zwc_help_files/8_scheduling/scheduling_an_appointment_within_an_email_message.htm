Programmation d'un rendez-vous à partir d'un mail

Dans le contenu d'un mail, une partie du texte est interprétée comme une date et offre la possibilité de cliquer avec le bouton droit pour créer un rendez-vous. Des textes tels que "« aujourd'hui », « demain », un jour de la semaine (par ex.« jeudi »), ou une date exacte sont mis en évidence dans les messages.

  1. Déplacez votre souris sur ce type de texte pour vérifier si vous avez un rendez-vous programmé.

  2. Cliquez sur le texte surligné pour ouvrir votre calendrier.

  3. Créez le rendez-vous.
