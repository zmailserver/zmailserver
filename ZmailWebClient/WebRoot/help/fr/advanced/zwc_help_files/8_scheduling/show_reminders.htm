Affichage des rappels

Vous pouvez créer des rappels de réunion. L'option par défaut est cinq minutes. Vous pouvez également choisir d'émettre un son avec le rappel. Si vous avez coupé le volume de votre ordinateur, vous n'entendrez aucun son. Sélectionnez Faire clignoter la barre de titre du navigateur si vous travaillez dans une fenêtre ou un onglet de navigateur différent et que vous choisissez de faire clignoter la barre de titre du navigateur pour vous rappeler.

  1. Ouvrez la page Préférences>Calendrier.

  2. Dans la section Général>Afficher les rappels, configurez combien de minutes avant les réunions le rappel doit vous être envoyé. Il s'agit de la configuration par défaut ; vous pouvez la modifier pour les réunions individuelles.

  3. Choisissez comment vous voulez être averti d'une réunion à venir. Sélectionnez l'une des options suivantes :

Option

Description

Émettre un son

Vous êtes averti par un signal sonore. Si vous avez coupé le volume de votre ordinateur, vous n'entendrez aucun son. Pour pouvoir entendre le signal sonore, le logiciel QuickTime ou Windows MediaCenter doit être installé.

Mettre en évidence l'onglet Mail

L'onglet Mail est mis en évidence si vous ne travaillez pas dedans à cet instant.

Afficher une notification

Des informations sur le mail s'affichent. Le logiciel Yahoo!BrowserPlus doit être installé sur votre ordinateur.

Pour en obtenir une copie, rendez-vous sur la page
http://browserplus.yahoo.com/
