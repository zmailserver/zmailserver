Suppression d'un dossier

Vous pouvez supprimer les dossiers définis par l'utilisateur mais pas les dossiers système. Lorsque vous supprimez un dossier, le dossier et son contenu sont déplacés vers la Corbeille. Si vous souhaitez sauvegarder certains messages dans un dossier avant de supprimer le dossier, déplacez les messages vers un autre dossier.

Vous pouvez récupérer un dossier et son contenu dans la Corbeille en le sélectionnant et en le faisant glisser vers un autre dossier.

Attention :  lorsque vous supprimez un dossier, une fenêtre contextuelle s'affiche en haut du panneau indiquant que le dossier est déplacé dans la corbeille. Cliquez sur Annuler si vous voulez annuler la suppression. Cette fenêtre contextuelle s'affichant un très court instant, cliquez immédiatement sur Annuler si vous ne voulez pas supprimer le dossier, sinon récupérez manuellement le dossier.

  1. Ouvrez la page Mail>Dossiers.

  2. Cliquez avec le bouton droit de la souris sur le dossier que vous voulez supprimer, puis sélectionnez Supprimer

Remarque :   le dossier Corbeille est vidé régulièrement selon les paramètres définis par l'administrateur, par conséquent les objets qu'il contient risquent d'être définitivement supprimés.
