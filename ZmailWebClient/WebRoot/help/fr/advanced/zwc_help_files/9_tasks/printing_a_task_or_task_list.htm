Impression d'une tâche ou d'une liste de tâches

Vous pouvez imprimer toutes les tâches de votre liste ou sélectionner des tâches spécifiques à imprimer.

  1. Ouvrez la page Tâches.

  2. Dans le panneau d'aperçu, sélectionnez la tâche ou la liste de tâches que vous souhaitez imprimer. Pour sélectionner plusieurs tâches, utilisez les cases à cocher à côté de la tâche.

  3. Dans le panneau de contenu, sélectionnez la liste de tâches ou les tâches à imprimer.

  4. Dans la barre d'outils Tâches, cliquez sur Imprimer et choisissez Imprimer la ou les tâches sélectionnées ou Imprimer le contenu du dossier des tâches.

  5. Dans la boîte de dialogue Impression, cliquez sur OK.
