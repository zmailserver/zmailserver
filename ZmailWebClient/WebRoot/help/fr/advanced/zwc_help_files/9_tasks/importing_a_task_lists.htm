Importation d'une liste de tâches

Vous pouvez importer des listes de tâches qui sont enregistrées dans un fichier .tgz.

  1. Ouvrez la page Préférences>Importer/Exporter.

  2. Dans Importer>Fichier, cliquez sur Parcourir pour localiser le fichier .tgz à importer.

  3. Pour la Destination, cliquez sur Tous les dossiers et sélectionnez la liste de tâches dans laquelle seront importées les tâches ou créer une nouvelle liste de tâches.

  4. Cliquez sur Importer.

Suivant la taille du fichier .tgz, l’importation peut prendre quelques minutes.
