Recherche d'éléments partagés avec vous

Il est possible que vous ne soyez pas notifié par mail qu'un objet est partagé avec vous, ou votre organisation peut disposer d'objets partagés avec le public que vous aimeriez trouver. Le lien Rechercher des partages vous permet de trouver des objets qui peuvent être partagés avec vous.

Le lien Rechercher des partages se trouve au bas de la section Dossiers dans le panneau d'aperçu.

  1. Cliquer sur Rechercher des partages.

  2. Dans la boîte de dialogue Ajouter dossier partagé, sélectionnez l'objet partagé avec vous que vous n'avez pas accepté, ou dans la zone de texte Rechercher des partages, entrez une adresse mail pour voir si un objet a été partagé avec vous à partir de cette adresse.

  3. Dans le menu déroulant Inclusion, sélectionnez l'application à rechercher. Sélectionnez Toutes les applications pour rechercher toutes les applications pour les objets partagés avec vous.

  4. Cliquez sur Rechercher.

  5. Sélectionnez les objets que vous souhaitez accepter.

  6. Cliquez sur Ajouter.
