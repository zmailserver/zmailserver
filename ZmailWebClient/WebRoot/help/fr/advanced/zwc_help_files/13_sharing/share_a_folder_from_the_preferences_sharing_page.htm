Partage d'un dossier depuis la page Préférences>Partage

  1. Ouvrez la page Préférences>Partage.

  2. Dans la section Partager un dossier, sélectionnez le type de dossier à partager à partir du menu déroulant.

  3. Cliquez sur Partager.

  4. Sélectionnez le dossier à partager et cliquez sur OK.

  5. Dans la boîte de dialogue Propriétés partagées, choisissez si vous voulez le partager avec des utilisateurs internes, externes ou publics.

  6. Selon le type d'utilisateur avec qui vous partagez le dossier, effectuez l'une des opérations suivantes :

    7. •    Pour les Utilisateurs et groupes internes, entrez les adresses mail des utilisateurs. Dans la zone Rôle, sélectionnez les privilèges d'accès.

    •    Pour les Invités externes, entrez les adresses mail des utilisateurs. Un mail contenant un lien vers l'objet partagé est envoyé aux invités externes. Ils créent un nom d'affichage et un mot de passe la première fois qu'ils voient l'objet partagé.

    •    Pour le Public, entrez l'URL à utiliser.

  7. Dans la zone Mail, sélectionnez le type de mail à envoyer aux personnes auxquelles vous accordez l'accès. Ce message peut être envoyé aux utilisateurs internes et aux invités externes. Remarque : si vous sélectionnez Public, vous devez notifier les utilisateurs publics de l'URL.

  8. Cliquez sur OK.
