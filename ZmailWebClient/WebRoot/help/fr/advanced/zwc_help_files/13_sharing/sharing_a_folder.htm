Partage d'un dossier

Vous pouvez partager des dossiers de messagerie, des carnets d'adresses, des calendriers et des dossiers de liste de tâches. Vous invitez un utilisateur à partager et l'utilisateur peut accepter ou refuser l'invitation.

Remarque :   lorsque vous partagez des dossiers de messagerie, vous ne pouvez pas partager les dossiers Corbeille ni Spams.

  1. Cliquez avec le bouton droit sur le dossier que vous souhaitez partager.

  2. Cliquez sur Partager dossier.

  3. Dans la boîte de dialogue Propriétés partagées, choisissez si vous voulez le partager avec des utilisateurs internes, externes ou publics.

  4. Selon le type d'utilisateur avec qui vous partagez le dossier, effectuez l'une des opérations suivantes :

    5. •    Pour les Utilisateurs et groupes internes, entrez les adresses mail des utilisateurs. Dans la zone Rôle, sélectionnez les privilèges d'accès.

    •    Pour les Invités externes, entrez les adresses mail des utilisateurs. Un mail contenant un lien vers l'objet partagé est envoyé aux invités externes. Ils créent un nom d'affichage et un mot de passe la première fois qu'ils voient l'objet partagé.

    •    Pour le Public, entrez l'URL à utiliser.

  5. Dans la zone Mail, sélectionnez le type de mail à envoyer aux personnes auxquelles vous accordez l'accès. Ce message peut être envoyé aux utilisateurs internes et aux invités externes. Remarque : Si vous avez sélectionné Public, vous devez notifier les utilisateurs de l'URL.

  6. Cliquez sur OK.

Si l'utilisateur accepte l'invitation à partager le dossier, une copie du dossier est placée dans son compte utilisateur.
