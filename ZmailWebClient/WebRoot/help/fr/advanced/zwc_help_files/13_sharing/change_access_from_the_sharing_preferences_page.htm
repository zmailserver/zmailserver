Modification de l'accès depuis la page Préférences>Partage

  1. Dans la section Dossiers partagés par moi de la page Partage, sélectionnez le nom de l'utilisateur à modifier.

  2. Dans la colonne Actions, vous pouvez modifier, retirer ou renvoyer un objet partagé.

Option

Description

Modifier

Pour ouvrir la boîte de dialogue Propriétés partagées et modifier le rôle.

Retirer

Pour ouvrir la boîte de dialogue Retirer le partage et annuler le dossier partagé.

Renvoyer

Pour renvoyer la notification de partage.
