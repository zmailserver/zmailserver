Gérer vos élément partagés

La page Préférences>Partage affiche les objets qui sont partagés avec vous et les objets que vous partagez. Vous pouvez également trouver des objets qui ont été partagés avec vous mais pour lesquels vous n'avez pas encore accepté le partage. La page est organisée en sections comme suit :

•      Dossiers partagés avec moi mais que je n'ai pas encore acceptés. Dans cette section, vous pouvez basculer entre deux vues :

•    Afficher la liste des dossiers partagés avec moi dans le cadre d'une liste de distribution. Cette option est la valeur par défaut. Si les dossiers sont partagés avec vous dans le cadre d'une liste de distribution, les champs sont remplis avec les détails des objets partagés.

•    Afficher la liste des dossiers partagés avec moi par l'utilisateur suivant. Lorsque vous sélectionnez cette option, vous devez spécifier le nom d'un utilisateur qui partage un objet avec vous. Si vous avez accepté l’invitation de partage, les détails sont affichés dans l’un des champs. Vous pouvez consulter les détails d’un seul utilisateur à la fois, et les détails ne sont pas enregistrés.

•      Dossiers partagés avec moi et que j'ai acceptés. Les dossiers qui sont partagés avec vous sont répertoriés dans cette section.

•      Dossiers partagés par moi. Cette section présente une liste de vos dossiers partagés, avec les détails concernant la façon dont le partage a été configuré, notamment le rôle que vous avez attribué à l'utilisateur et l'objet partagé. Vous pouvez modifier les propriétés d'un partage, retirer les autorisations ou renvoyer l'invitation.

•      Partager un dossier. Vous pouvez également partager les dossiers suivants : mails, carnet d'adresses, calendriers, tâches et porte-documents, à partir de cette section.
