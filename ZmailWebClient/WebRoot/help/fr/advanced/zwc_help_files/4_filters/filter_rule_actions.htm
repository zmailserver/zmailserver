Actions de règles de filtre

Une règle de filtre peut contenir une ou plusieurs actions. Si le mail correspond aux conditions spécifiées, toutes les actions sont appliquées dans l’ordre dans lequel elles apparaissent dans le filtre.

Les actions d’une règle de filtre peuvent comprendre ce qui suit :

•      Transférer le mail vers un dossier spécifié

•      Appliquer un tag ou un drapeau au mail

•      Supprimer le mail avant qu'il n'atteigne votre boîte de réception

•      Faire suivre le mail à l'adresse mail indiquée

Remarque : dans chaque règle de filtre, Ne pas appliquer d'autres filtres doit toujours être la dernière action. Elle empêche l'application d'autres règles de filtre aux mails entrants qui satisfont la règle actuelle. Si cette action n'est pas spécifiée, les règles de filtre ultérieures continuent à être appliquées aux mails, même si la ou les conditions du filtre actuel sont satisfaites. Vous ne voulez probablement pas qu'un même message corresponde à plusieurs règles de filtre et exécuter plusieurs actions éventuellement contradictoires.
