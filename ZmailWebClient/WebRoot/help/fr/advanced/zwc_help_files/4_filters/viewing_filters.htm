Affichage des filtres

Affichage des filtres existants

Vous pouvez afficher les filtres que vous avez définis et leur état.

  1. Ouvrez la page Préférences>Filtres.

  2.    Pour voir les informations sur un filtre en particulier, double-cliquez dessus.

Afficher l'ordre de filtre

Les filtres actifs sont appliqués dans l'ordre dans lequel ils sont listés. Vous pouvez modifier cet ordre.

  1. Ouvrez la page Préférences>Filtres.

  2. Utilisez les boutons Ascendant et Descendant pour modifier l'ordre.

Remarque :   un mail est testé par chaque filtre actif sauf si vous spécifiez qu'un message est traité uniquement par le premier filtre qui correspond. Généralement, une fois qu'il y a une correspondance, d'autres filtres ne sont pas appliqués car le mail peut subir plusieurs actions éventuellement contradictoires.
