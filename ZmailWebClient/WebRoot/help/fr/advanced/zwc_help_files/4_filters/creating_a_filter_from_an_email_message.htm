Création d'un filtre à partir d'un mail

Vous pouvez créer un filtre en vous basant sur un mail de votre boîte de réception. Les informations du filtre sont extraites du mail et vous pouvez alors modifier les conditions et les actions.

  1. Cliquez avec le bouton droit sur un mail et sélectionnez Nouveau filtre.

    2. La boîte de dialogue Ajouter un filtre apparaît remplie avec les informations du mail.

  2. Dans la zone de texte Nom du filtre, saisissez un nom unique pour le filtre.

  3. Modifiez les conditions et les actions comme vous le souhaitez à l'aide des menus déroulants.

  4. Cliquez sur OK.
