Filtrer à l'aide caractères de substitution

Les caractères de substitution peuvent être utilisés dans les comparaisons qui exécutent l'opérateur de comparaison « Modèle de correspondance ». Les deux caractères de substitution sont * et ?

Astérisque (*)

L'astérisque * est un paramètre de substitution pour « aucun ou plusieurs caractères de n'importe quel type ».

Exemple de recherche de caractères dans le champ d'objet : palette*bananes

avec Correspondances : Lignes d'objet telles que « palettebananes », « bananes », « palette de casserole de feuilles de bananes » mais pas « palette-super-bananes »

Exemple de recherche de caractères dans le champ d'objet : bandana*w

avec Correspondances : Lignes d'objet telles que « grand bandana » ou « gars avec le bandana », mais pas « gars avec le bandana de qui ? »

Point d'interrogation (?)

Le point d'interrogation est un paramètre de substitution pour « un caractère (et pas davantage !) ».

Exemple de recherche de caractères dans le champ d'objet : bateau?banane

avec Correspondances : Lignes d'objet telles que « bateaubanane », « bateau-banane », « bateau!banane » ou « bateau banane »

Le caractère d'échappement est la barre oblique : \

Vous pouvez vouloir spécifier une correspondance exacte pour une chaîne dont les caractères sont normalement considérés comme des caractères de substitution. Par exemple, si vous voulez spécifier une correspondance pour une ligne d'objet dans laquelle l'en-tête principal est entouré de trois astérisques, telle que

***MORE MONEY!!!*** ou

***FREE GIFT***

Pour spécifier un caractère de substitution en tant que tel plutôt qu'une substitution pour d'autres caractères, placez la barre oblique inverse (\) juste avant le caractère. Par exemple, le caractère de comparaison « \*\*\**\*\*\* » spécifie un sujet avec trois astérisques placés avant et après une chaîne de caractères (espaces compris).
