Conditions et actions de filtre prises en charge

Les règles de filtrage sont insensibles aux majuscules, ce qui signifie que les règles ne tiennent pas compte des majuscules.

Conditions

Les champs de comparaison incluent les éléments suivants:

  • De. À utiliser pour spécifier un nom dans l'en-tête De d'un mail.

  • À. Identique au champ De, mais la recherche des noms spécifiés s'effectue dans l'en-tête À: Cc:.

  • Cc. Identique au champ De, mais s'affiche dans l'en-tête Cc : .

  • À ou Cc. Identique au champ De, mais s'affiche dans l'en-tête À et Cc.

  • Objet. Recherche dans l'en-tête Objet du mail.

  • Taille. Utilisez cette option pour sélectionner les mails d'une taille supérieure ou inférieure à la taille spécifiée. La taille d'un mail inclut les pièces jointes. Vous pouvez utiliser cette option pour supprimer les mails trop volumineux.

  • Date. Utilisez cette option pour spécifier les mails envoyés avant ou après une date.

  • Corps du message. Les options de contenu sont Contient/Ne contient pas et vous pouvez spécifier les mots correspondants dans le corps du mail. Vous ne pouvez pas filtrer les mots des pièces jointes.

  • Accusé de lecture. Les mails d'accusé de lecture peuvent être filtrés.

  • Pièce jointe. Filtrez vos mails, qu'ils contiennent ou non des pièces jointes.

  • Adresse dans. Vérifie la présence d'une adresse dans vos contacts. Le champ suivant sert à indiquer l'adresse.

  • Invitation dans le calendrier. Filtre les mails relatifs au calendrier.

  • Social. Filtre les messages provenant de sites de réseautage social.

  • En-tête nommé. Lorsque cette option est sélectionnée, un champ d'entrée de texte supplémentaire apparaît avant l'opérateur de comparaison (la"partie" contient). Cette option vous permet d'indiquer l'en-tête mail de votre choix. Vous pouvez non seulement spécifier les champs standard À: ou De:, mais également les champs de Date, Répondre à et d'autres champs personnalisés qui peuvent être inclus dans l'en-tête du message. Vous pouvez utiliser cette option pour filtrer les e-mails qui ont des "en-têtes" mal formées, ce qui signifie qu'elles ne contiennent pas certaines informations qui sont normalement considérés comme standard pour un message e-mail. Parfois les spams (générés automatiquement) omettent des informations, telles que les champs Expéditeur et Répondre à. Utilisez la deuxième zone de texte (le champ situé immédiatement à droite de l'opérateur de comparaison) pour spécifier l'en-tête à tester.

Actions

  • Conserver dans la Boîte de réception. Enregistre les mails dans la boîte de réception. Si aucune règle de filtre ne correspond à un mail, cette action s'exécute par défaut.

  • Ranger dans un dossier. Transfère un mail vers un dossier défini.

  • Détruire. Supprime le mail sans le distribuer. Le mail ne fait pas partie de vos dossiers, même pas du dossier Corbeille.

  • Faire suivre. Fait suivre le mail à l'adresse indiquée.

  • Marquer. Sélectionnez comme lu ou avec drapeau.

  • Affecter d'une balise. Vous pouvez marquer les mails correspondants avec un tag sélectionné.

Actions multiples

Vous pouvez créer une règle de filtre constituée de plusieurs actions. Les combinaisons d'actions dans vos règles sont interprétées comme suit:

  • Détruire. Si elle est combinée avec d'autres actions, l'action de suppression est ignorée et les autres actions deviennent effectives.

  • Ranger dans un dossier. Plusieurs "fichiers" en action résultent en plusieurs copies du message étant stockées dans des dossiers différents. Si aucun dossier n'est spécifié, le mail sera conservé dans la boîte de réception.

  • Baliser/Marquer. Ces actions s'appliquent au mail envoyé par la dernière action exécutée. Dans le cas de plusieurs actions "Ranger dans", il peut arriver que plusieurs copies du mail soient sauvergardées sans tag, et d'autres avec.

  • Conserver dans la Boîte de réception. Plusieurs "actions de" maintient peuvent être spécifiées, mais une seule copie du message est sauvegardée dans la boîte de réception.

  • Faire suivre à l'adresse. Le mail est transféré vers l'adresse indiquée.
