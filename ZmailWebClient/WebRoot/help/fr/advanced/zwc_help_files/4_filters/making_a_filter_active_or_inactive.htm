Activation et désactivation d'un filtre

Vous pouvez activer ou désactiver un filtre. Si un filtre est inactif, il est ignoré. Si vous désactivez un filtre, vous le conservez pour une utilisation ultérieure plutôt que de le supprimer.

  1. Aller à la page Préférences>filtres.

  2. Cliquez sur Modifier filtre.

  3. Désélectionner la case Actif dans l'angle supérieur droit pour désactiver le filtre.

  4. Cliquez sur OK.
