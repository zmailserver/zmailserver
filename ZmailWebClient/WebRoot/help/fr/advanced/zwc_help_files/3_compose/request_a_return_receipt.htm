Demande d'accusé de réception

Les mails que vous envoyez à d'autres utilisateurs @ClientNameShort@ peuvent être marqués pour demander un accusé de réception lorsque le destinataire ouvre votre message. Cependant, les destinataires doivent avoir l'option Accusé de réception activée pour leurs comptes.

Pour définir cette option par défaut :

•       Ouvrez la page Préférences>Mail, et dans la section Créer des mails, sélectionnez Toujours demander un accusé de réception.

Pour définir cette demande par mail :

•       Avant d'envoyer le mail, sélectionnez Demander un accusé de lecture dans la liste déroulante Options.
