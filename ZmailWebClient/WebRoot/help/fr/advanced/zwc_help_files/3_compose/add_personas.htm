Ajout d'avatars

La création de plusieurs avatars vous permet d'utiliser plusieurs adresses mail dans votre boîte aux lettres. Par exemple, vous pouvez utiliser votre avatar de compte principal pour vos correspondances mail professionnelles, et créer un nouvel avatar pour vos correspondances mail privées.

  1. Aller à la page Comptes des >préférences. Votre compte par défaut et vos informations personnelles sont affichées comme compte principal.

  2. Cliquez sur Ajouter un avatar. Le nom de compte Nouvelle personnes'affiche dans la colonne Nom du compte et dans la zone de texte >Paramètres de la personneNom de la personne.

  3. Dans la zone de texte Noms des personnesentrez un mot descriptif pour identifier la personne dans la liste de l'expéditeur lorsque vous composez un courrier électronique. Ce nom n'apparaît pas dans le mail.

  4. Dans la section Paramètres des messages envoyés, précisez les informations du champ De pour cet avatar.

Dans la zone de texte De, saisissez le nom qui apparaît dans le champ De de vos mails sortants. Il s'agit du nom qui s'affiche devant votre adresse mail.

Dans le menu déroulant situé à côté de la zone de texte, sélectionnez l'adresse mail à partir de laquelle le mail doit être envoyé. Si le champ n'est pas éditable, vous n'avez pas de compte externe supplémentaire identifié.

  1. Pour adresser les réponses aux e-mails de cette personne à un nom et une adresse différente de celle que vous avez configurée dans l'expéditeursélectionner Définir le champ "Répondre à" des messages de courrier électronique à, puis entrez le nom à utiliser dans la zone de texte.

  1. (Facultatif) Pour associer une signature à l'avatar, cliquez sur Signature.

  2. Pour utiliser automatiquement cette personne lorsque vous répondez aux messages envoyés à une adresse email spécifique ou ou lorsque vous retransmettez les messages provenant de cette adresse, sélectionner Lorsque je réponds ou que je retransmet des messages envoyés à, et tapez l'adresse e-mail dans la zone de texte. Si vous entrez plus d'une adresse mail, séparez les adresses par une virgule ou un point-virgule.

  3. Pour utiliser automatiquement cette personne lorsque vous répondez aux messages dans un dossier spécifique ou lorsque vous retransmettes les messages à partir de ce dossier, sélectionner Répondre à ou retransmettre des messages dans le/les dossier(s). Cliquer sur l'icône dossier pour sélectionner un ou plusieurs dossiers ou pour créer un nouveau dossier.

Si vous sélectionnez plus d'un dossier, séparez les noms de dossier par une virgule ou un point-virgule.
