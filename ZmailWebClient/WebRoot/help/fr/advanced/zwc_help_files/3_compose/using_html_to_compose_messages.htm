Utilisation de HTML pour rédiger des mails

Les messages créés avec l'éditeur HTML peuvent avoir un formatage, une numérotation, des puces, des fonds colorés, des tableaux et des liens.  Cependant, certains clients de messagerie n'acceptent pas les mails formatés en HTML. Vous pouvez changer l'éditeur pour un mail en particulier.

Remarque :   vous n'avez pas besoin de connaître l'HTML pour utiliser l'éditeur HTML.

Pour changer le format du mail que vous envoyez, choisissez Format HTML ou Format texte simple dans Options.
