Recherche avec des mots-clés

Si vous savez où rechercher votre élément, vous pouvez entrer des mots clés suivis par deux points et rechercher l'élément dans le champ Rechercher.

Mot clé

Description
contenu :

Spécifie le texte que le message doit contenir. Par exemple, content:bananes recherche tous les éléments contenant le mot "bananes".

de :

Spécifie un nom d'expéditeur ou l'adresse e-mail qui est dans l'en-tête De. Ce peut être du texte, comme dans "John Smith III", une adresse e-mail telle que " joe@acme.com", ou un domaine tel que " zimbra.com".

à :

Identique à from: sauf qu'il spécifie l'une des personnes à qui le courriel a été adressé dans le champ À : .

cc :

Identique à from: sauf qu'il spécifie un destinataire dans le Cc : Cc: du mail.

objet :

Spécifie le texte qui doit apparaître dans l'en-tête de l'Objet du message. Un exemple pourrait être le sujet : nouvelle politique de vacanze.

dans :

Spécifie un dossier. Par exemple,dans : envoyémontre tous les éléments dans votre dossier Envoyer.

a :

Spécifie un attribut que le mail doit posséder. Les types d' objet que vous pouvez spécifier sont "pièce jointe", "téléphone" ou " url". Par exemple,a :pièce jointe trouve tous les messages qui contiennent une ou de plusieurs pièces jointes de tout type.

Nom de fichier :

Spécifie le nom d'une pièce jointe. Par exemple, nom de fichier : query.txt trouve des messages avec une pièce jointe nommée "query.txt".

type :

Spécifie une recherche dans les pièces jointes d'un type spécifique. Les types de pièces jointes que vous pouvez spécifier sont "text", "word", "excel" et " pdf". Par exemple, type : motbonjourtrouve des messages avec pièces jointes qui sont des documents Microsoft Word et recherche à l'intérieur de ces pièces jointes pour le mot "bonjour".

apièce jointe:

Spécifie tout élément d'un certain type de pièce jointe. Par exemple, pièce jointe : Word trouve tous les messages avec une pièce jointe Word.

Remarque : Si vous n'avez pas trouvé une pièce jointe qui a été écrite dans une langue non-occidentale, voir Rechercher les problèmes avec des pièces jointes de langue non-occidentale.

est :

Recherche les mails avec un statut spécifique. Les valeurs autorisées sont "non lu", "lu", "marqué", " non marqué", "envoyé", "ébauche", "reçu", "répondu", " non répondu", "transmis", non transmis", "partout", "à distance" (dans un dossier partagé), "local", "envoyé", et "en solo"

Par exemple,

est : non lutrouve tous les messages non lus.

est : en solo trouve des e-mails qui n'ont pas une réponse à eux pour le moment.

L'état menu déroulant dans les filtres avancés comprend ces options.

date :

Utiliser ce mot-clé pour spécifier une date. Vous pouvez utiliser une date absolue (jj/mm/aaaa ou aaaa/jj /mm) ou une date relative (+/- nnnn [heure, jour, semaine, mois, année]). Utiliser le format qui est par défaut sur ​​votre navigateur. Par exemple, format anglais des États-Unisdate :2/1/2012trouve les messages en date du 1 Février 2012.

Le symbole supérieur à (date :>DATE) est le même que après : DATE. Le symbole inférieur à (date :>DATE) est le même queavant : Date.

après :

Spécifie un mail envoyé après une certaine date. Par exemple, après : 2/1/2010précise les e-mail envoyés après le 1er Février 2010.

avant :

Même que après : sauf précise les e-mails envoyés avant la date spécifiée.

taille :

Spécifie les e-mail dont la taille totale, y comprises les pièces jointes, est un nombre spécifié d'octets, kilo-octets, ou méga-octets Par exemple, taille :12 kotrouve les e-mails qui sont exactement dans la taille 12K. Les symboles supérieur à (>) ou inférieur à (<) peuvent être utilisés à la place de bigger ou smaller.

plus grand :

Identique à size: hormis le fait qu'il permet de spécifier une taille supérieure à celle indiquée.

plus petit :

Identique à size: hormis le fait qu'il permet de spécifier une taille inférieure à celle indiquée.

balise :

Recherche les mails marqués par un tag spécifique. Par exemple, balise :ambretrouve les messages qui ont une balise appelée" "ambre"" appliquée.
