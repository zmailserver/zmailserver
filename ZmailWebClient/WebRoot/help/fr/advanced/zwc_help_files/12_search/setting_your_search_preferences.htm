Configuration de vos préférences de recherche

Par défaut, vos spam, corbeille et dossiers d'éléments partagés ne sont pas recherchés. Vous pouvez activer ces préférences. Pour afficher la syntaxe de requête de recherche dans le champ de recherche, vous pouvez activer la préférence de langue de recherche.

  1. Aller à la page Préférences>générale.

  2. Sélectionner Paramètres de recherche>Inclure un dossier spam dans les recherches, Inclure une corbeille dans les recherches, et inclure des éléments partagés dans les recherchespour inclure ces dossiers dans toute recherche que vous effectuez.

  3. Sélectionner Langue de recherche>Afficher les fonctions évoluées dans la barre d'outils de recherchepour afficher les détails chaîne de recherche dans la zone de texte de recherche en haut de la page. Par défaut, cette option n'est pas sélectionnée.

  4. Cliquez sur Enregistrer.
