Effacement des données de votre périphérique mobile

Si votre mobile est perdu ou volé, vous pouvez en effacer les données personnelles à l'aide du lien Supprimer le mobile, sur la page Préférences des mobiles. Cette fonction efface les données stockées dans le périphérique et rétablit les paramètres d'origine définis en usine.

  1. Ouvrez la page Préférences>Mobiles.

  2. Sélectionnez le périphérique à effacer et cliquez sur Effacer le contenu de ce mobile.

L'état Suppression en instance indique que le serveur attend que le périphérique soit connecté. Une fois le périphérique supprimé, l'état devient Suppression terminée.

Si vous trouvez le périphérique alors que l'état est Suppression en instance, vous pouvez annuler la demande de suppression.

N'exécutez pas cette commande pour nettoyer le périphérique que vous possédez toujours.
