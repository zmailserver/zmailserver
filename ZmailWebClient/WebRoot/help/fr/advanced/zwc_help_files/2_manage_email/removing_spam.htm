Suppression des spams

Le filtrage de la plupart des mails automatiques non sollicités (également appelés spam ou courrier indésirable) est effectué par un filtre anti-spam avant que ces mails ne soient délivrés dans votre Boîte de réception.

Les mails pouvant être considérés comme spam (mais avec le bénéfice du doute) sont placés dans le dossier Spams. Vous pouvez consulter ces mails, les déplacer hors du dossier Spams ou les supprimer. Si vous ne déplacez ou ne supprimez pas ces mails, ils seront purgés après un délai défini par votre administrateur.

Si vous recevez un nombre important de spams, contactez votre administrateur. Le fait de signaler des mails comme indésirables permet à votre administrateur de messagerie d'affiner le filtrage des spams.

Marquer le mail comme spam

  1. Sélectionnez le mail à marquer comme spam.

  2. Dans la barre d'outils, sélectionnez Spam.

Vous pouvez également effectuer double-cliquer sur le message et sélectionner Marquer comme Spam.

Le mail est déplacé vers le dossier Spams.

Marquer le mail comme non spam

Il peut arriver que des mails qui ne sont pas du courrier indésirable soient placés dans le dossier Spams. Vous pouvez déplacer ces mails dans le dossier approprié.

•      Faites glisser le message vers un autre dossier.

•      Cliquez avec le bouton droit sur le message et sélectionnez Marquer comme non spam. Le mail sélectionné est placé dans votre boîte de réception.

Vider le dossier Spams

Le dossier Spams est vidé automatiquement au bout d'un certain délai défini par l'administrateur. Vous pouvez vider le dossier Spams à tout moment.

  1. Cliquez avec le bouton droit de la souris sur le dossier Spams et sélectionnez Vider le dossier Spams.

  2. Cliquez ensuite sur OK.
