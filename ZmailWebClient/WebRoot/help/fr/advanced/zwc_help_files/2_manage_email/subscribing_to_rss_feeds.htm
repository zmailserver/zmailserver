S'abonner au fils RSS

Vous pouvez vous inscrire aux sites Web qui vous fournissent vos fils RSS/podcast favoris et envoyer des informations mises à jour directement vers votre boîte aux lettres. Vous devez connaître l'URL du fil pour configurer votre abonnement.

  1. Cliquez sur l'icône représentant un engrenage dans le titre Dossiers et sélectionnez Nouveau dossier.

    2. Remarque :  Vous devez créer un dossier séparé pour chaque filRSS.

  2. Dans la zone de texte Nom, saisissez le nom du dossier de fil RSS.

  3. Cochez S'inscrire à un fil RSS/Atom.

  4. Dans le champ URL, saisissez l’URL du fil.

  5. Sélectionnez l'emplacement où il faut ajouter le dossier de fil RSS dans le menu Dossier. Si vous choisissez l'onglet racine, le nouveau dossier sera placé dans la partie supérieure de votre affichage de dossiers, au même niveau que votre dossier Réception.

  6. Cliquez sur OK. Le nouveau dossier apparaît dans le panneau d’aperçu avec l'icône de fil RSS .

Votre fil de données RSS est automatiquement mis à jour toutes les 12 heures. Cliquez sur Charger fil RSS dans la barre d'outils pour mettre à jour le contenu à tout moment.

Remarque :   Les fils RSS que vous enregistrez sont décomptés de votre quota de compte.
