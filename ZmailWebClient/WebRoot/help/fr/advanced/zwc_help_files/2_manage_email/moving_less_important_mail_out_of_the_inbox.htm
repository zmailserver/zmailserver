Déplacer automatiquement les mails moins importants en dehors de la boîte de réception.

Vous pouvez configurer le filtre de flux d'activité afin que les mails moins importants, tels que les newsletters et les notifications des réseaux sociaux, soient acheminés vers le dossier de flux d'activité plutôt que vers votre boîte de réception. Le dossier de flux d'activité est ajouté à la liste de vos dossiers personnels dans le panneau d'aperçu.

  1. Cliquez sur l'icône représentant un engrenage dans le titre Dossiers du panneau de navigation et sélectionnez Flux d'activité.

    2. Vous pouvez également configurer cette fonction à partir du lien Paramètres de flux d'activité sur la page Préférences>Filtres.

  2. Dans la boîte de dialogue Flux d'activité, sélectionnez Activer le transfert des messages vers le flux d'activité.

  3. Sélectionnez le type de mails à déplacer. Vous pouvez déplacer les types de mails suivants :

    4. •    Messages où votre adresse ne se trouve pas dans le champ À ou À ou Cc

    •    les mails provenant de personnes qui ne figurent pas dans votre carnet d'adresses ;

    •    les mails envoyés à une liste de distribution dont vous êtes membre ;

    •    les messages publicitaires et électroniques envoyés en masse.

  4. Cliquez sur OK.

  5. Dans la zone d'informations qui s'ouvre, cliquez sur Oui.

Pour plus d'informations, consultez la rubrique Filtres.
