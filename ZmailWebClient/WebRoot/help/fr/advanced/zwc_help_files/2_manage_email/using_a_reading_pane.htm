Utilisation d'un panneau de lecture

Vous pouvez utiliser le panneau de lecture pour afficher un aperçu des mails présents dans votre boîte de réception et dans d'autres dossiers. Si le panneau de lecture est activé, lorsque vous cliquez sur un mail, celui-ci s'affiche dans le panneau de lecture et est marqué comme lu.

Pour régler la taille du panneau de lecture, placez le pointeur sur la ligne de séparation puis faites-la glisser.

Activer ou désactiver le panneau de lecture

Le panneau de lecture vous permet de lire vos mails dans votre boîte de réception sans les ouvrir. Vous pouvez afficher le panneau de lecture au bas de la page, à droite de la page ou le désactiver.

  1. Dans la barre d'outils de l'onglet Mail, cliquez sur le menu déroulant Affichage.

  2. Sélectionnez l'option Panneau de lecture.

Configurer le panneau de lecture pour définir le comportement de l'option Marquer comme lu.

Vous pouvez également choisir de marquer ou de ne pas marquer comme lus les mails que vous affichez dans le panneau de lecture.

  1. Ouvrez la page Préférences>Mail.

  2. Dans la section Affichage des mails>Panneau de lecture, choisissez la manière dont les mails sont gérés lorsqu'ils sont sélectionnés.

  3. Cliquez sur Enregistrer.
