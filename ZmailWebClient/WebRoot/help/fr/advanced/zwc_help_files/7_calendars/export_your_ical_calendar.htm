Exportation de votre calendrier iCal

Vous pouvez exporter les calendriers au format .ics.

  1. Ouvrez la page Préférences>Importer/Exporter.

  2. Dans Exporter, sélectionnez le type Calendrier. Vous exportez vos rendez-vous au format .ics standard.

    3. Pour les importer dans un autre programme de calendrier, consultez la documentation de ce dernier.

  3. Dans Source, sélectionnez le calendrier à exporter.

  4. Cliquez sur Exporter. Sélectionnez Enregistrer sur disque et cliquez sur OK.

  5. Sélectionnez où enregistrer le fichier et saisissez un nom de fichier.

  6. Cliquez sur Enregistrer.
