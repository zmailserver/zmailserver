Création d'un calendrier

Un calendrier est créé pour vous. Vous pouvez créer des calendriers supplémentaires pour suivre différents types de fonctions. Par exemple, vous pouvez créer un calendrier pour le travail, un pour les rendez-vous personnels et un pour les activités de la société.

Si vous gérez plusieurs calendriers, vous pouvez les regrouper dans un affichage hiérarchique. Par exemple, vous pouvez regrouper tous les calendriers partagés sous un dossier Calendrier que vous appelez Partagé. Vous pouvez également déplacer le calendrier

•       Dans le panneau d'aperçu du calendrier, sélectionnez un calendrier et faites-le glisser sur le calendrier dans lequel il doit être imbriqué.

•       Vous pouvez aussi cliquer avec le bouton droit de la souris sur le calendrier à déplacer et sélectionner Déplacer. Dans la boîte de dialogue Sélectionnez dossier, sélectionnez le calendrier à déplacer.

Ajouter un calendrier

  1. Cliquez sur l'onglet Calendrier.

  2. Dans la barre d'outils, sélectionnez Nouveau>Calendrier.

  3. Dans la zone de texte Nom, saisissez le nom du calendrier.

  4. Sélectionnez une couleur d'affichage des activités du calendrier.

  5. Afin d'empêcher d'autres personnes de voir votre calendrier lorsque vous planifiez des rendez-vous, sélectionnez Exclure ce calendrier pour le rapport des disponibilités.

  6. Pour synchroniser ce calendrier avec un calendrier distant, sélectionnez Synchroniser les rendez-vous avec le calendrier distant. Entrez l'URL du fichier iCal distant.

  7. Cliquez sur OK. Le nouveau calendrier s'affiche dans la liste des calendriers.
