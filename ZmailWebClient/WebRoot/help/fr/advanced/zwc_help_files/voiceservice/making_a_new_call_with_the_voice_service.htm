Faire un nouvel appel avec le service vocal

Remarque :  cette fonctionnalité est pour ZCS Network Edition uniquement et est disponible uniquement dans le client Web avancé.

Vous pouvez faire des appels de tous les appareils téléphoniques enregistrés, y compris les clients de téléphone doux, tels que Cisco ou Mitel.

  1. Sur la page vocale, cliquer sur Nouvel appel. Une nouvelle boîte de dialogue d'appel s'affiche.

  2. Sélectionner le numéro de l' Expéditeur à partir du menu déroulant. Les numéros énumérés sont ceux qui sont enregistrés pour vous.

  3. Entrez ou sélectionner le numéro du Destinataire pour lequel vous souhaitez effectuer l'appel. Le numéro peut être obtenu à partir de vos carnets d'adresses ou de cartes de contact, ou vous pouvez entrer un numéro manuellement.

  4. Cliquer sur OK pour effectuer l'appel. Utilisez le dispositif de téléphone associé au numéro de téléphone que vous utilisez pour effectuer l'appel

Fin d'un appel

Pour terminer un appel, raccrochez le téléphone ou cliquez dans la boîte de dialogue pour raccrocher.
