Travailler dans le Porte-documents

Le Porte-documents peut être utilisé pour partager et gérer les documents que vous créez dans votre porte-documents, ainsi que les documents et les images que vous y placez. Utilisez le Porte-documents pour enregistrer des fichiers dans votre compte afin d'accéder à ces fichiers lorsque que vous vous connectez à votre compte depuis n'importe quel ordinateur.

Vous pouvez partager les dossiers du Porte-documents avec d'autres personnes qui peuvent consulter et modifier les fichiers dans le Porte-documents partagé.

Vous pouvez créer plusieurs dossiers dans le Porte-documents, puis uploader tout type de fichier de votre ordinateur ou de votre réseau, enregistrer les pièces jointes envoyées avec vos mails et créer de nouveaux documents.

Les fichiers dans le Porte-documents sont des copies des fichiers originaux, que le fichier se trouve dans votre ordinateur ou en pièce jointe de votre mail. Les modifications apportées aux fichiers de votre Porte-documents ne modifient pas le fichier original.

Création de dossiers de Porte-documents

  1. Ouvrez Porte-documents>Dossiers.

  2. Cliquez sur l'icône d'engrenage et sélectionnez Nouveau Porte-documents.

  3. Entrez un Nom pour le nouveau dossier de Porte-documents.

    4. Le nom du dossier de Porte-documents doit être unique dans la hiérarchie de tous les dossiers de votre boîte aux lettres. Ce nom ne peut correspondre à aucun dossier de premier niveau dans votre boîte aux lettres, votre calendrier ou votre carnet d'adresses. Par exemple, si l'un de vos calendriers s'appelle Vacances, vous ne pouvez pas nommer un porte-documents de premier niveau du même nom. Vacances peut être le nom d'un porte-documents dans un autre porte-documents.

  4. Sélectionnez la Couleur du nouveau fichier de porte-documents.

  5. Sélectionnez la hiérarchie du dossier, soit sous Dossiers soit sous un autre dossier de Porte-documents. Vous pouvez créer un porte-documents de premier niveau ou l'insérer dans un porte-documents existant.

  6. Cliquez sur OK.

Uploader des fichiers à partir du Porte-documents

  1. Ouvrez l'onglet Porte-documents et sélectionnez le porte-documents où enregistrer le fichier.

  2. Cliquez sur Uploader le fichier et Parcourir pour trouver le fichier à uploader.  

  3. Dans le champ Remarques, ajoutez les informations concernant le fichier. Ces informations s'affichent dans la liste de fichiers.

  4. Pour ajouter un autre fichier, cliquez sur Ajouter.

    5. Vous ne pouvez pas uploader de fichier portant le même nom dans le même porte-documents. Si un fichier qui existe déjà dans le dossier est uploadé, une boîte de dialogue Conflit d'upload s'affiche. Vous pouvez cliquer sur Oui pour uploader le fichier et écraser le fichier existant, ou cliquer sur Non.

    Le numéro de version est mis à jour si un fichier est écrasé.

  5. Lorsque tous les fichiers sont uploadés, cliquez sur OK.

Remarque :   si vous avez sélectionné le mauvais porte-documents, vous pouvez utiliser glisser-déposer pour déplacer le fichier vers le bon dossier Porte-documents.

Si le fichier est trop volumineux, un avertissement s'affiche. Les fichiers que vous uploadez ont une incidence sur le quota de votre compte.
