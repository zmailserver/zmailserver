Renommer un fichier

Le changement de nom d'un fichier crée une autre version du fichier.

  1. Ouvrez l'onglet Porte-documents et sélectionnez le document à renommer.

  2. Cliquez avec le bouton droit sur le fichier, puis sélectionnez Renommer. Le champ Nom est changé en champ modifiable.

  3. Modifiez ce champ pour changer le nom de fichier. N'oubliez pas de conserver la même extension dans le nom de fichier sinon vous ne pourrez plus le modifier plus tard.

  4. Appuyez sur Entrée.
