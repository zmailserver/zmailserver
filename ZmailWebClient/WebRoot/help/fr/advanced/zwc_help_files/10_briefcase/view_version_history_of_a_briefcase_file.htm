Affichage de l'historique de version d'un fichier du Porte-documents

Le contrôle de version permet de gérer les changements apportés aux documents du porte-documents. Chaque fois que vous modifiez un fichier et que vous l'enregistrez, le numéro de version est mis à jour. Le dernier numéro de version figure dans la liste des documents.

•       Cliquez sur la flèche dans le panneau d'aperçu pour voir l'historique de version et les auteurs. Si le panneau d'aperçu est désactivé, vous verrez une colonne avec le numéro de version.

Restaurer un document à la version antérieure

•       Cliquez avec le bouton droit de la souris sur une autre version et sélectionnez Restaurer comme version actuelle.
