Accès à vos comptes externes

Au lieu de vous connecter à chacun des comptes de messagerie externes, vous pouvez récupérer les mails de votre compte directement à partir de @ClientNameShort@. Lorsque vous configurez votre accès à d'autres comptes mails externes, vous pouvez spécifier l'emplacement de sauvegarde des mails et créer l'avatar que vous voulez utiliser lorsque vous répondez ou transférez un mail reçu depuis ce compte externe.

Les messages provenant de votre compte externe ne sont pas automatiquement reçus dans votre compte @ClientNameShort@. Cliquez sur l'icône Actualiser dans la barre d'outils pour recevoir les derniers mails.

Si vous ajoutez un compte POP externe, vous pouvez configurer le comportement d'accès POP lorsque des mails sont téléchargés sur votre compte.

Ajouter des comptes externes

  1. Aller à la page Comptes des >préférences.

  2. Cliquez sur Ajouter un compte externe. Le nom du compte Nouveau accompte externes'affiche dans la colonne et dans la zone de texte Paramètres du compte>Nom du compte. Vous pouvez le modifier.

  3. Dans la section Paramètres du compte externe, entrez l'adresse mail du compte externe.

  4. Dans la zone de texte Nom du comptesaisir un mot descriptif pour identifier ce compte. Il est utilisé pour identifier le compte externe.

  5. Sélectionnez le type de compte : POP ou IMAP.

  6. Dans la zone de texte Nom d'utilisateur pour ce compte , saisissez le nom associé à ce compte de messagerie. Parfois votre nom d'utilisateur correspond à la partie de votre adresse mail précédant le caractère @. Dans d'autres cas, il s'agit de votre adresse complète.

  7. Dans la zone de texte Serveur de messagerie, entrez le nom de votre serveur de compte externe.

  8. Dans la zone de texte Mot de passe, entrez le mot de passe à utiliser pour vous connecter à ce compte externe. Pour vérifier que le mot de passe est correct, sélectionnez Afficher mot de passe.

  9. Si votre fournisseur de service utilise un port différent de celui défini par défaut, modifiez-le dans la section Paramètres avancés. Si vous utilisez une connexion sécurisée, activez Avec ce serveur, utiliser une connexion cryptée (SSL).

    10. Votre fournisseur doit vous prévenir si votre compte n'utilise pas un port standard IMAP (143) ou POP (110).

  10. Cliquez sur Tester les paramètres. Les informations sont vérifiées.

  11. Sélectionner le dossier à partir duquel les messages doivent être téléchargés.

  12. Pour supprimer des mails du compte externe après leur réception dans votre compte, cochez Supprimer les mails du serveur après les avoir téléchargés.

  13. Configurer le nom de l'expéditeur, répondre aux paramètres d'adresse et de signatures pour les messages que vous envoyez.

  • Saisissez le nom qui doit s'afficher dans le champ De de vos mails sortants (par exemple, John). Il s'agit du nom qui s'affiche devant votre adresse mail.

  • Pour adresser les réponses aux e-mails de cette personne à un nom et à une adresse différente de celle qui vous avez configuré dans Sélectionner expéditeur Configurer le champ "Répondre à" des messages e-mail à. Saisir le nom dans la zone de texte.

  • Pour associer une signature avec la personne, cliquer sur Signatureet ajouter une nouvelle signature.
