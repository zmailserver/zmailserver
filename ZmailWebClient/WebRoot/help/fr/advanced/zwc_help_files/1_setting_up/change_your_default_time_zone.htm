Modification de votre fuseau horaire par défaut

Le réglage du fuseau horaire par défaut qui est affiché dans le le dossier Préférences générales>est utilisé pour localiser le temps pour les messages reçus et les activités du calendrier dans le client Web standard.

En utilisant le client Web standard, le fuseau horaire sur l'ordinateur ne sert pas à définir l'heure à laquelle un message est reçu ou pour les activités du calendrier ; il s'agit de la configuration du fuseau horaire dans l'onglet Préférences>Général. Cependant, en utilisant le client Web avancé, la configuration du fuseau horaire de l'ordinateur sert d'horodatage pour les messages reçus et pour les activités du calendrier, pas la configuration du fuseau horaire sur l'onglet Général.

Le fuseau horaire de votre compte peut être différent du fuseau horaire de votre ordinateur.

  1. Aller à la page Préférences>générales.

  2. Dans le menu déroulant des Options de connexion>par défautdu fuseau horairesélectionner le fuseau horaire que vous souhaitez utiliser.

  3. Cliquez sur Enregistrer.

Remarque: étant donné que le client Web avancé et le client Web standard n'utilisent pas la même source de fuseau horaire pour indiquer les messages, vous pouvez remarquer que le même message comporte une heure différente quand il est affiché sur l'un ou l'autre des clients. Pour éviter cela, réglez le fuseau horaire de l'ordinateur et le fuseau horaire du client Web à la même heure.
