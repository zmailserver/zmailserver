タスクを追加する

新しいタスクは、単にタスクリストにタスク名を入力して作成するか、タスクページを開き、備考や添付ファイルなどを含め、タスクに関する詳細情報を追加して作成できます。 タスクページを開くと、アプリケーションのツールバーに[タスク]タブが表示され、内容ペインでタスクフォームが開きます。複数の[タスク]タブを開いて、アプリケーションのツールバーでタブ間を移動することができます。

タスクをすばやく追加するには

  1. 概要ペインでタスクリストを選択します。

  2. 内容ペインの「新しいタスクを追加するには、ここをクリック」と表示されている先頭の行をクリックします。    

  3. テキストフィールドが表示されます。タスクの名前を入力します。

  4. Enterキーを押します。そのタスクがタスクリストに追加されます。

備考：間違ったリストにタスクを作成した場合は、そのタスクを開き、[タスクリスト]ドロップダウンから正しいリストを選択するか、内容ペインから正しいリストにそのタスクをドラッグアンドドロップします。

タスクを詳細情報とともに追加するには

タスクリストを使用して、タスクを作成したり、タスクの進行状況を管理したりできます。また、タスク内に備考を記入することや、ファイルを添付してタスク内から簡単にアクセスできるようにすることも可能です。

開始日と期限を入力してプロジェクトの期間を予測したり、優先度を設定したりできます。また、タスクを開始するときには、[開始前]や[進行中]などのステータスを選択したり、パーセント値による完了度を指定したりできます。

  1. 概要ペインでタスクリストフォルダを選択します。

  2. タスクツールバーの[新規]をクリックします。 無題のタスクページが開き、ツールバーに[タスク]タブが表示されます。

  3. [詳細]領域で以下の設定を行います。

    • [件名]フィールドに、タスクの名前を入力します。必要に応じて、[場所]フィールドに場所を入力します。

    • [優先度]から[標準]、[低]、または[高]を選択します。

    • 目的のタスクリストを選択する前に[新規]をクリックした場合でも、[タスクリスト]ドロップダウンリストから正しいタスクリストを選択できます。

  4. [進行状況]領域で、[開始日]、[期限]を選択します。タスクが開始されている場合は、[ 開始前]をリスト内のオプションに変更して、完了度（％）を更新できます。

  5. リマインダー日時を設定します。[プリファレンス]>[通知]ページでメールアドレスまたはSMSアラートを設定している場合は、[メール]または[SMS]の該当するボックスをオンにします。 通知を設定していない場合は、[設定]をクリックして、[プリファレンス]の[通知]ページにアクセスして設定します。

  6. テキストフィールドに備考を記入します。また、添付ファイルを追加する場合は、ツールバーの[添付ファイルを追加]をクリックします。

  7. [保存]をクリックしてタスクを保存します。その新しいタスクがタスクリストに追加されます。
