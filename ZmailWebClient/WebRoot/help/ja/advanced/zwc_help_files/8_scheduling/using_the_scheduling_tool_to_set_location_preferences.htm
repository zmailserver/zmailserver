予定ツールを使用して場所の空き状態を表示する

カレンダー予定ツールは、組織全体の次に空いている予定のない時間および会議の場所を検索するのに役立ちます。優先される会議時間、建物、場所および部屋のサイズのプリファレンスを設定できます。

参加者の名前を入力すると、予定ツールが概要ペインに時間と場所を提案してくれます。また、参加者を追加する前に会議の場所を探すオプションもあります。場所のプリファレンスを設定している場合は、それらの場所だけが表示されます。

予定ツールを使用して場所のプリファレンスを設定する

特定の場所にある部屋をいつも予約したい場合は、カレンダーアプリケーションの[提案済みの場所]ツールを使用し、予約作成時にこれらの部屋の空き状態を提案することができます。

  1. [新しい予定ページで、[場所を提案]をクリックします。

  2. 開いたペインで、歯車アイコンをクリックします。

  3. [提案のプリファレンス]ダイアログボックスの[場所のプリファレンス]領域で、場所に関する情報を入力します。場所の名前だけを入力することも、さらに詳しい情報を入力することもできます。

  4. [OK]をクリックします。

右側の列には、要件に合った場所が表示されます。部屋の場所は、社内ディレクトリで設定されている場合に表示されます。

一度予約についてこれを設定すると、この情報が保存され、他の予約をするときに利用可能となります。
