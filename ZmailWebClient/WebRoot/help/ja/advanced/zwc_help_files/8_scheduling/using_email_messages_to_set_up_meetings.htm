メールメッセージを使用して会議を設定する

会議要求は、メッセージ、スレッドをミニカレンダー上の日付にドラッグアンドドロップすることによってすばやく作成できます。メッセージやスレッドをドラッグアンドドロップすると、そのメッセージ内の情報を使用して[予約]ページ内の多くのフィールドに情報が自動的に記入されます。

  1. 特定のメッセージまたはスレッドをクリックし、それをミニカレンダー上の日付にドラッグします。

  2. [参加者を追加]ダイアログが開きます。多くのフィールドにメッセージの情報が入力された状態で予約ページが開きます。

    3. •    予約の件名がメッセージの件名になります。

    •    メッセージ（またはスレッドの場合はスレッド内の最も新しいメッセージ）の[宛先:]欄と[Cc:]欄のすべてのメールアドレスが参加者になります。連絡先リストの連絡先をミニカレンダーにドラッグすると、その連絡先の最初のメールアドレスが参加者フィールドに追加されます。

    •    メッセージのテキストは、招待のテキストになります。メッセージの添付ファイルは予約要求に添付されません。

  3. 予約ページで、会議要求の詳細を見直し、必要な変更を加えます。

  4. 会議の時刻を設定します。これを繰り返し会議にする場合は、繰り返しパターンを設定します。

  5. [送信]をクリックします。
