予定ツールを使用して参加者の空き状況を検索する

会議を予定する場合、参加者の予定なし/予定あり状態を確認することができるため、すべての参加者に良い時間を見つけることができます。[参加者]フィールドに参加者を追加すると、概要ペインの[提案]パネルに参加者の空き状態が30分間のタイムスロットで示されます。

デフォルトの時間のプリファレンスでは、自分および参加者の勤務時間内の時間が提案されるようになっています。

•      自分の勤務時間のみ。カレンダーのプリファレンスで設定した勤務日と時間です。

•      他の参加者の勤務時間のみ。場所を検索するとき、すべての人の勤務時間を考慮に入れます。

これは変更できます。

  1. [新しい予定ページで、[ 時間を提案]をクリックします。

  2. 開いた[時間の候補]ペインで、歯車アイコンをクリックします。

  3. 参加者の空き状況の計算に使用しないボックスを選択解除します。

  4. 繰り返し会議の場合は、参加者が空いている必要がある繰り返し会議のインスタンス数を選択します。

  5. [OK]をクリックします。

ミニカレンダーの空き状態の表示

ミニカレンダーは、参加者の空き状態を色つきで表示します。

•      カレンダーの緑色はすべての参加者が参加でき、少なくとも1つの場所が使用可能であることを示します。

•      赤はそれらの日に半数以上の参加者が参加できないことを示します。
