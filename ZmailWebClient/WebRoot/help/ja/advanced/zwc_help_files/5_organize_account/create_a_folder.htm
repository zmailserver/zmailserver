フォルダを作成

スタンドアロンのユーザー定義フォルダまたはシステムフォルダを含む別のフォルダの中にあるフォルダを作成できます。

1つの親フォルダ内に同じ名前のフォルダを2つ作成することはできません。たとえば、[受信箱]フォルダの中に[ショッピング]という名前のフォルダを2つ配置することはできませんが、1つを[受信箱]フォルダの中に、もう1つを[送信済み]フォルダの中に配置することはできます。

  1. [メール] > [フォルダ]ページへアクセスし、[フォルダ]歯車アイコンをクリックします。

  2. [新しいフォルダ]を選択します。

  3. 新しいフォルダの[名前]を入力し、フォルダの[色]を選択します。

    4. フォルダ名には、:（コロン）、
    /（スラッシュ）、"（二重引用符）を除く任意の文字を使用できます。

    RSS/ATOMフィードの購読を選択するオプションがあります。

  4. [フォルダ] ツリーで、新しいフォルダの配置を選択します。

    5. •    [フォルダ]をクリックすると、リストの一番上に新しいフォルダが表示されます。

    •    既存のフォルダをクリックすると、そのフォルダの中に新しいフォルダが配置されます。

  5. [OK]をクリックします。
