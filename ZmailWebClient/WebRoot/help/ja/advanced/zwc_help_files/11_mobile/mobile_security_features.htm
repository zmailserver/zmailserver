携帯のセキュリティ機能

システム管理者が携帯のセキュリティポリシーを設定している場合、携帯デバイスの個人識別番号（PIN）を作成するように求められる可能性があります。PINを忘れた場合は、システム管理者に連絡してください。

PINがある場合でも、ポリシーの設定によっては正しいPINを入力しないとユーザーがロックアウトされる可能性があります。システム管理者は、何者かが不正なパスワードを使用して、設定された試行回数を超えてデバイスにログオンしようとした場合にデバイスのローカルワイプ操作が実行されるようにPINのロックを設定します。この操作が実行されると、デバイス上のすべてのデータが消去されます。このポリシー設定を実行できるのはシステム管理者だけで、他のユーザーが変更することはできません。詳細については、システム管理者にお問い合わせください。
