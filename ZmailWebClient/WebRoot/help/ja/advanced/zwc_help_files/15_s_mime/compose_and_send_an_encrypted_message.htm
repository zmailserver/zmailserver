暗号化されたメッセージを作成して送信

注意: これはZCS Network Editionの機能であり、アドバンストウェブクライアントでのみ使用できます。

公開キーおよび証明書を送信してきた受信者にはメッセージを暗号化して送信できます。受信者の証明書はアドレスブックの受信者の連絡先ページに保存されます。

  1. ツールバーの[新規]をクリックします。

  2. セキュリティプリファレンスのデフォルト設定が暗号化されたデジタル署名付きメッセージを送信しないとなっている場合、[セキュリティ]をクリックして[署名と暗号化]を選択します。

  3. [宛先：]にユーザーのアドレスを入力するか、[宛先：]リンクをクリックして、アドレスブックからアドレスを検索します。

  4. メッセージの件名を入力します。

  5. メッセージを作成します。

  6. 添付ファイルを追加するには、[添付ファイルを追加]をクリックしてフォルダを参照するか、お使いのコンピュータからファイルをメールヘッダーセクションにドラッグアンドドロップします。

  7. [送信]をクリックします。

    • 暗号化されたメッセージの下書きは保存できません。

    • 暗号化されたメッセージにはスペルチェックを実行できません。
