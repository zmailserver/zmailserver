詳細なタスクを作成

タスクリストを使用して、タスクを作成したり、タスクの進行状況を管理したりできます。また、備考を記入することや、ファイルを添付してタスク内から簡単にアクセスできるようにすることも可能です。このようにタスクに関連するすべての情報を一箇所に配置しておけば、他のユーザーとタスクリストを共有するときにとても便利です。

開始日と期限を入力してプロジェクトの期間を予測したり、優先度を設定したりできます。また、タスクを開始するときには、[開始前]や[進行中]などのステータスを選択したり、パーセント値による完了度を指定したりできます。

  1. [タスク]ページに移動して、[新しいタスク]を選択します。

  2. [詳細設定]領域で、タスク情報を入力します。タスクに名前を付けるには、[件名]のみが必須です。

    オプション

    説明

    件名

    （必須）タスクの簡単な説明を入力します。入力したテキストは、タスクリストの説明として使用されます。

    場所

    タスクを実行する場所を入力します。

    優先度

    このタスクの優先度を選択します。

    タスクリスト

    このタスクを表示させるタスクリストを選択します。

  3. [進行状況]領域で、タスクの進行状況を追跡するための情報を入力します。この情報はタスクの必須項目ではありません。

    オプション

    説明

    ステータス

    このタスクのステータスを選択します。いつでもこのタスクを開きなおしてステータスを変更できます。

    パーセンテージ

    タスクのステータスのパーセンテージを選択します。いつでもこのタスクを開きなおしてパーセンテージを変更できます。

    開始日

    タスクの開始日を定義します。

    期限

    タスクの期限を定義します。

    リマインダー

    リマインダーの日時を設定します。

    通知

    [プリファレンス] > [通知]ページでメールアドレスまたはSMSアラートを設定している場合は、該当するリマインダーのチェックボックスをオンにします。

    •通知を設定していない場合は、[設定]をクリックして、[プリファレンス] > [通知]ページにアクセスして設定します。

    添付ファイルを追加

    ツールバーの[添付ファイルを追加]をクリックし、テキストフィールドに備考を入力して、添付ファイルを追加します。

  4. [保存]をクリックします。
