使用語音服務撥打電話

備註： 這是 ZCS Network Edition 的專門功能，並只可用於 Advanced Web Client。

你可以從所有註冊的電話裝置（包括軟件電話用戶端如 Cisco 或 Mitel）撥打電話。

  1. 在「語音」頁面上，按一下 新通話。新通話對話方格隨即顯示。

  2. 從下拉選單中選擇 來電 號碼。列出的號碼是聯絡過你的號碼。

  3. 輸入或選擇 致電 號碼，以撥打想要撥打的號碼。你可從通訊錄或聯絡人卡片獲取號碼，或手動輸入號碼。

  4. 按一下「確定」撥打電話。利用你所使用電話號碼關聯的電話裝置撥打電話

結束通話

若要結束通話，請掛斷電話或按一下對話方格以掛斷。
