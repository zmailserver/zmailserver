自動使用署名

你可包括自動署名在郵件結尾。署名中可包括姓名和任何其他附加文字。如果你使用 HTML 編輯器，你可格式化你的署名、為 URL 加入連結，以及加入圖片。

你可建立多個署名。例如，你可準備一個正式的署名作傳送給客戶的郵件之用，而另一個非正式的署名則作傳送給朋友的郵件之用。

如果你有多個電郵身份 (亦稱身份)，則可建立不同的署名，並分別指派給具體的地址。

  1. 前往偏好設定>署名頁面。

  2. 在名稱字框中輸入一個描述性的名稱，用於識別此署名。你可建立多個署名，所以在此使用可識別的名稱是很有用的。

  3. （可選）要以純文字為格式，按HTML 格式並選擇純文字格式。

  4. 在字框中，按照你希望顯示的樣式輸入署名。如果你正使用 HTML 格式，你可加入圖片及為你的署名文字建立連結。

  5. 在使用署名部份，從下拉選單中選擇預設的署名，用於從你不同帳戶傳送的郵件。

  6. 選擇署名在郵件中的位置。

    7. •    若需將你的署名加入在你的回覆的結尾，但在所包括的郵件的前面，可選取包括的郵件正文上面。

    •    若需將你的署名加入在整個郵件的結尾，可選取包括的郵件正文下面。

  7. 按下儲存。
