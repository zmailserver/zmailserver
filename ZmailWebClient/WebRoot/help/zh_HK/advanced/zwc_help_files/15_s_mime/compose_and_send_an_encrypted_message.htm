撰寫及傳送已加密郵件

備註： 這是 ZCS Network Edition 的專門功能，並只可用於 Advanced Web Client。

郵件可加密傳送予已傳送公用鑰匙及憑證給你的收件人。收件人的憑證會被儲存到通訊錄中收件人的通訊錄頁面。 

  1. 按下工具列上的新郵件。

  2. 如果你的安全偏好設定並非預設為傳送已加密及電子簽署的郵件，按安全並選取簽署及加密。

  3. 在收信人：輸入用戶地址或按下收信人：連結，以從你的通訊錄中搜尋地址。

  4. 輸入會議主題。

  5. 撰寫郵件。

  6. 若要加入附件，按加入附件並瀏覽至檔案，或將該檔案從你的電腦拖曳到郵件標題部份。

  7. 按傳送。

    • 你不能儲存已加密郵件的草稿。

    • 你不能執行在已加密郵件中的拼字檢查。
