撰寫及傳送電子簽署郵件

備註： 這是 ZCS Network Edition 的專門功能，並只可用於 Advanced Web Client。

電子簽署郵件可傳送予已傳送公用鑰匙及憑證給你的收件人。收件人的憑證會被儲存到通訊錄中收件人的通訊錄頁面。 

  1. 在工具列中按一下。

  2. 在收信人中：輸入用戶地址或按下收信人：連結，以從你的通訊錄中搜尋地址。

  3. 如果你的安全偏好設定並非預設為傳送電子簽署的郵件，按安全並選取只限簽署。

  4. 輸入會議主題。

  5. 撰寫郵件。

  6. 若要加入附件，按加入附件並瀏覽至檔案，或將該檔案從你的電腦拖曳到郵件標題。

  7. 按傳送。
