自動完成地址功能

當你撰寫新郵件時直接在「收件人」、「副本」和「密件」欄中輸入地址，自動完成功能會提示一些名稱。 以你所輸入文字開頭、源自聯絡人清單或全網域地址清單的潛在地址之清單即會顯示。

隨著你繼續在收件人：欄位輸入，相符的地址清單就會越來越小，相符的地址也會越來越少。如果沒有相符的地址，清單即消失。

示例

例如，假定你要給 David Brinks 傳送郵件，他的電郵地址剛好是 dbrinks@acme.com。當你在地址欄位中輸入第一個字母「d」時，就會出現一個清單，顯示所有可能的相符項，並且凸顯相符部分。在本例情況下，有可能顯示：

Dale Edwards < ded@acme.com>

David Brinks < dbrinks@acme.com>

Erica Dodd <erica878@xyz.com>

當你在收件人：欄位中輸入其他字母時，相應的清單就會越來越小。

相符的地址

自動完成功能可將你輸入的文字與某聯絡人的以下部分進行匹配：

  • 名字

  • 姓氏

  • 電郵地址

尋找相符的地址之過程，總是從欄位的開頭進行的。「r;bob」將與「r;bob smith」相符，但不會與「r; billybob smith」相符。

使用自動完成功能

當你輸入的文字與一個或多個聯絡人相符時，相符項會顯示在目前編輯欄位下方的清單中。在預設狀態下，第一個相符項是凸顯的。

姓名將按「等級」順序顯示；你曾向其傳送過郵件的聯絡人，將排在你未傳送過或不常向其傳送郵件的聯絡人之上。即是，最常記起的聯絡人會排列在至頂。 如果你不希望該聯絡人的姓名始終排在最前，可按下略過，聯絡人將被重新排到靠後的位置。

有幾個鍵可用來觸發特殊功能。滑鼠也有可能用來選擇相符的項目。

  • 逗號、分號、Return/Enter 以及 Tab 鍵都具有完成功能。你所輸入的文字將被目前選定的相符項替代。

  • 按下 ESC 鍵，隱藏清單。

  • 上下箭咀可改變清單中的選擇。在清單中移動滑鼠也可改變清單中的選擇。

  • 在一個相符項上按一下滑鼠，即可選取該項。

單個鍵（如分號）可啟動自動完成功能，並可用於快速輸入地址。如果你知道用幾個字元就能顯示特定聯絡人的地址，就可輸入那幾個字元，然後輸入分號，相應聯絡人的地址隨即就會出現。例如，如果你知道「r; db」就是 David Brinks，則輸入「r; db;」，完整地址「r;David Brinks < dbrinks@acme.com>」 即會出現在地址方格中。

備註：你可透過偏好設定，通訊錄頁面變更自動完成功能的運行方式。

自動完成功能在其他應用程式中也有（如在記事簿的預約中加入出席者），而且使用方式相同。
