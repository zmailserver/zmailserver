設定你的桌面偏好設定

「偏好設定」索引標籤中包含有一些功能頁面，使得你可以為你在 Zimbra Desktop 建立的每個帳戶自訂你的郵箱、通訊錄和記事簿的工作方式。當你在 Zimbra Desktop 上建立帳戶時，預設的操作動作即得以設定。

在一般偏好設定資料夾中所作修改會影響所有帳戶基本的 Zimbra Desktop 版面。

全域

對此索引標籤的改動會影響 Zimbra Desktop 上的所有郵件帳戶。

  • 主題用於顯示一個清單，供你選用不同的 UI 面板。你可在任何時候改變郵箱的背景色。

  • 語言。指 Zimbra Desktop 中顯示文字所用的語言。（可在 Zimbra Web Client 中使用）

  • 如果在此索引標籤中顯示的時區不正確，可修改預設時區。時區功能可確定你傳送和接收郵件的正確日期和時間，並正確顯示你「記事簿」中的預約時間。 這可能與所用電腦上所設置的時區不同。

  • 在搜尋設定中，你可選擇自動在每次執行搜尋時包含雜件箱和垃圾桶。預設情況下，這兩個資料夾沒有包括在內。

  • 在搜尋語言中，剔選在搜尋工具列中顯示進階搜尋語言以便在搜尋字框中顯示搜尋字串的細節。

  • 啟用顯示核取方格後，你可快速選擇清單中的項目從而為內容窗格中的每個項目顯示一個核取方格。透過此核取方格，你可以選擇一項或多項同時執行形同的操作，例如刪除、移動、標記為已讀/未讀。

  • 系統預設。啟用此選項後，可使 Zimbra Desktop 成為你的預設郵件應用程式。當你傳送郵件或按網頁上的 Mail To（郵件傳送到）連結時，Zimbra Desktop 的「寫信」視窗隨即打開。

郵件

  • 顯示。選擇每頁顯示多少條目（郵件或對話）。預設值為每頁顯示 25 個條目，但你可選擇 10、50 或 100。

  • 顯示郵件。對於已經格式化為 HTML 的郵件，可指定是否以 HTML 方式查看。若關閉此項，郵件則以純文字形式顯示，即只顯示 HTML 標記，但不顯示標記語言版本。

  • 郵件預覽。

    • 顯示郵件片段。若啟用，當游標掠過郵件主題時，郵件清單中的郵件片段則會顯示出來。

    • 如果啟用了按兩下後在新視窗中開啟郵件，當按兩下郵件時，它就會在一個新視窗中打開

  • 圖片。啟用圖片後，HTML 郵件便可自動下載圖片。若啟用，你無需按下「顯示圖片郵件」就可以看見圖片。

  • 當我在閱讀窗格讀取郵件時，可用來在閱讀窗格中設定所查看郵件的標記，即將所查看的郵件註明為已讀，還是未讀。可選設定包括﹕立即標記為已讀、在「閱讀」窗格查看郵件時過了你定義的若干秒後再標記為已讀、或是讓在「閱讀」窗格中查看的郵件保持未讀。

  • 預設郵件搜尋。此設定決定了登入時執行的搜尋偏好設定。預設的搜尋目標是你的收件箱。 當你打開 Desktop，「收件箱」的搜尋結果會即刻顯示。你可修改此欄位，以使在剛開啟 Desktop 時顯示的是其他的資料夾、標記或儲存的搜尋。有關句法，請見查詢語言說明。

  • 接收郵件。

當一封郵件到達時：

    • 播放聲音。當有新郵件到達收件箱時發出「嗶」聲提醒你。如果你把音量調到靜音，你將聽不到聲音。

    • 凸顯郵件標籤。如果你沒有在「郵件」索引標籤操作，當新郵件到達收件箱時，郵件索引標籤會凸顯。

    • 閃爍顯示瀏覽器標題。當新郵件到達收件箱時，瀏覽器閃爍，直到你打開「郵件」索引標籤。

    • 顯示彈出的通知。當郵件到達時，一個小對話方格顯示有關該郵件的資訊。

以下功能只能在 Zimbra Web Client 帳戶中使用。

  • 指定一個郵件轉寄地址。你可指定將原始郵件從 Zimbra 郵箱中刪除。

  • 確定是否對發來的郵件傳送自動回覆郵件並設定開始和結束日期。 自動回覆訊息經常被用作外出不在訊息，或休假訊息，以表明你不在辦公室並交代相關事項。你還可指定可使用該郵件的開始和結束日期。

  • 我傳送的郵件。設定「我傳送的郵件」部分，用以處理以下兩種情況：你傳送的郵件，但你自己的地址是郵件的收件人之一；或所含地址為包括你自己在內的郵件清單。

寫信

  • 設定預設的文字編輯器，HTML 或純文字。

  • 設定以 HTML 格式撰寫郵件、文件和記事簿時所使用的預設字型。 你可確定字型的樣式、大小和顏色，用以作為預設的樣式。預設字型為 Arial、12 point black。

  • 總在新視窗中撰寫，這一選項可使你在撰寫郵件時開啟一個與你郵箱分開的新寫信視窗。在寫信視窗開啟的情況下，你可查看和瀏覽你的郵箱。

  • 每隔幾分鐘，你所撰寫的郵件都會自動儲存到草稿箱。如果不需要此自動儲存功能，可關閉此功能。

  • 選擇是否在你的回覆郵件正文中包含原始郵件。你可選擇包含原始郵件，不包含原始郵件，以附件形式包含原始郵件，在正文中包含但加上你選的字首，或只包含最近的郵件。 （也就是說，只引用寄件人寫下的內容，而不是前面的文字。）

  • 如果選擇引用原始文字，請指定是否需要為轉寄或回覆的前一封郵件的每一行加上字首 | 的 >。

  • 預設情況下，在寄件箱儲存副本為啟用狀態。你傳送郵件的副本被儲存在「寄件箱」。

  • 指定在轉寄郵件時如何包含郵件的原始文字：放入郵件正文、加上字首放入郵件正文或作為附件一併轉寄。

署名

可為你的郵件建立各種署名。署名中可包括姓名和其他結束語句，帳戶設定中對語句的長短和字元數量上限均有限制。如果你有多個身份，則可建立不同的署名，並分別指派給具體的地址。

通訊錄

  • 選擇以何種形式查看聯絡人，清單或名片。

  • 選擇每頁顯示的聯絡人的人數。預設值為 25。

  • 啟用此功能後，你可在傳送郵件時自動將地址加入電郵聯絡人資料夾。

    當傳送郵件到一個所有的通訊錄中都沒有的新地址時，該聯絡人將登記到你的電郵聯絡人通訊錄中。你可在偏好設定的通訊錄索引標籤中關閉這一功能。從設定中取消剔選，將新聯絡人加入「電郵聯絡人」。

E身份

帳戶標籤用於建立和管理你的身分（persona）。身份是用於「寄件人」中你傳送郵件中的欄位的電郵身份。

預設的樣式在基本帳戶設定中定義。在建立新的身份時，可使用與「預設身份」相同的設定，也可設定不同的偏好設定。

郵件過濾器

利用「郵件過濾器」索引標籤定義郵件過濾規則。若需瞭解如何設定郵件過濾規則，請參見過濾工作原理。

記事簿

  • 用於選擇如何查看 Zimbra 記事簿。預設值為按照工作週。

  • 指定在記事簿中把哪一天作為一週的第一天。

  • 一個小日曆顯示在記事簿中。 你可在任何視窗中啟用小日曆。

  • 可啟用在回覆之後自動刪除邀請函，否則邀請函滯留在收件箱中。

  • 設定會議的提醒通知。你可設定何時進行會議提醒，可以從不提醒，也可以是會議前的最多達 60 分鐘的某個時刻。你可選擇提醒的類型、聲音、閃爍顯示瀏覽器標題或顯示一個彈出的通知。

  • 如果你經常建立不包含參會者的預約，剔選使用快速加入對話方格。

  • 如果你在不同時區之間旅行，你可能需要選擇在預約視窗中顯示時區清單，這樣在安排預約時就可在必要時改變時區。

匯入/匯出。

  • 要匯出你帳戶中的所有資料夾或你帳戶中指定的資料夾，參見備份你帳戶的內容

  • 將匯出的資料夾匯入回你的帳戶

  • 搜尋指定檔案類型將之匯出

  • 匯入和匯出記事簿程式。

  • 匯入和匯出聯絡人清單

快捷方式

快速鍵是鍵或鍵的組合，當輸入該鍵或鍵的序列時，可快速選擇一項操作。  

前往你的快速鍵偏好設定資料夾，查看完整的快速鍵清單或在任何視窗中按 Ctrl Q，快速鍵清單隨即彈出。
