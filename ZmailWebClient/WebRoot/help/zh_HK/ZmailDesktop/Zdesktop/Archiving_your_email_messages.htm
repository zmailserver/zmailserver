郵件歸檔

你的電郵帳戶可以擁有最大的存儲空間，並在該配額下進行管理。當你的郵箱開始填滿時，則需要移除某些郵件。如果不刪除郵件，可在離線狀態下在 Zimbra Desktop 中保留這些檔案。這可透過在 Zimbra Desktop 中移動你的郵件和附件到一個資料夾來實現，此操作不會反向同步到你的帳戶。

備註：當你歸檔郵件後，該郵件將儲存到你的電腦中。如果電腦損毁，則歸檔的郵件可能會丟失。若要保護歸檔資料，你應該使用 Zimbra 備份工具。參見備份帳戶內容。

如何歸檔 Zimbra 帳戶中的郵件

對於 Zimbra Web Client，稱為 Local Folders（本機資料夾）的資料夾在 Zimbra Desktop 上建立。存入「本機資料夾」的郵件被儲存在你的電腦中，並同時從伺服器中刪除。

 

若要歸檔 Zimbra 郵件，請在「本機資料夾」中建立資料夾，並將郵件移至這些資料夾中。還可將整個私人資料夾拖曳到「本機資料夾」中。在下一次同步時，這些移至「本機資料夾」中的郵件將從伺服器中刪除。當直接登入你的 Zimbra 帳戶時，「本機資料夾」中的郵件不可用。

備註：如果從「本機資料夾」中刪除檔案，這些檔案不會轉移到「垃圾桶」，而是直接被刪除。

如何歸檔其他電郵帳戶中的郵件

對於你設定的其他電郵帳戶，你可在 Zimbra Desktop 上建立資料夾，此操作不會反向同步到伺服器。若一封郵件被移至了一個資料夾中而且沒有同步，那麼在下次 Zimbra Desktop 與帳戶同步時，該郵件將從伺服器中刪除。當直接登入你的帳戶時，這些資料夾中的郵件不可用。

  1. 若要建立一個新的歸檔資料夾，請選擇 Zimbra Desktop 上的帳戶，然後在「資料夾」選單中右鍵點擊並選取新資料夾。

  2. 給資料夾起名並選擇新資料夾存放位置。

  3. 按確定。

  4. 選擇此資料夾，右鍵點擊並選擇關閉同步。  

 

你亦可以關閉任何私人資料夾的同步功能。

若關閉了同步，資料夾旁將不再出現綠球。

 

移動到資料夾而沒有反向同步到帳戶的郵件，將從伺服器中刪除。

備註：若要再次將一個資料夾同步，請開啟同步。下次 Zimbra Desktop 同步到你的帳戶時，該資料夾中的內容將被同步。
