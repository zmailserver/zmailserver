使用事項清單

透過事項功能，你可建立待辦事項清單，並用該清單管理這些事項的完成情況。你可將事項加入預設事項清單中，還可以建立其他事項清單，以便透過更多的特定活動，如公司或私人項，來整理你的待辦事項清單。

當你進行某事項時，你可能需要加入資訊，更改狀態及完成的百分比，或標示該事項完成。

事項排序

你可依主題、狀態、完成百分比以及到期日對事項清單進行排序。

  • 在「事項」視窗中，選擇一個事項清單，並在「內容」窗格工具列中，按一下排序依據標題。

建立一個事項

  1. 在工具列中按新事項。

  2. 在詳細資料部份，輸入事項資訊。只有主題是必需，以提供給事項一個名稱。

  3. 在日期部份，輸入開始日期和到期日。

  4. 按下儲存。
