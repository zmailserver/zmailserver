搜尋的句法規則

你可搜尋任何項目，如單字、名、姓、電話號碼或網域名稱。當你搜尋內容時，要注意一些規則。搜尋內容是搜尋郵件的正文，以及任何系統可讀的附件。

選項

描述

文字完全相符

如果你搜尋短語，每一個當中的字都必須完全相符。不允許有變體。例如，如果你搜尋 bananas，含有 banana  的郵件則不相符。

不區分大小寫

搜尋功能不區分大小寫，South、south 和 SOUTH 都被視為是同一個詞。

特殊符號

以下特殊字元不能在搜尋文字中使用。 ~ ' ! @ # $ % ^ & * () _- + ? / { }[ ] ; : "

版權和商標符號 (

特殊字元，例如作為文字一部分的商標符號，在索引搜尋時會被移除。當這些特殊字元作為完整搜尋文字的一部分用於搜尋查詢時，它們會被忽略。例如，搜尋 Zimbra™ 時，會找到所有關於 Zimbra 的資訊，包括那些沒有商標符號的內容。

通配字元

支援在字首後使用星號 *  作為通配字元。例如，如果你搜尋 do*，返回的項目是 dog、door 等詞。

備註：搜尋查詢中的雙位元星號不會被視為通配字元，搜尋執行時將忽略。

附件

系統可讀附件會被搜尋。系統可讀附件是一種可以轉換成 HTML 格式的檔案類型，如 Microsoft Office 文件和文字檔案。

通訊錄中的聯絡人

你必須搜尋整個字或數字元串。例如，如果按照電話號碼搜尋聯絡人，你需要輸入完整的電話號碼，正如在聯絡人顯示的一樣。不能僅按照區碼進行搜尋。
