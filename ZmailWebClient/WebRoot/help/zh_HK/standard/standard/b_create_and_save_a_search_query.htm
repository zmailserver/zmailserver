建立和儲存一個搜尋條件

如果你頻繁地執行同樣的搜尋條件，可建立該搜尋條件並將之儲存到搜尋資料夾。當需要再次執行該搜尋時，簡單地按下已儲存的搜尋資料夾即可啟動搜尋，結果顯示在內容窗格中。

  1. 在概況窗格>搜尋標題中，按編輯。  

  2. 在工具列中，選擇新搜尋。

  3. 在名稱字框中，輸入你的搜尋名稱。

  4. 在父資料夾字框中，選擇何處儲存此搜尋。該資料夾被儲存在概況窗格中的搜尋標題下。如果你要把該搜尋儲存到另一個資料夾，從下拉清單中選擇該資料夾。可用放大的圖示標識搜尋資料夾。

  5. 在搜尋查詢字框中，建立你的搜尋查詢。參見查詢語言說明以獲得你可使用的搜尋語法的資訊。

  6. 按下搜尋。

若要執行你的搜尋，按概況窗格中的搜尋資料夾。搜尋結果將在內容窗格中顯示。
