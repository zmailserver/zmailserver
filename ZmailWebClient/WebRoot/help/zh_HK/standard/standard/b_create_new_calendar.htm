建立新的記事簿

系統為你建立了一個記事簿，你可以建立更多的記事簿以便追蹤不同類型的活動。例如，你可以為工作、私人、公司的各項活動建立各自的記事簿，也可訂閱互聯網上的記事簿，還可透過連結使用分享給你的記事簿。

建立一個記事簿

  1. 選擇「記事簿」標籤。

  2. 在「記事簿」窗格中，按下編輯。

  3. 在工具列中，按下新建記事簿。

  4. 輸入記事簿的名稱。

  5. 選擇「記事簿」活動的顯示顏色。這樣可為郵箱中的記事簿設定顏色，從而使記事簿的活動總是以選定的顏色顯示。

  6. 如果不想讓其他人看見記事簿中的活動，請剔選在報告忙/閑時排除這個記事簿。

  7. 預設情況下，已在 UI 剔選為啟用狀態。此記事簿中的活動將在開啟「記事簿」視窗時顯示。 如果不希望隨同其他記事簿一起查看此記事簿，請取消剔選該方格。在這種情況下，要查看此記事簿的活動，須選擇該記事簿並將其打開。

  8. 按下建立記事簿。

訂閱另一個記事簿

  1. 選擇記事簿索引標籤。

  2. 在「記事簿」窗格中，按下編輯。預設的記事簿資訊隨即顯示。  

  3. 在工具列中，按下訂閱記事簿。  

  4. 輸入該記事簿的名稱、一個該遠端記事簿的有效 iCal 檔案 URL，以及選擇一種顯示該記事簿活動的顏色。

  5. 如果不想讓其他人看見記事簿中的活動，請 剔選報告空閒/忙碌時間時排除此記事簿。

  6. 預設情況下，已在 UI 剔選為啟用狀態。此記事簿中的活動在你開啟記事簿視窗時顯示。 如果不希望隨同其他記事簿一起查看此記事簿，請取消剔選該方格。在這種情況下，要查看此記事簿的活動，須選擇該記事簿並將其打開。

  7. 按建立記事簿。
