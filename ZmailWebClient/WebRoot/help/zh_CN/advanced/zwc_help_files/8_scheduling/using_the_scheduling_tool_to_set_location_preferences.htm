使用安排工具查看地点可用性

行事历安排工具能够帮助您在贵组织中找到下一个可用的空闲时间和会议地点。可为首选的会议时间、建筑物、地点和房间大小设置首选项。

输入参会者姓名时，日程安排工具将开始在“概述”窗格中显示建议的时间和地点。您也可以选择在添加参会者之前查找会议地点。如果设置了地点首选项，则仅显示这些地点。

使用安排工具设置地点首选项

如果您始终需要在某个特定地点预订会议室，则在创建约会时，可使用“行事历”应用程序中的“建议的地点”工具显示这些会议室是否空闲。

  1. 在新约会页面中，单击建议一个地点。

  2. 在打开的窗格中，单击齿轮图标。

  3. 在建议首选项对话框的“地点首选项”部分，输入地点的相关信息。可以只输入地点名称，也可以输入更加具体的信息。

  4. 单击确定。

右侧的列显示符合您要求的地点。如果已在公司目录中配置了会议室地点，则会显示这些地点。

一旦为某个约会设置了地点首选项，便会保存相应的信息，当您设置其他约会时，这些首选项便可供您使用。
