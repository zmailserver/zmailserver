搜索语法规则

您可以搜索任何项，例如，字词、名、姓、电话号码或域名。对内容进行搜索时，需要牢记一些规则。对内容进行搜索时会对邮件的正文以及所有系统可读取的文件附件进行搜索。

选项

描述

完全文本匹配

如果您搜索短语，则必须与该短语中的每个字都完全匹配。不允许拼写变体。例如，如果您搜索 bananas，则包含 banana 的邮件就不是匹配项。

不区分大小写

搜索功能不区分大小写；South、south 和 SOUTH 将被视为同一个词。

特殊字符

不能在搜索文本中使用以下特殊字符。 ~ ' ! @ # $ % ^ & * () _- + ? / { }[ ] ; : "

版权和商标符号 (

对词建立搜索索引时，将删除词中包含的特殊字符（如商标符号）。当在搜索查询中使用这些特殊字符作为整个词搜索的一部分时，将忽略这些字符。例如，如果搜索 Zimbra™，将找到出现的所有 Zimbra，包括不带商标符号的词。

通配符

可在前缀后使用星号 * 作为通配符。例如，如果您搜索 do*，则会返回包含 dog、door 等单词的项目。

注意：搜索查询中的双字节星号不视为通配符，在搜索运行时将被忽略。

附件

搜索功能会对系统可读取的文件附件进行搜索。系统可读文件是指可以转换为 HTML 的文件，例如，Microsoft Office 文档和文本文件。

地址簿中的联系人

您必须搜索整个字词或数字字符串。例如，要按电话号码搜索联系人，您必须输入与联系人条目中显示的号码完全相同的完整电话号码，不能仅按照区号进行搜索。

注意：在 @ClientNameLong@ 中搜索内容与文本匹配搜索或文字处理功能（如“查找”）略有不同。
