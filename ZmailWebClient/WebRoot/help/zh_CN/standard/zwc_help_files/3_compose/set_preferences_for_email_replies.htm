设置电子邮件回复的首选项

要以与所收到邮件相同的格式回复邮件，请选择回复/转发时使用原始邮件的格式。这可确保收件人收到的邮件采用最适宜他们阅读的格式。

  1. 转到首选项 > 邮件页面的编写邮件部分。

  2. 要以与所收到邮件相同的格式回复邮件，请选择回复/转发时使用原始邮件的格式。

  3. 选择是否在您的回复邮件中包含原始邮件。如果要包含原始邮件：

    4. •   选择是使用字符标记原始邮件还是在标题中包括原始详细信息（“收件人”、“发件人”和“主题”字段）。还可以附件形式包含原始邮件。

    •    如果想要为原始邮件添加前缀，请选择尖括弧 (>) 或竖线 (|)。

  4. 单击保存。
