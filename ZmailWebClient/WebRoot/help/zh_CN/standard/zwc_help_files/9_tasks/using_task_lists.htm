使用任务列表

利用任务功能，您可创建待办事项列表，并管理这些任务直至完成。您可将任务添加到默认任务列表中，还可以创建其他任务列表，以便通过更多的特定活动（如工作或个人项目）来整理您的待办事项列表。

在处理某项任务时，您可能想要添加信息、更改状态和完成百分比或者将任务标记为完成。

任务排序

您可按照主题、状态、完成百分比以及结束日期对任务进行排序。

  • 在“任务”视图中，选择一个任务列表，并在内容窗格工具栏上单击排序所要依据的标题。

创建任务

  1. 在工具栏中，单击“新任务”。

  2. 在详细资料部分中，输入任务信息。为任务指定名称时，仅主题为必填字段。

  3. 在日期部分，选择开始日期和结束日期。

  4. 单击保存。
