创建并保存搜索查询

如果您频繁地执行同样的搜索查询，可创建该搜索查询并将其保存到“搜索”文件夹。需要再次运行该搜索时，只需单击已保存的搜索文件夹即可运行搜索，并在内容窗格中显示结果。

  1. 在概述窗格 > 搜索标题处，单击编辑。  

  2. 在工具栏中，选择新搜索项。

  3. 在名称文本框中，输入搜索的名称。

  4. 在父文件夹文本框中，选择将此搜索保存到何处。此文件夹将保存在“概述”窗格的“搜索”标题下。如果要把搜索保存到另一个文件夹，从下拉列表中选择该文件夹。可用放大镜图标标识搜索文件夹。

  5. 在搜索查询文本框中，创建您的搜索查询。请参见查询语言说明以获得您可使用的搜索语法的信息。

  6. 单击搜索。

要运行搜索，单击概述窗格中的搜索文件夹。搜索结果将在内容窗格中显示。
