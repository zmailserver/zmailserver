关于编写电子邮件

您可以在编写完电子邮件后立即将其发送，也可以先编写一份草稿，然后在以后某个时间再回来将其完成并发送。

编写新邮件时，首先单击工具栏上的 打开一个空白写信页面。也可右键单击电子邮件发件人部分的人名，然后选择新电子邮件以打开一个空白写信页面。

根据您的邮件首选项，此时可能会出现两种情况：一种情况是“应用程序”工具栏上出现一个“写信”选项卡，写信页面随即打开；另一种情况是有一个独立的写信窗口打开。您可以打开多个“写信”选项卡，而且可以在工具栏上的不同选项卡之间移动。

在写信页面上，将收件人的电子邮件地址键入到收件人:字段。

  • 若需查找某个人的电子邮件地址，可单击收件人以在您的联系人或可用的全局地址列表中进行搜索。

  • 若需添加密送地址，可单击显示密送（在邮件标题中抄送:文本字段的旁边）。

如果设置了不同的身份，写信页面中就会显示发件人字段。选择要用作“发件人”地址的身份。

书写邮件既可以采用 HTML 格式，也可采用纯文本格式。默认格式在首选项>常规页面中配置。若仅需快速更改此邮件的格式，可单击写信工具栏上的选项并选择“HTML”或“纯文本”。

  • HTML 格式使您可以使用不同的字体样式设置邮件的格式、创建表格、添加颜色和插入超链接。

  • 纯文本格式所生成的文本没有式样或格式。任何计算机都可以读取这种类型的邮件。

在 Zimbra 帐户中工作时，可以请求回条。单击写信工具栏上的选项，然后选择请求读信回条。

您可以向邮件附加文件，包括图片、文档、电子表格和音频文件。

请参见编写新邮件。
