邮件归档

您的电子邮件帐户可以拥有最大的存储空间，并在该配额下进行管理。当您的邮箱开始填满时，则需移除某些邮件。如果不删除邮件，可在离线状态下在 Zimbra Desktop 中保存这些文件。这可通过在 Zimbra Desktop 中移动您的邮件和附件到一个文件夹来实现，此操作不会反向同步到您的帐户。

备注：当您归档邮件后，该邮件将保存到您的计算机中。如果计算机死机，则归档的邮件可能会丢失。若需保护归档数据，您应该使用 Zimbra 备份工具。请参见备份您的帐户内容。

如何归档 Zimbra 帐户中的邮件。

对于 Zimbra Web Client，会在 Zimbra Desktop 上创建一个本地文件夹文件夹。存入“本地文件夹”的邮件被保存在您的计算机中，并同时从服务器中删除。

 

若需归档 Zimbra 邮件，请在“本地文件夹”中创建文件夹，并将邮件移动到这些文件夹中。还可将整个私人文件夹拖动到“本地文件夹”中。在下一次同步时，这些移动到“本地文件夹”中的邮件将从服务器中删除。当直接登录到您的 Zimbra 帐户时，“本地文件夹”中的邮件不可用。

备注：如果从“本地文件夹”中删除文件，这些文件不会转移到“已删除邮件”文件夹，而是直接被删除。

如何归档其他电邮账户中的邮件

对于您设置的其他电子邮件帐户，您可在 Zimbra Desktop 上创建文件夹，此操作不会反向同步到服务器。若一封邮件被移动到了一个文件夹中而且没有同步，那么在下次 Zimbra Desktop 与帐户同步时，该邮件将从服务器中删除。当直接登录到您的帐户时，这些文件夹中的邮件不可用。

  1. 要创建新的归档文件夹，请选择 Zimbra Desktop 上的帐户，然后在“文件夹”选单中右键单击并选择新建文件夹。

  2. 给文件夹起名并选择新文件夹存放位置。

  3. 单击确定。

  4. 选择此文件夹，右键单击并选择关闭同步。  

 

对于任何的私人文件夹而言，您都可以关闭同步功能。

若关闭了同步，文件夹旁将不再出现绿球。

 

移动到不能反向同步到帐户的文件夹中的邮件，将从服务器中删除。

备注：若要再次将一个文件夹同步，请开启同步。下次 Zimbra Desktop 同步到您的帐户时，该文件夹中的内容将被同步。
