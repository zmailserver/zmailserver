Zimbra Desktop 功能概述

电子邮件

  • 写信、编辑、删除、回复或编写草稿

  • 通过会话功能跟踪来往邮件，也可转换到传统邮件视图

  • 在邮件中加入附件

  • 将邮件转发给一个或多个收件人

  • 通过特征或指定的文字搜索邮件和附件

  • 创建您自己的文件夹用以整理邮件

  • 创建标记用以整理邮件

  • 创建过滤器以便将来信导入指定的文件夹

  • 设置不同的帐户身份和地址，用以管理您在工作和个人生活中的不同角色。

  • 在离线时工作，您所发送的邮件将保存到发件箱，当您再次连接到网络时既可发送

  • 将邮件保存到本地文件夹，以便留出更多的帐户存储量

地址簿

  • 在同一个地方存储您所有账户中的联系人

  • 创建组合标签以便进行管理

  • 添加联系人个人相片

  • 书写邮件时自动完成邮件地址

  • 以 .csv 文件格式从其他应用程序导入新联系人，或将联系人导出为 .csv 文件以便进行备份

行事历

  • 创建和管理多彩色编码的行事历

  • 创建约会、会议和事件

  • 导入公网行事历（iCal 格式）

  • 邀请他人参加会议并查看空闲/忙碌时间

任务、公文包

  • 追踪您的任务，包括开始和结束日期、完成情况百分比

  • 编辑文档，其中包括添加图像、表格和电子表单，并通过邮件与他人分享

  • 将附件保存在公文包中而不是作为邮件附件

管理您的账户

  • 将文件夹归档到您的计算机，然后将其从服务器中移除

  • 备份您的帐户或帐户中指定的文件夹，以便将来导入到帐户中以恢复丢失的信息
