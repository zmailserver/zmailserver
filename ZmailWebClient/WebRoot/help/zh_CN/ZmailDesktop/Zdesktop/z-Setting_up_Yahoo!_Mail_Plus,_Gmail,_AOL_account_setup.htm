设置 Yahoo!Mail 和 Gmail 帐户

您须在您现有的帐户中首先输入设置值，然后方可使用 Zimbra Desktop。首次打开 Zimbra Desktop 时，会要求您设置一个帐户。您可随时设置其他帐户。

  • Yahoo!Mail 帐户。您可设置 Yahoo!Mail、Yahoo!Mail Plus、Ymail 或 Rocketmail 帐户。您的电子邮件、电子邮件文件夹和地址簿可以在 Zimbra Desktop 中使用。您还可选择同步行事历和联系人。

  • Gmail。您的 Gmail 帐户必须设置为允许 IMAP 访问。若选择了 Gmail，您可能需要首先登录到 Gmail，方可启用 IMAP。您的邮件在 Zimbra Desktop 中可用。您还可选择同步行事历和联系人。

要设置您的账户

开始设置前，确认已连接到互联网。

  1. 打开 Zimbra Desktop 并在右上角单击设置。  

  1. 单击添加新帐户。在“帐户类型”列表中选择帐户类型。“Zimbra 帐户设置”页面随即显示。输入以下信息。

帐户名称。这是作为帐户名称显示在 Zimbra Desktop 中的名称。若需设置多个帐户，确保创建能够很容易区分的名称以便识别各个帐户。例如，对于您的 Yahoo! 帐户，可用 Yahoo 这个名称。

您的全名。输入您的名字。（可选）

电子邮件地址。输入此账户的电子邮件地址。

密码。输入登录您的帐户时使用的密码。

  1. 在同步设置中，选择 Zimbra Desktop 检查您帐户中邮件的频率。默认情况下为每 15 分钟进行一次同步处理。

-如果您选择手动同步，您必须单击工具栏上的“发送/接收”，从而将您的帐户与 Zimbra 服务器同步。

-如果您选择的间隔时间太短，如 1 分钟，您的计算机性能可能要受到影响。

-您可随时更改这个设置，方法是单击“帐户设置”，选择“帐户”然后修改该设置。

从邮箱选择要同步的文件夹。

* 同步所有行事历。

* 同步所有联系人和组。

  1. 完成后，单击验证&保存。

Zimbra Desktop 随即开始验证您的帐户信息。如果您的帐户顺利通过验证，您将看到“服务已创建”对话框。如果您的帐户未能通过验证，须返回到“设置”对话框查看并更正详细信息。

您的帐户随即开始与 Zimbra Desktop 同步。

  1. 单击“启动 Desktop”以打开 Desktop。在您的帐户进行同步期间，您可在 Zimbra Desktop 中工作。
