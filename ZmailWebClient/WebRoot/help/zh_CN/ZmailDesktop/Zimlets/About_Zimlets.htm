关于 Zimlet

Zimlet 是小型程序，是为了将第三方信息和内容与 Zimbra Web Client 的功能进行集成而创建的一种机制。Zimlet 可使您在邮件中使用不同的内容类型，并与之互动。  

  • 单击邮件中的 URL 后，您便可启动一个浏览器窗口，前往该 URL。将鼠标悬停在 URL 上，便能以预览方式看网站内容。

  • 右键单击一个电话号码后，即可从所用计算机的 softphone（如 Skype 或 Cisco 的 VOIP 电话）拨号。

  • 右键单击邮件中的一个日期，便可看到您的行事历，并可在不离开邮件的情况下安排会议。

  • 右键单击一个姓名、地址或电话号码，即可更新您的地址簿。

概述窗格中可能还有其他可用的 Zimlet。您可以从首选项> Zimlet 中管理您的帐户中显示的 Zimlet。

备注：如果您的概述窗格中没有显示 Zimlet 部分，说明您的系统管理员已禁用您帐户中的 Zimlet 功能。
