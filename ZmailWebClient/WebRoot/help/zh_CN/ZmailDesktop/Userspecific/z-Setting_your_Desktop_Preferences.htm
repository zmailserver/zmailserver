设置您的 Desktop 首选项

“首选项”选项卡包含一些功能页面，使您可以为您在 Zimbra Desktop 建立的每个帐户自定义您的邮箱、地址簿和行事历的工作方式。当您在 Zimbra Desktop 上建立帐户时，默认的操作即得以配置。

您在常规首选项文件夹中所作的更改将影响所有帐户的基本 Zimbra Desktop 版面。

全局

对此选项卡所作的更改将影响 Zimbra Desktop 上的所有邮件帐户。

  • 主题用于显示一系列不同的 UI 面板。您可随时更改邮箱的背景色。

  • 语言。指 Zimbra Desktop 中显示文本所用的语言。（可在 Zimbra Web Client 中使用）

  • 如果在此选项卡中显示的时区不正确，您可更改默认时区。默认时区决定了您收发邮件的日期和时间以及行事历中的约会所显示的时间。这可能与您计算机上所配置的时区不同。

  • 在搜索设置中，您可选择在所执行的搜索中自动包括“垃圾邮件”和“已删除邮件”文件夹。默认情况下，这两个文件夹没有包括在内。

  • 在搜索语言中，勾选在搜索工具栏中显示高级搜索语言以便在搜索文本框中显示搜索字符串的细节。

  • 启用显示复选框后，便快速选择列表中的内容，以便为内容窗格中的每项内容显示一个复选框。通过此复选框，您可以选择一项或多项内容来同时执行相同的操作，例如，删除、移动、标记为已读/未读。

  • 系统默认值。启用此选项后，可使 Zimbra Desktop 成为您的默认邮件应用程序。当您发送邮件或单击网页上的“邮件发送到”链接时，Zimbra Desktop 的“写信”窗口随即打开。

邮件

  • 显示。选择每页显示多少条目（邮件或会话）。默认设定值为每页显示 25 个条目，但您可选择 10、50 或 100。

  • 显示邮件。对于已经格式化为 HTML 的邮件，可指定是否以 HTML 方式查看。若关闭此项，邮件则以纯文本形式显示，即只显示 HTML 标记，但不显示标记语言版本。

  • 邮件预览。

    • 显示邮件片段。若启用，当光标掠过邮件主题时，邮件列表中的邮件片段则会显示出来。

    • 如果启用了双击后在新窗口中打开邮件，则当您双击一封邮件时，邮件会在一个新窗口中打开。

  • 图像。启用图像后，HTML 邮件便可自动下载图片。启用此选项后，您无需单击显示图像信息即可看到图像。

  • 当我在阅读窗格读取邮件时可用来在“阅读”窗格中设定所查看邮件的标记，即标记为已读，还是未读。可选设置有：立即标记为已读、在阅读窗格查看邮件后过了您定义的若干秒后再标记为已读，或是让在阅读窗格查看的邮件始终保持未读状态。

  • 默认邮件搜索。此设置决定了登录时执行的搜索首选项。默认的搜索就是您的收件箱。当您打开 Desktop 时，您的收件箱中的结果即被显示出来。您可更改此字段，以便在刚打开 Desktop 时显示其他文件夹、标记或保存的搜索项。有关句法，请见查询语言说明。

  • 接收邮件。

当一封邮件到达时：

    • 播放声音。当有新邮件到达收件箱时就会发出“哔”声提醒您。如果把音量调到静音，则听不到声音。

    • 高亮显示“邮件”选项卡。如果没有在“邮件”选项卡操作，则在新邮件到达收件箱时，邮件选项卡会高亮显示。

    • 闪烁显示浏览器标题。当新邮件到达收件箱时，浏览器就会闪烁，直到您打开“邮件”选项卡。

    • 显示弹出的通知。当邮件到达时，一个小对话框就会显示有关该邮件的信息。

以下功能只能在 Zimbra Web Client 帐户中使用。

  • 指定一个邮件转发地址。您可指定将原始邮件从 Zimbra 邮箱中删除。

  • 指定是否对发来的邮件发送自动回复邮件并设置开始和结束日期。自动回复邮件经常被用作外出不在邮件，或休假邮件，以表明您不在办公室并交代相关事项。您还可指定可使用该邮件的开始和结束日期。

  • 我发的邮件。配置我发送的邮件部分，用以处理您发送的邮件的以下两种情况：您自己的地址是邮件的收件人之一；或被包含在一个地址中，如一个包括您自己在内的邮件列表中。

写信

  • 设置默认的文本编辑器，HTML 或纯文本。

  • 设置以 HTML 格式书写邮件、文档和行事历时所使用的默认字体。您可确定字体的样式、大小和颜色，用以作为默认的样式。默认字体为 Arial、12 point black。

  • 每次写信时都打开一个新窗口选项可使您在编写邮件时打开一个独立于您邮箱的新写信窗口。在书写窗口打开的情况下，您可查看和浏览您的邮箱。

  • 每隔几分钟，您所书写的邮件都会自动保存到草稿箱。如果不需要此自动保存功能，可关闭此功能。

  • 选择是否在您的回复邮件正文中包含原始邮件。您可选择包含原始邮件，不包含原始邮件，以附件形式包含原始邮件，在正文中包含但加上您选的前缀，或只包含最近的邮件。（也就是说，只引用发件人写下的内容，而不是以前的文本。）

  • 如果选择引用原始文本，请指定是否需要为转发或回复的前一封邮件文本的每一行加上前缀 > 或 |。

  • 默认情况下，保存邮件副本到“已发送邮件”文件夹处于启用状态。您发送邮件的副本被保存在“已发送邮件”文件夹。

  • 指定在转发邮件时如何包含邮件的原始文本：放入邮件正文、加上前缀放入邮件正文或作为附件一并转发。

签名

可为您的邮件创建各种签名。签名中可包括姓名和其他结束语句，帐户设定中对语句的长短和字符数量上限均有限制。如果您有多个身份，则可创建不同的签名，并分别指派给具体的地址。

联系人

  • 选择以何种形式查看联系人，列表或名片。

  • 选择每页显示的联系人的人数。默认值为 25。

  • 启用可在发送邮件时自动将地址添加到电子邮件联系人文件夹中的功能。

    当发送邮件到一个所有的地址簿中都没有的新地址时，该联系人将登记到您的电子邮件联系人地址簿中。您可在首选项的地址簿标签中关闭这一功能。从设置中取消勾选的将新联系人添加到“电子邮件联系人”。

身份

帐户选项卡用于创建和管理您的身份。身份是您所发送的邮件的“发件人”字段中使用的邮件身份。

默认的样式在基本账户设置中定义。创建新身份时，可利用与“默认身份”相同的设置，也可设置不同的首选项。

邮件过滤器

利用“邮件过滤器”选项卡定义邮件过滤规则。若需了解如何设置邮件过滤规则，请参见过滤工作原理。

行事历

  • 用于选择如何查看 Zimbra 行事历。默认值为按照工作周。

  • 指定在行事历中把哪一天作为一周的第一天。

  • 一个小日历显示在行事历中。您可在任何视图中启用小日历。

  • 可启用在回复之后自动删除邀请，否则，邀请会滞留在收件箱中。

  • 设置会议的提醒通知。您可设置何时进行会议提醒，可以是从不提醒以及最早在会议前 60 分钟提醒。您可选择提醒的类型，声音、闪烁显示浏览器标题或显示一个弹出的通知。

  • 如果您经常创建不包含参会者的约会，请勾选使用快速添加对话框。

  • 如果您在不同时区之间旅行，您可能需要选择在约会视图中显示时区列表，这样在安排约会时就可在必要时更改时区。

导入/导出。

  • 要导出您帐户中的所有文件夹或您帐户中特定的文件夹，参见备份您帐户的内容

  • 将导出的文件夹导入回您的账户

  • 搜索指定文件类型将之导出

  • 导入和导出行事历计划。

  • 导入和导出联系人列表

快捷方式

快捷键是键或键的组合，当输入该键或键的序列时，可快速选择一项操作。  

转到快捷键首选项文件夹，查看快捷键的完整列表，或从任何视图单击 Ctrl Q，随即便会弹出快捷键列表。
