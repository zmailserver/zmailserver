设置桌面提示和提醒

您可以创建新邮件通知的桌面提示并创建约会提醒。要接收提示，您的邮箱必须打开。

  • 电子邮件提示。在“首选项”>“邮件”页面中设置首选项。下面是邮件到达时的提示选项：

    • 播放声音。当有新邮件到达收件箱时就会发出“哔”声提醒您。 如果将音量设为静音，您将听不到声音。须安装 QuickTime 或 Windows Media。

    • 高亮显示“邮件”选项卡。如果没有在“邮件”选项卡上操作，则在新邮件到达收件箱时，邮件选项卡会高亮显示。

    • 显示弹出的通知。 当新邮件到达时，就会弹出提示消息。您必须安装 Yahoo!BrowserPlus 。

  • 在行事历中显示提醒。在首选项>行事历页面上设置首选项。 为您的会议设置提醒时间后，才能设置提醒。提醒对话框将在会议开始之前的指定时间显示。如果您将提醒设置为从不，即便已启用提醒选项，您也不会得到会议提醒。

如果您没有使用行事历，页面顶部不会弹出提醒。启用下列选项之一有助于提示您有提醒信息弹出：

    • 播放声音。当有约会提醒信息弹出时发出“哔”声提醒您。如果您把音量调到静音，您将听不到声音。

    • 闪烁显示浏览器标题。当有约会提醒信息弹出时，浏览器将会闪烁，直到您关闭弹出的窗口。

    除了对您的计算机设置提醒外，您还可以从约会内部设置电子邮件或 SMS 通知。您可以在“首选项”>“通知”页面中配置电子邮件地址和 SMS 号码。
