Изменение часового пояса по умолчанию

Настройка часового пояса по умолчанию, отображаемая в папке «Настройки>Общие», используется для определения местного времени полученных сообщений и мероприятий ежедневника в стандартном веб-клиенте.  

При использовании стандартного веб-клиента часовой пояс компьютера не применяется для установки времени получения сообщения или мероприятий ежедневника. Вместо этого применяется настройка часового пояса на вкладке «Настройки>Общие». Однако при использовании расширенного веб-клиента в качестве отметки времени для полученных сообщений и мероприятий ежедневника используется настройка часового пояса компьютера, а не настройка на вкладке «Общие».

Часовой пояс вашей учетной записи может отличаться от часового пояса компьютера.

  1. Перейдите на страницу Настройки>Общие.

  2. В раскрывающемся меню Параметры входа >Часовой пояс по умолчанию выберите необходимый часовой пояс.

  3. Нажмите кнопку Сохранить.

Примечание. Поскольку расширенный и стандартный веб-клиенты используют разные источники временных поясов для отображения сообщений, одно и то же сообщение может иметь разное время при отображении в разных клиентах.  Этого можно избежать, установив одинаковое время для часового пояса компьютера и часового пояса веб-клиента.
