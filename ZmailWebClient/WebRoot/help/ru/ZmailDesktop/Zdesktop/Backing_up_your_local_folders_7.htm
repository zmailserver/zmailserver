Архивация учетных записей

Архивацию учетных записей Zimbra Desktop можно выполнить на странице «Настройки > Архивация» в разделе «Все учетные записи».  По умолчанию выполняется архивация только локальных папок, поскольку локальные папки содержат элементы, которые не архивируются на других почтовых серверах.  

Архивация учетных записей

  1. Перейдите на страницу Настройки > Архивация.

  2. В разделе Настройка архивации выберите учетные записи, которые следует архивировать. Не забудьте, что элементы локальных папок и учетной записи POP сохраняются на вашем компьютере. Архивацию этих учетных записей следует выполнить в первую очередь. Можно архивировать все учетные записи, созданные в Zimbra Desktop, но следует иметь в виду, что файлы могут иметь довольно большой размер. Что приблизительно узнать, сколько необходимо места на диске, установите курсор над именем учетной записи, и отобразится размер почтового ящика.

  3. Если необходимо запланировать архивацию, укажите, когда следует ее выполнять: Каждый день, Каждую неделю или Каждый месяц.

  4. В поле Расположение введите путь к каталогу, в котором следует сохранить архив. Например, чтобы сохранение выполнялось в папке «ZD_backups» на внешнем диске «E», введите «E:\ZD_backups».

  5. В поле Сохранение выберите количество архивных копий, которые необходимо сохранять. По умолчанию сохраняется 1 архивный файл. Можно выбрать сохранение до 5 архивных файлов, после чего самый старый файл будет перезаписываться.

Если запланирована автоматическая архивация, она выполняется согласно расписанию. Для выполнения архивации программа Zimbra Desktop должна быть открыта. Если сеанс архивации запланирован на время, когда пользователь не выполнил вход в Zimbra Desktop, архивация начнется при следующем входе пользователя в программу.

Если необходимо выполнить архивацию учетных записей немедленно, или для расписания установлено значение Никогда, щелкните Архивировать учетные записи, чтобы запустить сеанс архивации. В зависимости от размера архивируемых учетных записей этот процесс может занять некоторое время.  Не закрывайте Zimbra Desktop во время выполнения архивации. Можно продолжать работу в Zimbra Desktop.

Примечание. Выбор немедленной архивации учетных записей не сбрасывает запланированное время следующей архивации.

Сведения о восстановлении учетной записи см. в разделе Восстановление учетных записей из архивных файлов.
