Настройка учетных записей POP/IMAP, включая Microsoft Exchange IMAP

Для настройки учетной записи необходимо иметь сведения о настройках учетной записи, включая данные серверов входящей и исходящей почты. Эти сведения можно получить у системного администратора или поставщика услуг Интернета.

Для синхронизации можно указать все сообщения в почтовом ящике или только сообщения в папке «Входящие».

Как настроить учетную запись

Прежде чем начать, убедитесь в том, что есть подключение к интернету.

  1. Откройте Zimbra Desktop и в правом верхнем углу нажмите кнопку Установка.  

  2. Нажмите кнопку ДОБАВИТЬ НОВУЮ УЧЕТНУЮ ЗАПИСЬ.  В списке «Тип учетной записи» выберите необходимый тип. Откроется страница «Установка учетной записи Zimbra». Введите следующие данные.

Имя учетной записи. Это имя учетной записи, отображаемое в Zimbra Desktop. При настройке нескольких учетных записей старайтесь использовать легко различимые имена для каждой учетной записи. Например, для учетной записи Yahoo! можно использовать имя «Yahoo».

Ваше полное имя. Введите ваше имя. (Необязательно).

Адрес электронной почты. Введите ваш электронный адрес для данной учетной записи.

  1. В разделе Получение почты введите следующие данные.

Имя пользователя. Введите имя пользователя для проверки почты. Обычно эта запись имеет следующий формат: username@example.com.

Пароль. Введите пароль для входа в эту учетную запись.

Сервер входящей почты. Введите адрес сервера входящей почты. Обычно эта запись имеет следующий формат: mail.example.com.

Безопасность. При необходимости установите флажок Во время доступа к серверу используется шифрование SSL. Если сервер входящей почты использует другой порт, нажмите кнопку Изменить, чтобы изменить номер порта. Если вы не знаете номер порта, используемого сервером входящей почты, обратитесь к системному администратору.

  1. В разделе Отправка почты введите следующие данные.

SMTP-сервер. Введите адрес сервера исходящей почты. Обычно эта запись имеет следующий формат: smtp.example.com.

Безопасность. При необходимости установите флажок Во время доступа к серверу используется шифрование SSL. Если сервер входящей почты использует другой порт, нажмите кнопку Изменить, чтобы изменить номер порта. Если вы не знаете номер порта, используемого сервером входящей почты, обратитесь к системному администратору.

Выберите пункт Проверка подлинности, если для отправки сообщений требуются имя пользователя и пароль. Если этот флажок установлен, укажите имя пользователя и пароль.

Обратный адрес. Если вы хотите использовать для обратного адреса имя и электронный адрес, отличные от имени и электронного адреса исходящей почты, введите нужные имя и электронный адрес.

  1. В разделе Настройки синхронизации укажите график синхронизации с сервером. По умолчанию опрос выполняется каждые 15 минут. При выборе параметра вручную необходимо нажать кнопку «Отправить/Получить» на панели инструментов Zimbra Desktop, чтобы синхронизировать учетную запись с сервером Zimbra. Выбор слишком малого периода времени, например одной минуты, может снизить производительность компьютера.

Этот параметр можно изменить в любое время, нажав кнопку Установка и изменив соответствующее значение для выбранной учетной записи.

  1. По окончании нажмите кнопку Подтвердить и сохранить.

Клиент Zimbra Desktop проверит данные вашей учетной записи. В случае успешной проверки откроется диалоговое окно Служба создана. Если проверка окажется неудачной, снова отобразится диалоговое окно «Установка», в котором можно просмотреть и откорректировать введенные данные.

  1. Нажмите кнопку Запустить Desktop, чтобы открыть Zimbra Desktop. Вы можете работать в Zimbra Desktop во время синхронизации вашей учетной записи.
