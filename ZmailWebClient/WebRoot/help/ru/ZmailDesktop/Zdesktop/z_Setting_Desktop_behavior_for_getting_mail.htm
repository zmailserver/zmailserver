Настройка действий Desktop при получении почты

В разделе «Настройки > Почта > Отображение сообщений» включение одного из параметров в разделе При нажатии кнопки "Отправить/Получить" определяет, следует ли изменять вид текущей страницы при нажатии кнопки «Отправить/Получить».

  • Если установлен флажок Запустить поиск по стандартным критериям, то при нажатии кнопки Отправить/Получить запускается поиск сообщений, которые необходимо доставить в папку «Входящие», и вид автоматически сменяется с папки, которую вы просматривали, на папку «Входящие».  

  • Если выбран параметр Обновить текущий вид, то при нажатии кнопки Отправить/Получить в любом виде папки запрос на получение новой почты ищет обновления в текущем виде папки и новую почту в других папках. Текущий вид папок при этом не меняется.  При получении новой почты в другие папки имена таких папок выделяются полужирным шрифтом и обновляется количество непрочитанных сообщений.
