Поиск сообщений

Можно выполнять поиск сообщений, разговоров или контактов

  • В текстовом окне поиска введите запрос, как описано в разделе Настройки быстрого поиска.

  • Функция расширенного поиска используется для определения более подробных критериев поиска.

Поиск сообщений с помощью текстового окна поиска

  1. Введите критерии поиска в окне Поиск, например has:attachment или имя отправителя.

  2. Щелкните стрелку поиска и выберите тип поиска в списке.

  3. Нажмите кнопку Поиск.

Поиск сообщения с помощью подробных критериев

  1. Выберите пункт Дополнительно, чтобы открыть область расширенного поиска.

  2. Щелкните значки на панели задач, чтобы открыть панель ввода критериев, которые вы хотите использовать для поиска.

  3. Введите критерии поиска.

Поиск выполняется после ввода критериев поиска в области поиска, после выбора других вариантов в какой-либо другой области поиска, например, при смене переключателя или варианта поиска в раскрывающемся списке.

Элементы, соответствующие критериям поиска отображаются внизу справа. Если включена функция «Разговор», отображается список разговоров, которые содержат сообщения, соответствующие критериям поиска.

Примечание. Чтобы обновить область результатов поиска, нажмите кнопку Поиск.
