Работа в портфеле

Портфель можно использовать для общего доступа к документам, создаваемым в Портфеле, или документам и изображениям, загруженным в Портфель, и управления этими документами. Портфель позволяет сохранять файлы в учетной записи, поэтому доступ к этим файлам можно получить с любого компьютера, войдя в свой почтовый ящик.

В портфель можно загружать документы, электронные таблицы, презентации, изображения, pdf-файлы с компьютера или из сети. Кроме того, в нем можно сохранять вложения почтовых сообщений.  В портфеле можно создавать разные папки для систематизации файлов,.

В портфеле хранятся переданные копии исходных файлов на компьютере.  Изменения, вносимые в файл в портфеле, не сохраняются в исходном файле.

Создание папок портфеля

  1. Откройте вкладку Портфель.

  2. Выберите пункт Портфель из списка Создать на панели инструментов.

  1. Откроется диалоговое окно Создать новый портфель. Введите имя новой папки портфеля.

Это имя должно быть уникальным в иерархии папок вашего почтового ящика. Папки портфеля не могут иметь такие же имена, как у папок верхнего уровня в почте, ежедневнике или адресной книге Zimbra. Например, если у вас есть ежедневник с именем «Праздники», то дать имя «Праздники» портфелю верхнего уровня нельзя. Имя «Праздники» можно дать портфелю в другом портфеле.

  1. Выберите иерархию папки. Можно создать новый портфель верхнего уровня или поместить его в уже существующий портфель.

  2. Нажмите кнопку ОК. Новый портфель появится в области обзора.

Загрузка файлов

В процессе загрузки файлы передаются с компьютера в почтовый ящик. Это позволяет получить доступ к файлам в любое время, для этого нужно только войти в почтовый ящик. Количество файлов, которое можно загрузить в портфель, определяется лимитом учетной записи.

Загрузка файлов

  1. Откройте портфель и на панели инструментов щелкните Загрузить файл.

  2. В диалоговом окне Загрузить новый файл в портфель нажмите кнопку Обзор, чтобы найти файлы для загрузки. Можно выбрать несколько файлов для одновременной загрузки, если эти файлы расположены в одной папке. Для добавления других файлов нажмите кнопку Добавить и найдите необходимые файлы.

При загрузке файла, имя которого совпадает с именем существующего файла, откроется диалоговое окно «Информация»

    • Если нажать кнопку «Нет», файл не будет загружен.

  1. Если нужно загрузить все файлы, нажмите кнопку ОК. Файлы отображаются в портфеле.  

Просмотр содержимого в областях просмотра

Можно выбрать режим просмотра файлов, которые содержатся в папках Портфеля, в области просмотра.

  • Область просмотра внизу / область просмотра справа. Файлы, типы которых распознаются, отображаются в области просмотра. Область просмотра можно отключить.

Общий доступ к портфелю

Доступ к папкам портфеля может быть предоставлен другим пользователям внутри и за пределами организации. Другим пользователям можно предоставить разрешение на создание и редактирование содержимого в портфеле или только на просмотр страниц. См. раздел «Общий доступ к папкам Zimbra».
