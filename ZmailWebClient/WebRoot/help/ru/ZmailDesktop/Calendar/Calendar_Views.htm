Параметры просмотра ежедневника

Вид ежедневника можно настроить для отображения рабочих дней недели и часов.  На странице настроек ежедневника можно настроить приведенные ниже параметры.

  • Вид по умолчанию, который будет отображаться после открытия ежедневника. Ниже перечислены возможные виды ежедневника.

    • День отображает мероприятия одного дня

    • Рабочая неделя отображает мероприятия с понедельника по пятницу

    • Неделя отображает мероприятия  семи дней. На вкладке «Ежедневник» страницы настроек можно выбрать, какой день является первым днем недели.

    • Месяц отображает мероприятия одного месяца

    • Расписание отображает каждый ежедневник в отдельном столбце.

  • Первый день рабочей недели.
