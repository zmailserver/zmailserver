Изменение доступа к общей папке

Примечание. Только для учетных записей Zimbra.

Изменить или аннулировать право доступа можно в любое время.

  1. Щелкните правой кнопкой мыши общую папку и выберите пункт Изменить свойства. В разделе Общий доступ в диалоговом окне отображаются пользователи с правом доступа.

  2. Щелкните ссылку Изменить для изменения разрешений или
    щелкните ссылку Отменить для отмены разрешений.

  3. Выберите, необходимо ли отправить сообщение с описанием изменений.

  4. Нажмите кнопку ОК.
