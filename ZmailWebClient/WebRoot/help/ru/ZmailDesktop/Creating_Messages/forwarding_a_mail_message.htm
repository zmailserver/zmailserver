Пересылка сообщения

С помощью функции Переслать можно переслать сообщение другому пользователю. При пересылке сообщение пересылается вместе с вложениями, а при использовании функции Ответить сохраняется только текст сообщения без вложений.

Пересылка сообщений настраивается на вкладке Настройки > Создание сообщений. В сообщение можно включить либо исходный текст пересылаемого сообщения с данными полей заголовка сообщения «Кому», «От» и «Тема», либо исходное сообщение в виде вложения.

Пересылка сообщения

    1. Откройте сообщение, которое необходимо переслать.

    2. На панели инструментов нажмите кнопку .  Откроется окно создания сообщения. Текст пересылаемого сообщения будет находиться в теле нового сообщения, если в настройках создания сообщений не выбрана пересылка сообщений в виде вложений.   

    3. Введите адреса в поля Кому, Копия и Скрытая копия. .

    4. При необходимости вверху сообщения можно ввести несколько строк нового текста, чтобы получатель знал причину пересылки сообщения.

    5. Нажмите кнопку Отправить.
