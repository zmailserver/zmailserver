О программе Zimbra Desktop

Zimbra Desktop — это офлайновый клиент, позволяющий управлять всеми вашими учетными записями электронной почты в одной программе. Почта Zimbra, Yahoo!, Gmail, Outlook и любые другие рабочие или личные почтовые ящики, работающие через протокол POP или IMAP, легко настраиваются в Zimbra Desktop. Каждая учетная запись обслуживается отдельно от других.

После настройки учетных записей ваши почтовые сообщения загружаются на компьютер и будут доступны всегда, независимо от подключения к Интернету. См. раздел Работа в автономном режиме. В Zimbra Desktop можно читать, отвечать, пересылать и удалять сообщения во всех ваших почтовых ящиках. Общий объем сообщений зависит только от доступного места на диске вашего компьютера, а не от заранее установленного лимита учетной записи.

Когда в любой ваш почтовый ящик приходит сообщение, оно, при наличии доступа к интернету, автоматически загружается в Zimbra Desktop. Для этого программу даже не надо открывать. На панели задач Windows или в строке прикрепления на Macintosh появляется значок конверта с указанием общего количества непрочитанных сообщений во всех ваших почтовых ящиках. Щелкните этот значок, чтобы открыть программу.

Основные возможности Zimbra Desktop

Кроме доступа к разным учетным записям электронной почты в одном окне, Zimbra Desktop позволяет также управлять контактами и содержит встроенный ежедневник, средство для работы со списками задач и «портфель» для хранения вложений.

Список основных функций см. в разделе Обзор функций Zimbra Desktop.

Поддерживаемые типы учетных записей

Можно настроить следующие учетные записи электронной почты.

  • Учетная запись Zimbra. Zimbra Desktop можно использовать для доступа к своей учетной записи электронной почты, если используется Zimbra. Сюда входит электронная почта, папки, адресные книги, ежедневники, списки задач и документы портфеля. Для доступа к учетной записи Zimbra с помощью браузера следует использовать Zimbra Web Client. Настройки, сделанные в ZWC для учетной записи Zimbra, доступны в Zimbra Desktop. Примечание. Для доступа ко всем функциям Zimbra через Zimbra Desktop версия Zimbra должна быть ZCS 5.0 или выше. Версию Zimbra можно узнать у системного администратора Zimbra.

  • Yahoo! Mail. Можно настроить учетные записи Yahoo! Mail, Yahoo! Mail Plus, Yahoo! Small Business, Ymail и Rocketmail.  К вашим сообщениям, папкам почты и адресной книге можно обращаться из Zimbra Desktop.

  • Gmail. В учетной записи Gmail должен быть включен IMAP-доступ. Возможно, для включения IMAP понадобится войти в учетную запись Gmail. К вашим сообщениям электронной почты и контактам можно будет обращаться из Zimbra Desktop.

  • Учетная запись Microsoft Exchange IMAP. К вашим сообщениям в почтовом ящике Exchange IMAP можно будет обращаться из Zimbra Desktop.

Если ваш почтовый ящик не принадлежит ни к одному из этих типов, но поддерживает доступ по протоколу POP3 или IMAP4, его также можно настроить в Zimbra Desktop. Для этого необходимо иметь полные сведения о настройках учетной записи, включая данные серверов входящей и исходящей почты. Эти сведения можно получить у системного администратора или поставщика услуг Интернета.
