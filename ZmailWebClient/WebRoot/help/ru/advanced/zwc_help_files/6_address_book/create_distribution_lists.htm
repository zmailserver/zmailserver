Создание списка рассылки с управляемым доступом

  1. Перейдите к раскрывающемуся меню Адресная книга>Новый контакт и выберите вариант Список рассылки.

  2. Введите имя списка рассылки в заголовке страницы. Это адрес электронной почты списка рассылки. Имя должно быть доступно. Если оно недоступно, отобразится сообщение об ошибке и придется ввести другое имя.

  3. Введите Короткое имя для списка рассылки. Чтобы использовать имя, а не адрес электронной почты списка рассылки, введите короткое имя.

  4. Введите описание списка рассылки.

  5. Выберите настройку Личная, если список рассылки необходимо скрыть в списке глобальных адресов (GAL).

  6. Выберите параметры почты . В параметрах почты можно установить правила относительно пользователей, которые могут просматривать список рассылки.

  •  Отправка в этот список разрешена всем пользователям

  • Отправка в этот список разрешена только участникам

  • Отправка в этот список разрешена только внутренним пользователям

  • Отправка в этот список разрешена только этим пользователям. В текстовом поле введите имена и адреса электронной почты пользователей, которые могут осуществлять отправку в этот список.

  1.   Выберите тип новых запросов на подписку, которые необходимо разрешить для этого списка рассылки.

  2.  Установите правила запросов на отмену подписки для этого списка рассылки.

  3.  Если необходимо разрешить другим пользователям управлять списком, добавьте их адреса в текстовом поле имен и адресов электронной почты владельца списка.  Если вы не хотите управлять списком, удалите свое имя перед добавлением других адресов.  

  4.  На вкладке «Участники» выберите новых участников группы, которые будут добавлены в список. Можно выполнять поиск адресов в списках контактов или в глобальном списке адресов, а также можно добавлять адреса, которые не включены в эти списки.  Введите адрес электронной почты в текстовом поле Или введите адреса ниже. Разделяйте адреса запятыми (,) или точками с запятыми (;). Адреса электронной почты, которые вы вводите, не проверяются.

  5.  Нажмите кнопку Сохранить. Учетные записи добавляются в список рассылки, а адресная книга списка рассылки учетной записи обновляется для отображения имени списка рассылки.
