Отправка файлов из портфеля

Во время работы в портфеле можно отправить ссылку на файлы, содержащиеся в портфеле, или отправить собственно файл в виде вложения в сообщение. Чтобы открыть файл, получатели должны иметь соответствующее приложение.

Отправка сообщения электронной почты со ссылкой на файл

Можно отправить сообщение со ссылкой на файл в портфеле. Для работы с этим файлом адресат должен иметь соответствующее разрешение, а также соответствующее приложение, чтобы его открыть. Все вносимые в связанный файл изменения отображаются в файле, который находится в папке вашего портфеля.

  1. Откройте вкладку Портфель и выберите файл, к которому необходимо предоставить общий доступ.

  2. Щелкните Действия и выберите Отправить ссылки. Откроется область подтверждения с напоминанием о том, что у адресата должно быть разрешение на просмотр содержимого папки. Чтобы продолжить, нажмите кнопку Да.

  3. Откроется окно нового сообщения электронной почты со ссылкой в теле письма. Когда сообщение будет готово, нажмите кнопку Отправить, чтобы отправить его.

Отправка файла как вложения в сообщение электронной почты

Файл можно отправить как вложение в сообщение электронной почты непосредственно из портфеля. Чтобы открыть файл, получатели должны иметь соответствующее приложение.

  1. Откройте вкладку Портфель и выберите файл для вложения.

  2. Щелкните Действия и выберите Отправить как вложение. Откроется страница создания сообщения с вложенным файлом.

  3. Добавьте адрес электронной почты адресата и завершите составление сообщения.

  4. Нажмите кнопку Отправить.
