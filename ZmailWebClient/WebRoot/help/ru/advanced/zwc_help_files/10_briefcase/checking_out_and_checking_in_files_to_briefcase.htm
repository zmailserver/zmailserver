Извлечение и возвращение файлов в портфеле

Файлы портфеля можно открывать в любое время, для этого нужно только войти в свою учетную запись. Для открытия, чтения и обновления файлов, созданных без применения документов и инструментов портфеля, необходимо соответствующее приложение.

Если к портфелю предоставлен общий доступ, можно использовать функции извлечения и возврата файлов портфеля для управления изменением файлов. Процесс извлечения несколько отличается от процесса возврата, в зависимости от того, был ли файл загружен в портфель или создан как документ портфеля.

  1. Откройте вкладку Портфель и выберите файл, который необходимо извлечь.

  2. Выберите Действия>Извлечь файл.

  3. В открывшемся диалоговом окне выберите вариант сохранения файла на компьютере. Поскольку файл извлечен, для изменения его необходимо сохранить на своем компьютере; данный файл доступен только для чтения, поэтому изменения не могут быть сохранены.

  4. С помощью соответствующего инструмента внесите в файл изменения и сохраните его.

  5. Когда файл сохранен на компьютере, правой кнопкой мыши щелкните заблокированный файл в списке файлов портфеля и выберите действие Вернуть файл.

  6. В открывшемся диалоговом окне Вернуть файл в портфель перейдите в папку компьютера, где сохранен измененный файл. Чтобы добавить сведения об изменениях, укажите их в области текста Примечания.

  7. Нажмите кнопку Возврат. Файл загружен. Блокировка будет снята, и сведения о файле, включая номер новой версии и примечания, будут загружены в список файлов.
