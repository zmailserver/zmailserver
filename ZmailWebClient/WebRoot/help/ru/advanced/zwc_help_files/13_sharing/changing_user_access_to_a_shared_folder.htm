Изменение доступа пользователя к общей папке

Вы можете изменить права на общий доступ, назначенные пользователю или группе.

  1. Щелкните правой кнопкой мыши общую папку, которую следует изменить, и выберите пункт Изменить свойства. В диалоговом окне Свойства папки отобразятся имена пользователей, которые совместно используют папку.

  2. В разделе Общий доступ для этой папки рядом с именем, которое следует изменить, выберите пункт Изменить.

  3. В диалоговом окне Свойства профиля общего доступа измените права.

    4. Можно временно отключить доступ к общей папке без отмены прав на общий доступ, выбрав вариант Нет.

  4. Выберите, необходимо ли уведомить пользователя по электронной почте.

  5. Нажмите кнопку ОК.
