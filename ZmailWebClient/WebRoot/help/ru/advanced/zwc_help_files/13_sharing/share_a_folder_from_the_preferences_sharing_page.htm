Предоставление общего доступа к папке на странице «Настройки общего доступа»

  1. Перейдите на страницу Настройки >Общий доступ.

  2. В разделе Предоставление общего доступа к папке в раскрывающемся меню выберите тип папки, к которой следует предоставить общий доступ.

  3. Нажмите кнопку Общий доступ.

  4. Выберите папку, которую следует использовать в качестве общего ресурса, и нажмите кнопку OK.

  5. В диалоговом окне Свойства профиля общего доступа выберите предоставление общего доступа внутренним, внешним или открытым пользователям.

  6. В зависимости от выбранной категории пользователей выполните следующие действия.

    7. •    Для категории Внутренние пользователи или группы введите адреса электронной почты пользователей. В области Роль выберите права для доступа.

    •    Для категории Внешние гости введите адреса электронной почты пользователей. Внешним гостям отправляется сообщение электронной почты со ссылкой на общий ресурс. Они создают отображаемое имя и пароль при первом просмотре общедоступного ресурса.

    •    Для категории Открытый введите URL-адрес, который следует использовать.

  7. В области Сообщение укажите тип сообщения, которое следует отправить пользователям, которым вы предоставляете доступ. Это сообщение может быть отправлено внутренним пользователям и внешним гостям. Примечание. При выборе варианта Открытый необходимо предоставить пользователям URL-адрес.

  8. Нажмите кнопку ОК.
