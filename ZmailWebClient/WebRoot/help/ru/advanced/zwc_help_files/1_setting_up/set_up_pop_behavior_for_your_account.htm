Настройка параметров POP для учетной записи

Если настраивается дополнительная внешняя учетная запись POP, можно указать параметры доступа по протоколу POP при загрузке сообщений в ваш почтовый ящик.

  1. Перейдите на страницу Настройки>Почта, в раздел Доступ из других почтовых клиентов.

  2. Выберите параметры обработки сообщений, полученных из учетных записей POP.

    3. Параметр

    Описание

    Загрузка сообщений

    Выберите, будет ли в вашу учетную запись загружена вся почта, которая хранится в учетной записи POP, или только новая почта, полученная с момента настройки доступа к внешней учетной записи.

    Входящие нежелательные сообщения

    Выберите, требуется ли передавать в вашу учетную запись сообщения, отправленные в папку «Спам» учетной записи POP. Сообщения будут добавлены в папку учетной записи POP.

    Входящие удаленные сообщения

    Выберите, как сообщения, удаленные в учетной записи POP, будут сохраняться в вашей учетной записи.

  3. Нажмите кнопку Сохранить.
