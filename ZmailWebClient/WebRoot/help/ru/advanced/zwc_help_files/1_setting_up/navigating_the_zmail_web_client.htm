Перемещение по интерфейсу @ClientNameLong@

При входе отображается экранное изображение, подобное показанному ниже. Это представление «Просмотр почты». Оно изменится, если открыть другое приложение, например адресную книгу или ежедневник.

 

 

Параметр

Описание

Заголовок окна

В заголовке окна отображается:

•      поле «Поиск»

•      имя пользователя, выполнившего вход. С помощью меню имени пользователя можно воспользоваться справкой по продукту, выбрать новые функции, отобразить информацию о продукте, изменить пароль и использовать функции выхода.

Вкладки приложений

На вкладках в этой строке отображаются приложения, к которым можно получить доступ, например приложения «Почта» и «Ежедневник».

Область обзора

В области обзора отображаются следующие элементы.

•      Папки — включая системные папки («Входящие», «Отправленные», «Черновики»,

•      «Спам» (и «Корзина»), а также созданные пользователем папки.

•      Профили поиска — созданные поисковые запросы, которые вы сохранили для дальнейшего использования.

•      Теги — любые созданные вами теги. Щелкните тег для быстрого просмотра всех помеченных тегом сообщений.

•      Модули Zimlet — созданные для интеграции приложений сторонних производителей с почтовым ящиком.

•      В ежедневнике в области обзора отображаются список ежедневника и модули Zimlet.

Календарик

Календарик является необязательным элементом. Его можно отключить на странице «Настройки>Ежедневник».

Панель инструментов

Здесь отображаются действия, которые можно выполнять в используемом в данный момент приложении.

Область содержимого

Область содержимого меняется в зависимости от используемого приложения. Вид «Входящие» отображает все сообщения и разговоры в папке «Входящие».

Область чтения

В области чтения отображаются сообщения электронной почты, если эта функция включена. Включите эту функцию с помощью раскрывающегося меню вида на панели инструментов почты.

Панель поиска

В данной области отображаются параметры поиска. Также в этой области можно сохранять поисковые запросы.

Отображение справки

Если провести курсором над элементом, например кнопкой управления, разговором или почтовыми адресами в заголовках сообщений, появится небольшое окно с текстом описания элемента.

Информация о папке

Если навести курсор на папку, отобразится количество сообщений в папке и размер папки.
