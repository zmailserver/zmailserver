Использование горячих клавиш

Для многих функций и команд доступны горячие клавиши.

Просмотр горячих клавиш

•       Чтобы отобразить список горячих клавиш во время работы с почтовым ящиком, нажмите Ctrl+Q.

Если требуется, чтобы окно горячих клавиш оставалось открытым во время использования учетной записи, щелкните Новое окно.

Печать горячих клавиш

•       Перейдите на страницу Настройки>Горячие клавиши, чтобы отобразить список горячих клавиш. Щелкните Печать в верхней правой части окна, чтобы распечатать список.
