Удаление задачи или списка задач

Удаление задачи

  1. Перейдите на страницу Задачи.

  2. В области обзора выберите список задач, содержащий задачи, которые следует удалить.

  3. В области содержимого выберите задачи, которые следует удалить. Можно выбрать несколько задач, установив флажок рядом с каждой задачей.

  4. На панели инструментов нажмите кнопку Удалить.

Можно сразу отменить это действие, щелкнув ссылку Отменить, которая отображается вверху страницы. Задача восстанавливается в исходном расположении.

Удаление списка задач

Удалить можно только те списки задач, которые вы сами создали.

  1. Перейдите на страницу Задачи.

  2. В области обзора выберите список задач, который следует удалить.

  3. В раскрывающемся меню выберите Удалить.

Можно сразу отменить это действие, щелкнув ссылку Отменить, которая отображается вверху страницы. Список задач перемещается в исходное расположение.
