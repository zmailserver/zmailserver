Создание подробной задачи

Списки задач можно использовать для создания задачи и отслеживания ее выполнения. Кроме того, можно писать заметки и вкладывать файлы для простого доступа к ним из задачи. Наличие всей необходимой информации в задаче полезно при предоставлении общего доступа к списку задач.

Рейтинг длительность проекта можно с помощью даты начала и окончания проекта и установки приоритета. Начиная выполнение задачи, можно выбрать для нее статус, Не началась  или Выполняется, и указать процент выполнения.

  1. Перейдите на страницу Задача и выберите пункт Новая задача.

  2. В разделе Подробности введите информацию о задаче. Чтобы присвоить задаче имя, необходима только тема.

    Параметр

    Описание

    Тема

    (Обязательное поле.) Введите краткое описание задачи. Введенный текст становится описанием в списке задач.

    Место

    Введите место осуществления задачи.

    Приоритет

    Выберите приоритет для задачи.

    Список задач

    Выберите список задач, в котором данная задача должна отображаться.

     

  3. В разделе Выполнение введите сведения для отслеживания выполнения задачи. Эти сведения необязательны для задачи.

    Параметр

    Описание

    Cтатус

    Выберите статус задачи. В любой момент можно вновь открыть задачу и изменить статус.

    Процент

    Выберите процент, используемый со статусом задачи. В любой момент можно вновь открыть задачу и изменить процент.

    Дата начала

    Задайте дату начала для задачи.

    Срок

    Задайте дату выполнения для задачи.

    Напоминание

    Установите дату и время для напоминания.    

    Уведомление

    Если на странице Настройки > Уведомления настроено уведомление по электронной почте или SMS, установите соответствующий флажок для напоминания.

    •      Если уведомление не настроено, щелкните Настройка, чтобы перейти на страницу Настройки > Уведомления и выполнить настройку.

    Добавить вложения

    Щелкните Добавить вложения на панели инструментов для ввода каких-либо примечаний в текстовом поле и добавления вложений.

  4. Нажмите кнопку Сохранить.
