Печать сообщения электронной почты 

При печати сообщения по умолчанию используется шрифт размером 12 пунктов. Эти настройки можно изменять на странице «Настройки>Общие».

Примечание.   Не используйте функцию браузера Файл>Печать. Страница может быть плохо отформатирована и сообщение трудно будет прочитать.

  1. Щелкните правой кнопкой мыши сообщение и выберите пункт «Печать». Сообщение откроется в новом окне. Откроется стандартное диалоговое окно печати.

  2. Выберите принтер.

  3. Нажмите кнопку ОК. Сообщение будет отправлено на выбранный принтер.

  4. Закройте окно сообщения.

Также можно выбрать печать с помощью ссылки «Действия» на панели инструментов.
