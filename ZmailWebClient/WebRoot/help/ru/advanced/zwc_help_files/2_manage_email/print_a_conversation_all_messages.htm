Печать разговора (все сообщения)

  1. Щелкните правой кнопкой разговор, который следует распечатать, и выберите пункт Печать. Разговор откроется в новом окне. Откроется стандартное диалоговое окно печати.

  2. Выберите принтер.

  3. Нажмите кнопку ОК. Разговор будет отправлен на выбранный принтер.

  4. Закройте окно разговора.

Также можно выбрать печать с помощью ссылки «Действия» на панели инструментов.

Печать одного сообщения в разговоре

  1. Откройте разговор и выберите сообщение для печати.

  2. В конце сообщения щелкните значок с шестеренкой и выберите пункт «Печать». Сообщение откроется в новом окне. Откроется стандартное диалоговое окно печати.

  3. Выберите принтер.

  4. Нажмите кнопку ОК. Сообщение будет отправлено на выбранный принтер.

  5. Закройте окно сообщения.
