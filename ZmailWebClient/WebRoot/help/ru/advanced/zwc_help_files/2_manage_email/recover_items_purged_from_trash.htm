Восстановление элементов, удаленных из папки корзина

Если после щелчка правой кнопкой папки «Корзина» отображается пункт Восстановить удаленные элементы , значит есть возможность восстановить сообщения электронной почты, которые были удалены из Корзины.

  1. Щелкните правой кнопкой мыши почтовую папку Корзина и выберите пункт Восстановление удаленных элементов.

  2. Выберите одно или несколько сообщений для восстановления.

    3. С помощью текстового поля Поиск можно найти определенный элемент, который требуется восстановить.

  3. Щелкните значок Папка под текстовым полем «Поиск» и выберите папку назначения или создайте новую папку.

  4. Нажмите кнопку ОК.
