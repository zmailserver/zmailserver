Функции безопасности мобильного устройства

Администратор может создавать политику безопасности мобильных устройств, требующую от пользователя создания личного идентификационного номера (PIN-кода) на мобильном устройстве. Если вы забыли PIN-код, обратитесь к своему системному администратору.

При использовании PIN-кода политика может включать функцию блокирования пользователей, если они вводят неправильный PIN-код.  Системный администратор настраивает блокировку PIN-кода таким образом, что, если кто-либо пытается войти в устройство с неправильным паролем более указанного количества раз, выполняется очистка устройства, запускаемая локально. В результате все данные удаляются с устройства. Настройки политики устанавливаются только администратором, и пользователи не могут их изменять. Для получения дополнительной информации обратитесь к системному администратору.
