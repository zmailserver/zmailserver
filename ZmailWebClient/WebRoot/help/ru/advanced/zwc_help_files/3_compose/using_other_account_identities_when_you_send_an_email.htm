Использование других личных данных учетной записи при отправке сообщения электронной почты

При отправке сообщения электронной почты оно определяется при помощи учетной записи электронной почты. Когда настроена учетная запись электронной почты, имя учетной записи является основными личными данными пользователя. Однако для управления различными типами электронной почты можно создавать другие личные данные электронной почты, называемые также образами.

Например, можно создать один образ для деловой переписки по электронной почте, а другой — для личной.

Если создано несколько образов или добавлены внешние учетные записи, то при отправке сообщения электронной почты можно выбрать учетную запись, которую необходимо использовать в качестве адреса отправителя.
