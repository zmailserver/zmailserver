Проверка правописания сообщения

Функцию проверки правописания можно всегда использовать для проверки правильности написания.

  1. На панели инструментов щелкните ссылку Abc. Будут выделены слова в сообщении, неизвестные программе проверки правописания.

  2. Щелкните выделенное слово правой кнопкой мыши. Откроется раскрывающееся меню с вариантами исправления.

  3. Выберите правильное слово. Слово будет выделено другим цветом.

  4. Чтобы принять изменения и закрыть программу проверки правописания, нажмите кнопку Продолжить редактирование.

Перед закрытием программы проверки правописания можно вернуть исходное написание исправленного слова. Щелкните выделенное слово и выберите его исходное написание вверху всплывающего окна.

Проверка правописания всех сообщений перед их отправкой

  1. Перейдите на страницу Настройки>Почта>Написать письмо.

  2. Выберите параметр Обязательно проверять правописание перед отправкой сообщения.

  3. Нажмите кнопку Сохранить.
