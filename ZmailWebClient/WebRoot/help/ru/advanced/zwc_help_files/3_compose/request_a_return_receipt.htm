Запрос подтверждения прочтения

Сообщение, отправляемое другим пользователям @ClientNameShort@, может быть отмечено как требующее отправки обратно подтверждения прочтения, после того как получатель откроет сообщение. Однако в учетных записях получателей должна быть включена функция Подтверждение прочтения.

Чтобы установить ее по умолчанию:

•       перейдите на страницу Настройки>Почта  и в разделе Создание сообщений выберите параметр Всегда запрашивать подтверждение прочтения.

Чтобы установить запрос для отдельного сообщения:

•       прежде чем отправлять сообщение, в раскрывающемся списке Параметры выберите параметр Запросить подтверждение прочтения.
