Выполнение быстрого поиска

Для быстрого поиска следует ввести слово или имя, не указывая никаких ключевых слов. Например, чтобы найти все почтовые сообщения, содержащие слово " tim" в любом месте в строке темы, в поле «Кому», «От», «Копия». или «Скрытая копия», в теле сообщения или вложенном файле, можно просто ввести слово «Петр» в поле поиска и нажать кнопку Поиск.

Можно ввести любое значение: слово, имя, фамилию, номер телефона или домен. Для контактов из адресных книг в поле необходимо вводить целое слово. Например, для поиска контактов по номеру телефона, необходимо ввести полный номер телефона как он указан в записи контакта, выполнять поиск только по коду города нельзя.

Прежде чем воспользоваться функцией быстрого поиска, укажите тип элемента, который нужно искать. Стрелка списка слева от поля поиска позволяет выбирать тип элементов для поиска. Можно выполнять поиск сообщений, личных контактов, личных и общедоступных контактов, контактов из глобального списка адресов, страниц или элементов всех типов.

Выполнение быстрого поиска

  1. В текстовом поле Поиск введите содержимое, которое необходимо найти. С параметрами поиска можно ознакомиться в разделе Синтаксис поиска.

  2. В раскрывающемся меню рядом с текстовым полем «Поиск» выберите часть учетной записи, в которой необходимо выполнить поиск.

  3. Нажмите клавишу Enter.

После окончания поиска перейдите на вкладку «Поиск».
