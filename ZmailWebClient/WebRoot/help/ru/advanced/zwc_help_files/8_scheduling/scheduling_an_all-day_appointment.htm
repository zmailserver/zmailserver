Планирование встречи на весь день

С помощью ежедневника можно запланировать встречу на весь день, которая называется событием.

  1. В заголовке ежедневника нажмите кнопку Новая встреча.

  2. Если над строкой темы отображается поле От, для учетной записи создано несколько образов. Выберите образ, который следует использовать для создания этой встречи.

  3. В текстовом поле Тема введите короткое описание собрания.  Введенный здесь текст становится описанием в ежедневнике.

  4. Выберите Весь день, а затем дату начала и окончания.

  5. В текстовом поле «Участники» введите адреса электронной почты, разделяя адреса точкой с запятой (;). При вводе имен открывается планировщик для отображения информации о доступности участников.

Чтобы найти время, когда доступны все участники, нажмите кнопку Предложить время. Отобразится ежедневник с информацией о доступности. При вводе имен участников средство планирования начинает предлагать время, когда доступны все участники.

Примечание. Если участник разрешил отображение информации о доступности только определенным пользователям и вы в этот список не входите, возможность просматривать расписание занятости будет отсутствовать.

Если некоторые из участников собрания не являются обязательными, но их необходимо пригласить, установите флажок Показать «Необязательно» и введите адреса электронной почты в поле. Это также можно сделать с помощью планировщика. Щелкните значок рядом с именем и выберите «Необязательный участник».  После отправки приглашения их имена будут перечислены как необязательные.

  1. В текстовом поле Место укажите место проведения собрания. Если данное место содержится в каталоге компании, после начала ввода появляется список соответствующих мест. Можно щелкнуть Предложить расположение, чтобы просмотреть список мест, доступных в компании.

  2. В разделе Отображать выберите параметр доступности для этой встречи, в какой ежедневник следует добавить встречу, и нужно ли отметить эту встречу как личную.

  3. Напоминание. Установка уведомления для напоминания участникам о встрече.

  4. Если на странице Настройки > Уведомления настроено уведомление по электронной почте или SMS, установите соответствующий флажок: Электронная почта или SMS.  Если уведомление не настроено, щелкните Настройка, чтобы перейти на страницу Уведомления для его настройки.

  5. В текстовом поле под планировщиком введите сообщение электронной почты, которое будет включено в приглашение на встречу. Чтобы добавить вложения, щелкните значок скрепки на панели инструментов.

  6. По умолчанию от участников требуется ответ на приглашение. Если ответ от участников не нужен, на панели инструментов ежедневника выберите Параметры > Запросить ответ и снимите флажок.

  7. Если приглашение в данный момент отправлять не нужно, нажмите кнопку Сохранить. В противном случае нажмите кнопку Отправить.
