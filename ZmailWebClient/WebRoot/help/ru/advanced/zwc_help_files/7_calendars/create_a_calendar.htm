Создание ежедневника

Для пользователя создан один ежедневник. Можно создавать дополнительные ежедневники для ведения учета функций различных типов. Например, можно создать один ежедневник для рабочих мероприятий, один — для личных и еще один — для мероприятий компании.

При ведении нескольких ежедневников можно группировать их для отображения по иерархии. Например, можно группировать все общие ежедневники в папке ежедневников с названием «Общие». Также можно перемещать ежедневники.

•       В области обзора ежедневника выберите ежедневник и перетащите его в тот ежедневник, в который его необходимо вложить.

•       Или правой кнопкой мыши можно щелкнуть ежедневник, который необходимо переместить, и выбрать действие Переместить. В диалоговом окне Выбрать папку выберите ежедневник, в который необходимо выполнить перемещение.

Добавление ежедневника

  1. Откройте вкладку Ежедневник.

  2. На панели инструментов выберите Создать>Ежедневник.

  3. В текстовом поле Имя укажите имя ежедневника.

  4. Выберите цвет для отображения мероприятий ежедневника.

  5. Чтобы другие пользователи не могли просматривать ваш ежедневник при планировании встреч, выберите команду Исключить этот ежедневник при составлении отчета о свободном/занятом времени.

  6. Для синхронизации этого ежедневника с удаленным выберите Синхронизировать встречи из удаленного ежедневника. Введите URL-адрес для удаленного файла iCal.

  7. Нажмите кнопку ОК. Новый ежедневник появится в списке ежедневников.
