Составление и отправка зашифрованного сообщения

Примечание. Эта функция доступна только в выпуске ZCS Network Edition на расширенной версии веб-клиента.

Можно отправлять зашифрованные сообщения адресатам, которые предоставили вам свой открытый ключ и сертификат. Сертификат адресата сохраняется в разделе адресатов на странице контактов в адресной книге. 

  1. На панели инструментов нажмите кнопку Создать.

  2. Если для параметра безопасности не установлена по умолчанию отправка зашифрованных сообщений с цифровой подписью, щелкните вариант Безопасность и выберите Использовать подпись и выполнить шифрование.

  3. В поле Кому: введите адрес пользователя или выберите ссылку Кому, чтобы искать адрес в своих адресных книгах.

  4. Введите тему сообщения.

  5. Составьте сообщение.

  6. Чтобы добавить вложение, выберите команду Вложить файл и перейдите к файлу или перетащите файл из папки на компьютере в раздел заголовка сообщения электронной почты.

  7. Нажмите кнопку Отправить.

    • Нельзя сохранять черновик зашифрованного сообщения.

    • Нельзя проверять правописание зашифрованных сообщений.
