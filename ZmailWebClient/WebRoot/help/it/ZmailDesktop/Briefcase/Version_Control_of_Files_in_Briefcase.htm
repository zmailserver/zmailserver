Controllo delle versioni dei file nella Valigetta

Il controllo delle versioni consente di gestire le modifiche apportate ai file nella Valigetta. Il numero di versione è indicato nell'elenco dei documenti. Quando si visualizzano i documenti in base alla cronologia delle versioni, è possibile vedere le versioni predenti del file e il relativo autore. È possibile ripristinare una versione precedente del file nonché eliminare le versioni non più necessarie.

Nota: Ogni versione di un file occupa parte della quota dell'account dell'utente.  

I file caricati possono essere estratti (checkout) per impedire che più persone lo utilizzino contemporaneamente nella Valigetta. Questi file vengono scaricati sul desktop. Quando si carica il file aggiornato, esso viene archiviato (checkin) e ne viene modificata la versione.
