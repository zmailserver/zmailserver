Backup degli account

È possibile eseguire il backup degli account di Zimbra Desktop  dalla pagina Preferenze > Backup nella sezione Tutti gli account.  Per impostazione predefinita viene eseguito solo il backup delle cartelle locali, in quanto contengono gli elementi di cui non viene eseguito il backup nei server di email.  

Per eseguire il backup degli account

  1. Passa alla pagina Preferenze > Backup.

  2. Nella sezione Configura backup, seleziona gli account di cui eseguire il backup. Gli elementi nelle cartelle locali e nelle cartelle degli account POP vengono salvati nel computer. È consigliabile eseguire il backup di questi account. È possibile eseguire il backup di tutti gli account creati in Zimbra Desktop, tuttavia questi file possono essere di dimensioni notevoli. Per sapere la quantità approssimativa di spazio su disco necessaria, posiziona il puntatore del mouse sul nome dell'account per visualizzare le dimensioni della casella postale.

  3. Se desideri pianificare il backup, seleziona Ogni giorno, Ogni settimana o Ogni mese.

  4. Nel campo Percorso digita il percorso della directory in cui salvare il backup. Ad esempio, per salvare nella cartella ZD_backups nell'unità esterna E, digita E:\ZD_backups.

  5. Nel campo Conversazione seleziona il numero di backup da salvare. Per impostazione predefinita viene salvato 1 solo file di backup. È possibile scegliere di salvare fino a 5 backup prima che venga sovrascritto il backup meno recente.

Se è stata pianificata l'esecuzione automatica del backup, quest'ultimo viene eseguito in base alla pianificazione. Zimbra Desktop deve essere aperto affinché vengano eseguiti i backup. Se una sessione di backup è pianificata in un orario in cui non sei connesso a Zimbra Desktop, esso verrà eseguito all'accesso successivo.

Se si desidera eseguire il backup degli account immediatamente o se la pianificazione è impostata su  Mai, clicca su Esegui backup degli account ora per eseguire una sessione di backup. A seconda delle dimensioni degli account in fase di backup, l'operazione può richiedere tempo.  Non chiudere Zimbra Desktop  durante l'esecuzione del backup. È possibile continuare a utilizzare Zimbra Desktop.

Nota: Se si seleziona di eseguire il backup degli account l'ora di backup pianificata successiva non viene reimpostata.

Per ripristinare l'account, vedi Ripristino degli account dai file backup.
