Eliminazione di un account da Desktop

  1. Clicca su Imposta in Zimbra Desktop. Trovi il link all'angolo destro.

Viene visualizzata la pagina I miei account che li elenca tutti.

  1. Per eliminare un account, clicca su Elimina. Viene visualizzata una finestra di dialogo contenente il seguente messaggio: Le impostazioni dell'account e i dati scaricati verranno eliminati. L'operazione non influirà sui dati sul server. Vuoi continuare comunque?

Tutti i dati vengono eliminati dal computer, compresi i messaggi email, gli elenchi di contatti, le agende, le pagine del Blocco note e i file salvati nella Valigetta. La funzione Elimina rimuove i file dal computer, ma non dal server.

Nota:  I file salvati nella cartella locale vengono salvati solo in Zimbra Desktop. Se vuoi salvare questi file prima di eliminare il tuo account devi esportare i file nella Cartella locale. Puoi farlo dalla scheda Preferenze>Importa/Esporta.
