Allegare file al messaggio

Quando componi un messaggio email, puoi allegare file dal computer o dalla Valigetta, contatti della Rubrica e messaggi email.

Nota – I destinatari del tuo messaggio email devono avere il software appropriato per aprire e leggere il file. I file creati in software di uso comune come Microsoft Office, potranno essere aperti anche da altri utenti sui sistemi desktop più diffusi. I formati di file comuni, come i file di testo o HTML e i file di immagini, ad esempio GIF o JPG, possono essere aperti con diversi programmi. Dipende comunque sempre dal tipo di sistema usato dall'utente.

Per aggiungere allegati mediante il link Aggiungi allegato:

  1. Scrivi il messaggio nel modo descritto in Composizione e invio di un nuovo messaggio email.

  2. Clicca su  . Nella finestra di dialogo Allega file vengono visualizzate le schede Il mio computer, Valigetta, Mail e Contatti. Seleziona la scheda relativa al tipo di allegato.

    • Il mio computer. Individua una cartella nel computer e seleziona il file. Seleziona il file e clicca su Apri, oppure trascina i file da una cartella sull'intestazione del messaggio.

    • Valigetta. Seleziona la cartella desiderata. I relativi file verranno visualizzati a destra. Seleziona i file da allegare.

    • Mail. Se il messaggio email da allegare non si trova in In arrivo, seleziona la cartella. Puoi cercare un messaggio nella cartella selezionata. I messaggi email sono visualizzati a destra. Seleziona il messaggio da allegare.

    • Contatti. Seleziona la Rubrica. I nomi nella Rubrica sono visualizzati a destra. È possibile cercare un nome nelle rubriche. Seleziona i nomi da allegare. Verranno allegati al messaggio come file .vcf.

Nota: Utilizza CTRL-clic o MAIUSC-clic per aggiungere più allegati contemporaneamente. Nella finestra di dialogo Allega file vengono aggiunti più allegati su una riga.

  1. Dopo aver aggiunto tutti gli allegati in tale scheda, clicca su Allega.

  2. Clicca su Invia per inviare il messaggio e gli allegati.

Per aggiungere allegati dal computer mediante trascinamento

  1. Seleziona uno o più file e tieni premuto il cursore sui file file da allegare.

  2. Trascina i file file nell'area di intestazione del messaggio e rilascia il cursore. I nomi dei file verranno visualizzati nell'intestazione.

 

 

 

Rimozione di un allegato

Per rimuovere un allegato, deseleziona la casella corrispondente prima di inviare il messaggio email.
