Tipi di file allegati supportati

È possibile aprire qualsiasi allegato a un messaggio direttamente da Zimbra Web Client, a condizione che nel computer sia installata l'applicazione appropriata e che l'estensione non sia stata bloccata dall'amministratore di sistema.  Per aprire ad esempio un documento creato con Microsoft Word, è necessario poter eseguire Microsoft Word nel computer.

Nella maggior parte dei computer, il tipo di file è indicato dall'estensione, rappresentata in genere da un punto seguito da un codice di tre lettere. Ad esempio: expenses.xls o New-Items.doc. Se il tipo di file è supportato dal software installato sul tuo computer, in genere è sufficiente cliccare due volte sul file e il computer avvierà automaticamente l'applicazione in grado di leggerlo.

Molti tipi di file possono essere aperti direttamente dal browser mediante diversi plug-in. In genere, se clicchi sul link di un file in una pagina, il browser rileva automaticamente se il plug-in adatto è disponibile e in caso contrario ti chiede di scaricarlo. Tieni presente che i criteri dell'amministratore del sistema potrebbero non permettere agli utenti finali di scaricare e installare alcun tipo di software.

Note:
in Windows, il computer presuppone che il tipo di file indicato nel nome del file corrisponda al formato effettivo. Anche se la formattazione interna è corretta, se nel nome del file non è indicata l'estensione giusta, il computer potrebbe non riconoscere il file e non aprire quindi l'applicazione adatta.

Un numero sempre più elevato di virus informatici viene progettato per essere diffuso tramite gli allegati. Pertanto, in alcuni ambienti, gli amministratori del sistema hanno scelto di bloccare i messaggi in entrata contenenti determinati tipi di allegati, in particolare i file con l'estensione EXE o ZIP come parte del nome di file. Se invii un messaggio email a una persona che usa un sistema configurato per bloccare determinati tipi di allegati, è possibile che la notifica del blocco non venga inviata.

Le estensioni file più usate sono le seguenti

  • .TXT - Indica un file di testo, ovvero un file contenente solo informazioni di testo. I file di testo possono essere aperti e letti mediante una vasta gamma di editor di testo, tra cui Blocco note, WordPad, Word, editor della riga di comando, browser e persino programmi di fogli di calcolo.

  • .HTM - Indica un file HTML, ovvero uno speciale tipo di file di testo che in genere viene aperto mediante un browser web.

  • .ZIP - Indica un file compresso mediante la compressione ZIP. È necessario disporre di WinZip o di un altro programma di compressione in grado di leggere ed estrarre il file ZIP.

  • .EXE - Indica un eseguibile o un file di programma. Cliccando due volte su un file del genere, viene avviata l'applicazione corrispondente.

  • .DOC - Indica un file di un documento Microsoft Word.

  • .JPG - Indica un'immagine grafica in formato JPEG. Puoi aprire i file JPEG in un browser oppure usare uno dei numerosi programmi di grafica come Paint o Adobe Photoshop.

  • .GIF - Indica un'immagine grafica in formato GIF. I file GIF possono essere aperti in un browser o con un programma di grafica.

  • .MPG - Indica un filmato o un'animazione nel formato MPEG. spesso riproducibile con un plug-in del browser o con un lettore come Windows Media Player.

  • .WMV - Indica un filmato nel formato WMV, supportato solo da Windows.

  • .PDF - Indica un documento in formato PDF. In genere può essere aperto all'interno del browser tramite un plug-in.
