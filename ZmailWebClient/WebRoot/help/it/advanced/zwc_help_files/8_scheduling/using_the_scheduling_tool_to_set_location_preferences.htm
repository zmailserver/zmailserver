Uso dello strumento di pianificazione per visualizzare la disponibilità dei luoghi

Lo strumento di pianificazione dell'agenda è utile per individuare i primi orari e le sedi per le riunioni disponibili nella tua azienda. Puoi anche configurare le tue preferenze per gli orari della riunione, gli edifici, le località e la dimensione della sala.

Mentre inserisci i nomi dei partecipanti, lo strumento di pianificazione inizia a suggerire gli orari e i luoghi nel riquadro Panoramica. Puoi anche decidere di cercare un luogo per la tua riunione prima di aggiungere i partecipanti. Se configuri le tue preferenze sul luogo, vengono mostrate solo le località preferite.

Uso dello strumento di pianificazione per impostare le preferenze di luoghi

Se vuoi sempre prenotare delle sale di un luogo specifico, puoi utilizzare lo strumento Luoghi suggeriti nell'applicazione Agenda per scoprire la disponibilità di queste sale quando crei un appuntamento.

  1. Nella pagina Nuovo appuntamentot, clicca su Suggerisci una località.

  2. Nel riquadro che si apre, clicca sull'icona con la ruota dentata.

  3. Nella finestra di dialogo Preferenze suggerimenti , nella sezione omonima inserisci le informazioni sul luogo. Puoi inserire solo il nome del luogo o puoi essere più specifico.

  4. Clicca su OK.

La colonna a destra mostra le località che soddisfano i tuoi requisiti. Le ubicazioni delle sale sono visualizzate se sono state configurate nella directory aziendale.

Dopo aver configurato tutto ciò per un appuntamento, le informazioni vengono salvate e rimangono disponibili per quando configuri altri appuntamenti.
