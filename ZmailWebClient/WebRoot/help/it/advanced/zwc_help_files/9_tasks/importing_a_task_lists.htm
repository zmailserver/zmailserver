Importazione di elenchi impegni

Puoi importare gli elenchi di impegni salvati in un file .tgz.

  1. Passa alla pagina Preferenze>Importa/Esporta.

  2. In Importa>File, clicca su Sfoglia per individuare il file .tgz dell'elenco di impegni da importare.

  3. Per Destinazione, clicca su Tutte le cartelle e seleziona l'elenco di impegni in cui saranno importati gli impegni o creane uno nuovo.

  4. Clicca su Importa.

Le dimensioni del file .tgz determinano la durata dell'importazione che potrebbe anche richiedere alcuni minuti.
