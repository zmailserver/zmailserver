Funzioni di sicurezza per i dispositivi cellulari

L'amministratore può impostare delle opzioni di sicurezza per i dispositivi cellulari che richiedono la creazione di un numero di identificazione personale (PIN) sul tuo dispositivo cellulare. Se dimentichi il tuo PIN, contatta il tuo amministratore di sistema.

Se hai un PIN, le opzioni di sicurezza potrebbero prevedere una funzione di blocco per quegli utenti che non immettono il PIN corretto.  L'amministratore di sistema può anche configurare l'impostazione di blocco PIN in modo che, se qualcuno tenta di accedere a un dispositivo con una password errata per un numero di volte superiore a quello impostato, viene avviata automaticamente la cancellazione locale del dispositivo. In questo modo vengono cancellati tutti i dati su tale dispositivo. Queste impostazioni possono essere configurate soltanto dall'amministratore e non possono essere modificate dagli utenti.  Per ulteriori informazioni, rivolgiti al tuo amministratore di sistema.
