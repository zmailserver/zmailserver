Salvataggio di una query di ricerca

Se  crei una ricerca che vuoi usare in seguito, hai anche la possibilità di salvarla.

  1. Crea la query di ricerca.

  2. Ne menu a tendina della scheda Cerca, seleziona Salva come cartella di ricerca.

  3. Inserisci un Nome per la ricerca.

  4. Seleziona un Colore per la ricerca.

  5. Seleziona il percorso in cui salvare la ricerca.

  6. Clicca su OK.

Le ricerche vengono salvate nel pannello di navigazione.

Per usare una ricerca salvata, clicca sulla cartella della ricerca. I risultati vengono visualizzati immediatamente nel riquadro del contenuto.
