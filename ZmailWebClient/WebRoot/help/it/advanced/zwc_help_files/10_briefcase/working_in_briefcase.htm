Uso della Valigetta

La Valigetta può essere usata per condividere e gestire i documenti che si creano nella Valigetta o i documenti e le immagini che carichi nelle tue cartelle della valigetta. Usa la Valigetta per salvare file nell'account in modo da potervi accedere ogni volta che entri nell'account da qualsiasi computer.

Puoi condividere le tue cartelle della valigetta con altri che possono visualizzare e modificare i file nella Valigetta condivisa.

Puoi creare più cartelle della valigetta e quindi caricare qualsiasi tipo di file dal tuo computer o dalla rete di computer, salvare gli allegati inviati con i tuoi messaggi email e creare nuovi documenti.

I file nella Valigetta sono copie del file originale, indipendentemente dal fatto che il file si trovi sul tuo computer o sia un allegato alla tua email.  Pertanto, le modifiche apportate ai file nella Valigetta non si riflettono sul file originale.

Creazione di cartelle Valigetta

  1. Passa a Valigetta>Cartelle.

  2. Clicca sull'icona con la ruota dentata e seleziona Nuova valigetta.

  3. Inserisci un Nome per la nuova cartella della valigetta.

    4. Il nome della cartella della valigetta deve essere univoco all'interno della gerarchia di tutte le cartelle della casella di posta. Questo nome non può coincidere con quello di una cartella di livello superiore di posta, agenda o rubrica. Se ad esempio hai un'Agenda chiamata Festività, non puoi assegnare lo stesso nome a una Valigetta di livello superiore. Si può però denominare Festività una Valigetta all'interno di un'altra Valigetta.

  4. Seleziona il Colore del nuovo file della valigetta.

  5. Seleziona la gerarchia di cartelle, in Cartelle o in un'altra cartella della valigetta. Puoi creare una nuova Valigetta di livello superiore o inserirla in una Valigetta esistente.

  6. Clicca su OK.

Caricamento dei file dalla valigetta

  1. Passa alla scheda Valigetta e seleziona la valigetta in cui salvare il file.

  2. Clicca su Carica file e Sfoglia per trovare il file da caricare.  

  3. Nel campo di testo Note aggiungi tutte le informazioni sul file. Queste informazioni sono mostrate nell'elenco dei file.

  4. Per aggiungere un altro file, clicca su Aggiungi.

    5. Non puoi caricare due file con lo stesso nome nella stessa Valigetta. Se si carica un file che esiste già nella cartella, viene mostrata una finestra di dialogo Conflitto di caricamento. Puoi cliccare su Sì per caricare il file, sovrascrivendo il file esistente o cliccare su No.

    Il numero della versione viene aggiornato se un file viene sovrascritto.

  5. Dopo aver caricato tutti i file, clicca su OK.

Nota:   Se hai selezionato la valigetta errata, puoi usare la funzione di trascinamento per spostare il file nella cartella Valigetta corretta.

Se un file è troppo grande, viene visualizzato un avviso. I file caricati influiscono sulla quota di spazio disponibile dell'account.
