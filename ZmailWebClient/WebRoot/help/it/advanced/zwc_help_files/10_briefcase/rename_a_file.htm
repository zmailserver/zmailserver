Ridenominazione di un file

Rinominando un file se ne crea un'altra versione.

  1. Passa alla scheda Valigettae seleziona il documento che vuoi rinominare.

  2. Clicca con il tasto destro del mouse sul file e scegli Rinomina. Il campo Nome viene modificato in un campo modificabile.

  3. Modifica questo campo per cambiare il nome del file. Ricorda che è necessario conservare la stessa estensione nel nome del file per consentirne la modifica in un secondo tempo.

  4. Premi Invio.
