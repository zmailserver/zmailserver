Aggiunta di file alle cartelle Valigetta

Il caricamento di un file lo copia dal tuo PC al tuo account. In questo modo il file è disponibile ogni volta che accedi al tuo account. Puoi caricare i file nei seguenti modi:

•      Facendo clic sugli allegati che ricevi con i messaggi email.

•      Usando lo strumento di caricamento del file quando lavori nella Valigetta

•      Usando gli strumenti del client WebDav
