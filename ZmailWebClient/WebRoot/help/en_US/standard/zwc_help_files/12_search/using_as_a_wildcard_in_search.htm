Using * as a Wildcard in Search

The asterisk (*) is a wildcard character that can be used to match multiple characters.  Use the wildcard to create searches where there are unknown characters or various endings. The * wildcard cannot be used as the first character in a search term.

To use the asterisk as a wildcard, enter the first characters of a search term and add the * at the end. For example, do* returns items with the word dog, door, etc.
