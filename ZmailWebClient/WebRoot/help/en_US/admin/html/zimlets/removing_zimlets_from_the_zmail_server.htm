Removing Zimlets from the Zimbra Server

Only remove your custom Zimlets. You should not remove Zimlets that are shipped with Zimbra Collaboration Server.  If you do not want to use the Zimlets, you can disable access.

To remove Zimlets

  1. In the Navigation pane's Configuration section, click Zimlets. The Zimlets content pane opens.

  2. Select the Zimlet and click Undeploy. An Information box asks you to verify that you want to remove the Zimlet.  

  3. Click Yes.  The Zimlet and all associated files are uninstalled. The Zimlet zip file is not deleted.  You can remove the Zimlet from /opt/ zimbra/zimlets.
