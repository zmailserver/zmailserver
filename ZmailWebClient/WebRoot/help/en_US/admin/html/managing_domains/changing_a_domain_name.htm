Changing a Domain Name

A domain name can be changed with the CLI tool, zmprov -l re [olddomain.com] [newdomain.com].

Renaming a domain actually creates a new domain, moves all addresses to the new domain and deletes the old domain. All account, aliases, distribution lists, and resource addresses are changed to the new domain name. The LDAP is updated to reflect the changes.

See the Administration Guide, Managing ZCS Configuration chapter.
