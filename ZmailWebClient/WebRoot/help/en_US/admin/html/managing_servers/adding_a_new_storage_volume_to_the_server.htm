Adding a New Storage Volume to the Server

Each ZCS mailbox server is configured with one primary message volume and an index volume. Additional secondary message volumes can be added.

Before adding a new volume, the directory path must already exist.

  1. Select the server from the Servers list.

  2. Go to the Volumes tab and click Add.  

  3. Enter the volume name, root path for the new volume, and select the volume type:  

  • Primary Message

  • Secondary Message

  • Index

  1. If you want to compress blobs, check Compress Blobs and set the compression threshold.

  2. Click OK..
