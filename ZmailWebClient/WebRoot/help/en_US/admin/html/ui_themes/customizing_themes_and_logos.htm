Customizing Themes

You can change themes either as a global setting or by domain. For information about how changing the colors works, see Customize theme colors .

Procedure to change the background color

  1. Go to Configure>Global Settings, or Configure>Domains.

  2. From the Themes page, change a base color for Background, Secondary, Selection and Foreground in the color pallet next to each base color. You can also enter a hexadecimal color code for an exact color.  

  3. Click Save.

  4. Flush the cache. In the Informational dialog that opens, select Yes.

  5. Click Flush and then OK.

When users log on they see the new colors.

  • To reset colors back to the ZCS defaults, click Reset all color of the themes.
