Calling Back a Voice Mail Message

Note:  This feature is for ZCS Network Edition only and is available only in the Advanced Web Client.

Reply to a voice mail message by calling the sender back using the Click-To-Call feature.

  1. Select the Voice tab.

  2. In the call history list, select the voice mail message you want to reply to.

  3. Click Call Back. The number that is associated with the caller is dialed.
