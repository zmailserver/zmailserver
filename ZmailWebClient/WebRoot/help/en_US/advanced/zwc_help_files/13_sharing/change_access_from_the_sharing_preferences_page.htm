Change Access From the Sharing Preferences Page

  1. In the Folders shared by me section on the Sharing page select the name of the user to change.

  2. In the Actions column you can edit, revoke or resend a shared item.

Option

Description

Edit

To open the Shared Properties dialog and change the role.

Revoke

To open the Revoke Share dialog and cancel the shared folder.

Resend

To resend the share notice.
