Create a Distribution List with Managed Access

  1. Go to the Address Book>New Contact drop-down menu and select Distribution List.

  2. Enter a Distribution List name in the page heading. This is the email address of the distribution list. The name must be available. If it is not available, you will received an error notice and will have to enter another name.

  3. Enter a Display Name for the distribution list. If you want a name other than the email address of the distribution list, enter a display name.

  4. Enter the Description for the distribution list.

  5. Select if the distribution list if Private, and the list is hidden in the Global Address List (GAL).

  6. Select the Mail options. Mail options set rules about who can view your distribution list.

  •  Anyone can send to this list

  • Only members can send to this list

  • Only internal users can send to this list

  • Only these users can send to this list. Enter the names and email address of the users that can send to the list in the text box.
