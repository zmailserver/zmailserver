E-mailberichten archiveren

Het is mogelijk dat voor je e-mailaccounts een maximale opslagruimte is toegewezen en dat je e-mailberichten uit je mailbox moet verwijderen om onder de toegewezen opslaglimiet te blijven. In plaats van het verwijderen van deze berichten kun je de bestanden offline opslaan in Zimbra Desktop. Je doet dit door je berichten en bijlagen te verplaatsen naar een map in Zimbra Desktop die niet met je account wordt gesynchroniseerd.

Let op: Wanneer je berichten archiveert, worden de berichten op je computer opgeslagen. Als je computer vastloopt, gaan je gearchiveerde berichten mogelijk verloren.  Ter bescherming van je gearchiveerde gegevens wordt het gebruik van het back-upprogramma van Zimbra aanbevolen. Zie Een back-up maken van de inhoud van je account.

Berichten archiveren voor de accounts van Zimbra

Voor Zimbra Web Client wordt de map Plaatselijke mappen gemaakt in Zimbra Desktop. Berichten die je in deze map plaatst, worden op je computer opgeslagen en van de server verwijderd.

 

Als je berichten voor Zimbra wilt archiveren, dien je onder Plaatselijke mappen submappen te maken waarin je de berichten plaatst.   Je kunt tevens een hele persoonlijke map naar de plaatselijke map slepen.  Wanneer je de volgende keer synchroniseert, worden de berichten die je in de map Plaatselijke mappen hebt geplaatst van de server verwijderd. Berichten in de map Plaatselijke mappen zijn niet beschikbaar wanneer je je rechtstreeks bij je Zimbra-account aanmeldt.

Opmerking:  Wanneer je bestanden uit de plaatselijke map verwijdert, worden deze niet in de Prullenbak geplaatst maar onmiddellijk verwijderd.

Het archiveren van berichten voor je overige e-mailaccounts

Voor de andere e-mailaccounts die je instelt, kun je in Zimbra Desktop mappen maken die niet met de server worden gesynchroniseerd.  Wanneer een bericht wordt verplaatst naar een map die niet met de server wordt gesynchroniseerd, worden deze berichten van de server verwijderd wanneer je Zimbra Desktop de volgende keer met je account synchroniseert. De berichten in deze mappen zijn niet beschikbaar wanneer je je rechtstreeks bij je account aanmeldt.

  1. Als je een nieuwe archiefmap wilt maken, selecteer dan de gewenste account in Zimbra Desktop en klik in het menu Mappen op de rechtermuisknop en selecteer Nieuwe map.

  2. Geef de map een naam en selecteer waar je de nieuwe map wilt maken.

  3. Klik op OK.

  4. Selecteer de desbetreffende map, klik op de rechtermuisknop en selecteer Synchroniseren uitschakelen  

 

Verder kun je het synchroniseren voor elke persoonlijke map uitschakelen.

Wanneer je het synchroniseren uitschakelt, wordt naast de map geen groen bolletje meer weergegeven.

 

Berichten die naar een map zijn verplaatst die niet met je account wordt gesynchroniseerd, worden van de server verwijderd.

Let op: Als je de mappen opnieuw wilt synchroniseren, dien je het synchroniseren opnieuw in te schakelen. Wanneer Zimbra Desktop de volgende keer met je account wordt gesynchroniseerd, wordt de inhoud van de map gesynchroniseerd.
