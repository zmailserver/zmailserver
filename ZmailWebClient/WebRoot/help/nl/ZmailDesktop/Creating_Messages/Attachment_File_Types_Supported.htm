Ondersteunde bestandstypen als bijlage

Je kunt een bestandsbijlage in een e-mailbericht rechtstreeks vanuit Zimbra Web Client openen, mits je werkstation de juiste toepassing heeft en de bestandsextensie niet wordt geblokkeerd door de systeembeheerder.  Als je bijvoorbeeld een document opent dat is gemaakt in Microsoft Word, moet je in staat zijn om Microsoft Word op je computer uit te voeren.

Op de meeste computers wordt het bestandstype opgegeven door de bestandsuitbreiding. Deze is meestal een punt gevolgd door een code van drie letters. Voorbeelden zijn bestanden zoals uitgaven.xls en Nieuwe-items.doc. Als het bestandstype een type is dat door de geïnstalleerde software op je computer wordt ondersteund, kun je meestal dubbelklikken op het bestand; je computer start dan automatisch de juiste toepassing op om dat bestand te kunnen lezen.

Veel bestandstypen kunnen rechtstreeks vanuit je browser worden geopend met verscheidene plug-ins. Meestal detecteert je browser automatisch of je wel of niet de juiste plug-in hebt als je op een bestandskoppeling op een pagina klikt. Als dat niet het geval is, volgt er meestal een link naar de plug-in. Het is mogelijk dat de systeembeheerrichtlijnen eindgebruikers niet toestaan software te downloaden en te installeren.

Opmerkingen:
Op Windows-computers wordt aangenomen dat het bestandstype in de bestandsnaam overeenkomt met de werkelijke indeling. Zelfs als het bestand intern juist is ingedeeld maar niet het juiste type bestandsextensie heeft in de bestandsnaam, wordt het bestand mogelijk niet herkend door je computer en wordt mogelijk niet de juiste toepassing geopend.

Een toenemend aantal computervirussen is ontworpen om zich te verspreiden via bestandsbijlagen. Daarom kiezen systeembeheerders er in bepaalde omgevingen voor om binnenkomende e-mail met bepaalde typen bijlagen te blokkeren. Dit geldt vooral voor bestanden met de extensie . EXE of .ZIP als onderdeel van de bestandsnaam. Als je een e-mailbericht naar iemand stuurt wiens e-mailsysteem is geconfigureerd om bepaalde soorten bijlagen te blokkeren, ontvang je mogelijk een kennisgeving dat de bijlage is geblokkeerd.

Ter verwijzing enkele veelgebruikte bestandsuitbreidingen

  • .TXT - geeft een tekstbestand aan, d.w.z. een bestand met alleen tekstgegevens. Tekstbestanden kunnen worden geopend en gelezen in allerlei verschillende teksteditors, zoals Kladblok, WordPad, Word, opdrachtregeleditors, browsers en zelfs spreadsheetprogramma's.

  • .HTM - geeft een HTML-bestand aan, een speciaal soort tekstbestand dat meestal wordt geopend in een webbrowser.

  • .ZIP - geeft een bestand aan dat is gecomprimeerd met behulp van ZIP-compressie. Je hebt WinZip of een ander compressieprogramma nodig dat het ZIP-bestand kan lezen en uitpakken.

  • .EXE - geeft een uitvoerbaar bestand of programmabestand aan. Als je op een dergelijk bestand dubbelklikt, wordt de toepassing meestal geopend.

  • .DOC - geeft een Microsoft Word-document aan.

  • .JPG - geeft een grafische afbeelding in de JPEG-indeling aan. Je kunt JPEG's openen in een browser of in een van de vele grafische programma's zoals Paint of Adobe Photoshop.

  • .GIF - geeft een grafische afbeelding in de GIF-indeling aan. Je kunt GIF's openen in een browser of grafisch programma.

  • .MPG - geeft een filmclip of animatie in de MPEG-indeling aan. Deze zijn vaak afspeelbaar met een plug-in voor de browser of een speler zoals Windows Media Player.

  • .WMV - geeft een filmclip in de WMV-indeling aan. Deze indeling is uitsluitend bestemd voor Windows.

  • .PDF - geeft een document in de Acrobat Reader-indeling aan. Je kunt deze meestal vanuit je browser openen met een plug-in.
