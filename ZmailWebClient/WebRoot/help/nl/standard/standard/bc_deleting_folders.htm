Mappen verwijderen

Je kunt zelfgemaakte mappen verwijderen. Systeemmappen kunnen niet worden verwijderd.

Wanneer je een map verwijdert, wordt de inhoud ervan verplaatst naar de Prullenbak. De inhoud van de Prullenbak kan worden hersteld door de inhoud naar een andere map te verslepen.

Opmerking: De Prullenbak wordt regelmatig leeggemaakt; dus items in de Prullenbak gaan op een gegeven moment verloren.

Ga als volgt te werk om een map te verwijderen:

  1. Selecteer de map die je wilt verwijderen

  2. Klik in de koptekst Mappen in het deelvenster Overzicht op Bewerken.

  3. Selecteer Deze map verwijderen.

  4. Klik op Map verwijderen.

Als er in de map berichten zijn die je wilt bewaren, verplaats je deze vóór het verwijderen naar een andere map.

De inhoud van een map verwijderen

Je kunt in elk van je mappen alles verwijderen, maar niet de map zelf.

  1. Selecteer de map  waarvan je de inhoud wilt verwijderen.

  2. Klik in de koptekst Mappen in het deelvenster Overzicht op Bewerken.

  3. Schakel het selectievakje Alle items in deze map permanent verwijderen in. Deze bewerking kan niet ongedaan worden gemaakt.

  4. Klik op Alle items verwijderen. De inhoud van de map wordt onmiddellijk verwijderd.
