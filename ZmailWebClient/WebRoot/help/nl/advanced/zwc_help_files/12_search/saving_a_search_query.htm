Een zoekopdracht opslaan

Als je een zoekopdracht maakt die je mogelijk later opnieuw wilt gebruiken, kun je deze opslaan.

  1. Maak de zoekopdracht.

  2. Selecteer Opslaan als zoekmap in het vervolgkeuzemenu op het tabblad Zoeken.

  3. Voer een Naam voor de zoekopdracht in.

  4. Selecteer een Kleur voor de zoekopdracht.

  5. Selecteer de locatie waar je de zoekopdracht wilt opslaan.

  6. Klik op OK.

Zoekopdrachten worden in het deelvenster Navigatie opgeslagen.

Als je een opgeslagen zoekopdracht wilt gebruiken, klik je op de map Zoeken. De zoekresultaten worden onmiddellijk weergegeven in het deelvenster Inhoud.
