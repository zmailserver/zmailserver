Een map maken

Je kunt een zelfstandige door een gebruiker gedefinieerde map of een map in een andere map, inclusief systeemmappen, maken.

Het is niet mogelijk om twee mappen met dezelfde naam te maken binnen dezelfde hogergelegen map. Je kunt bijvoorbeeld niet twee mappen met de naam Expeditie maken in het Postvak IN, maar je kunt wel één map met de naam Expeditie maken in het Postvak IN en een tweede map met die naam in de map Verzonden.

  1. Ga naar de pagina E-mail>mappen en klik op het tandwielpictogram Mappen.

  2. Selecteer Nieuwe map.

  3. Voer de nieuwe Naam voor de map in en selecteer een Kleur voor de map.

    4. Mapnamen kunnen elk teken bevatten behalve een puntkomma (:), schuine streep vooruit
    (/) en vraagteken (").

    Je kunt je abonneren op RSS/ATOM-feed.

  4. Selecteer in de boomstructuur Mappen de locatie voor de nieuwe map.

    5. •    Klik op Mappen om de nieuwe map boven in de lijst weer te geven.

    •    Klik op een bestaande map om je nieuwe map in die map te plaatsen.

  5. Klik op OK.
