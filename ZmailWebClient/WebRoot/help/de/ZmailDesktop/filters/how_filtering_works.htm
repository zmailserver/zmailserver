Filterregeln einrichten

In Mailfiltern können Sie Regeln definieren, um eingehende und ausgehende Mails zu verwalten.  Filter sortieren Ihre Mails automatisch gemäß den von Ihnen eingerichteten Regeln.

Sie können Ihre eingehenden Mails so filtern, dass:

  • Eingehende Mails in Ordner sortiert werden

  • Mails automatisch mit Tags gekennzeichnet werden

  • Mails weitergeleitet werden

  • Mails als gelesen oder als markiert gekennzeichnet wird

  • Mails entsorgt werden

Sie können beispielsweise eine Filterregel definieren, die alle Mails erkennt, die von Ihrem direkten Vorgesetzten gesendet werden. Diese Mails werden dann in dem Ordner "Vom Chef" abgelegt. Oder Sie lassen die Mails einer bestimmten Adresse direkt in den Papierkorb wandern.

Sie können Ihre ausgehenden Mails so filtern, dass:

  • Mails, die im Ordner „Gesendet“ gespeichert sind, in andere Ordner sortiert werden

  • Mails automatisch mit Tags gekennzeichnet werden

  • Mails weitergeleitet werden

  • Mails als gelesen oder als markiert gekennzeichnet wird

  • Mails entsorgt werden

Filterbedingungen

Jeder Filter ist eine Regel mit einer oder mehreren Bedingungen sowie einem Ziel. Jede Filterregel kann mehrere Bedingungen enthalten. Wenn Ihre Vorgesetzte Ihnen beispielsweise Mails von mehreren Adressen sendet, z. B. von jane.smith@company.com oder von jane-smith23@yahoolcom, können Sie den Filter "Chef" mit zwei Bedingungen, eine für jede Mail-Adresse, definieren.

Die Bedingungen sind u. a.:

  • Bestimmte Adressen in den Feldern „Von:“, „An:“, „CC:“ in der Kopfzeile der Mail

  • Wörter bzw. Zeichenfolgen im Betreff oder Inhalt der Mail

  • Vorhandensein bzw. Nichtvorhandensein von Dateianhängen

Für alle Bedingungen können Sie auch angeben, dass diese "nicht" erfüllt sind. Sie können z. B. Mails filtern, die ein bestimmtes Wort nicht enthalten.

Sie können Bedingungen kombinieren, um anhand von komplexeren Kriterien nach Mails zu suchen.

Eine oder Alle

Bedingungen in einer Filterregel können mithilfe der Operatoren Eine oder Alle gruppiert werden. Die Verwendung dieser Begriff entspricht den Suchen vom Typ "AND" und "OR", die in der Suchfunktion beschrieben sind, wobei Eine OR entspricht und Alle AND.

Wenn Sie bei der Definition der Bedingungen für eine neue Filterregel den Operator Eine angeben, dann gilt eine Mail, die eine der Bedingungen erfüllt, als Entsprechung. Wenn Sie jedoch Alle angeben, dann muss die Mail alle in der Filterregel definierten Bedingungen erfüllen, um dem Filter zu entsprechen.

Aktionen für Filterregeln

Jede Filterregel kann eine oder mehrere Aktionen angeben. Die Aktionen sind u. a.:

  • Mail im Posteingang belassen (keine Aktion)

  • Mail in einen anderen, angegebenen Ordner verschieben

  • Mail mit Tag markieren

  • Mail als gelesen oder als markiert kennzeichnen

  • Mail entsorgen. Diese Aktion löscht die Mail einfach. Die Aktion entspricht nicht der Löschfunktion im Menü. Wenn Sie eine Mail löschen, wird sie im Papierkorb abgelegt. Bei der Aktion "Verwerfen" wird verhindert, dass die Mail Ihre Mailbox überhaupt erreicht.

  • Mail an eine andere Adresse weiterleiten

Die Aktion Keine weiteren Filter anwenden  sollte in jeder Filterregel die letzte Aktion sein. Dadurch wird verhindert, dass auf Mails, die der aktuellen Regel entsprechen, weitere Filterregeln angewendet werden.  Wird diese Aktion nicht angegeben, werden nachfolgende Filterregeln auf die Mail angewendet, auch wenn die Mail dem aktuellen Filter entspricht. Sie werden wahrscheinlich nicht wollen, dass eine Mail mehreren Filterregeln entspricht und mehrere unter Umständen widersprüchliche Aktionen darauf angewendet werden.

Zeitpunkt der Filteranwendung

Jede eingehende Mail wird zum Zeitpunkt der Zustellung der Mail anhand der Filterregeln geprüft, und die Filteraktionen werden zu diesem Zeitpunkt auf entsprechende Mails angewendet.

Sie können die Filterregeln auch für eine vorhandene Gruppe der Mailobjekte ausführen. Unter "Mailfilter-Einstellungen" können Sie über die Schaltfläche  Filter ausführen mehrere Filter gegen die ausgewählten Mail-Ordner ausführen.

Filterreihenfolge

Die Filter werden in der Reihenfolge angewendet, in der sie unter Mailfilter-Einstellungen aufgeführt sind. Diese Reihenfolge können Sie ändern.

Jede Mail wird anhand der Bedingungen in jedem Filter geprüft. Wenn die Mail die Bedingungen (entsprechend dem Operator Eine oder Alle der Filterregel) insgesamt erfüllt, gilt sie als Entsprechung. Die Reihenfolge der Bedingungen in einem Filter ist nicht relevant.

Wenn die Mail den Bedingungen entspricht, werden alle der Filterregel zugewiesenen Aktionen in der Reihenfolge ausgeführt, in der sie in dem Filter aufgeführt sind.

Aktiv und: Inaktive Filter

Alle aktuell definierten Filter werden auf der Seite "Einstellungen, Mailfilter" aufgeführt. Über das Kontrollkästchen Aktiv können Sie Filter ein- bzw. ausschalten, ohne die Filterregeln zu löschen.
