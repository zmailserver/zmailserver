Mails filtern

Beim Filtern werden eingehende Mails anhand von Übereinstimmungsregeln geprüft und entsprechende Maßnahmen durchgeführt.

Sie können eingehende Mails filtern, um sie in bestimmte Ordner zu verteilen, um sie automatisch zu kennzeichnen, weiterzuleiten oder zu löschen. Sie können z. B. folgende Regel erstellen:

  • Alle Mails von Ihrem Vorgesetzten werden im Ordner "Vorgaben vom Chef" abgelegt.

  • Alle Mails der Mailing-Liste "Firmenveranstaltungen" werden mit dem Tag "Veranstaltungen"gekennzeichnet.

So erstellen oder ändern Sie eine Filterregel

  1. Klicken Sie auf die Registerkarte Einstellungen.

  2. Öffnen Sie Filter, und klicken Sie auf Neuer Filter.

  3. Daraufhin wird das Dialogfeld Filter hinzufügen angezeigt. Hier definieren Sie die Regel und das Ziel für den neuen Filter.

  4. Geben Sie unter Filtername einen eindeutigen Namen für die Filterregel ein.

  5. Wählen Sie unter Wenn xxx der folgenden Bedingungen erfüllt ist/sind eine Gruppierungseinstellung aus.  

    • Eine bedeutet, dass die Aktion angewendet wird, wenn eine der Bedingungen im Filter erfüllt ist.

    • Alle bedeutet, dass alle Bedingungen im Filter erfüllt sein müssen, damit die Filteraktion angewendet wird.  

Sie können die folgenden Schritte wiederholen, um mehrere Bedingungen und Aktionen in einem Filter einzurichten. Klicken Sie auf den Link "+", um mehrere Bedingungen festzulegen.

    • Wählen Sie in der ersten Dropdown-Liste aus, welchen Teil der Mail die Filterbedingung betreffen soll.  

    • Wählen Sie aus, welcher Vergleich vorgenommen werden soll. Die angezeigten Optionen richten sich nach Ihrer Auswahl in der vorherigen Dropdown-Liste. Es wird beispielsweise "enthält" oder "enthält nicht" angezeigt.

    • Geben Sie im Textfeld ein Wort oder einen Satzteil für den Vergleich ein.

Klicken Sie auf , wenn Sie weitere Bedingungen definieren möchten. Sie können weitere Bedingungen hinzufügen oder mit dem nächsten Bereich fortfahren, in dem Sie eine oder mehrere Aktionen hinzufügen.

  1. Wählen Sie unter Folgende Aktionen ausführen in der Dropdown-Liste eine Aktion aus.

Geben Sie ggf. einen Ordner bzw. ein Tag an. Was Sie angeben, richtet sich nach der ausgewählten Aktion.

Klicken Sie auf , wenn Sie weitere Aktionen definieren möchten. Sie können weitere Aktionen hinzufügen oder Ihre Eingaben mit OK beenden.

  1. Im oberen Bereich des Dialogfeldes „Filter hinzufügen“ befindet sich das Kontrollkästchen Aktiv. Aktivieren Sie dieses Kontrollkästchen, um die Filterregel einzuschalten. Sie können das Kontrollkästchen deaktivieren, wenn keine Regel ausgeführt werden soll.

  2. Aktivieren Sie Keine weiteren Filter anwenden, wenn dies der einzige Filter ist, der für Mails, die die Bedingungen dieses Filters erfüllen, ausgeführt werden soll. Die Aktion Keine weiteren Filter anwenden sollte in jeder Filterregel die letzte Aktion sein. Dadurch wird verhindert, dass auf Mails, die der aktuellen Regel entsprechen, weitere Filterregeln angewendet werden.

     Hinweis:: Wird diese Aktion nicht angegeben, werden nachfolgende Filterregeln auf die Mail angewendet, auch wenn die Mail dem aktuellen Filter entspricht. Sie werden wahrscheinlich nicht wollen, dass eine Mail mehreren Filterregeln entspricht und mehrere unter Umständen widersprüchliche Aktionen darauf angewendet werden.

Änderungen an Filterregeln werden sofort gespeichert. Der Filter wird automatisch auf alle neu empfangenen Mails angewendet, sobald diese eingehen.  

Sie können die Filterregeln auch für bestehende Mails ausführen. Klicken Sie unter „Filtereinstellungen“ auf Filter ausführen und wählen Sie die Mailordner aus, die gegen die ausgewählten Filter ausgeführt werden sollen.

Hinweis: Sie können einen neuen Filter auch aus einer Mail heraus definieren. Klicken Sie mit der rechten Maustaste auf die Mail und wählen Sie Neuer Filter. Das Dialogfeld Filter bearbeiten wird angezeigt. Dabei sind die Felder Von, An und Betreff mit den Daten aus dieser Mail ausgefüllt. Wenn die Filterbedingungen korrekt sind, geben Sie einen Namen für den Filter ein und bestätigen mit "OK".
