Aufgaben verschieben

Sie können eine Aufgabe auf eine der folgenden Arten in eine andere Aufgabenliste verschieben:

  • Klicken Sie mit der rechten Maustaste auf die Aufgabe und wählen Sie Verschieben. Wählen Sie dann im Dialogfeld Aufgabe verschieben die entsprechende Aufgabenliste aus oder legen Sie eine neue Aufgabenliste an. Klicken Sie auf OK.

  • Wählen Sie die zu verschiebende Aufgabe aus, halten Sie die linke Maustaste gedrückt und ziehen Sie die Aufgabe mit der Maus auf die Liste im Übersichtsfenster.

  • Öffnen Sie die Aufgabenseite und wählen Sie im Bereich Details unter Aufgabenlisten die Zielaufgabenliste aus.
