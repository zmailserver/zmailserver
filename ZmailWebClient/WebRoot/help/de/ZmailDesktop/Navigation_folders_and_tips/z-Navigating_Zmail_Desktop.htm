Navigation in Zimbra Desktop

Wenn Sie Zimbra Desktop aufrufen, wird Ihnen eine Vollbildansicht ähnlich der folgenden dargestellt.

Eine Seite des Zimbra Desktop-Clients umfasst die folgenden Bereiche:

  1. Suchleisten-Zeile. In diesem Bereich werden Suche und erweiterte Suche angezeigt. Die Links für Account-Einrichtung und Hilfe finden Sie rechts.

  2. Registerkarten der Anwendungen: Die Anwendungen, auf die Sie Zugriff haben, werden in den oberen Registerkarten angezeigt.

  3. Symbolleiste: Sie zeigt Befehle für die aktuelle Anwendung an. In diesem Beispiel wird die Symbolleiste der Anwendung Mail angezeigt.

  4. Inhaltsfenster. Der Inhalt in diesem Bereich richtet sich nach der jeweils aufgerufenen Anwendung. Im Posteingang zeigt das Inhaltsfenster alle Mails und Unterhaltungen Ihres Posteingangs an. Bei der Anzeige dieser Seite ist das Lesefenster der Bereich unterhalb der Mails in Ihrem Posteingang. Wenn Sie auf eine Mail klicken, wird diese in einem Lesefenster unterhalb der Mail-Liste angezeigt. Um die Größe des Lesefensters anzupassen, klicken Sie den Fensterteiler an und ziehen ihn mit der Maus nach oben oder unten. Standardmäßig ist die Anzeige von Mails im Lesefenster eingeschaltet. Sie können das Lesefenster nach links verschieben oder es über den Link „Ansicht“ in der Symbolleiste ausblenden.

  5. Übersichtsfenster. Zeigt Ihre verschiedenen Accounts an. Es wird jeder Accountname angezeigt. Dieses Beispiel zeigt das Layout „Alle Mailboxen“. Alle Mails aus jedem Ihrer Accounts für Posteingang, Gesendet und Entwürfe werden in diesem Abschnitt zusammengefasst. Diese Einstellung kann auf der Seite “Einstellungen>Allgemein“ festgelegt werden.

    Jeder Account umfasst die folgenden Ordner:

    • Ordner, einschließlich der Systemordner Posteingang, Gesendet, Entwürfe, Spam und Posteingang sowie alle von Ihnen möglicherweise angelegten, benutzerdefinierten Ordner.

    • Suchvorgänge, die Sie möglicherweise gespeichert haben.

    • Tags, die Sie angelegt haben.

    • Zimlet-Links, die für die Integration von Fremdanwendungen in Ihre Mailbox erstellt wurden.

  6. Unterhalb des Übersichtsfensters kann ein Mini-Kalender angezeigt werden. Die Anzeige des Mini-Kalenders ist optional und kann auf der Seite Einstellungen>Kalender aktiviert werden.

Zimbra Desktop öffnen

Wenn Sie auf das Internet zugreifen und Ihr Account auf automatische Synchronisierung eingestellt ist, werden neue Mails automatisch auf Ihren Zimbra Desktop heruntergeladen. Zu diesem Zweck braucht Zimbra Desktop nicht geöffnet zu sein. Im Infobereich unter Windows (oder im Dock unter Mac) wird ein Briefumschlagsymbol und die Gesamtzahl neuer Mails in allen Ihren Accounts seit der letzten Zimbra Desktop-Synchronisation angezeigt. Klicken Sie auf das Briefumschlagsymbol, um die Desktop-Anwendung zu öffnen.

 

Mail- und Terminbenachrichtigung

Sie können die automatische Mail- und Terminbenachrichtigung einrichten. Es stehen verschiedene Benachrichtigungsoptionen zur Verfügung.

  • Unter Windows wird die Benachrichtigung als Symbol in Form eines Briefumschlags im Infobereich angezeigt. Auf dem Mac müssen Sie zunächst Growl installieren, damit die Mail- und Terminbenachrichtigung funktioniert. Außerdem müssen Sie Popup-Benachrichtigung anzeigen in Einstellungen>Mail und >Kalender aktivieren. Hinweis: Diese Funktion ist unter Linux nicht verfügbar.

 

  • Unter Einstellungen>Mail und > Kalender können Sie weitere Arten von Benachrichtigungen einrichten, z. B. das Abspielen von Sounds, das Hervorheben der Registerkarten "Mail" oder "Kalender" oder das Blinken der Browser-Überschrift. Die Popup-Benachrichtigungen gelten für die einzelnen Account-Einstellungen. So können Sie die Einstellungen für einige Accounts aktivieren und für andere nicht.
