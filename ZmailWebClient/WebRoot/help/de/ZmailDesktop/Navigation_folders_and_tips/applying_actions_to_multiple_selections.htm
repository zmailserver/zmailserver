Für mehrere ausgewählte Objekte Aktionen durchführen

Wenn Sie auf der linken Seite auf einen Ordnernamen klicken, um einen Ordner anzuzeigen, wird der Inhalt des Ordners angezeigt, wobei ein Element ausgewählt (hervorgehoben) ist. Sie können zur Durchführung von Aktionen ein oder mehrere Elemente wie folgt auswählen:

  • Klicken Sie auf ein Objekt, um es auszuwählen. Wenn Sie auf ein anderes Objekt klicken, wird das vorherige wieder abgewählt.

  • Doppelklicken Sie, um ein Objekt zu öffnen.

  • Klicken Sie mit der rechten Maustaste auf ein Objekt, um ein Kontextmenü mit Befehlen für dieses Objekt aufzurufen. Sie können auch mehrere Elemente auswählen und dann mit der rechten Maustaste klicken, um denselben Befehl für mehrere Objekte gleichzeitig durchzuführen.

  • Drücken Sie die Strg-Taste und klicken Sie mit der Maus, um mehrere Objekte auszuwählen. Auf diese Weise können Sie auch das aktuelle Objekt abwählen, so dass kein Objekt mehr ausgewählt ist.

  • Drücken Sie die Umschalttaste und klicken Sie mit der Maus, um mehrere aufeinander folgende Objekte auszuwählen. Klicken Sie auf ein Element und dann weiter unten in der Liste auf ein anderes Element und halten Sie dabei die Umschalttaste gedrückt, um alle Elemente zwischen diesen beiden Elementen auszuwählen.

Sie können mehrere Elemente auswählen und durch Rechtsklick alle Elemente als ungelesen kennzeichnen, mit einem Tag versehen, löschen oder verschieben.
