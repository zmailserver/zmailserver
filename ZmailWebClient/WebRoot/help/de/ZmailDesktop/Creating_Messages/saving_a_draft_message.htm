Mails als Entwürfe speichern

Sie können Ihre Mail als halbfertigen Entwurf speichern und zu einem späteren Zeitpunkt vervollständigen und senden. Entwurfsmails werden in Ihrem Ordner "Entwürfe" gespeichert, bis Sie sie senden oder löschen.

Sie können das System so einrichten, dass Ihre Mail, während Sie die Nachricht schreiben, automatisch als Entwurf gespeichert wird. Sie können während des Schreibens auch auf Entwurf speichern klicken, um die Mail-Nachricht als Entwurf zu speichern.

System so einrichten, dass Mails automatisch als Entwürfe gespeichert werden

Gehen Sie zur Registerkarte "Einstellungen>Verfassen" und aktivieren Sie die Option Mail automatisch während des Schreibens als Entwurf speichern. Mails, die Sie verfassen, werden automatisch in Abständen von wenigen Minuten als Entwürfe gespeichert.

So speichern Sie eine Mail-Nachricht manuell:

  1. Verfassen Sie die Mail wie unter Neue Mail verfassen beschrieben.

  2. Klicken Sie auf Entwurf speichern.

Der Mailentwurf wird dann im Ordner Entwürfe gespeichert.

Mailentwurf abrufen

  • Öffnen Sie den Ordner Entwürfe und klicken Sie auf die Mail oder wählen Sie die Mail aus und klicken Sie in der Symbolleiste auf Bearbeiten.

Sie können die Mail wie gewohnt zu Ende schreiben und senden. Nach dem Senden wird die Mail im Ordner "Entwürfe" gelöscht.
