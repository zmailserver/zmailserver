Als Anhänge unterstützte Dateitypen

Sie können jeden Anhang direkt aus dem Zimbra Web Client öffnen, vorausgesetzt, auf Ihrem Computer ist erforderliche Anwendung installiert und die Erweiterung wurde nicht von Ihrem Systemadministrator blockiert.  Wenn Sie z. B. ein Dokument öffnen möchten, das mit Microsoft Word erstellt wurde, müssen Sie Microsoft Word auf Ihrem Computer ausführen können.

Auf den meisten Computern ergibt sich der Dateityp aus der Dateierweiterung. Sie besteht üblicherweise aus einem Punkt gefolgt von einem Code aus drei Buchstaben. Dies können z. B. Dateien sein, wie expenses.xls oder New-Items.doc. Wenn der Dateityp von der auf Ihrem Computer installierten Software unterstützt wird, reicht es normalerweise, auf die Datei doppelzuklicken, um die richtige Anwendung aufzurufen und die Datei zu öffnen.

Viele Dateitypen können mittels verschiedener Plug-ins direkt in Ihrem Browser aufgerufen werden. Wenn Sie auf einer Seite auf eine Dateiverknüpfung klicken, erkennt Ihr Browser normalerweise automatisch, ob Sie das Plug-in haben. Falls nicht, werden Sie aufgefordert, es herunterzuladen. Beachten Sie, dass die Richtlinien der Systemadministration es möglicherweise nicht zulassen, dass Endbenutzer Software herunterladen und installieren.

Hinweise:
Auf Windows-Computern setzt der Computer voraus, dass der im Dateinamen angegebene Dateityp dem tatsächlichen Format entspricht. Wenn eine Datei selbst zwar korrekt formatiert ist, aber nicht die richtige Erweiterung für den Dateityp im Dateinamen hat, erkennt Ihr Computer möglicherweise die Datei nicht und kann deshalb die entsprechende Anwendung nicht öffnen.

Es gibt inzwischen immer mehr Computerviren, die sich über Dateianhänge verbreiten. Deshalb haben sich viele Systemadministratoren dazu entschieden, eingehende Mails mit bestimmten Arten von Anhängen zu blockieren, meistens Dateien mit der Erweiterung . EXE oder .ZIP in ihrem Dateinamen. Wenn Sie eine Mail an einen Nutzer senden, dessen Mailsystem so konfiguriert wurde, dass bestimmte Arten von Anhängen blockiert werden, erhalten Sie in manchen Fällen eine Benachrichtigung dass Ihre Mail blockiert wurde.

Die im Folgenden aufgeführten Dateierweiterungen sind weit verbreitet

  • .TXT - weist auf eine Textdatei hin, die nur Text enthält. Textdateien können mit einer Vielzahl von Texteditoren geöffnet und gelesen werden, darunter der Windows-Editor, WordPad, Word, Befehlszeileneditoren, Browser und sogar Tabellenkalkulationsprogramme.

  • .HTM - weist auf eine HTML-Datei hin, eine besondere Art von Textdatei, die üblicherweise in einem Webbrowser geöffnet wird.

  • .ZIP - weist auf eine Datei hin, die als ZIP-Datei komprimiert wurde. Sie benötigen WinZip oder ein anderes Komprimierungsprogramm, das die ZIP-Datei lesen und entpacken kann.

  • .EXE - weist auf eine ausführbare Datei bzw. eine Programmdatei hin. Durch einen Doppelklick wird die Anwendung üblicherweise gestartet.

  • .DOC - weist auf ein mit Microsoft Word erstelltes Dokument hin.

  • .JPG - weist auf ein Bild im Format JPEG hin. Sie können JPEG-Dateien in einem Browser oder mit einer Reihe von Grafikprogrammen wie z. B. Paint oder Adobe Photoshop öffnen.

  • .GIF - weist auf ein Bild im Format GIF hin. Sie können GIF-Dateien in einem Browser oder mit einem Grafikprogramm öffnen.

  • .MPG - weist auf einen Filmausschnitt oder auf eine Animation im Format MPEG hin. Diese Dateien lassen sich häufig mit Browser-Plugins oder in einem Player wie Windows Media Player abspielen.

  • .WMV - weist auf einen Filmausschnitt im Format WMV hin. Dieses Format wird nur von Windows unterstützt.

  • .PDF - weist auf ein Dokument im Portable Document Format (Transportables Dokumentenformat) hin. Sie können eine solche Datei üblicherweise mit einem Plug-in direkt im Browser öffnen.
