Zimbra Desktop als Standard-Mailprogramm einrichten

Wenn Sie in Ihrem Browser oder in einer anderen Anwendung auf einen Mail-Link klicken, wird das Standard-Mailprogramm Ihres Computers aufgerufen.  

Hierfür können Sie Zimbra Desktop als primäre Anwendung festlegen. Wenn Sie in anderen Programmen auf einen Mail-Link klicken, wird das Fenster zum Verfassen einer Mail für den Standard-Account in Zimbra Desktop aufgerufen.

Die Einrichtung verläuft wie folgt:  

  1. Wählen Sie Allgemein Einstellungen.

  2. Aktivieren Sie im Bereich Andere unter Standardeinstellungen für das System das Kontrollkästchen Zimbra Desktop als Standardanwendung für Mail verwenden.
