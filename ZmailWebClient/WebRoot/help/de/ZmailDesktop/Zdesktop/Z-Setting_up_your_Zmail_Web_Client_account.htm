Ihren Zimbra Web Client-Account einrichten

Bevor Sie Zimbra Desktop nutzen können, müssen Sie für Ihren vorhandenen Account Einstellungen eingeben. Beim ersten Öffnen von Zimbra Desktop werden Sie aufgefordert, einen Account einzurichten. Sie können jederzeit weitere Accounts einrichten.

Prüfen Sie vor dem Einrichten eines Accounts, ob eine Internetverbindung besteht.

  1. Öffnen Sie Zimbra Desktop und klicken Sie in der rechten oberen Ecke auf Einrichten.  

  2. Klicken Sie auf NEUEN ACCOUNT HINZUFÜGEN.  Wählen Sie in der Liste "Account-Typ" die Option "Zimbra" aus. Die Seite zum Einrichten eines Zimbra-Accounts wird angezeigt. Geben Sie die folgenden Daten ein.

    • Accountname: Dies ist der Name, der in Zimbra Desktop als Accountname angezeigt wird. Jeder Account benötigt einen eigenen Namen. Da es sich hier um einen ZWC-Account handelt, können Sie ihn beispielsweise Zimbra nennen.

    • Mail-Adresse: Geben Sie die Mail-Adresse Ihres ZWC-Accounts ein.

    • Passwort: Geben Sie das Passwort für die Anmeldung an Ihrem ZWC-Account ein.

    • Posteingangsserver: Geben Sie die Adresse des Zimbra-Servers ein. Dies ist typischerweise ein Name in der Form mail.beispiel.com. Dies ist der Server, auf dem sich Ihr Account befindet.

    • Sicherheit: Wenn Sie immer eine sichere Verbindung nutzen sollen, aktivieren Sie SSL-Verschlüsselung für den Zugriff auf diesen Server verwenden. Wenn Sie nicht sicher sind, ob Sie diese Option aktivieren sollen, wenden Sie sich an Ihren Systemadministrator.

    • Geben Sie in den Synchronisierungseinstellungen an, wie häufig Zimbra Desktop die Daten von Ihrem Account abrufen soll. Standardmäßig werden Mails geprüft und eine Synchronisierung durchgeführt, wenn neue Mails eingehen.

Wenn Sie die Option Manuell auswählen, müssen Sie in der Symbolleiste von Zimbra Desktop auf Senden/Empfangen klicken, um Ihren Account mit dem Zimbra-Server zu synchronisieren.

Die Auswahl eines zu kurzen Zeitraums – z. B. eine Minute – beeinträchtigt unter Umständen die Leistungsfähigkeit Ihres Computers.

Sie können diese Einstellung jederzeit ändern, indem Sie auf "Einrichten" klicken und den Account bearbeiten.

  1. Klicken Sie abschließend auf Prüfen und Speichern.

Zimbra Desktop prüft Ihre Account-Daten. Wurde Ihr Account erfolgreich geprüft, wird das Dialogfeld Dienst erstellt angezeigt. Wurde Ihr Account nicht erfolgreich geprüft, wird erneut das Dialogfeld "Account einrichten" angezeigt, sodass Sie Ihre Eingaben korrigieren können.

  1. Um den Desktop zu öffnen, klicken Sie auf Desktop starten. Während der Synchronisierung Ihres Accounts können Sie in Zimbra Desktop arbeiten.
