Account in Zimbra Desktop löschen

  1. Klicken Sie in Zimbra Desktop auf Setup. Sie finden diesen Link in der rechten Ecke.

Daraufhin wird die Seite "Meine Accounts" aufgerufen, auf der Ihre Accounts angezeigt werden.

  1. Klicken Sie für den zu löschenden Account auf Löschen. Ein Dialogfeld zeigt folgende Meldung an: Die Einstellungen des Accounts und heruntergeladene Daten werden gelöscht. Dies hat keine Auswirkung auf die Daten auf dem Server. Möchten Sie fortfahren?

Alle Daten werden vom Computer gelöscht: Mails, Kontaktlisten, Kalender, Notizen im Notizbuch, Dateien im Ordner "Dateien".  Durch das Löschen werden die Daten vom Computer entfernt, nicht jedoch auf dem Server.

Hinweis:  Dateien, die Sie im lokalen Ordner gespeichert haben, sind nur in Zimbra Desktop gespeichert. Wenn Sie diese Dateien speichern möchten, bevor Sie Ihren Account löschen, exportieren Sie die Dateien in einen lokalen Ordner. Hierzu öffnen Sie die Registerkarte Einstellungen > Importieren/Exportieren.
