Desktop-Einstellungen einrichten

Auf der Registerkarte „Einstellungen“ gibt es Funktionsseiten, auf denen Sie die Funktionsweise von Mailbox, Adressbuch und Kalender für jeden von Ihnen eingerichteten Account in Zimbra Desktop festlegen können. Das Standardverhalten wird konfiguriert, wenn Sie einen Account unter Zimbra Desktop.

Änderungen, die Sie im Ordner Allgemeine Einstellungen vornehmen, betreffen das grundlegende Layout in Zimbra Desktop für alle Accounts.

Global

Änderungen auf dieser Registerkarte betreffen alle Mail-Accounts inZimbra Desktop.

  • Unter Stile wird eine Liste der verschiedenen Skins für die Oberfläche angezeigt. Sie können die Hintergrundfarbe Ihrer Mailbox jederzeit ändern.

  • Sprache. Dies ist die Sprache, in der Text in Zimbra Desktop. (Verfügbar für Zimbra-Web-Client.)

  • Wenn die auf dieser Registerkarte angezeigte Zeitzone nicht korrekt ist, können Sie die Standardzeitzone hier ändern. Auf diese Weise legen Sie Datum und Uhrzeit für die von Ihnen gesendeten und empfangenen Mails fest, und Termine im Kalender werden korrekt angezeigt. Diese Zeitzone kann sich von der Zeitzone, die für Ihren Computer eingerichtet ist, unterscheiden.

  • Unter Sucheinstellungen können Sie angeben, dass bei jeder Suche automatisch auch im Papierkorb und im Spam-Ordner gesucht wird. Standardmäßig werden diese Ordner nicht durchsucht.

  • Unter Sprache der Suche können Sie die Option Sprache für erweiterte Suche in Symbolleiste der Suche anzeigen aktivieren, wenn die Details der Suchabfrage im Suchtextfeld angezeigt werden sollen.

  • Aktivieren Sie die Option Kontrollkästchen für die schnelle Auswahl von Objekten in Listen anzeigen, um für jedes Objekt im Inhaltsfenster ein Kontrollkästchen anzuzeigen. Über die Kontrollkästchen können Sie eines oder mehrere Objekte für einen Bearbeitungsschritt auswählen, z. B. Löschen, Verschieben, als gelesen/ungelesen markieren.

  • Standardeinstellungen für das System. Aktivieren Sie die Option, wenn Sie Zimbra Desktop als Standard-Mailanwendung nutzen möchten. Wenn Sie eine Mail senden oder auf einer Webseite auf einen Mail-Link klicken, wird das Fenster „Verfassen“ in Zimbra Desktop.

Mail

  • Anzeige. Wählen Sie, wie viele Objekte (Mails oder Unterhaltungen) pro Seite angezeigt werden sollen. Standardmäßig werden 25 Objekte je Seite angezeigt, aber Sie können auch auswählen, dass 10, 50 oder 100 Objekte je Seite angezeigt werden.

  • Mails anzeigen. Geben Sie an, ob Sie Mails, die im HTML-Format gesendet wurden, als HTML anzeigen möchten. Wenn Sie diese Option ausschalten, werden alle Mails als reiner Text (Plain-Text) angezeigt. Dabei sehen Sie die HTML-Tags an sich, die Formatierung wird nicht umgesetzt.

  • Mail-Vorschau

    • Mailauszüge in Mailliste anzeigen. Wenn diese Option aktiviert ist, werden Mailauszüge in der Mailliste angezeigt, wenn Sie den Mauszeiger auf dem Betreff einer Mail positionieren.

    • Wenn Durch Doppelklick die Mail in einem neuen Fenster öffnen aktiviert ist, wird die Mail bei Doppelklick in einem neuen Fenster geöffnet.

  • Bilder. Aktivieren Sie Bilder, um Bilder in HTML-Mails automatisch herunterzuladen. Wenn diese Option aktiviert ist, müssen Sie nicht extra klicken, um das Bild in der Mail anzuzeigen.

  • Unter Beim Lesen im Lesefenster legen Sie das Verhalten zum Markieren von Mails als gelesen oder ungelesen fest, wenn Sie die Mails im Lesefenster anzeigen. Sie können bestimmen, ob Mails sofort oder nach einer festgelegten Anzahl von Sekunden der Anzeige im Lesefenster als gelesen markiert werden oder trotz Anzeige im Lesefenster als ungelesen markiert bleiben.

  • Standardmailsuche. Diese Einstellung definiert die Suche, die bei Ihrer Anmeldung durchgeführt werden soll. Standardmäßig wird nach Ihrem Posteingang gesucht. Wenn Sie Zimbra Desktop öffnen, wird der Inhalt Ihres Posteingangs angezeigt. Sie können dieses Feld ändern und einen anderen Ordner, einen Tag oder eine gespeicherte Suche angeben, damit diese Suche durchgeführt wird, wenn Sie zuerst Zimbra Desktop öffnen. Informationen zur Syntax finden Sie unter Beschreibung der Sprache für Suchabfragen.

  • Mails empfangen

Wenn eine Mail eingeht:

    • Sound abspielen. Sie werden durch einen Signalton benachrichtigt, wenn in Ihrem Posteingang eine neue Mail eingeht. Wenn Sie die Lautstärke stumm geschaltet haben, hören Sie keinen Signalton.

    • Registerkarte „Mail“ hervorheben. Die Registerkarte „Mail“ wird hervorgehoben, wenn im Posteingang eine neue Mail eingeht und Sie nicht auf der Registerkarte „Mail“ arbeiten.

    • Browser-Titel blinkend darstellen. Wenn im Posteingang eine neue Mail eingeht, blinkt der Browser, bis Sie die Registerkarte „Mail“ öffnen.

    • Popup-Benachrichtigung anzeigen. Wenn eine Mail eingeht, werden Informationen zu der Mail in einem kleinen Dialogfeld angezeigt.

Die folgenden Optionen stehen nur beim Zimbra Web Client-Account zur Verfügung.

  • Geben Sie eine Adresse an, an die Ihre Mails weitergeleitet werden. Die ursprüngliche Mail kann dabei aus Ihrer Zimbra-Mailbox gelöscht werden.

  • Geben Sie an, ob auf eingehende Mails eine automatische Antwort gesendet werden soll. Richten Sie hierfür ein Anfangs- und Enddatum ein. Diese Funktion wird häufig als Abwesenheitsmeldung genutzt. Sie können den Text selbst eingeben. Sie können auch ein Anfangs- und Enddatum für das Senden dieser Abwesenheitsmails festlegen.

  • Mails von mir. Unter „Mails von mir ...“ legen Sie fest, wie mit Mails verfahren werden soll, die von Ihnen gesendet werden, die Ihre Adresse als einen der Empfänger der Mail enthalten oder die an eine Adresse wie einen Verteiler gehen, in dem auch Sie enthalten sind.

Verfassen

  • Geben Sie an, welchen Texteditor Sie standardmäßig nutzen möchten: HTML oder Plain-Text.

  • Richten Sie die Standardschriftart ein, die im HTML-Format zum Verfassen von Mails, Notizen und Kalendereinträgen verwendet wird. Sie können die Schriftart, Größe und Farbe für Ihre Standardschrift definieren. Standardmäßig wird Arial, 12 Punkt, schwarz verwendet.

  • Die Option Mails immer in neuem Fenster verfassen bewirkt, dass beim Verfassen neuer Mails ein neues Fenster unabhängig von Ihrer Mailbox geöffnet wird, in dem Sie die Mail schreiben können. Sie können, während das Fenster zum Verfassen geöffnet ist, Ihre Mailbox anzeigen und in ihr navigieren.

  • Mails, die Sie verfassen, werden automatisch in Abständen von wenigen Minuten als Entwürfe gespeichert. Sie können diese Funktion ausschalten, wenn Sie nicht möchten, dass automatisch gespeichert wird.

  • Wählen Sie aus, ob der ursprüngliche Text im Textkörper Ihrer Antwortmail enthalten sein soll. Sie können angeben, dass der ursprüngliche Text enthalten sein soll, dass er nicht enthalten sein soll, Sie können den Text als Anhang oder im Textkörper der Mail mit einem von Ihnen ausgewählten Präfix beifügen, oder Sie können nur die letzte Mail einschließen. (Dann wird nur zitiert, was die Person, von der Sie die Mail erhalten haben, geschrieben hat. Frühere Mails sind nicht enthalten.)

  • Geben Sie an, ob dem zitierten Text der weitergeleiteten oder beantworteten Mail in jeder Zeile eines der Zeichen > oder | als Präfix vorangestellt werden soll.

  • Standardmäßig ist die Option Kopie im Ordner "Gesendet" speichern aktiviert. Kopien der von Ihnen gesendeten Mails werden in Ihrem Ordner "Gesendet" gespeichert.

  • Wählen Sie aus, wie der ursprüngliche Text einer Mail beigefügt werden soll, wenn Sie eine Mail weiterleiten: im Textkörper der Mail, im Textkörper der Mail mit Präfix oder als Anhang zu der Mail.

Signaturen

Sie können für Ihre Mails Signaturen anlegen. Ihre Signatur kann Ihren Namen und weiteren Text mit der maximal für Ihren Account zulässigen Anzahl von Zeichen enthalten. Wenn Sie mehrere Identitäten anlegen, können Sie verschiedene Signaturen anlegen und diese den jeweiligen Adressen zuordnen.

Kontakt

  • Geben Sie an, ob Kontakte in Form einer Liste oder in Form von Karten angezeigt werden.

  • Wählen Sie aus, wie viele Kontakte pro Seite angezeigt werden sollen. Die Standardeinstellung ist 25.

  • Sie können die Funktion aktivieren, dass beim Senden von Mails die Adressen automatisch zum Ordner Mail-Kontakte hinzugefügt werden sollen.

    Standardmäßig wird das Adressbuch Mailempfänger erweitert, wenn Sie eine Mail an eine neue Adresse senden, die sich noch nicht in einem Ihrer Adressbücher befindet. Sie können diese Funktion über die Registerkarten "Einstellungen, Adressbuch" abstellen. Deaktivieren Sie die Option „Einstellungen“, Neue Kontakte zu "Mailempfänger" hinzufügen.

Persönlichkeiten

Auf der Registerkarte "Accounts" erstellen und verwalten Sie Ihre Persönlichkeiten. Eine Persönlichkeit ist die Mail-Identität, die in von Ihnen gesendeten Mails im Feld „Von“ verwendet wird.

Der Standardstil wird in den Einstellungen für den primären Account definiert. Wenn Sie neue Identitäten anlegen, können Sie dieselben Einstellungen wie bei der Standardidentität nutzen oder Sie können andere Einstellungen einrichten.

Mailfilter

Auf der Registerkarte "Mailfilter" definieren Sie die Regeln für Mailfilter. Weitere Informationen dazu, wie Sie Regeln für Mailfilter einrichten, finden Sie unter Funktionsweise von Filtern.

Kalender

  • Wählen Sie aus, wie Ihr Zimbra-Kalender angezeigt werden soll. Standardmäßig wird die Arbeitswoche angezeigt.

  • Geben Sie an, welcher Wochentag in Ihrem Kalender der erste Tag der Woche sein soll.

  • Dann wird ein Mini-Kalender in Ihrem Kalender angezeigt. Den Mini-Kalender können Sie in jeder Ansicht aktivieren.

  • Sie können festlegen, dass eine Einladung automatisch gelöscht wird, nachdem Sie sie beantwortet haben. Wenn Sie diese Option nicht aktivieren, bleibt die Einladung in Ihrem Posteingang.

  • Richten Sie Erinnerungen für Besprechungen ein. Sie können festlegen, wann Sie an eine Besprechung erinnert werden möchten. Optionen hierfür reichen von Nie bis zu 60 Minuten vor der Besprechung. Sie können die Art der Erinnerung auswählen: Sound abspielen, Browser-Titel blinkend darstellen oder eine Popup-Benachrichtigung anzeigen.

  • Wenn Sie häufig Termine ohne Teilnehmer anlegen, aktivieren Sie die Option Schnelleingabe von Terminen verwenden.

  • Wenn Sie sich häufig in verschiedenen Zeitzonen aufhalten, können Sie die Option „Liste der Zeitzonen in Terminansicht“ aktivieren, damit Sie bei der Planung von Terminen ggf. die Zeitzone wechseln können.

Importieren/Exportieren

  • Sie können wahlweise alle oder bestimmte Ordner in Ihrem Account exportieren. Informationen hierzu finden Sie unter Inhalt Ihres Accounts sichern.

  • Sie können exportierte Ordner wieder in Ihren Account importieren.

  • Sie können bestimmte Dateitypen für den Export suchen.

  • Kalenderprogramme importieren und exportieren.

  • Kontaktlisten importieren und exportieren

Tastenkürzel

Tastenkürzel sind Tasten oder Tastenkombinationen, mit denen Sie rasch einen Befehl ausführen können.  

Wechseln Sie zu Ihrem Ordner Tastenkürzel Einstellungen, um eine vollständige Liste der Tastenkürzel zu sehen. Alternativ können Sie in einer beliebigen Ansicht auf Strg Q klicken und die Tastenkürzel-Listen werden eingeblendet.
