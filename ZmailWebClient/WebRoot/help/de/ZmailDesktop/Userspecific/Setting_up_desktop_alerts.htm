Desktop-Benachrichtigungen und Erinnerungen einrichten

Sie können zur Benachrichtigung bei neuen Mails und Erinnerungen an Termine Desktop-Benachrichtigungen erstellen. Um Benachrichtigungen zu erhalten, muss Ihre Mailbox geöffnet sein.

  • Mail-Benachrichtigungen. Sie richten Ihre Einstellungen auf der Seite "Einstellungen>Mail" ein. Nachfolgend sehen Sie die Benachrichtigungsoptionen, wenn eine Mail eintrifft:

    • Sound abspielen. Sie werden durch einen Signalton benachrichtigt, wenn in Ihrem Posteingang eine neue Mail eingeht.  Wenn Sie die Lautstärke stumm geschaltet haben, hören Sie keinen Signalton. Sie müssen QuickTime oder Windows Media installiert haben.

    • Registerkarte „Mail“ hervorheben. Die Registerkarte  "Mail" wird markiert, wenn im Posteingang eine neue Mail eingeht und Sie nicht auf der Registerkarte "Mail" arbeiten.

    • Popup-Benachrichtigung anzeigen:  Wenn eine neue Mail eintrifft, wird eine Meldung eingeblendet. Sie müssen Yahoo! BrowserPlus installiert haben.

  • Benachrichtigungsoptionen in Kalendern anzeigen. Sie legen Ihre Einstellungen auf der Seite Einstellungen>Kalender fest.  Bevor Sie eine Benachrichtigung einstellen können, müssen Sie die Uhrzeit festlegen, zu der Sie an Ihren Termin erinnert werden möchten.  Eine Erinnerungsmeldung wird zu der festgelegten Uhrzeit vor dem Termin angezeigt. Wenn Sie für Erinnerungen die Option Nie auswählen, werden Sie auch dann nicht an den Termin erinnert, wenn Sie die Erinnerungsoptionen aktiviert haben.

Die Erinnerung wird nicht oben auf der Seite eingeblendet, wenn Sie sich nicht in der Kalenderansicht befinden. Wenn Sie eine der folgenden Optionen aktivieren, nehmen Sie die Erinnerung besser wahr:

    • Sound abspielen. Sie werden durch einen Signalton benachrichtigt, wenn eine Erinnerung an einen Termin eingeblendet wird. Wenn Sie die Lautstärke stumm geschaltet haben, hören Sie keinen Signalton.

    • Browser-Titel blinkend darstellen. Wenn eine Erinnerung an einen Termin eingeblendet wird, blinkt der Browser, bis Sie das Popup-Fenster schließen.

    Zusätzlich zu Benachrichtigungen auf Ihrem Computer können Sie eine Mail- oder SMS-Benachrichtigung aus dem Termin heraus einrichten.  Sie konfigurieren die Mail-Adresse und die SMS-Nummer auf der Seite "Einstellungen>Benachrichtigung".
