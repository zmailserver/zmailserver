Freigegebene Objekte verwalten

Sie können die folgenden Objekte für andere Nutzer in Ihrer Organisation, für externe Nutzer oder öffentlich freigeben: Mail, Adressbuch, Kalender, Aufgaben und Ordner "Dateien".  Sie können die Freigabe internen Nutzern erteilen, die vollständigen Manager-Zugriff auf die Ordner erhalten können, Sie können die Freigabe externen Nutzern erteilen, die sich den Ordnerinhalt nach Eingabe eines Passworts ansehen können, und Sie können öffentlichen Zugriff auf die Ordner erteilen, so dass jeder, der die URL kennt, sich den Inhalt der Ordner ansehen kann.

Sie können Ihre freigegebenen Objekte auf der Seite Einstellungen > Freigabe anzeigen und verwalten. Auf dieser Seite haben Sie folgende Möglichkeiten:

    • Ordner anzeigen, die für Sie freigegeben wurden, auch wenn Sie die Freigabe nicht angenommen haben

    • Ordner anzeigen, die Sie für andere Nutzer freigegeben haben. Aktionen wie Bearbeiten, Freigabe entziehen oder neu senden sind möglich.

    • Ordner für andere Nutzer freigeben

Von der Seite "Freigabe" aus verwalten

Für mich freigegebene Ordner, die jedoch noch nicht angenommen wurden

In diesem Abschnitt sind die Ordner, die für Sie freigegeben, jedoch noch nicht angenommen wurden, aufgeführt.  So wechseln Sie zwischen zwei Ansichten:

  • Ordner anzeigen, die für mich über eine Verteilerliste freigegeben wurden. Diese Option ist standardmäßig ausgewählt.

  • Ordner anzeigen, die folgender Nutzer für mich freigegeben hat. Wenn Sie diese Option auswählen, geben Sie den Namen eines Nutzers ein, der ein Objekt für Sie freigegeben hat.  Sie können die Daten jeweils für einen Nutzer anzeigen. Die Details werden nicht gespeichert.

Für mich freigegebene Ordner, die ich angenommen habe

In diesem Abschnitt werden die Ordner aufgeführt, für die Sie die Freigabe angenommen haben. Eine Beschreibung des freigegebenen Ordners, einschließlich Ihrer Rolle, dem Ordnertyp und für wen der Ordner freigegeben sowie dem Namen der Verteilerliste.

Von mir freigegebene Ordner

In diesem Abschnitt werden die von Ihnen freigegebenen Ordner mit den folgenden Details angezeigt: Adresse der Nutzer, für die Sie Ordner freigegeben haben, Speicherort des freigegebenen Objekts, Typ des freigegebenen Objekts (Mail, Adressbuch, Kalender, Aufgabenliste oder Ordner „Dateien“) und die Rolle, die Sie dem Nutzer zugewiesen haben.

Sie können die Freigabeeigenschaften bearbeiten, Berechtigungen für die Freigabe entziehen oder die Einladung erneut senden.

Ordner freigeben

Sie können die folgenden Objekte direkt unter „Einstellungen“ freigeben: Mail, Adressbuch, Kalender, Aufgaben und Ordner "Dateien".

  1. Wählen Sie den Ordnertyp aus.

  2. Klicken Sie auf Freigabe.

  3. Wählen Sie das freizugebende Objekt aus und klicken Sie auf OK. Daraufhin wird das Dialogfeld "Eigenschaften der Freigabe" angezeigt.

  4. Wählen Sie im Dialogfeld "Eigenschaften der Freigabe" aus, für wen Sie den Ordner freigeben möchten.

Interne Nutzer oder Gruppen

  1. Wenn Sie "Interne Nutzer oder Gruppen" wählen, können Sie aus der beim Eingeben angezeigten Liste Adressen auswählen oder Sie können Adressen eingeben, die nicht in der Liste stehen. Bei dem Nutzer bzw. der Gruppe muss es sich um eine gültige Adresse auf dem Mailserver Ihres Unternehmens handeln.

  2. Im Bereich Rolle wählen Sie die Zugriffsrechte aus.

  3. Wenn Sie Informationen zu Ihren privaten Terminen freigeben möchten, aktivieren Sie die Option Die Anzeige privater Termine durch andere Nutzer zulassen.

  1. Wählen Sie im Bereich Mail aus, welche Art von Mail gesendet werden soll.  Es stehen folgende Optionen zur Auswahl:

  • Standardmail senden. Die Standardmail mit der Freigabebenachrichtigung wird an die Mail-Adresse gesendet.  

  • Notiz zur Standardmail hinzufügen. Sie können die Standardmail um zusätzliche Informationen erweitern.

  1. Klicken Sie auf OK.

 

Externe Nutzer

  1. Wenn Sie externe Nutzer auswählen, geben Sie die Mail-Adressen der Nutzer ein.   Sie können mehrere Adressen in das Feld „Mail“ eingeben. Trennen Sie die Adressen durch Semikolons (;).

  2. Geben Sie das Passwort ein, das die Nutzer eingeben müssen, um Ihren Kalender anzuzeigen.

  3. Wenn Sie Informationen zu Ihren privaten Terminen freigeben möchten, aktivieren Sie die Option Die Anzeige privater Termine durch andere Nutzer zulassen.

  1. Wählen Sie im Bereich Mail aus, welche Art von Mail gesendet werden soll.  Die URL für Ihren Kalender und das Passwort werden in die Mail eingefügt.

    • Standardmail senden. Die Standardmail mit der Freigabebenachrichtigung wird an die Mail-Adresse gesendet.

    • Notiz zur Standardmail hinzufügen. Sie können die Standardmail um zusätzliche Informationen erweitern.

  2. Klicken Sie auf OK.

Öffentlich

Wenn Sie "Öffentlich" auswählen, kann jeder, der die URL kennt, Ihren Kalender anzeigen.  Es ist kein Passwort erforderlich.
