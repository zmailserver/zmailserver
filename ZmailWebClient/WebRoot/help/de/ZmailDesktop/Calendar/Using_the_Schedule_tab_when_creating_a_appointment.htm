Verfügbarkeit von Teilnehmern und Orten im Vorschlagbereich anzeigen

Beim Planen von Besprechungen können Sie überprüfen, ob die Teilnehmer belegt oder frei sind. So können Sie einen günstigen Termin finden, an dem alle Teilnehmer zu der Besprechung kommen können. Wenn Sie Teilnehmer zum Feld "Teilnehmer" hinzufügen, werden in einem Vorschlagbereich im Überblicksfenster die Verfügbarkeit der Teilnehmer und die Verfügbarkeit von mindestens einem Raum (Ort) in Zeitfenstern mit halbstündigen Abständen angezeigt. Die Zahl rechts von der Zeit zeigt die Anzahl an Orten, die für diesen Zeitpunkt zur Verfügung stehen. Sie können auf diese Zahl klicken, um eine Liste der vorgeschlagenen Orte aufzurufen und einen Ort auszuwählen. Der ausgewählte Ort wird zum Feld "Ort" auf der Seite "Termin" hinzugefügt.

Hinweis: Räume (Orte) werden angezeigt, wenn sie im Firmenverzeichnis konfiguriert wurden.

Im Mini-Kalender wird die Verfügbarkeit der Teilnehmer farblich gekennzeichnet angezeigt.

  • Grün bedeutet, dass alle Teilnehmer und mindestens ein Raum für den betreffenden Tag verfügbar sind.

  • Orange bedeutet, dass nicht alle Teilnehmer verfügbar sind, dass aber mehr als die Hälfte der Teilnehmer teilnehmen können und dass mindestens ein Raum verfügbar ist. Sie können den Mauszeiger über das Datum bewegen, um die Namen von maximal 10 Personen anzuzeigen, die nicht verfügbar sind.

  • Rot bedeutet, dass mehr als die Hälfte der Teilnehmer an den betreffenden Tagen nicht an der Besprechung teilnehmen kann.

In dem nachstehenden Beispiel sind alle Teilnehmer den ganzen Monat verfügbar, außer am 18. Januar. Am 18. Januar sind weniger als die Hälfte der Teilnehmer verfügbar.

Orte für den Organisator anzeigen

Um verfügbare Orte für Sie als Organisator zu finden, öffnen Sie eine Terminseite. Bevor Sie einen eingeladenen Teilnehmer im Bereich "Vorgeschlagene Zeiten" zur Übersicht hinzufügen, klicken Sie auf Vorschläge anzeigen für (Datum).

Vorschläge verwalten

Sie können konfigurieren, wie die Informationen im Vorschlagbereich angezeigt werden sollen. Standardmäßig werden Teilnehmer und Ortsverfügbarkeit angezeigt, sobald Sie ersten Teilnehmernamen in das Feld "Teilnehmer" eingeben. Sie können die Standardeinstellungen ändern und andere Einstellungen im Dialogfeld Einstellungen für Vorschläge konfigurieren.

Klicken Sie im Übersichtsfenster auf die Überschrift Vorgeschlagene Zeiten, um das Dialogfeld "Einstellungen für Vorschläge" aufzurufen.

Konfigurieren Sie die folgenden Optionen:

  • Vorschläge. Die Optionen sind Automatisch (Standard)  oder Manuell.

    • Um Teilnehmerzeitpunkt und -orte manuell anzuzeigen, klicken Sie auf Manuell. Klicken Sie zum Aufrufen der  Teilnehmer-/Ortsdaten auf Vorschläge anzeigen für (Datum) im Bereich "Vorgeschlagene Zeiten".

  • Ortseinstellungen.  In diesem Bereich können Sie festlegen, welche Art von Orten Sie in Ihre Suche aufnehmen möchten.  Lassen Sie diesen Bereich leer, wenn Sie keine Ortspräferenzen haben.

    • In das Feld "Name" können Sie einen Teil eines Ortsnamens eingeben.  Orte mit dieser Zeichenfolge im Namen werden dann in die Suche eingeschlossen.  Beispiel: Räume im Gebäude 3 haben  "Gbd3" im Ortsnamen.  Wenn Sie "Gbd3" in das Feld Name eingeben, werden nur Orte mit "Gbd3" im Namen vorgeschlagen.

    • Wenn Sie nicht möchten, dass Orte mit Teilnehmerverfügbarkeit vorgeschlagen werden, deaktivieren Sie das Kontrollkästchen Verfügbare Räume vorschlagen .

  • Zeiteinstellungen. Sie können angeben, welche Zeiten bei der Suche nach Zeitpunkten, an denen die Teilnehmer verfügbar sind, einbezogen werden sollen.

    • Nur meine Arbeitszeiten. Mit dieser Option wird nur nach Zeiten gesucht, die innerhalb Ihrer konfigurierten Arbeitszeiten liegen. Das sind die wöchentlichen Arbeitszeiten, die Sie in Ihren Kalendereinstellungen festgelegt haben.

    • Arbeitszeiten aller Teilnehmer. Bei der Suche nach einem Ort werden die Arbeitszeiten aller Teilnehmer berücksichtigt. Hinweis:  Wenn Sie Teilnehmer aus verschiedenen Zeitzonen einschließen, insbesondere solche, die um viele Zeitzonen differieren, werden Sie niemals einen Ort finden, an dem sich alle Teilnehmer treffen können.  In diesem Fall werden im Vorschlagbereich keine Raumressourcen angezeigt.

    • Arbeitsfreie Zeiten. Bei dieser Suche werden Zeiten geplant, die nicht als Arbeitszeiten gekennzeichnet sind, zum Beispiel Abende und Wochenenden.
