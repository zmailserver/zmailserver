iCal-Kalender importieren

Sie können iCalendars-Kalender (iCal), die im Format .ics gespeichert wurden, importieren. iCalendar ist das Standardformat für den Austausch von Kalenderdaten über das Internet, das Dateiformat lautet .ics. Die Importfunktion steht auf der Registerkarte Einstellungen > Importieren/Exportieren zur Verfügung.

Sie können Ihren iCal-Kalender in einen vorhandenen Kalender importieren oder einen neuen Kalender anlegen und die .ics-Datei in diese Datei importieren.

So importieren Sie einen iCal-Kalender

  1. Wählen Sie Einstellungen, und klicken Sie auf den Ordner Importieren / Exportieren.

  2. Suchen Sie im Feld „Datei“ die zu importierende .ics-Datei.

  3. Klicken Sie im Feld Ziel auf Kalender. Wählen Sie dann den Ordner, in den der Kalender importiert werden soll. Wählen Sie entweder einen Ihrer vorhandenen Kalender oder klicken Sie auf Neu, um einen neuen Kalender anzulegen.

  4. Klicken Sie auf Importieren.  

  5. Klicken Sie auf OK.

iCal-Kalender werden regelmäßig entsprechend den vom Systemadministrator festgelegten Abfrageintervallen für dezentrale Kalender aktualisiert.
