Termin planen

In der Ansicht "Termin" können Sie einzelne oder wiederkehrende Termine, Besprechungen oder Ereignisse anlegen und eine Mail schreiben. Wenn Sie auf Speichern klicken, werden die Teilnehmer per Mail eingeladen und die geforderten Ressourcen reserviert.

Auf der Terminseite geben Sie die Informationen zum Termin wie folgt ein:

  • Betreff der Besprechung:  Der hier eingegebene Text wird als Beschreibung im Kalender angezeigt.

  • Ort  der Besprechung: In diesem Feld können Sie den Ort der Besprechung, wie beispielsweise eine Adresse, eine Gebäudenummer, eine Etagennummer, festlegen.

  • In der Dropdown-Liste Kalender werden alle Kalender in Ihrer Kalenderliste angezeigt.  Der zuerst angezeigte Kalender ist Ihr ursprünglicher Kalender. Wenn Sie mehrere Kalender besitzen, können Sie einen Termin anlegen und den Kalender auswählen. Der Termin wird zu dem jeweils ausgewählten Kalender hinzugefügt.

  • Mit Markieren als können Sie einen Termin als privat oder öffentlich kennzeichnen.

  • Unter Uhrzeit geben Sie an, wann der Termin stattfindet: Tag, Stunden, ganztägige Ereignisse und wiederkehrende Termine.  Sie können eine Zeit für die Benachrichtigung durch eine Besprechungserinnerung festlegen. Alle Teilnehmer werden vor der eigentlichen Besprechungszeit über die Besprechung informiert.

  • Teilnehmer:   Sie fügen Teilnehmer hinzu, indem Sie entweder einen Namen in das Feld "Teilnehmer" eingeben, auf der Registerkarte "Teilnehmer suchen" nach Namen suchen oder sich auf der Registerkarte "Planung" die freien/belegten Zeiten der Teilnehmer ansehen.

  • Wenn das Kontrollkästchen Antworten auf Anfragen markiert ist, wenn der Eingeladene akzeptiert, der Eingeladene als vorläufig gekennzeichnet wird, oder wenn er die Einladung ablehnt, wird Ihnen eine Statusmeldung zugesendet.

  • Wenn das Kontrollkästchen Benachrichtigungsmail senden nicht aktiviert ist, erhalten die Teilnehmer keine Benachrichtigung über die Besprechung oder über die Änderungen, die Sie während der Besprechung vorgenommen haben.  Heben Sie die Auswahl dieses Kontrollkästchens auf, wenn Sie geringfügigere Änderungen an der Besprechung vorgenommen haben, wovon die Teilnehmer nichts zu wissen brauchen.
