Wissenswertes zum Planen von Terminen

Sie können einzelne oder wiederkehrende Termine, Besprechungen und Ereignisse anlegen, vor der Einladung der Teilnehmer ermitteln, ob diese frei sind, einen Raum und Geräte reservieren und eine Mail schreiben, die zusammen mit der Einladung gesendet wird. Sie können die Einladung speichern und später senden. Wenn Sie auf Senden klicken, werden die Teilnehmer per Mail eingeladen und die angeforderten Ressourcen reserviert.

Sie können folgende Aktivitäten planen:

  • Termine: Termine sind Aktivitäten, bei denen keine anderen Personen oder Ressourcen beteiligt sind. Wenn Sie einen Termin planen, wird keine Mail zur Bestätigung des Termins gesendet. Sie können wiederkehrende Termine einrichten. Über die Schnelleingabe können Sie rasch einen Termin anlegen.

  • Ereignisse: Ein Ereignis ist eine Aktivität, die einen ganzen Tag in Anspruch nimmt. Ereignisse werden im Kalender nicht mithilfe von Uhrzeiten angezeigt, sie erscheinen als Banner im oberen Bereich des Kalenderplans.

  • Besprechungen: Besprechungen sind Termine, bei denen andere Personen beteiligt sind. Wenn Sie eine Besprechung anlegen, wählen Sie Teilnehmer und Ressourcen aus und senden diesen per Mail eine Einladung. Sie können Besprechungen verlegen und wiederkehrende Besprechungen einrichten.

Alle drei Arten von Aktivitäten können auf verschiedene Weise eingegeben werden. Sie können in jeder Ansicht, mit Ausnahme der Ansicht "Einstellungen", einen Termin einrichten. Um einen Termin anzulegen, gehen Sie auf eine der folgenden Arten vor:

  • Klicken Sie in der Symbolleiste auf den Pfeil von und wählen Sie Termin. Diese Vorgehensweise ist in jeder Ansicht, mit Ausnahme der Ansicht "Einstellungen", möglich.

  • Geben Sie Termine direkt im Kalender ein. Doppelklicken Sie auf ein Datum und eine Uhrzeit im Kalender bzw. ziehen Sie diese, um den Dialog Schnelleingabe Termin zu öffnen.

  • Klicken Sie mit der rechten Maustaste entweder auf den Mini-Kalender oder die Anfangsuhrzeit in einer der Kalenderansichten. Sie können auswählen, welche Art von Termin Sie anlegen möchten: Neuer Termin oder Neuer ganztägiger Termin.

  • Erstellen Sie die Kopie einer Besprechung und ändern Sie die Informationen. Klicken Sie mit der rechten Maustaste auf die Besprechung, die kopiert werden soll, und wählen Sie Eine Kopie erstellen.

  • In einer Mail wird bestimmter Text als Datum interpretiert. Dann ist es möglich, mit der rechten Maustaste auf den Text zu klicken und einen Termin anzulegen. Ausdrücke wie "heute", "morgen" oder ein Wochentag bzw. ein genaues Datum werden in Mails hervorgehoben. Bewegen Sie die Maus auf einen solchen Text, um zu ermitteln, ob Sie einen Termin geplant haben. Klicken Sie auf den hervorgehobenen Text, um Ihren Kalender zu öffnen.

  • Ziehen Sie eine Nachricht auf das Datum des Mini-Kalenders. Die Terminseite wird geöffnet und zeigt die hinzugefügten Informationen der Mail-Nachricht an. Der Betreff der Nachricht wird der Betreff der Besprechung. Die Adressen "An:" "Cc:" und "Von:" werden dem Feld "Teilnehmer" hinzugefügt und der Text im Textkörper der Nachricht wird dem Textfeld "Termin" hinzugefügt.
