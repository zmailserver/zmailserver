Platzhalter * bei der Suche nutzen

Das Sternchen * kann als Wildcard in Suchvorgängen verwendet werden, um Begriffe mit ähnlicher Schreibweise zu finden.  

Nutzen Sie das Sternchen * als Wildcard im Anschluss an eine bestimmte Zeichenfolge. Beispielsweise gibt Ha* Objekte mit dem Wort Hand, Haus usw. zurück.
