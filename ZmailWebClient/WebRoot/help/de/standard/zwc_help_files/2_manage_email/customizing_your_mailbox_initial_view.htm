Startansicht der Mailbox anpassen

Auf der Seite "Einstellungen" > "Mail" wird mit Standardmailsuche die erste Ansicht nach der Anmeldung bei der Mailbox angegeben. Standardmäßig wird in Ihrem Posteingang gesucht. Wenn Sie sich anmelden, wird deshalb der Inhalt Ihres Posteingangs angezeigt.

Sie können dieses Feld ändern, damit ein anderer Ordner, ein Tag oder eine gespeicherte Suche beim Öffnen des Webclients angezeigt wird.

  1. Öffnen Sie die Seite Einstellungen > Mail.

  2. Geben Sie im Textfeld Mails anzeigen > Standardmailsuche an, welche Mails beim Öffnen der Mailbox angezeigt werden sollen. Geben Sie das Schlüsselwort für die Suche gefolgt von einem Doppelpunkt (:) und dem Namen eines Ordners, eines Tags oder einer gespeicherten Suche ein.

    3. Die folgende Tabelle enthält Beispiele für die Eingabe in diesem Textfeld.

    ist:ungelesen

    Zeigt nur ungelesene Mails an.

    tag:<Tagname>

    Zeigt Mails an, die mit bestimmten Tags markiert wurden. Beispiel: Um nur Mails mit dem ToDo-Tag anzuzeigen, geben Sie "tag:ToDo" ein.

    in:<Ordnername>

    Zeigt Mails an, die über einen Mail-Filter an einen bestimmten Ordner gesendet wurden.

    Hinweis: Nutzen Sie Anführungszeichen, wenn der Ordnername oder Tag aus mehreren Wörtern besteht. Beispiel: in:"Ungelesene Mails"

  3. Klicken Sie auf Speichern.
