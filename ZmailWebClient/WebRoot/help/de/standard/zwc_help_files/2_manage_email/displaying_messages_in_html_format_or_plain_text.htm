Mails im HTML-Format oder als Plain-Text anzeigen

Sie können Mails in zwei Formaten anzeigen: HTML oder Plain-Text.

•      HTML-Mails können formatierten Text, Nummerierungen, Aufzählungszeichen, farbigen Hintergrund und Links enthalten, wodurch der Text leichter lesbar wird.

•      Plain-Text-Mails können in allen anderen Mail-Clients angezeigt werden. Im Plain-Text-Format werden keine Textformatierungen, Stile, Links oder Bilder unterstützt. Sie können sich Mails auch dann im Plain-Text-Format anzeigen lassen, wenn sie HTML-Code enthalten. In der Plain-Text-Ansicht werden die HTML-Tags im Textkörper der Mail angezeigt.

Anzeigeformat für Mails festlegen

  1. Öffnen Sie die Seite Einstellungen > Mail.

  2. Wählen Sie unter Mails anzeigen > Mails anzeigen entweder Als HTML (falls möglich) oder Als Plain-Text aus.

  3. Klicken Sie auf Speichern.
