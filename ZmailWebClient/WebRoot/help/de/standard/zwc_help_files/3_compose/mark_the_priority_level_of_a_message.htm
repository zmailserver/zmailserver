Prioriätsstufe einer Mail kennzeichnen

Sie können die Wichtigkeit einer Mail angeben. Dies kann z. B. hilfreich sein, wenn Sie eine Anfrage senden, die so schnell wie möglich beantwortet werden muss.

•       Wählen Sie vor dem Senden der Mail im Dropdown-Menü Priorität in der Kopfzeile der Mail die Prioritätsoption Hoch, Normal oder Niedrig aus.

Beim Empfänger wird die Prioritätskennzeichnung in der Mailbox und in der Mail angezeigt.
