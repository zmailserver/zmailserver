Mails und Kontakte mit Tags versehen

Tags sind Ihr persönliches Klassifizierungssystem für Mails, Kontakte und Termine. Tags helfen Ihnen, diese Objekte zu finden. Sie können beispielsweise ein Tag "Sofort erledigen" einrichten und ein anderes "Mittlere Priorität". Die mit Tags markierten Mails können sich in unterschiedlichen Ordnern befinden. Mithilfe von Tags haben Sie die Möglichkeit, alle Mails mit einem bestimmten Tag anzuzeigen, ganz gleich, wo diese abgelegt sind.

Sie können mehrere Tags erstellen und einem einzigen Objekt zuordnen, um es in mehrere Kategorien gleichzeitig einzuordnen. Nachdem Sie ein Tag erstellt haben, können Sie damit Unterhaltungen, Mails und Kontakte markieren.

Neuen Tag anlegen:

  1. Klicken Sie im Übersichtsfenster für das Tag auf Bearbeiten.

  2. Klicken Sie im Inhaltsfenster auf Neuer Tag.

  3. Geben Sie einen Namen für das Tag ein und wählen Sie eine Farbe aus.

  4. Klicken Sie auf Neuen Tag erstellen. Das neue Tag wird im Übersichtsfenster für das Tag aufgeführt.

Mehrere Objekte gleichzeitig mit Tags versehen bzw. Tags entfernen:

  1. Aktivieren Sie die Kontrollkästchen der Objekte, die Sie mit einem Tag markieren möchten.

  2. Wählen Sie im Dropdown-Menü Aktionen das gewünschte Tag aus.

  3. Klicken Sie auf Los.

Ein Tag von mehreren Objekten löschen:

  1. Aktivieren Sie die Kontrollkästchen der Objekte, die mit dem zu löschenden Tag markiert sind.

  2. Wählen Sie im Dropdown-Menü Aktionen unter "Tag entfernen" das gewünschte Tag aus. Um mehrere Tags zu löschen, wählen Sie ALLE aus.

  3. Klicken Sie auf Los.
