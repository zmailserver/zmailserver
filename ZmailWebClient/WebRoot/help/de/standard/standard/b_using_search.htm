Suche durchführen

Die @CompanyName@ -Suche ist eine leistungsstarke Funktion. Sie können zwar lernen, wie Sie komplexe Textabfragen eingeben (dies wird beschrieben unter Beschreibung der Sprache für Suchabfragen), doch es ist viel schneller, einfach einen Namen oder ein Wort ohne Schlüsselwörter einzugeben. Wenn Sie z. B. alle Mails finden möchten, die das Wort "tim" in einer der Zeilen "Betreff", "An", "Von" "CC": oder "BCC" , im Textkörper oder im Dateianhang enthalten, geben Sie einfach tim in das Textfeld Suchen ein und klicken Sie auf Suchen.

Sie geben an, wo gesucht werden soll: Mail, Kontakte, Termine, Aufgaben und Dateien im Ordner "Dateien". Sie können auch alle Ordner nach der Information durchsuchen.

Um nach Kontakten in Ihrem Adressbuch zu suchen, müssen Sie für die Abfrage ein vollständiges Wort eingeben. Um z. B. in den Kontakten nach einer Telefonnummer zu suchen, müssen Sie die vollständige Telefonnummer so eingeben, wie sie im Kontakteintrag vorhanden ist. Sie können nicht z. B. nur nach der Vorwahl suchen.

  1. Wählen Sie in der Suchzeile im Dropdown-Menü die Option Persönliche Kontakte aus. Damit werden alle Ihre Adressbücher durchsucht.

  2. Geben Sie einen vollständigen Namen oder andere Kontaktdaten in das Textfeld "Suchen" ein.

  3. Klicken Sie auf Suchen.

Sie können jeden Wert, z. B. ein Wort, einen Vornamen, Nachnamen, eine Telefonnummer oder einen Domainnamen eingeben. Zur größeren Flexibilität nutzen Sie das Sternchen * als Platzhalter.  

Platzhalter * bei der Suche nutzen

Das Sternchen * kann in einer Suche als Platzhalter eingesetzt werden, um Inhalte mit Wörtern ähnlicher Schreibweise zu finden.  

Nutzen Sie das Sternchen * als Platzhalter im Anschluss an eine bestimmte Zeichenfolge. Beispiel: Die Suche nach Hu* findet Objekte mit den Wörtern "Hund", "Hut" usw.

  1. Wählen Sie in der Suchzeile im Dropdown-Menü das zu suchende Objekt oder den zu suchenden Objekttyp aus.

  2. Geben Sie ein Wort, einen Namen oder andere mailspezifische Daten in das Textfeld "Suchen" ein.

  3. Klicken Sie auf Suchen.
