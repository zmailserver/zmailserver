Im Ordner "Dateien" arbeiten

Der Ordner "Dateien" kann zum Freigeben und Verwalten von Dokumenten, die Sie unter "Dateien" erstellen, sowie von Dokumenten und Bildern verwendet werden, die Sie in Ihren Ordner "Dateien" hochladen. Im Ordner "Dateien" können Sie Dateien in Ihrem Account speichern und jederzeit nach dem Anmelden bei Ihrem Account von einem beliebigen Computer aus aufrufen.

Sie können Dokumente, Kalkulationstabellen, Präsentationen, Bilder, PDF-Dateien von Ihrem Computer oder Ihrem Computer-Netzwerk hochladen, und Sie können mit Ihren Mails versendete Anhänge in einem Ordner "Dateien" speichern. Sie können weitere Ordner "Dateien" zum Organisieren der Datei erstellen.

Eine Datei, die Sie in den Ordner "Dateien" hochladen, ist eine Kopie der ursprünglichen Datei auf Ihrem Computer. Änderungen, die Sie an einer Datei im Ordner "Dateien" vornehmen, haben keine Auswirkungen auf die ursprüngliche Datei.

Ordner "Dateien" erstellen

  1. Wählen Sie die Registerkarte Dateien aus.

  2. Klicken Sie unter der Überschrift Ordner im Übersichtsfenster auf Bearbeiten.

  3. Klicken Sie auf Neue Dateien.

  4. Eine Seite Neue Dateien wird geöffnet. Geben Sie einen Namen für den neuen Ordner "Dateien" ein.

Dieser Name muss innerhalb der Hierarchie Ihrer Mailbox-Ordner eindeutig sein. Ordner unter "Dateien" dürfen nicht denselben Namen haben wie ein übergeordneter Ordner in der Mail, im Kalender oder im Adressbuch von @CompanyName@. Wenn Sie beispielsweise einen Kalender mit dem Namen "Urlaub" haben, dürfen Sie keinem übergeordneten Ordner "Dateien" den Namen "Urlaub" geben. Ein Ordner "Dateien", der sich in einem untergeordneten Ordner "Dateien" befindet, darf "Urlaub" heißen.

  1. Wählen Sie unter Übergeordneter Ordner die Ordnerhierarchie aus. Sie können einen neuen übergeordneten Ordner "Dateien" im Stammordner (Root) anlegen oder in einen vorhandenen Ordner "Dateien" einfügen.

  2. Klicken Sie auf Dateien erstellen. Der neue Ordner "Dateien" wird im Übersichtsfenster angezeigt.

Dateien hochladen

Beim Hochladen wird eine Datei von Ihrem Computer in Ihren Mail-Account hochgeladen. Dadurch steht die Datei jederzeit zur Verfügung, wenn Sie sich an Ihrem Account anmelden. Ihr Account-Kontingent bestimmt, wie viele Dateien unter "Dateien" hochgeladen werden können.

  1. Öffnen Sie den Ordner "Dateien" und klicken Sie in der Symbolleiste auf Datei hochladen.

  2. Um eine Datei hochzuladen, klicken Sie auf Durchsuchen, um die gewünschte Datei auszuwählen.

  3. Wählen Sie die Datei aus und klicken Sie auf Öffnen.

  4. Weitere Anhänge laden Sie auf die gleiche Weise hoch.

  5. Wenn Sie alle Anhänge hinzugefügt haben, klicken Sie auf Fertig.

Hinweis: Für Ihren Account ist möglicherweise eine maximale Größe für hochgeladene Dateien definiert. Wenn eine Datei zu groß ist, wird eine Warnung angezeigt. Dateien, die Sie hochladen, wirken sich auf das Kontingent Ihres Accounts aus.
