Neue Kalender erstellen

Ein Kalender ist bereits vorhanden. Sie können weitere Kalender erstellen, um einen Überblick über verschiedene Arten von Ereignissen zu haben. Sie können z. B. einen Kalender für die Arbeit erstellen, Sie können einen privaten Kalender und einen für Aktivitäten der Firma erstellen, Sie können einen Kalender aus dem Internet abonnieren und Sie können einen Link zu einem Kalender einrichten, der für Sie freigegeben ist.

Kalender erstellen

  1. Öffnen Sie die Registerkarte "Kalender".

  2. Klicken Sie im Übersichtsfenster des Kalenders auf Bearbeiten.

  3. Klicken Sie in der Symbolleiste auf Neuer Kalender.

  4. Geben Sie den Namen des Kalenders ein.

  5. Wählen Sie eine Farbe aus, die die Kalenderaktivitäten widerspiegelt. So können Sie Kalender in Ihrer Mailbox farblich kennzeichnen, sodass Aktivitäten aus einem Kalender stets in der ausgewählten Farbe angezeigt werden.

  6. Wenn die Aktivitäten dieses Kalenders nicht von anderen Nutzern eingesehen werden sollen, aktivieren Sie Diesen Kalender bei Frei-/Belegt-Zeiten ignorieren.

  7. Standardmäßig ist Auf der Benutzeroberfläche markiert aktiviert. Die Aktivitäten dieses Kalenders werden angezeigt, wenn Sie die Ansicht "Kalender" öffnen. Soll dieser Kalender nicht zusammen mit dem anderen Kalender angezeigt werden, deaktivieren Sie dieses Kontrollkästchen. Dann müssen Sie, um die Aktivitäten in diesem Kalender anzuzeigen, den Kalender auswählen und öffnen.

  8. Klicken Sie auf Kalender erstellen.

Einen anderen Kalender abonnieren

  1. Öffnen Sie die Registerkarte Kalender.

  2. Klicken Sie im Übersichtsfenster des Kalenders auf Bearbeiten. Die Standardinformationen des Kalenders werden angezeigt.  

  3. Klicken Sie in der Symbolleiste auf Kalender abonnieren.  

  4. Geben Sie einen Namen für den Kalender ein, geben Sie die gültige URL einer iCal-Datei eines externen Kalenders ein und wählen Sie eine Farbe für die Aktivitäten in diesem Kalender aus.

  5. Wenn die Aktivitäten dieses Kalenders nicht von anderen Nutzern eingesehen werden sollen, aktivieren Sie Diesen Kalender bei Frei-/Belegt-Zeiten ignorieren.

  6. Standardmäßig ist Auf der Benutzeroberfläche markiert aktiviert. Die Aktivitäten dieses Kalenders werden angezeigt, wenn Sie die Ansicht "Kalender" öffnen. Soll dieser Kalender nicht zusammen mit dem anderen Kalender angezeigt werden, deaktivieren Sie dieses Kontrollkästchen. Dann müssen Sie, um die Aktivitäten in diesem Kalender anzuzeigen, den Kalender auswählen und öffnen.

  7. Klicken Sie auf Kalender erstellen.
