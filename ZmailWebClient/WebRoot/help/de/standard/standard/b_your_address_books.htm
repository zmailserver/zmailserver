Adressbücher

Über die Funktion "Adressbuch" können Sie Kontaktdetails in persönlichen Adressbüchern speichern. Standardmäßig ist für Sie ein Adressbuch Kontakte und ein Adressbuch Mailempfänger vorhanden. Sie können weitere Adressbücher anlegen.

Um einen Kontakt in Ihren Adressbüchern anzulegen, ist nur ein Name erforderlich. Sie können auch ausführliche Kontaktkarten anlegen, die den vollständigen Namen, mehrere Mail-Adressen, geschäftliche, private und andere Anschriften, Telefonnummern und Notizen zu dem Kontakt enthalten. Sie können auch Kontaktgruppen erstellen.

Standardmäßig wird das Adressbuch Mailempfänger automatisch ergänzt, wenn Sie eine Mail an eine neue Adresse senden, die sich noch nicht in einem Ihrer Adressbücher befindet. Sie können diese Funktion über die Registerkarte Einstellungen>Adressbuch deaktivieren. Deaktivieren Sie das Kontrollkästchen Automatisches Hinzufügen von Kontakten aktivieren.

Ihr Mail-Administrator kann eine Obergrenze für die Anzahl der Einträge in Ihren Adressbüchern festlegen. Wenn Sie diese maximale Anzahl erreichen, können Sie keine weiteren Kontakte hinzufügen. Sie haben jedoch die Möglichkeit, nicht benötigte Kontakte zu löschen, um Speicherplatz freizugeben.

  1. Öffnen Sie die Registerkarte "Adressbuch". Im Übersichtsfenster wird eine Liste Ihrer Adressbücher angezeigt.

  2. Klicken Sie im Übersichtsfenster der Registerkarte Adressbuch auf Bearbeiten.

  3. Klicken Sie auf Neues Adressbuch.

  4. Geben Sie einen Namen ein und wählen Sie die Hintergrundfarbe für das neue Adressbuch aus.

  5. Klicken Sie auf Adressbuch erstellen.

Das neue Adressbuch wird im Übersichtsfenster aufgeführt.
