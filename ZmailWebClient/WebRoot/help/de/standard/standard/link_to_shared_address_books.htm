Links zu freigegebenen Adressbüchern einrichten

Wenn Sie per Mail informiert werden, dass Ihnen der Zugriff auf das Adressbuch einer anderen Person gewährt wird, müssen Sie einen Link zu dem freigegebenen Adressbuch einrichten. Sie benötigen hierfür die vollständige Mail-Adresse des Besitzers und den genauen Namen des freigegebenen Adressbuchs.

So richten Sie einen Link zu einem freigegebenen Adressbuch ein

  1. Klicken Sie im Übersichtsfenster der Adressbuch-Kopfzeile auf Bearbeiten.

  2. Wählen Sie in der Symbolleiste die Option Link zu Adressbuch aus.

  3. Geben Sie einen Namen für das freigegebene Adressbuch ein. Dies ist der Name, der in der Adressbuchliste angezeigt wird.

  4. Geben Sie die Mail-Adresse des Besitzers und den Kalendernamen ein.

  5. Wählen Sie für dieses freigegebene Adressbuch eine Farbe aus.

  6. Klicken Sie auf Adressbuch erstellen.
