Anhänge hinzufügen

Über die Schaltfläche "Durchsuchen" können Sie jede Datei aus Ihrem Dateisystem an eine Mail anhängen.

Sie können jeden Dateianhang direkt aus Ihrer Mailbox öffnen, vorausgesetzt, Sie verfügen über die erforderliche Anwendung und die Erweiterung wurde nicht von Ihrem Administrator blockiert.

Wenn der Dateityp von der auf Ihrem Computer installierten Software unterstützt wird, reicht es normalerweise, auf die Datei doppelzuklicken, um automatisch die richtige Anwendung aufzurufen und die Datei zu öffnen.

Computerviren verbreiten sich häufig über Dateianhänge. Daher können Systemadministratoren eingehende Mails mit bestimmten Arten von Anhängen blockieren, meistens Dateien mit der Erweiterung .EXE oder .ZIP im Dateinamen. Wenn Sie eine Mail an einen Nutzer senden, dessen Mailsystem so konfiguriert wurde, dass bestimmte Arten von Anhängen blockiert werden, erhalten Sie möglicherweise keine Benachrichtigung über die Blockierung.

Einen Anhang zu einer Mail hinzufügen

Sie können Dokumente, Arbeitsblätter aus Tabellenkalkulationen, Bilder, Diashows und andere Dateitypen an Mails anhängen.

  1. Verfassen Sie die Mail.

  2. Klicken Sie unter dem Feld Betreff auf Anhängen und wählen Sie Mein Computer aus.

  3. Wählen Sie die Dateien aus und klicken Sie auf Öffnen. Die Dateinamen werden unter dem Textfeld "Betreff" angezeigt.

  4. Klicken Sie auf Senden, um die Mail samt Anhängen zu senden.

Hinweis:   Die Empfänger benötigen die entsprechende Software, um die Datei zu öffnen. Gängige Dateiformate wie Textdateien, HTML-Dateien und Bilder in den Formaten GIF oder JPEG können in einer Vielzahl von Programmen geöffnet werden.

Anhänge mit Drag & Drop hinzufügen

Sie können einer Mail bequem einen Anhang hinzufügen, indem Sie die Datei aus einem Ordner in die Mail ziehen.

  1. Wählen Sie eine oder mehrere Dateien aus, die angehängt werden sollen, und halten Sie die Maustaste gedrückt.

  2. Ziehen Sie die Dateien in den Kopfzeilenbereich der Mail und lassen Sie die Maustaste los. Die Dateinamen werden in der Kopfzeile angezeigt.

Anhang entfernen

Wenn Sie einen Anhang entfernen möchten, klicken Sie in der Blase für den Anhang auf X.
