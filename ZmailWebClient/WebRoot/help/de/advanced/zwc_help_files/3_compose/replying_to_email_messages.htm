Mails beantworten

Sie können eine Antwort an den Absender einer Mail oder an den Absender und alle Empfänger senden.

Wenn Sie auf eine Mail antworten, wird die ursprüngliche Mail standardmäßig im Textkörper eingeschlossen, die ursprünglichen Anhänge jedoch nicht. Sie können festlegen, dass jede Antwort in der Mail mit einem Präfix versehen wird, um die Antworten besser verfolgen zu können.

Beispielsweise kann jede Antwort in der Mail mit einer eckigen Klammer (>) markiert werden. Bei der letzten Antwort wird eine Klammer (>) vor jeder Zeile angezeigt, bei der Antwort davor zwei Klammern (>>), davor drei (>>>):

> Vorherige Mail

> > Mail vor der vorherigen Mail

> > > Und so weiter

Eine Mail beantworten

  1. Wählen Sie die Mail aus, die Sie beantworten möchten.

  2. Klicken Sie in der Symbolleiste auf Antworten oder Allen antworten. Es wird ein neues Fenster zum Verfassen einer Mail mit den Adressen in der Kopfzeile und der ursprünglichen Mail im Textkörper angezeigt.

    3. Wenn Sie die Mail in der Unterhaltungsansicht anzeigen, können Sie unten in der Mail auf "Antworten" klicken.

  3. Geben Sie Ihre Antwort ein.

  4. Klicken Sie in der Symbolleiste auf Optionen und wählen Sie eine Option aus, wenn Sie ändern möchten, wie die ursprüngliche Mail in der Antwort eingeschlossen wird.

  5. Klicken Sie auf Senden.

Hinweis:  Wenn Sie Ihre Signatur automatisch an Ihre Mails anfügen, wird sie entsprechend den Einstellungen in der Mail platziert.
