Persönlichkeiten hinzufügen

Wenn Sie mehrere Persönlichkeiten anlegen, können Sie in Ihrer Mailbox mehrere Mail-Adressen nutzen. Sie können z. B. die Hauptpersönlichkeit für den Account für geschäftliche Mails verwenden und eine neue Persönlichkeit für private Mails erstellen.

  1. Öffnen Sie die Seite Einstellungen > Accounts. Der Standard-Account und die persönlichen Informationen werden als primärer Account angezeigt.

  2. Klicken Sie auf Persönlichkeit hinzufügen. Der Accountname Neue Persönlichkeit wird in der Spalte "Accountname" und im Textfeld Persönlichkeitseinstellungen>Name der Persönlichkeit angezeigt.

  3. Geben Sie im Textfeld Name der Persönlichkeit ein anschauliches Wort zum Identifizieren der Persönlichkeit ein, das beim Verfassen einer Mail in der Liste "Von" angezeigt wird. Dieser Name wird in der Mail nicht angezeigt.

  4. Geben Sie im Abschnitt Einstellungen für gesendete Mails die Von-Informationen für diese Persönlichkeit ein.

Geben Sie im Textfeld Von den Namen ein, der im Feld Von Ihrer ausgehenden Mails angezeigt wird. Dies ist der Name, der vor Ihrer Mail-Adresse angezeigt wird.

Wählen Sie im Dropdown-Menü neben dem Textfeld die Mail-Adresse aus, von der die Mails gesendet werden sollen. Wenn dieses Feld nicht bearbeitet werden kann, haben Sie keine zusätzlichen externen Accounts definiert.

  1. Um Antworten auf Mails von dieser Persönlichkeit an einen anderen Namen und eine andere Adresse als unter Von angegeben weiterzuleiten, wählen Sie Als Antwortadresse von Mails einstellen aus und geben Sie dann im Textfeld den zu verwendenden Namen ein.

  1. (Optional) Wenn Sie der Persönlichkeit eine Signatur zuordnen möchten, klicken Sie auf Signatur.

  2. Wenn beim Beantworten von Mails, die an eine bestimmte Adresse gesendet wurden, oder beim Weiterleiten von Mails von dieser Adresse automatisch diese Persönlichkeit verwendet werden soll, wählen Sie Beim Antworten oder Weiterleiten einer Mail an aus und geben Sie im Textfeld die Mail-Adresse ein. Wenn Sie mehrere Mail-Adressen eingeben, trennen Sie die Adressen durch Komma oder Semikolon.

  3. Um beim Beantworten von Mails in einem bestimmten Ordner oder beim Weiterleiten von Mails aus diesem Ordner automatisch diese Persönlichkeit zu verwenden, wählen Sie Antworten oder Weiterleiten von Mails in folgenden Ordnern aus. Klicken Sie auf das Ordnersymbol, um einen oder mehrere Ordner auszuwählen oder um einen neuen Ordner anzulegen.

Wenn Sie mehrere Ordner auswählen, trennen Sie die Ordnernamen durch Komma oder Semikolon.

  1. Klicken Sie auf Speichern.
