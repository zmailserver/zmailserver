Mit Kalendern arbeiten

Mithilfe von Kalendern können Sie Termine, Besprechungen und Ereignisse überwachen und planen. Sie können mehrere Kalender verwalten und Ereignisse zwischen diesen verschieben. Außerdem können Sie Kalender für andere Personen freigeben.

Um einen Kalender zu öffnen, klicken Sie auf die Registerkarte Kalender. Standardmäßig wird zunächst die Arbeitswoche angezeigt. Arbeitstage und -zeiten werden in weiß angezeigt. Freie Zeit wird in grau angezeigt.

Die Einstellungen für Ihre Kalender können auf der Seite "Einstellungen" > "Kalender" geändert werden.

Standardkalenderansicht festlegen

Standardmäßig wird die Arbeitswochenansicht angezeigt. Wenn Sie den Kalender öffnen, können Sie auch eine andere Ansicht auswählen.

  1. Öffnen Sie die Seite Einstellungen > Kalender.

  2. Wählen Sie im Dropdown-Menü Standardansicht die gewünschte Ansicht aus.

    3. Option

    Beschreibung

    Tagesansicht

    Zeigt die Aktivitäten für den ausgewählten Tag an.

    Arbeitswochenansicht

    Zeigt die Aktivitäten von Montag bis Freitag an.

    Wochenansicht 

    Zeigt die Aktivitäten von Montag bis Sonntag an. Im Dropdown-Menü Woche beginnen am können Sie angeben, mit welchem Tag Ihre Woche beginnen soll.

    Monatsansicht

    Zeigt die Aktivitäten eines Monats an. In der Monatsansicht eines Kalenders können Sie auf ein Datum doppelklicken, um eine detaillierte Ansicht der Termine nach Stunden anzuzeigen.

    Listenansicht

    Zeigt eine Liste der Termine aus einem angegebenen Datumsbereich für die im Übersichtsfenster ausgewählten Kalender an.

  3. Klicken Sie auf Speichern.
