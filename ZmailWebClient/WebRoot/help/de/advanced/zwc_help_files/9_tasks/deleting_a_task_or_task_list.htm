Aufgabe oder Aufgabenliste löschen

Aufgabe löschen

  1. Öffnen Sie die Seite Aufgaben.

  2. Wählen Sie im Übersichtsfenster die Aufgabenliste mit der Aufgabe aus, die Sie löschen möchten.

  3. Wählen Sie im Inhaltsfenster die Aufgaben aus, die Sie löschen möchten. Um mehrere Aufgaben zu löschen, aktivieren Sie einfach die Kontrollkästchen neben den Aufgaben, die Sie löschen möchten.

  4. Klicken Sie in der Symbolleiste auf Löschen.

Sie können diese Aktion sofort rückgängig machen, indem Sie oben auf der Seite auf den Link Rückgängig klicken. Die Aufgabe wird zurück an ihre ursprüngliche Position verschoben.

Aufgabenliste löschen

Sie können nur Aufgabenlisten löschen, die Sie selbst erstellt haben.

  1. Öffnen Sie die Seite Aufgaben.

  2. Wählen Sie im Übersichtsfenster die Aufgabenliste aus, die Sie löschen möchten.

  3. Wählen Sie im Dropdown-Menü die Option Löschen aus.

Sie können diese Aktion sofort rückgängig machen, indem Sie oben auf der Seite auf den Link Rückgängig klicken. Die Aufgabenliste wird zurück an ihre ursprüngliche Position verschoben.
