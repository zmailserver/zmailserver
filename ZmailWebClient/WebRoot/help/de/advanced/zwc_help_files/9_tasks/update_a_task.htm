Aufgabe aktualisieren

Während Sie eine Aufgabe bearbeiten, können Sie Informationen hinzufügen, den Status sowie den Prozentwert der Fertigstellung ändern und die Aufgabe als abgeschlossen markieren.

  1. Öffnen Sie die Seite Aufgaben und wählen Sie die Aufgabe aus, die Sie bearbeiten möchten.

  2. Klicken Sie auf Bearbeiten und ändern Sie die Informationen dieser Aufgabe.

  3. Klicken Sie auf Speichern.
