Ordner auf der Seite "Einstellungen" > "Freigaben" freigeben

  1. Öffnen Sie die Seite Einstellungen > Freigaben.

  2. Wählen Sie im Abschnitt Ordner freigeben im Dropdown-Menü den Typ des Ordners aus, der freigegeben werden soll.

  3. Klicken Sie auf Freigabe.

  4. Wählen Sie den Ordner aus, den Sie freigeben möchten, und klicken Sie auf OK.

  5. Geben Sie im Dialogfeld Eigenschaften der Freigabe an, ob die Freigabe für interne, externe oder öffentliche Nutzer erfolgen soll.

  6. Führen Sie abhängig von der Art der Freigabe folgende Aktionen aus:

    7. •    Geben Sie für Interne Nutzer oder Gruppen die Mailadressen der Nutzer ein. Legen Sie im Bereich Rolle die Zugangsberechtigungen fest.

    •    Geben Sie fürExterne Nutzer die Mailadressen der Nutzer ein. Externe Nutzer erhalten eine Mail mit einem Link zum freigegebenen Objekt. Wenn sie zum ersten Mal auf das freigegebene Objekt zugreifen, müssen Sie einen Anzeigenamen sowie ein Passwort erstellen.

    •    Geben Sie unter Öffentlich die URL ein, die verwendet werden soll.

  7. Wählen Sie im Bereich Mail den Typ von Nachricht aus, die an Nutzer gesendet werden soll, denen Sie Zugriff gewähren. Diese Mail kann sowohl an interne als auch an externe Nutzer gesendet werden. Hinweis: Wenn Sie Öffentlich ausgewählt haben, müssen Sie die URL öffentlich bekanntgeben.

  8. Klicken Sie auf OK.
