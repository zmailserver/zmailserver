Datei umbenennen

Durch das Umbenennen einer Datei wird eine neue Version erstellt.

  1. Öffnen Sie die Registerkarte Dateien und wählen Sie das Dokument aus, das Sie umbenennen möchten.

  2. Klicken Sie mit der rechten Maustaste auf die Datei und wählen Sie Umbenennen aus. Das Feld "Name" wird in ein bearbeitbares Feld geändert.

  3. Bearbeiten Sie das Feld, um den Dateinamen zu ändern. Die Erweiterung darf nicht geändert werden. Andernfalls kann die Datei später nicht mehr bearbeitet werden.

  4. Drücken Sie die Eingabetaste.
