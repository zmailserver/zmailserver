Im Ordner "Dateien" arbeiten

Mithilfe des Ordners "Dateien" können Sie Dokumente freigeben und verwalten, die in diesem Ordner erstellt wurden, und Sie können Dokumente und Bilder freigeben und verwalten, die in den Ordner hochgeladen wurden. Mit dem Ordern "Dateien" können Sie außerdem Dateien in Ihrem Account speichern und von jedem Computer darauf zugreifen, wenn Sie sich bei Ihrem Account anmelden.

Sie können Ihre Ordner "Dateien" freigeben, damit diese von anderen Personen angezeigt und bearbeitet werden können.

Sie können mehrere Ordner "Dateien" erstellen und beliebige Dateitypen von Ihrem Computer oder Netzwerk hochladen, Anhänge aus Mails speichern oder neue Dokumente erstellen.

Dateien im Ordner "Dateien" sind Kopien der Originaldateien, die auf Ihrem Computer gespeichert oder einer Mail als Anhang beigefügt wurden. Änderungen, die Sie an den Dateien in diesem Ordner vornehmen, haben keine Auswirkungen auf die ursprüngliche Datei.

Ordner "Dateien" erstellen

  1. Öffnen Sie Dateien > Ordner.

  2. Klicken Sie auf das Zahnradsymbol und wählen Sie Neue Dateien aus.

  3. Geben Sie einen Namen für den neuen Ordner "Dateien" ein.

    4. Der Name des Ordners "Dateien" muss innerhalb der Hierarchie aller Mailboxordner eindeutig sein. Dieser Name darf nicht mit dem Namen eines Ordners der obersten Ebene in Ihrer Mailbox, einem Kalender oder einem Adressbuch übereinstimmen. Wenn Sie beispielsweise einen Kalender mit dem Namen "Urlaub" haben, dürfen Sie keinem Ordner "Dateien" of oberster Ebene den Namen "Urlaub" geben. Ein Ordner "Dateien", der sich in einem untergeordneten Ordner "Dateien" befindet, darf "Urlaub" heißen.

  4. Wählen Sie die Farbe des neuen Ordners "Dateien" aus.

  5. Wählen Sie die Ordnerhierarchie (unter "Ordner" oder unter einem anderen Ordner "Dateien") aus. Sie können einen neuen übergeordneten Ordner anlegen oder den Ordner in einen vorhandenen Ordner einfügen.

  6. Klicken Sie auf OK.

Dateien in den Ordner "Dateien" hochladen

  1. Öffnen Sie die Registerkarte Dateienund wählen Sie den Ordner "Dateien" aus, in dem die Datei gespeichert werden soll.

  2. Klicken Sie auf Datei hochladen und auf Durchsuchen..., um die Datei zu suchen, die Sie hochladen möchten.  

  3. Im Textfeld Notizen können Sie Informationen zu der Datei hinzufügen. Die Informationen werden in der Dateiliste angezeigt.

  4. Wenn Sie weitere Dateien hinzufügen möchten, klicken Sie auf Hinzufügen.

    5. Sie können keine Datei hochladen, die denselben Namen hat wie eine Datei, die bereits unter "Dateien" vorhanden ist. Wenn Sie eine Datei hochladen, die bereits in dem Ordner vorhanden ist, wird das Dialogfeld "Konflikt beim Hochladen" angezeigt. Sie können auf Ja klicken, um die Datei hochzuladen und die vorhandene Datei zu überschreiben, oder Sie können auf Nein klicken, um den Vorgang abzubrechen.

    Wenn die Datei überschrieben wird, wird ihre Versionsnummer aktualisiert.

  5. Nachdem alle Dateien hochgeladen wurden, klicken Sie auf OK.

Hinweis:   Wenn Sie den falschen Ordner "Dateien" ausgewählt haben, können Sie die Datei mit der Maus in den richtigen Ordner "Dateien" verschieben.

Bei zu großen Dateien wird eine Warnung angezeigt. Dateien, die Sie hochladen, wirken sich auf auf die Nutzung Ihres Account-Kontingents aus.
