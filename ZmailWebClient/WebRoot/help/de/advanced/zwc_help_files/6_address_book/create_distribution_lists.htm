Verteiler mit verwaltetem Zugriff erstellen

  1. Öffnen Sie das Dropdown-Menü Adressbuch>Neuer Kontakt und wählen Sie Verteiler aus.

  2. Geben Sie in der Seitenüberschrift einen Namen für den Verteiler ein. Dies ist die Mail-Adresse des Verteilers. Der Name muss verfügbar sein. Wenn er nicht verfügbar ist, wird eine Fehlermeldung angezeigt, und Sie müssen einen anderen Namen eingeben.

  3. Geben Sie in Anzeigename einen Anzeigenamen für den Verteiler ein. Wenn Sie einen anderen Namen als die Mail-Adresse des Verteilers wünschen, geben Sie einen Anzeigenamen ein.

  4. Geben Sie die Beschreibung des Verteilers ein.

  5. Geben Sie an, ob der Verteiler Privat ist. In diesem Fall wird der Verteiler nicht in der globalen Adressliste (GAL) angezeigt.

  6. Wählen Sie die Mail-Optionen aus. Mit Mail-Optionen legen Sie Regeln fest, die bestimmen, für wen die Verteilerliste angezeigt wird.

  • Jeder darf Mail an diesen Verteiler senden

  • Nur Mitglieder dürfen Mail an diesen Verteiler senden

  • Nur interne Nutzer dürfen Mail an diesen Verteiler senden

  • Nur diese Nutzer dürfen Mail an diesen Verteiler senden. Geben Sie im Textfeld die Namen und Mail-Adressen der Nutzer ein, die Mail an den Verteiler senden dürfen.

  1. Wählen Sie den Typ für Neue Abonnementanfragen aus, den Sie für diesen Verteiler zulassen möchten.

  2. Legen Sie die Regeln für Kündigungsanfragen für den Verteiler fest.

  3. Wenn der Verteiler von einer anderen Person verwaltet werden soll, fügen Sie im Textfeld Listenbesitzer Namen und Mail-Adressendie Adressen hinzu. Wenn Sie den Verteiler nicht verwalten möchten, entfernen Sie Ihren Namen, bevor Sie andere Adressen hinzufügen.

  4. Wählen Sie auf der Registerkarte "Mitglieder" neue Gruppenmitglieder aus, die dem Verteiler hinzugefügt werden sollen.. Sie können in den Kontaktlisten oder in der globalen Adressliste Adressen suchen und Sie können Adressen hinzufügen, die nicht in diesen Listen enthalten sind. Geben Sie die Mail-Adresse im Textfeld Oder geben Sie die Adressen unten ein ein. Trennen Sie einzelne Adressen durch ein Komma (,) oder Semikolon (;). Von Hand eingegebene Mail-Adressen werden nicht überprüft.

  5. Klicken Sie auf Speichern. Die Accounts werden dem Verteiler hinzugefügt und das Verteileradressbuch des Accounts wird mit dem Namen des Verteilers aktualisiert.
