Ordner löschen

Sie können benutzerdefinierte Ordner löschen, nicht jedoch Systemordner. Wenn Sie einen Ordner löschen, werden der Ordner und sein Inhalt in den Papierkorb verschoben. Wenn Sie einige Mails in einem Ordner vor dem Löschen behalten möchten, verschieben Sie die Mails in einen anderen Ordner.

Sie können einen Ordner und seinen Inhalt aus dem Papierkorb wiederherstellen, indem Sie den Ordner per Drag & Drop in einen anderen Ordner verschieben.

Wichtig:  Wenn Sie einen Ordner löschen, wird oben ein Popup angezeigt, in dem darauf hingewiesen wird, dass der Ordner in den Papierkorb verschoben wird. Klicken Sie auf Rückgängig, wenn Sie den Löschvorgang rückgängig machen möchten. Dieses Popup wird nur sehr kurz angezeigt. Klicken Sie also sofort auf "Rückgängig", wenn Sie den Ordner nicht löschen möchten. Andernfalls müssen Sie den Ordner manuell wiederherstellen.

  1. Öffnen Sie die Seite Mail > Ordner.

  2. Klicken Sie mit der rechten Maustaste auf den zu löschenden Ordner und wählen Sie Löschen aus.

Hinweis:   Der Ordner "Papierkorb" wird in regelmäßigen Abständen geleert, die vom Administrator festgelegt werden, d. h. darin enthaltene Objekte bleiben nicht für immer verfügbar.
