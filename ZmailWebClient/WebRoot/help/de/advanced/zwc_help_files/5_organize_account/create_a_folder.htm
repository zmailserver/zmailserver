Ordner erstellen

Sie können einen eigenständigen benutzerdefinierten Ordner oder einen Ordner in einem anderen Order erstellen, auch in Systemordnern.

Sie können in einem übergeordneten Ordner keine zwei Ordner gleichen Namens anlegen. Es kann z. B. in Ihrem Posteingang nicht zwei Ordner mit dem Namen Shopping geben, aber Sie können einen Ordner Shopping in Ihrem Posteingang und einen anderen Ordner mit dem gleichen Namen im Ordner "Gesendet" anlegen.

  1. Öffnen Sie die Seite Mail > Ordner und klicken Sie auf das Zahnradsymbol für Ordner.

  2. Wählen Sie Neuer Ordner aus.

  3. Geben Sie in Name den Namen des neuen Ordners ein und wählen Sie eine Farbe für den Ordner aus.

    4. Ordnernamen dürfen jedes Zeichen außer Doppelpunkt (:), Schrägstrich
    (/) und Anführungszeichen (") enthalten.

    Sie haben die Möglichkeit, RSS-/ATOM-Feeds zu abonnieren.

  4. Wählen Sie in der Struktur Ordner die Position des neuen Ordners aus.

    5. •    Klicken Sie auf Ordner, damit der neue Ordner am Anfang der Liste angezeigt wird.

    •    Klicken Sie auf einen vorhandenen Ordner, um den neuen Ordner in diesen Ordner einzufügen.

  5. Klicken Sie auf OK.
