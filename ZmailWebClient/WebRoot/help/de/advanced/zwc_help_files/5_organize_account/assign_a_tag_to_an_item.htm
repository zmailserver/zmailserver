Tag einem Objekt zuweisen

Nachdem Sie einen Tag erstellt haben, können Sie ihn einem Objekt zuweisen. Sie können einem Objekt mehrere Tags zuweisen, um es in unterschiedlichen Kategorien zu klassifizieren. Wenn ein Objekt über mehrere Tags verfügt, wird das Tag-Symbol mit unterschiedlichen Farben angezeigt. Es werden nur Tags aufgeführt, die dem Objekt noch nicht zugewiesen wurden.

  1. Klicken Sie im Inhaltsfenster mit der rechten Maustaste auf ein Objekt.

  2. Wählen Sie im Dropdown-Menü mit Tag versehen aus. Das Thema kann z. B. eine Unterhaltung sein, dies wird als Unterhaltung mit Tag versehen angezeigt.

  3. Klicken Sie auf den Tag, um ihn dem Objekt zuzuweisen. Neben dem Objekt wird ein Tag-Symbol angezeigt.
