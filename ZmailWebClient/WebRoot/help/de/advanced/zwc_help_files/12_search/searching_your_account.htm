Eigenen Account durchsuchen

Sie können nach Mails, Kontakten, Terminen, Aufgaben und Dateien suchen. Dabei können Sie nach bestimmten Wörtern, nach Datumsangaben, Uhrzeiten, URLs, Größenangaben, Tags, gelesenen und ungelesenen Mails, nach Mails mit Dateianhängen und sogar nach Mails mit Anhängen bestimmten Typs suchen.

Die Funktion "Suchen" führt die derzeit im Suchtextfeld angezeigte Suchabfrage aus. Im Suchfeld können Sie eines oder mehrere Schlüsselwörter für die Suche eingeben. Eine Liste möglicher Suchargumente wird automatisch angezeigt. Über den Dropdown-Pfeil links neben dem Suchfeld können Sie außerdem auswählen, nach welcher Art von Objekt Sie suchen möchten. Sie können in Ihren Mails, Kontakten, im Adressbuch der Firma, in Ihren Terminen, Aufgaben und Dateien suchen.

Darüber hinaus können Sie mithilfe spezifischer Suchkriterien wie Filter und Bedingungen komplexere Suchvorgänge durchführen. Die Suchabfragen können gespeichert und später erneut ausgeführt werden.

Objekt schnell suchen

  1. Wählen Sie den zu durchsuchenden Ordner aus oder wählen Sie im Dropdown-Menü "Suchen" aus, welchen Teil des Accounts Sie durchsuchen möchten.

  2. Geben Sie im Textfeld Suchen den Suchtext ein. Informationen zu den Suchoptionen finden Sie unter Suchsyntax.

  3. Drücken Sie die Eingabetaste. Die Registerkarte "Suchen" wird geöffnet und die Suchergebnisse werden angezeigt.

Auf der Seite "Suchen" eine Suche durchführen

  1. Klicken Sie im Feld "Suchen" auf die Lupe. Die Seite "Suchen" wird angezeigt.

  2. Wählen Sie im Navigationsfenster den Dropdown-Link zum Erstellen der Suche aus. Im Suchfeld oben auf der Seite wird in einer Blase die Suchsyntax angezeigt.
