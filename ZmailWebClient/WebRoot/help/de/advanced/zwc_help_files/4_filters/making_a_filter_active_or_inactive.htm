Filter aktivieren oder deaktivieren

Sie können Filter aktivieren oder deaktivieren. Wenn ein Filter deaktiviert wurde, wird er ignoriert. Wenn Sie einen Filter deaktivieren und nicht löschen, können Sie ihn später wieder verwenden.

  1. Öffnen Sie die Seite Einstellungen > Filter.

  2. Klicken Sie auf Filter bearbeiten.

  3. Deaktivieren Sie das Kontrollkästchen Aktiv in der rechten oberen Ecke, um den Filter auszuschalten.

  4. Klicken Sie auf OK.
