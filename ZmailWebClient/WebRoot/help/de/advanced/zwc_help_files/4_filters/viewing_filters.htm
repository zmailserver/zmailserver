Filter anzeigen

Vorhandene Filter anzeigen

Sie können die von Ihnen definierten Filter und ihren Status anzeigen.

  1. Öffnen Sie die Seite Einstellungen > Filter.

  2.    Wenn Sie Informationen zu einem bestimmten Filter anzeigen möchten, doppelklicken Sie darauf.

Filterreihenfolge anzeigen

Aktive Filter werden in der Reihenfolge angewendet, in der sie aufgeführt werden. Diese Reihenfolge können Sie ändern.

  1. Öffnen Sie die Seite Einstellungen > Filter.

  2. Ändern Sie die Reihenfolge mit den Schaltflächen "Nach oben" und "Nach unten".

Hinweis:   Eine Mail wird mit jedem aktiven Filter überprüft, es sei denn, Sie geben an, dass eine Mail nur mit dem ersten Filter mit Übereinstimmung behandelt wird. Normalerweise sollten keine weiteren Filter auf eine Mail angewendet werden, sobald eine Übereinstimmung vorliegt, da sonst für eine Mail mehrere möglicherweise widersprüchliche Aktionen ausgeführt werden.
