Mit Mailfiltern arbeiten

Sie können Filter zum Verwalten eingehender und ausgehender Mails definieren, u. a. auch für kalenderbezogene Mails, und mit dem Filter "Aktivitätenstream" weniger wichtige Mails verwalten. Ein Filter besteht aus einer oder mehreren Bedingungen und einer oder mehreren Aktionen. Eine Bedingung kann sich auf den Betreff, einen bestimmten Absender oder Mails eines bestimmten Datums beziehen. Wenn die Mail die Bedingungen erfüllt, werden die angegebenen Aktionen ausgeführt.

Sie können Filter für die folgenden Aktionen erstellen:

•      Eingehende Mails in Ordner sortieren. Es können z. B. alle Mails von Ihrem Vorgesetzten nach Eingang automatisch in den Ordner "Vorgaben vom Chef" verschoben werden.

•      Mails mit Tags versehen. Beispielsweise können Mails von der Mailing-Liste für Firmenveranstaltungen mit dem Tag "Veranstaltungen" markiert werden.

•      Mails weiterleiten oder verwerfen. Sie können z. B. alle Mails von einer bestimmten Mail-Adresse in den Papierkorb verschieben.

•      Gesendete Mails in bestimmte themenbezogene Ordner sortieren

•      Mails nach Wichtigkeit markieren

Filter werden in der Reihenfolge angewendet, in der sie aufgeführt werden. Sie können die Reihenfolge jederzeit ändern.

Hinweis:   Eine Mail wird mit jedem aktiven Filter überprüft, es sei denn, Sie geben an, dass eine Mail nur mit dem ersten Filter mit Übereinstimmung behandelt wird. Normalerweise sollten keine weiteren Filter auf eine Mail angewendet werden, sobald eine Übereinstimmung vorliegt, da sonst für eine Mail mehrere möglicherweise widersprüchliche Aktionen ausgeführt werden.

Filterregeln einrichten

Filterregeln bestehen aus einer oder mehreren Bedingungen und einer oder mehreren Aktionen. Wenn die Mail die Bedingungen erfüllt, werden die angegebenen Aktionen ausgeführt.

Sie können auch Regeln für "Aktivitätenstream" einrichten. Mit Regeln für "Aktivitätenstream" werden weniger wichtige Mails aus Ihrem Posteingang in den Ordner "Aktivitätenstream" verschoben, den Sie bei Bedarf überprüfen können.
