Filter erstellen

Sie können Filter definieren, um eingehende und ausgehende Mails zu verwalten. Ein Filter besteht aus einer oder mehreren Bedingungen und einer oder mehreren Aktionen. Wenn die Mail die Bedingungen erfüllt, werden die angegebenen Aktionen ausgeführt.

  1. Öffnen Sie die Seite Einstellungen > Filter.

  2. Klicken Sie auf Filter für eingehende Mails oder auf Filter für ausgehende Mails.

  3. Wählen Sie Neuer Filter aus.

  4. Geben Sie im Textfeld Filtername einen eindeutigen Namen für den Filter ein.

  5. Um den Filter zu aktivieren, aktivieren Sie das Feld Aktiv oben rechts.

  6. Wählen Sie aus, ob die Mail eine oder alle der aufgeführten Bedingungen erfüllen muss.

    7. •    eine bedeutet, dass die Aktion ausgeführt wird, wenn die Mail irgendeine der Bedingungen erfüllt.

    •    alle bedeutet, dass die Mail alle Bedingungen erfüllen muss, damit die Aktion ausgeführt wird.

  7. Wählen Sie im Dropdown-Menü aus, auf welchen Teil der Mail der Filter angewendet werden soll, z. B. auf das Feld Betreff oder Von.

  8. Wählen Sie die gewünschte Art der Übereinstimmung aus, z. B. enthält oder entspricht genau.

  9. Geben Sie die zu überprüfenden Informationen ein oder wählen Sie sie aus, z. B. Datum, Mail-Adresse oder Ausdruck. Bei den Bedingungen wird die Groß- und Kleinschreibung nicht beachtet.

    10. Wenn Sie z. B. Betreff und enthält ausgewählt und Mitarbeiterbesprechung eingegeben haben, werden alle Mails gefiltert, deren Betreff "Mitarbeiterbesprechung" enthält. Mit dem Filter werden auch "Mitarbeiterbesprechungen" und "Heutige Mitarbeiterbesprechung" gefunden, da Sie enthält und nicht entspricht genau ausgewählt haben.

  10. (Optional) Wenn Sie weitere Bedingungen hinzufügen möchten, klicken Sie auf die Schaltfläche + und wiederholen Sie die oben genannten Schritte. Die Reihenfolge der Bedingungen spielt keine Rolle.

  11. Wählen Sie unter Folgende Aktionen ausführen im Dropdown-Menü eine Aktion aus.

    12. Je nach der von Ihnen ausgewählten Aktion müssen Sie eventuell zu einem Ordner- oder Tagnamen navigieren.

  12.    (Optional) Um weitere Aktionen hinzuzufügen, klicken Sie auf die Schaltfläche + und wiederholen Sie den obigen Schritt. Die Aktionen werden in der Reihenfolge ausgeführt, in der sie angegeben wurden.

  13. Standardmäßig wird nach der ersten Übereinstimmung nicht nach weiteren Übereinstimmungen gesucht. Wenn mehrere Filter auf die Mail angewendet werden sollen, deaktivieren Sie Keine weiteren Filter anwenden.

    14. Hinweis:   Normalerweise sollten keine weiteren Filter auf eine Mail angewendet werden, sobald eine Übereinstimmung vorliegt, da sonst für eine Mail mehrere möglicherweise widersprüchliche Aktionen ausgeführt werden.

  14. Wenn Sie mit dem Erstellen des Filters fertig sind, klicken Sie auf OK.
