Termin oder Besprechung ändern

Die Angaben zu einer Besprechung oder zu einem Termin können geändert werden. Wenn Sie eine wiederkehrende Besprechung erstellt haben, können Sie einzelne Termine oder die gesamte Besprechungsserie anpassen.

Außerdem können Sie die Teilnehmer der Besprechung bei Bedarf per Mail über etwaige Änderungen informieren.

  1. Doppelklicken Sie auf die Besprechung, die Sie ändern möchten.

    2. Wenn es sich um eine wiederkehrende Besprechung handelt, wird das Dialogfeld Wiederkehrendes Objekt öffnen geöffnet.
    Sie können einzelne Instanzen einer wiederkehrenden Besprechung oder die gesamte Besprechungsreihe anpassen.

  2. Klicken Sie auf OK, um die Seite Termin zu öffnen.

  3. Ändern Sie die gewünschten Informationen.

  4. Standardmäßig werden alle Teilnehmer gebeten, die empfangene Einladung zu beantworten. Wenn Sie dies nicht möchten, klicken Sie in der Symbolleiste "Kalender" auf Optionen > Antworten auf Anfragen, um das Kontrollkästchen zu deaktivieren.

  5. Klicken Sie auf Senden, um die überarbeiteten Besprechungsdetails an die Teilnehmer zu senden.
