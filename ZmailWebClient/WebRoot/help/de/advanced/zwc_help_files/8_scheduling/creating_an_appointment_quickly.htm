Schnell einen Termin erstellen

Mit der Option "Schnelleingabe" können Sie schnell und einfach einen Termin für sich erstellen.

Hinweis:   Die Funktion "Schnelleingabe" ist optional. Wenn Sie möchten, dass beim Doppelklicken oder Ziehen im Kalender immer die Terminseite geöffnet wird, deaktivieren Sie diese Funktion unter "Einstellungen" > "Kalender".

  1. Klicken Sie mit der rechten Maustaste auf den Mini-Kalender oder auf die Startzeit in einer Kalenderansicht. Wählen Sie Neuer Termin oder Neuer ganztägiger Termin aus.

  2. Geben Sie im Textfeld Betreff eine kurze Beschreibung der Besprechung ein. Diese Informationen werden als Beschreibung im Kalender übernommen.

  3. Geben Sie im Textfeld Ort an, wo die Besprechung stattfinden soll. Passende Orte, die im Unternehmensverzeichnis enthalten sind, werden bei der Eingabe automatisch vorgeschlagen. Sie können auch auf Ort vorschlagen klicken, um eine Liste der verfügbaren Orte in Ihrem Unternehmen anzuzeigen.

  4. Konfigurieren Sie im Dropdown-Menü Anzeige die Frei/Belegt-Option für diesen Termin.

  5. Wählen Sie den Kalender aus, dem der Termin hinzugefügt werden soll, wenn Sie nicht den Standardkalender verwenden möchten.

  6. Geben Sie im Dropdown-Menü Markieren als an, ob es sich um einem privaten oder um einen öffentlichen Termin handelt. Bei privaten Terminen können andere Personen in freigegebenen Kalendern keine näheren Informationen zu dem Termin anzeigen. Angezeigt wird nur die Start- und Endzeit, die Dauer, ob es sich um einen wiederkehrenden Termin handelt sowie die Identität des Organisators.

  7. Legen Sie den Tag und die Uhrzeit fest, an denen der Termin starten bzw. enden soll. Bei wiederkehrenden Terminen können Sie unter Wiederholung angeben, wie oft ein Termin wiederholt werden soll.

  8. (Optional) Wenn Sie möchten, können Sie weitere Teilnehmer hinzufügen, eine Ressource planen oder Notizen zu einem geplanten Termin hinzufügen. Klicken Sie dazu auf Mehr Details.

  9. Klicken Sie auf OK.
