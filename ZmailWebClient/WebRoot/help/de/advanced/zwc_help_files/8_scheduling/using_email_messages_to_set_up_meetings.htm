Besprechungen mit Mails einrichten

Sie können rasch eine Besprechungsanfrage erstellen, indem Sie eine Mail oder eine Unterhaltung auf ein Datum im Mini-Kalender ziehen. Wenn Sie eine Mail oder Unterhaltung auf ein Datum ziehen, werden die Informationen in der Mail verwendet, um die Felder auf der Terminseite auszufüllen.

  1. Klicken Sie auf eine bestimmte Mail oder Unterhaltung und ziehen Sie sie auf ein Datum im Mini-Kalender.

  2. Das Dialogfeld "Teilnehmer hinzufügen" wird geöffnet. Die Terminseite wird geöffnet, und viele Felder sind bereits mit Informationen aus der Mail gefüllt.

    3. •    Der Betreff des Termins entspricht dem Betreff der Mail.

    •    Die Teilnehmer ergeben sich aus den Mailadressen in den Feldern "An" und "CC" der Mail bzw. den neuesten Mail der Unterhaltung. Wenn Sie einen Kontakt aus der Kontaktliste auf den Mini-Kalender ziehen, wird die erste Mailadresse des Kontakts dem Teilnehmerfeld hinzugefügt.

    •    Der Text der Nachricht wird als Text für die Einladung übernommen. Anhänge einer Mail werden nicht an die Terminanfrage angehängt.

  3. Überprüfen Sie die Details der Besprechungsanfrage auf der Terminseite, und nehmen Sie eventuell erforderliche Änderungen vor.

  4. Stellen Sie die Uhrzeit für die Besprechung ein. Handelt es sich um eine wiederkehrende Besprechung, richten Sie die Wiederholungen ein.

  5. Klicken Sie auf Senden.
