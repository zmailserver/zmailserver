Erinnerungen anzeigen

Sie können Erinnerungen für Besprechungen einrichten. Standardmäßig werden Sie fünf Minuten vorher erinnert. Für die Erinnerung können Sie auch Sound abspielen auswählen. Wenn Sie die Lautstärke stummgeschaltet haben, hören Sie keinen Signalton. Wählen Sie Browser-Titel blinkend darstellen aus, wenn Sie auf einer anderen Registerkarte des Browsers oder in einem anderen Browserfenster arbeiten und der Browser-Titel zur Erinnerung blinken soll.

  1. Öffnen Sie die Seite Einstellungen > Kalender.

  2. Unter Allgemein > Erinnerungen anzeigen können Sie angeben, wie viele Minuten vor einer Besprechung die Erinnerung gesendet werden soll. Dies ist die Standardeinstellung. Sie kann bei Bedarf für einzelne Besprechungen angepasst werden.

  3. Geben Sie an, wie Sie über eine bevorstehende Besprechung informiert werden möchten. Wählen Sie aus den folgenden Optionen aus.

Option

Beschreibung

Sound abspielen

Sie werden mit einem Signalton benachrichtigt. Wenn Sie die Lautstärke stummgeschaltet haben, hören Sie keinen Signalton. Hierfür muss QuickTime oder Windows MediaCenter installiert sein.

Mail-Register hervorheben

Die Registerkarte "Mail" wird hervorgehoben, wenn Sie zum Zeitpunkt des Maileingangs nicht auf der Registerkarte "Mail" arbeiten.

Popup-Benachrichtigung anzeigen

Es werden Informationen zu der Mail angezeigt. Hierfür muss Yahoo! BrowserPlus installiert sein.

Der Download ist erhältlich unter
