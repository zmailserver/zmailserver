Mails verschieben

Sie können Mails oder ganze Unterhaltungen aus einem Ordner verschieben.

•      Ziehen Sie das Objekt mit der Maus auf den gewünschten Ordner im Übersichtsfenster auf der linken Seite.

•      Klicken Sie mit der rechten Maustaste auf die Mail und wählen Sie Verschieben aus. Daraufhin wird das Dialogfeld Mail verschieben angezeigt.

Wenn Sie einige Mails einer Unterhaltung verschieben, erscheint die Unterhaltung an beiden Stellen. In der Detailansicht der Unterhaltung wird der neue Speicherort der verschobenen Mail angezeigt.

Hinweis:   Sie können das Verschieben sofort rückgängig machen. Klicken Sie auf die oben auf der Seite angezeigte Option Rückgängig, um die Mail wieder an den ursprünglichen Speicherort zu verschieben.
