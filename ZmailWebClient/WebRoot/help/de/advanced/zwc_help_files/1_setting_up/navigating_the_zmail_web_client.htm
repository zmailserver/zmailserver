Bei der Anmeldung wird der Bildschirm ähnlich wie der folgende angezeigt. Bei dieser Ansicht handelt es sich um die Mailansicht. Die Ansicht wird geändert, wenn Sie eine andere Anwendung verwenden, z. B. Adressbuch oder Kalender.

Option

Beschreibung

Kopfzeile des Fensters

In der Kopfzeile des Fensters wird Folgendes angezeigt:

•Feld "Suchen"

•Nutzername, unter dem Sie angemeldet sind. Über das Dropdown-Menü Ihres Nutzernamens können Sie auf Produkthilfe, neue Funktionen, Informationen und die Funktionen zum Ändern des Passworts und Abmelden zugreifen.

Registerkarten der Anwendungen

Die Anwendungen, die Sie aufrufen können, sind auf den Registerkarten dieser Zeile aufgeführt, z. B. die Anwendungen Mail und Kalender.

Übersichtsfenster

Im Übersichtsfenster wird Folgendes angezeigt:

•Ordner, darunter die Systemordner ("Posteingang", "Papierkorb", "Entwurf",

•"Spam" und "Papierkorb") sowie von Ihnen erstellte Nutzerordner

•Suchvorgänge: Suchabfragen, die Sie erstellt und zur weiteren Verwendung gespeichert haben

•Tags: alle von Ihnen erstellten Tags. Klicken Sie auf einen Tag, um alle Mails mit diesem Tag anzuzeigen.

•Zimlets: Links, die für die Integration von Fremdanwendungen in Ihre Mailbox erstellt wurden.

•In der Kalenderanwendung zeigt das Übersichtsfenster Ihre Kalenderliste und Zimlets an.

Mini-Kalender

DerMini-Kalender ist optional. Sie können den Mini-Kalender auf der Seite "Einstellungen > Kalender" deaktivieren.

Symbolleiste

Auf der Symbolleiste werden Befehle für die aktuelle Anwendung angezeigt.

Inhaltsfenster

Der Inhalt in diesem Bereich richtet sich nach der jeweils aufgerufenen Anwendung. Im Posteingang zeigt das Inhaltsfenster alle Mails und Unterhaltungen Ihres Posteingangs an.

Lesefenster

Im Lesefenster werden Mails angezeigt, wenn Sie diese Funktion aktiviert haben. Sie können diese Funktion über das Dropdown-Menü "Ansicht" in der Mail-Symbolleiste aktivieren.

Suchleiste

In diesem Bereich werden Suchoptionen angezeigt. Sie können die Suchabfragen auch von hier aus speichern.

Hilfe aufrufen

Wenn Sie Ihren Mauszeiger über Elemente wie Befehlsschaltflächen, Unterhaltungen oder Mail-Adressen in Mailkopfzeilen bewegen, wird ein kleines Textfeld mit einer Beschreibung zu dem Element angezeigt.

Ordnerinformationen

Wenn Sie die Maus auf einem Ordner positionieren, werden die Anzahl der Mails in dem Ordner und die Ordnergröße angezeigt.
