Tastenkürzel verwenden

Für viele Funktionen und Befehle gibt es Tastenkürzel.

Tastenkürzel anzeigen

• Wenn Sie beim Arbeiten mit Ihrer Mailbox eine Liste der Tastenkürzel anzeigen möchten, drücken Sie Strg+Q.

Wenn das Fenster mit den Tastenkürzeln geöffnet bleiben soll, während Sie mit Ihrem Account arbeiten, klicken Sie auf Neues Fenster.

Tastenkürzel drucken

• Öffnen Sie die Seite Einstellungen > Tastenkürzel, um eine Liste der Tastenkürzel anzuzeigen. Klicken Sie rechts oben auf Drucken, um die Liste zu drucken.
