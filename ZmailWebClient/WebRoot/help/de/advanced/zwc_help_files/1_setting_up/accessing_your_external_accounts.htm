Auf externe Accounts zugreifen

Statt sich bei jedem Ihrer externen Mail-Accounts anzumelden, können Sie die Mails dieser Accounts direkt im @ClientNameShort@-Account abrufen. Wenn Sie den Zugriff auf andere externe Mail-Accounts einrichten, können Sie angeben, wo die Mails gespeichert werden sollen. Sie können außerdem eine Persönlichkeit anlegen, die verwendet wird, wenn Sie Mails des externen Accounts beantworten oder weiterleiten.

Mails Ihres externen Accounts werden nicht automatisch in Ihrem @ClientNameShort@-Account empfangen. Klicken Sie auf das Symbol Aktualisieren auf der Symbolleiste, um die aktuellen Mails anzuzeigen.

Wenn Sie einen externen POP-Account hinzufügen, können Sie das POP-Zugriffsverhalten beim Herunterladen von Mails in Ihren Account konfigurieren.

Externe Accounts hinzufügen

  1. Öffnen Sie die Seite Einstellungen > Accounts.

  2. Klicken Sie auf Externen Account hinzufügen. Der Accountname Neuer externer Account wird in der Spalte "Accountname" und im Textfeld unter Einstellungen für externen Account > Accountname angezeigt. Sie können dies ändern.

  3. Geben Sie im Abschnitt Einstellungen für externen Account die Mail-Adresse des externen Accounts ein.

  4. Geben Sie im Textfeld Accountname ein Wort ein, das diesen Account beschreibt. Hiermit wird der externe Account identifiziert.

  5. Wählen Sie den Account-Typ aus: POP oder IMAP.

  6. Geben Sie im Textfeld Nutzername des Accounts den Namen für diesen Mail-Account ein. In manchen Fällen ist der Nutzername der Teil der Mail-Adresse vor dem Zeichen @, in anderen Fällen ist der Nutzername die vollständige Mail-Adresse.

  7. Geben Sie im Textfeld Mail-Server den Namen für den Server des externen Accounts ein.

  8. Geben Sie im Textfeld Passwort das Passwort ein, mit dem Sie sich bei diesem externen Account anmelden. Um zu überprüfen, ob das eingegebene Passwort korrekt ist, wählen Sie Passwort anzeigen aus.

  9. Wenn Ihr Mail-Anbieter einen anderen Port als den Standardport nutzt, passen Sie ihn im Bereich Erweiterte Einstellungen an. Wenn Sie eine sichere Verbindung nutzen, aktivieren Sie Für den Zugriff auf diesen Server eine verschlüsselte Verbindung (SSL) nutzen.

    10. Ihr Provider kann Ihnen mitteilen, falls er nicht den Standardport für IMAP (143) oder für POP (110) nutzt.

  10. Klicken Sie auf Einstellungen testen. Die Informationen werden überprüft.

  11. Wählen Sie den Ordner aus, in den die Mails heruntergeladen werden sollen.

  12. Wenn Mails aus dem externen Account gelöscht werden sollen, nachdem sie dort empfangen wurden, aktivieren Sie Mails nach dem Herunterladen auf dem Server löschen.

  13. Konfigurieren Sie die Einstellungen für den Namen im Feld "Von", die Antwortadresse und Signaturen für die von Ihnen gesendeten Mails.

  • Geben Sie den Namen ein, der in Ihren ausgehenden Mails im Feld Von angezeigt werden soll (z. B. Peter Meier). Dies ist der Name, der vor Ihrer Mail-Adresse angezeigt wird.

  • Um Antworten auf Mails von dieser Persönlichkeit an einen anderen Namen und eine andere Adresse als unter Von angegeben weiterzuleiten, wählen Sie Als Antwortadresse von Mails einstellen aus. Geben Sie den Namen im Textfeld ein.

  • Wenn Sie der Persönlichkeit eine Signatur zuordnen möchten, klicken Sie auf Signatur und fügen Sie eine neue Signatur hinzu.

  1. Klicken Sie auf Speichern.
