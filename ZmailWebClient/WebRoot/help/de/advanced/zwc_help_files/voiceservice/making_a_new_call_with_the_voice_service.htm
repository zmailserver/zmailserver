Neuen Anruf mit dem Sprachnachrichten-Dienst durchführen

Hinweis: Diese Funktion ist nur für ZCS Network Edition vorgesehen und nur im erweiterten Webclient verfügbar.

Sie können von allen registrierten Telefonen, einschließlich Softphone-Clients wie Cisco oder Mitel, Telefonanrufe tätigen.

  1. Klicken Sie auf der Seite "Sprachnachrichten" auf Neuer Anruf. Das Dialogfeld "Neuer Anruf" wird angezeigt.

  2. Wählen Sie im Dropdown-Menü unter Von eine Nummer aus. Die aufgeführten Nummern sind die Nummern, die für Sie registriert sind.

  3. Geben Sie in An die Nummer ein, die Sie anrufen möchten, oder wählen Sie sie aus. Die Nummer kann aus Ihren Adressbüchern oder Kontaktkarten übernommen werden oder Sie können sie eingeben.

  4. Klicken Sie auf "OK", um den Anruf durchzuführen. Verwenden Sie das Telefon, das der Telefonnummer zugeordnet ist, die Sie für den Anruf verwenden.

Anruf beenden

Verwenden Sie zum Beenden eines Anrufs die Funktion oder das Dialogfeld zum Auflegen.
