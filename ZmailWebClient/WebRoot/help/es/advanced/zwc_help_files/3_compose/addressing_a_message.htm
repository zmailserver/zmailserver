Añadir una dirección a un mensaje

La función de autocompletado sugiere nombres a medida que escribes en los campos Para, CC y CCO en un nuevo mensaje. Se mostrará una lista de las posibles direcciones de tus contactos en la que aparecerá el nombre, el apellido o la dirección de correo que comiencen con el texto que has escrito.

A medida que continúes escribiendo, la lista de direcciones que coinciden se reducirá, ya que habrá menos coincidencias disponibles. La lista desaparecerá si no hay ninguna coincidencia.

Por ejemplo, para enviar un mensaje de correo a Miguel Leugim, cuya dirección de correo es mleugim@acme.com, al escribir la letra m, aparecerá una lista de todas las posibles direcciones que coinciden, y la parte que coincide exactamente aparecerá resaltada. En este caso, la lista podría incluir:

•      Mela Olano <molano@acme.com>

•      Miguel Leugim <mleugim@acme.com>

•      Érica Moreno

Al añadir una “i” (mi) en el campo Para, la lista mostraría:

•      Miguel Leugim <mleugim@acme.com>

Uso del autocompletado de direcciones

Para seleccionar una coincidencia de la lista, puedes hacer clic en la coincidencia con el ratón o utilizar una de las teclas siguientes para realizar una acción:

•      Las teclas de coma, punto y coma, Intro y Tab completan la primera coincidencia de la lista y la colocan en el campo.

•      La tecla ESC ocultará la lista.

•      Las flechas arriba y abajo cambiarán la coincidencia resaltada. Mover el cursor del ratón sobre la lista también cambiará la coincidencia resaltada.

Para seleccionar una dirección al redactar un correo

  1. Escribe caracteres suficientes hasta que el contacto que deseas aparezca como la primera coincidencia.

  2. Pulsa una tecla rápida, como Intro o punto y coma. La dirección aparece en el campo de destinatarios.
