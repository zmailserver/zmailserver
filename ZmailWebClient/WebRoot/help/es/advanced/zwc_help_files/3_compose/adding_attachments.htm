Añadir archivos adjuntos 

Puedes adjuntar cualquier archivo que encuentres mediante Examinar en tu ordenador.

Puedes abrir cualquier archivo adjunto directamente desde el buzón de correo, siempre y cuando tengas la aplicación correcta y el administrador no haya bloqueado la extensión de ese tipo de archivo.

Si tienes instalado el software para ese tipo de archivo, basta con hacer doble clic en el archivo para que el ordenador inicie automáticamente el programa adecuado para poder leer ese archivo.

Los virus informáticos a menudo se pueden transmitir a través de archivos adjuntos. Por lo tanto, los administradores del sistema pueden bloquear el correo entrante que contenga ciertos tipos de archivos adjuntos, a menudo con las extensiones .EXE o .ZIP como parte del nombre de archivo. Si envías un mensaje de correo a un destinatario cuyo sistema se ha configurado para bloquear determinados tipos de archivos adjuntos, es posible que no recibas una notificación si el mensaje se ha bloqueado.

Añadir un archivo adjunto a un mensaje

Puedes adjuntar documentos, hojas de cálculo, fotos, presentaciones y otros tipos de archivos a tu mensaje de correo.

  1. Redacta el mensaje del correo.

  2. Debajo del campo Asunto, haz clic en Adjuntar y selecciona Mi PC.

  3. Selecciona los archivos y haz clic en Abrir. Los nombres de los archivos aparecen debajo del cuadro de texto Asunto.

  4. Haz clic en Enviar para enviar el mensaje y los archivos adjuntos.

Nota:   los destinatarios deberán tener el software apropiado para abrir el archivo. También se puede usar una variedad de programas para abrir otros formatos comunes, como los archivos de texto, HTML e imágenes como GIF o JPEG.

Añadir archivos adjuntos con la función de arrastrar y soltar

Puedes añadir fácilmente un archivo adjunto a un mensaje de correo si arrastras el archivo desde una carpeta hasta tu correo.

  1. Selecciona uno o más archivos y mantén pulsado el cursor en el o los archivos que quieres adjuntar.

  2. Arrastra los archivos hasta el encabezado del mensaje y suelta el cursor. Verás los nombres de los archivos en el encabezado.

Eliminar un archivo adjunto

Para quitar un archivo adjunto, haz clic en la x en el globo de archivos adjuntos.
