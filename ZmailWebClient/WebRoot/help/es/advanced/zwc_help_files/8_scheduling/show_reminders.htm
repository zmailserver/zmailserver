Mostrar recordatorios

Puedes configurar recordatorios para reuniones. Por defecto, se envía cinco minutos antes. También puedes elegir Reproducir un sonido con el recordatorio. Si has silenciado el volumen del ordenador, no escucharás esta alerta. Selecciona Título del navegador intermitente si estás trabajando en una pestaña o ventana distinta del navegador y elige que el título del navegador parpadee para hacerte acordar.

  1. Ve a la página Preferencias>Agenda.

  2. En la sección General>Mostrar recordatorios, define cuántos minutos antes de las reuniones deseas recibir un recordatorio. Este es el valor predeterminado y puedes cambiarlo para reuniones individuales.

  3. Selecciona cómo quieres que se te notifiquen las próximas reuniones. Selecciona cualquiera de las opciones siguientes.

Opción

Descripción

Reproducir un sonido

Se te notificará mediante un pitido. Si has silenciado el volumen, no escucharás esta alerta. Para que esta opción funcione correctamente, debes tener instalado QuickTime o Windows MediaCenter.

Resaltar la pestaña Correo

La pestaña Correo se resaltará si no estás trabajando en dicha pestaña en ese momento.

Mostrar una notificación emergente

Se muestra información sobre el mensaje. Debes tener instalado Yahoo!BrowserPlus.
