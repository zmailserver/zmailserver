Usar las versiones avanzada y estándar del cliente

Hay dos versiones disponibles de @ClientNameLong@.

La versión Avanzada ofrece todas las funciones para la colaboración en la web. Funciona con la mayoría de los navegadores actuales y conexiones a Internet más rápidas.

Si tienes una conexión a Internet lenta o prefieres un programa basado en HTML, puedes utilizar la versión Estándar, que tiene muchas de las mismas funciones que la versión avanzada pero permite acceder de manera diferente.

Herramientas del navegador

•El botón Atrásdel navegador te llevará a la página que veías anteriormente. También puedes usar el botón Adelante.

•No utilice el botón Recargar(Actualizar) del navegador. Si lo haces, se reiniciará la sesión.

Cambiar de versión de cliente

Puedes configurar la versión del cliente predeterminada para iniciar sesión. Si alternas con frecuencia entre un cliente y otro, puedes cambiarlas.

  1. Ve a la página Preferencias>General.

  2. En la sección Opciones de inicio de sesión>Iniciar sesión con, selecciona cómo deseas iniciar sesión.

    3. •Avanzado. Ofrece todas las funciones para la colaboración en la web. El programa funciona mejor con los navegadores más recientes y las conexiones a Internet más rápidas.

    •Estándar. Se recomienda si la conexión a Internet es lenta, si utilizas un navegador antiguo o para que el acceso sea más fácil.

  3. Haz clic en Guardar.

La próxima vez que inicies sesión, tu selección aparecerá como la predeterminada.
