Configurar alertas de escritorio para los mensajes nuevos

Puedes configurar alertas de escritorio para recibir notificaciones de correo nuevo. Para recibir estas alertas, tu buzón de correo debe estar abierto.

  1. Ve a la página Preferencias>Correo.

  2. En la sección Recibir mensajes>Al recibir un mensaje, selecciona como deseas que se te avise cuando llega un mensaje nuevo. Selecciona cualquiera de las opciones siguientes.

Opción

Descripción

Reproducir un sonido

Se te notificará mediante un pitido. Si has silenciado el volumen, no escucharás esta alerta. Para este trabajo, deberás tener instalado QuickTime o Windows MediaCenter.

Resaltar la pestaña Correo

La pestaña Correo se resaltará si no estás trabajando en dicha pestaña en ese momento.

Título del navegador intermitente

El navegador parpadeará hasta que abras la pestaña Correo.

Mostrar una notificación emergente

Se muestra información sobre el mensaje. Debes tener instalado Yahoo!BrowserPlus. Para obtener una copia, visita http://browserplus.yahoo.com/
