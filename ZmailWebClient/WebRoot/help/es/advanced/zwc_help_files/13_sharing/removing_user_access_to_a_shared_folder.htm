Eliminar el acceso de un usuario a una carpeta compartida

  1. Haz clic con el botón derecho en la carpeta compartida y selecciona Editar propiedades. El cuadro de diálogo Propiedades de la carpeta muestra los nombres que comparten esa carpeta.

  2. Haz clic en la opción Revocar situada junto al nombre del usuario que se está eliminando.

  3. En el cuadro de diálogo Revocar compartir, selecciona si deseas enviar un mensaje estándar o añadir una nota al mensaje.

  4. Haz clic en Sí.

El usuario ya no podrá acceder a tu carpeta.
