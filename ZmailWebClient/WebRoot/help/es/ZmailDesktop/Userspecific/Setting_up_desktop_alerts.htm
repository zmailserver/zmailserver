Configurar las alertas y los recordatorios de Desktop

Puedes crear alertas para recibir notificaciones de mensajes nuevos y recordatorios de citas. Para recibir estas alertas, tu buzón de correo debe estar abierto.

  • Alertas de mensajes. Puedes establecer tus preferencias en la página Preferencias>Correo. A continuación se enumeran las opciones de aviso que se muestran cuando se recibe un mensaje:

    • Reproducir un sonido. Escucharás un pitido cuando recibas un mensaje nuevo en tu bandeja de entrada.  Si has silenciado el volumen, no escucharás esta alerta. Se requiere tener QuickTime o Windows Media instalado en el ordenador.

    • Resaltar la pestaña Correo. La pestaña  correo se resalta cuando se recibe correo electrónico nuevo en la Bandeja de entrada si no estás trabajando en la pestaña Correo en ese momento.

    • Mostrar una notificación emergente.  Al recibir un mensaje nuevo, se abrirá una ventana emergente. Debes disponer de Yahoo! BrowserPlus instalado.

  • Mostrar avisos de recordatorio en las Agendas. Establece tus preferencias en la página Preferencias>Agenda.  Para poder configurar una alerta, debes indicar cuándo notificarte la reunión.  Aparecerá un cuadro de diálogo de recordatorio en el momento designado antes de la reunión. Si configuras esta opción como Nunca, no recibirás ningún recordatorio incluso si has habilitado la opción de recordatorios.

El recordatorio no se abrirá en la parte de arriba de la página si no estás en la Agenda. Si activas una de las siguientes opciones, será más difícil que pases por alto el recordatorio emergente:

    • Reproducir un sonido. Se te notificará mediante un pitido al abrirse un recordatorio de una cita. Si has silenciado el volumen, no escucharás esta alerta.

    • Título del navegador intermitente. Al abrirse un recordatorio, parpadeará el navegador hasta que cierres la ventana emergente.

    Además de alertas en tu ordenador, puedes configurar notificaciones mediante correo electrónico o SMS desde dentro de la cita.  Configura la dirección de correo electrónico y el número de SMS en la página Preferencias>Notificación.
