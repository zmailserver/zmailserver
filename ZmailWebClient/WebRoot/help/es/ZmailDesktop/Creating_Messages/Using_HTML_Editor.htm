Utilizar el editor de HTML

Puedes redactar tus mensajes utilizando HTML.  Los mensajes con HTML te permiten añadir fuentes, estilos, listas numeradas, listas con viñetas, fondos de color, tablas y enlaces para que tus mensajes sean más fáciles de leer.

Nota: Algunos programas de correo no permiten recibir mensajes con formato HTML.

Para redactar los mensajes con formato HTML de manera predeterminada y configurar la fuente predeterminada, incluido el estilo, tamaño y color, ve a la página Preferencias de redacción.
