Realizar copias de seguridad de las cuentas

Puedes realizar una copia de seguridad de tus cuentas Zimbra Desktop  desde la página Preferencias>Copias de seguridad de la sección Todas las cuentas.  La opción predeterminada es realizar una copia de seguridad de las Carpetas locales únicamente, ya que las Carpetas locales contienen elementos a los que no se les puede realizar una copia de seguridad en otros servidores de correo electrónico.  

Para realizar una copia de seguridad de las cuentas

  1. Ve a la página Preferencias>Copias de seguridad.

  2. En la sección Configurar copias de seguridad, selecciona la cuenta a la que deseas realizar una copia de seguridad. Recuerda que los elementos de las carpetas de la cuenta POP y las Carpetas locales se guardan en el ordenador. Estas cuentas son buenas opciones para la copia de seguridad. Puedes realizar una copia de seguridad de todas las cuentas que has creado en Zimbra Desktop, pero ten en cuenta que los archivos pueden ser bastante grandes. Si deseas saber cuánto espacio en disco necesitas aproximadamente, mantén el cursor sobre el nombre de la cuenta; se mostrará el tamaño del buzón de correo.

  3. Si deseas programar la copia de seguridad, selecciona cuándo deben ejecutarse las copias de seguridad, Cada día, Cada semana, Cada mes.

  4. En el campo Ubicación, especifica la ruta hacia el directorio en el que debería guardarse la copia de seguridad. Por ejemplo, para guardar la carpeta ZD_backups en la unidad externa E, teclea E:\ZD_backups.

  5. En el campo Retención, selecciona el número de copias de seguridad que deseas guardar. La opción predeterminada es guardar únicamente 1 archivo de copia de seguridad. Puedes seleccionar que se guarden hasta 5 copias de seguridad antes de que se sobrescriba la copia de seguridad más antigua.

Si has seleccionado que la copia de seguridad se ejecute automáticamente, la copia de seguridad se ejecutará de acuerdo a tu programa. Zimbra Desktop debe estar abierto para que las copias de seguridad puedan ejecutarse. Si se programa una sesión de copia de seguridad para una hora durante la cual no estás conectado a Zimbra Desktop, la próxima vez que inicies sesión, empezará la copia de seguridad.

Si deseas realizar una copia de seguridad de tus cuentas inmediatamente, o si el programa está ajustado en Nunca, haz clic en Realizar copias de seguridad de cuentas ahora para ejecutar una sesión de copia de seguridad. En función del tamaño de las cuentas a las que se van a realizar copias de seguridad, esta operación puede tardar un poco.  No cierres Zimbra Desktop  durante la ejecución de la copia de seguridad. Puedes continuar trabajando en Zimbra Desktop.

Nota: Si seleccionas realizar una copia de seguridad de las cuentas ahora no se restablece la siguiente hora de copia de seguridad programada.

Para restaurar la cuenta, consulta Restaurar cuentas desde los archivos de copia de seguridad.
