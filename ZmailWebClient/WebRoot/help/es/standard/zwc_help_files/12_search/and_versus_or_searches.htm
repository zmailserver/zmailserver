Búsquedas con las variables AND y OR

Para buscar mensajes que contienen los criterios X e Y, utiliza AND (y) para indicar que el mensaje debe cumplir todos los criterios especificados para que se considere una coincidencia. Para buscar mensajes que coinciden con cualquiera de los criterios especificados, utiliza OR (o) y, si el mensaje contiene el criterio X o Y, se considerará una coincidencia. Consulta

  • Trabajar con un panel Búsqueda básica permite crear una consulta de búsqueda con la variable AND. Si abres varias instancias de paneles de Búsqueda básica, crearás una consulta de búsqueda con la variable OR entre ambos paneles.

  • Seleccionar varios cuadros en los paneles Etiquetas o Estado te permite crear una consulta de búsqueda con la variable AND. Si abres varias instancias de los paneles Etiquetas o Estado y seleccionas diferentes casillas de verificación en cada uno, puedes crear una consulta de búsqueda con la variable OR entre los paneles.

  • En los paneles de búsqueda de Carpetas, Zimlets y Hora, la realización de varias selecciones dentro de un solo panel permite crear una búsqueda con la variable OR para dichos elementos.

Nota: Sólo verás "OR" en la búsqueda. Si seleccionas la opción de mostrar la consulta de búsqueda en la barra de búsqueda, se actualizará el cuadro de texto de búsqueda para mostrar la consulta resultante. Con la variable AND, la palabra AND no aparecerá.
