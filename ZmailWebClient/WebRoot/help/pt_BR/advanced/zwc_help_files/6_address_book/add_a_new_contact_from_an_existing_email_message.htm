Adicionar um novo contato a partir de uma mensagem de e-mail existente

Você pode adicionar endereços de e-mail a um catálogo de endereços diretamente nos campos Enviado por, Para, Cc e Cco.

  1. Na mensagem de e-mail, clique com o botão direito no nome para adicionar e selecione Adicionar aos contatos.

    2. Um novo formulário de contato é aberto pré-preenchido com as informações disponíveis no cabeçalho do e-mail.

  2. Adicione outras informações e verifique a precisão dos campos pré-preenchidos.

  3. Selecione como e onde você deseja salvar o contato.

    4. •    Para Arquivar como, selecione como arquivar o nome. O padrão é por sobrenome, nome.

    •    Para Local, selecione a lista de endereços para salvar o contato.

  4. Clique em  Salvar.
