Alterando o acesso do usuário para uma pasta compartilhada

Você pode alterar os privilégios de compartilhamento atribuídos a um usuário ou grupo.

  1. Clique com o botão direito do mouse na pasta compartilhada a alterar e selecione Editar propriedades. A caixa de diálogo Propriedades da pasta exibe os nomes que compartilham a pasta.

  2. Em Compartilhamento para esta pasta, clique em Editar ao lado do nome a ser alterado.

  3. Na caixa de diálogo Propriedades do compartilhamento, altere os privilégios.

    4. Você pode desativar temporariamente o acesso a uma pasta compartilhada sem revogar os privilégios de compartilhamento selecionando Nenhum.

  4. Selecione se deseja informar o usuário através de e-mail.

  5. Clique em OK.
