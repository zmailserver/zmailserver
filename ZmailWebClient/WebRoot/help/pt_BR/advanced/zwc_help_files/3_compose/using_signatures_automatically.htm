Usando assinaturas automaticamente

Você pode incluir uma assinatura automática no final de um e-mail. Uma assinatura pode incluir o seu nome e qualquer texto adicional. Se você está usando um editor HTML, é possível formatar a sua assinatura, adicionar um link para uma URL e adicionar imagens. 

Você pode criar mais de uma assinatura. Por exemplo, você pode ter uma assinatura formal para e-mails enviados para clientes e uma assinatura informal para email-s enviados para amigos.

Caso crie várias identidades de e-mail (também conhecidas como personagens), poderá criar diferentes assinaturas e atribuí-las a endereços específicos.

  1. Vá até a página Preferências>Assinaturas.

  2. Na caixa de texto Nome, digite um nome descritivo para identificar a assinatura. Você pode criar várias assinaturas, portanto é útil usar um nome identificável aqui.

  3. (Opcional) Para formatar em texto simples, clique em Formatar como HTML e selecione Formatar como texto simples.

  4. Na caixa de texto, digite o texto da assinatura como deseja que seja exibida. Se estiver usando Formatar como HTML, é possível adicionar imagens e criar links no texto da sua assinatura.

  5. Na seção Usando assinaturas, selecione a assinatura padrão a partir do menu suspenso para usar em mensagens enviadas de suas diversas contas.

  6. Selecione a posição da sua assinatura em uma mensagem.

    7. •    Selecione Acima das mensagens incluídas para adicionar a sua assinatura no final da sua respostas e antes das mensagens incluídas.

    •    Selecione Abaixo das mensagens incluídas para adicionar a sua assinatura no final da mensagem.

  7. Clique em Salvar.
