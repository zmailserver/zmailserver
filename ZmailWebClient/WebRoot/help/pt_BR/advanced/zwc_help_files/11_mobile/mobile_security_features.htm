Recursos de segurança móvel

Seu administrador pode ter configurado uma política de segurança móvel que exija que você crie um número de identificação pessoal (PIN) no seu dispositivo móvel. Se você esquecer o seu PIN, entre em contato com o administrador do sistema.

Quando você tiver um PIN, a política poderá estar configurada com um recurso para bloquear usuários caso não digitem o PIN correto.  O administrador do sistema configura a definição de bloqueio do PIN para que, se alguém tentar fazer login no dispositivo com uma senha incorreta além do número de tentativas configurado, uma limpeza do dispositivo seja executada. Isso apaga todos os dados do dispositivo. As definições da política só podem ser configuradas pelo administrador e não podem ser modificadas por usuários.  Solicite mais informações ao administrador do sistema.
