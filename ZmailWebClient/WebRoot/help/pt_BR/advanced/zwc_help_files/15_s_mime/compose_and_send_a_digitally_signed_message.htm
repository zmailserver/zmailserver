Redigir e enviar uma mensagem assinada digitalmente

Observação: Este recurso é exclusivo do ZCS Network Edition e está disponível somente no Cliente Web Avançado.

Mensagens assinadas digitalmente podem ser enviadas para destinatários que enviaram suas chaves e certificados públicos. O certificado do destinatário é armazenado na página de contatos de destinatários na Lista de endereços. 

  1. Clique na barra de ferramentas.

  2. Em Para: insira o endereço do usuário ou clique no link Para: para pesquisar por um endereço das listas de endereços.

  3. Se a sua preferência de segurança não estiver definida por padrão para enviar mensagens assinadas digitalmente, clique em Segurança e selecione Assinar somente.

  4. Insira um assunto para a mensagem.

  5. Redija a mensagem.

  6. Para adicionar um anexo, clique em Adicionar anexoe vá até o arquivo, ou arraste e solte o arquivo do seu computador para o cabeçalho do e-mail.

  7. Clique em Enviar.
