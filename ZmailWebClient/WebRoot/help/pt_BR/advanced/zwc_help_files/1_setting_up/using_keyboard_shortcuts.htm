Usando os atalhos de teclado

Muitas das funções e comandos possuem atalhos de teclado.

Visualizar teclas de atalho

•       Para visualizar uma lista de atalhos enquanto trabalha na sua caixa de entrada, pressione Ctrl+q.

Se você deseja manter a janela de atalhos aberta enquanto trabalha na sua conta, clique em Nova janela.

Imprimir teclas de atalho

•       Vá até Preferências>página Atalhos para obter uma lista das teclas de atalho. Clique em Imprimir no canto superior direito para imprimir a lista.
