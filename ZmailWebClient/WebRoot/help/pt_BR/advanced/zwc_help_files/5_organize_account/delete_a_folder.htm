Excluir uma pasta

Você pode excluir pastas definidas pelo usuário, mas não pastas de sistema. Quando você exclui uma pasta, ela e seu conteúdo são movidos para a Lixeira. Se quiser salvar algumas das mensagens em uma pasta antes de excluí-la, mova as mensagens para outra pasta.

É possível recuperar uma pasta e seu conteúdo da Lixeira, arrastando e soltando-a em outra pasta.

Importante:  Quando você exclui uma pasta, um pop-up é exibido no topo do painel, indicando que a pasta foi movida para a Lixeira. Clique em Desfazer se quiser reverter a exclusão. Esse pop-up é exibido por um tempo muito curto, portanto, clique em Desfazer imediatamente se não quiser excluir a pasta, caso contrário, será preciso recuperar a pasta manualmente.

  1. Vá para E-mail>página Pastas.

  2. Clique com o botão direito do mouse na pasta que deseja excluir e selecione Excluir.

Observação:   A pasta Lixeira é esvaziada regularmente, o que é ajustado pelo administrador, para que os itens na Lixeira não permaneçam lá para sempre.
