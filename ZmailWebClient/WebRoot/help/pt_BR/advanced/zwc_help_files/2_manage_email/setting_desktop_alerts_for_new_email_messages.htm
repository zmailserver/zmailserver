Definindo alertas de desktop para novas mensagens de e-mail

Você pode definir alertas de área de trabalho para notificações de novos e-mails. Para receber alertas, sua caixa de entrada deve estar aberta.

  1. Vá até a página Preferências>E-mail.

  2. Na seção Recebendo mensagens>Entrada de mensagens ,selecione como deseja ser notificado quando novos mensagens chegarem. Selecione qualquer uma dessas opções.

Opção

Descrição

Reproduzir um som

Você é notificado por um som. Se o volume estiver definido como mudo, você não ouvirá nenhum som. Para que isto funcione, você precisa ter o QuickTime ou o Windows MediaCenter instalado.

Realçar a aba E-mail

A guia E-mail é realçada se você não estiver trabalhando nesta guia no momento.

Piscar o título do navegador

O navegador pisca até que você abra a guia E-mail

Exibir uma notificação popup

Informações sobre a mensagem são exibidas. Você deve ter o Yahoo!BrowserPlus instalado. Para obter uma cópia, vá até http://browserplus.yahoo.com/
