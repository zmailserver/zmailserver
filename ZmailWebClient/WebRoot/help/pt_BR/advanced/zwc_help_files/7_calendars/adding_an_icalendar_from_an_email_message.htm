Adicionando um iCalendar a partir de uma mensagem de e-mail

Quando recebe uma mensagem de e-mail contendo um link para um arquivo .ics, você pode adicionar a agenda à sua Lista de agendas no painel Visão geral.

  1. Na mensagem, clique com o botão direito do mouse no link de agenda .ics.

  2. Selecione Criar nova agenda.

  3. Digite o nome da agenda e selecione a cor para ela.

  4. Se não quiser incluir essa agenda ao informar horários livres/ocupados, selecione Excluir esta agenda ao informar horários livres/ocupados.

  5. Clique em OK.
