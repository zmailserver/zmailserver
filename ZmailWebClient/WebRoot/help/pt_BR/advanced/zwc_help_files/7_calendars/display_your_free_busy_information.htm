Exibir a sua informação livre/ocupado

O recurso de livre/ocupado permite que os outros vejam os horários de reuniões agendadas nas agendas. Esse recurso fica ativo por padrão, mas você pode desativá-lo ou limitar quem pode ver as informações.

  1. Vá para Preferências>página Agenda.

  2. Role até a seção Permissões, selecione a opção Livre/Ocupado que deseja definir para essa agenda.

    3. Caso selecione Permitir que apenas os usuários internos a seguir vejam minhas informações livre/ocupado, no campo de texto, digite o endereço de e-mail dos usuários internos.

  3. (Opcional) Na seção Convites, especifique quem pode convidá-lo para as reuniões.

    4. Caso selecione Permitir que apenas os usuários internos a seguir vejam minhas informações livre/ocupado, no campo de texto, digite os nomes de usuário internos ou o endereço de e-mail completo.

  4. Se limitar quem pode convidá-lo para as reuniões, selecione Enviar respostas automáticas para usuários que estão impedidos de me convidar.

  5. Clique em Salvar.

Observação:   Essas configurações não afetam as agendas configuradas para não exibir as informações de livre/ocupado.
