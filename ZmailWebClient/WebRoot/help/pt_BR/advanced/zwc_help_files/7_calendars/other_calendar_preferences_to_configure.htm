Outras preferências de agenda para configurar

É possível definir diferentes preferências para gerenciar como você trabalha na Agenda.

Configurar sua agenda como uma agenda particular

Todos os convites de agenda recebidos são marcados como públicos por padrão, a menos que o criador do compromisso tenha marcado a notificação de reunião como particular. Quando Particular está definido, todos os convites de agenda recebidos são marcados como particulares. Detalhes sobre eventos que são marcados como particulares não são exibidos.

  1. Vá para Preferências>página Agenda.

  2. No menu suspenso Visibilidade padrão do compromisso, selecione Particular.

  3. Clique em Salvar.

Um pequeno ícone de bloqueio aparece próximo à mensagem em sua agenda para mostrar que se trata de um compromisso particular. As agendas compartilhadas com outras pessoas mostram o compromisso, mas não os detalhes.
