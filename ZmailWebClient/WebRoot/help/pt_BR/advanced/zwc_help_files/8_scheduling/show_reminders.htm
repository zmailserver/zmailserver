Exibir lembretes

Você pode configurar lembretes de reunião. O padrão é cinco minutos. É possível também optar por Reproduzir um som com o lembrete. Se o volume do computador estiver definido como mudo, você não ouvirá som algum. Selecione Piscar o título do navegador se estiver trabalhando em uma guia ou janela de navegador diferente e optar por fazer o título piscar para lembrá-lo.

  1. Vá para Preferências>página Agenda.

  2. Na seção Geral>Exibir lembretes, defina quantos minutos antes das reuniões o lembrete deve ser enviado. Esse é o padrão e você pode alterá-lo para cada reunião.

  3. Selecione como deseja ser notificado sobre uma reunião que está para ocorrer. Selecione qualquer uma dessas opções.

Opção

Descrição

Reproduzir um som

Você é notificado por um som. Se o volume estiver definido como mudo, você não ouvirá nenhum som. Para que isto funcione, você precisa ter o QuickTime ou o Windows MediaCenter instalado.

Realçar a aba E-mail

A guia E-mail é realçada se você não estiver trabalhando nesta guia no momento.

Exibir uma notificação popup

Informações sobre a mensagem são exibidas. Você precisa ter o Yahoo!BrowserPlus instalado.

Para obter uma cópia, vá para
http://browserplus.yahoo.com/
