Usando os clientes avançado e padrão

Duas versões do @ClientNameLong@ estão disponíveis.

OCliente Web Avançado oferece o conjunto completo de recursos de colaboração na Web. Ele funciona com a maioria dos navegadores atuais e conexões de Internet mais rápidas.

Se você possui uma conexão de Internet lenta ou prefere mensagens com base em HTML, é possível usar a versão do Cliente Web Padrão , que oferece muitos dos recursos do Cliente Web Avançado, mas pode ser acessado de forma diferente. Você pode alterar o padrão para o Cliente Web Padrão na sua página Preferências>Geral após efetuar o login.

Ferramentas do navegador

•      O botão Voltar do navegador leva você de volta à página anterior que estava visualizando. Você também pode usar o botão Avançar.

•      Não use o botão Recarregar (Atualizar) do navegador. Isso fará com que a sua sessão seja reiniciada.
