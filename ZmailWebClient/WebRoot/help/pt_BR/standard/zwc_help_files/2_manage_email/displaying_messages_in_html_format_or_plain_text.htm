Exibindo mensagens no formato HTML ou Texto simples

Você pode visualizar mensagens de e-mail em dois formatos, HTML ou texto simples.

•      As mensagens em HTML podem ter formatação de texto, numeração, marcadores, plano de fundo colorido e links que podem facilitar a leitura das mensagens.

•      As mensagens em texto simples sempre podem ser lidas por outros clientes de e-mail. Texto simples não suporta nenhuma formatação de texto, estilo, links ou imagens. Você pode visualizar uma mensagem no modo texto simples mesmo que ela contenha HTML. Os marcadores HTML são exibidos no corpo da mensagem na visualização de texto sem formatação.

Definir o formato de exibição da mensagem

  1. Vá até a página Preferências>E-mail.

  2. Em Exibindo mensagens>Exibir e-mail, selecione Como HTML (quando possível) ou Como texto.

  3. Clique em Salvar.
