Criando reuniões recorrentes

Reuniões recorrentes são compromissos que se repetem em algum tipo de programação. Você pode programar reuniões recorrentes para que elas se repitam diariamente, semanalmente, mensalmente ou anualmente. Além disso, você pode personalizar programações de compromisso recorrente.

  1. Quando você criar um novo compromisso, clique em Repetir.

  2. Na seção Repetição>básica, clique no botão de opção para selecionar a frequência com que a reunião deve ser repetida.

  3. Se a reunião recorrente tiver uma data final, na seção Término>previsto, selecione o término das reuniões após um número específico de reuniões ou até uma data específica.

  4. A seção Avançado permite que se programem padrões mais específicos.

  • Diariamente. Personalize repetições diariamente para selecionar se é para todos os dias ou alguma outra sequência, como a cada dois dias ou quatro dias.

  • Semanalmente. Personalize as repetições semanalmente para selecionar se é um compromisso recorrente semanal ou algum outro padrão semanal, como a cada duas semanas.  Você pode selecionar que dia da semana deseja a reunião.

  • Mensalmente. Personalize as repetições mensalmente para selecionar se é um compromisso recorrente mensal ou algum outro padrão mensal, como a cada dois meses.  Uma agenda é exibida e com isso é possível selecionar que dia(s) deseja programar a reunião.

  • Anualmente. Personalize as repetições anualmente para selecionar o mês e a data.

  1. Clique em Concluído para retornar à página de compromissos.
