Exportando um iCalendar

Você pode exportar agendas salvas no formato .ics. iCalendars é o padrão para troca de dados de agenda na Internet, e .ics é o formato padrão das informações iCalendar.

  1. Selecione a aba Agenda.

  2. No painel Visão geral da agenda, clique em Editar.

  3. Selecionar a agenda para exportar

  4.  No campo Exportar .ICS, clique no link da agenda.

  5. Na caixa de diálogo, selecione Salvar arquivo.

  6. Clique em OK.
