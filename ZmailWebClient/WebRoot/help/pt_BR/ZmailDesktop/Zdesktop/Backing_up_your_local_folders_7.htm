Backup das contas

Você pode fazer backup de contas do Zimbra Desktop na página Preferências>Backups da seção Todas as contas.  O padrão é fazer backup apenas das pastas locais, já que elas contêm itens que não têm backup em outros servidores de e-mail.  

Para fazer backup das contas

  1. Vá para a página Preferências>Backups.

  2. Na seção Configurar backups, selecione as contas para backup. Lembre-se de que os itens nas pastas locais e nas pastas da conta POP são salvos no computador. É uma boa ideia fazer backup dessas contas. Você pode fazer backup de todas as contas criadas no Zimbra Desktop, mas não se esqueça de que os arquivos podem ser bem grandes. Para descobrir aproximadamente quanto espaço em disco é necessário, passe o mouse sobre o nome da conta para ver o tamanho da caixa de entrada.

  3. Se preferir programar o backup, selecione quando quer que o backup seja executado, Todo dia, Toda semana, Todo mês.

  4. No campo Localização, digite o caminho para o diretório em que o backup deve ser salvo. Por exemplo, para salvar na pasta ZD_backups na unidade externa E, digite E:\ZD_backups.

  5. No campo Retenção, selecione o número de backups que quer salvar. O padrão é salvar apenas um arquivo de backup. Você pode selecionar salvar até cinco backups até que o mais antigo seja substituído.

Se você programar a execução automática, o backup será executado de acordo com a programação. O Zimbra Desktop deve estar aberto para que se executem backups. Se estiver programada para uma hora em que você não estiver conectado ao Zimbra Desktop, a sessão de backup na próxima vez que você se conectar.

Se você quiser backup suas contas imediatamente ou se a programação estiver definida para Nunca, clique em Fazer backup de contas agora para executar uma sessão de backup. Dependendo do tamanho das contas, o backup pode levar algum tempo.  Não feche o Zimbra Desktop durante a execução do backup. Você pode continuar a trabalhar com o Zimbra Desktop.

Observação: Se você selecionar o backup imediato das contas, a hora programada para o próximo backup não será redefinida.

Para restaurar uma conta, consulte Restaurando contas a partir de arquivos de backup.
