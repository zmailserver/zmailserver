Usando agendas no Zimbra Desktop

Você pode manter uma agenda no Zimbra Desktop para cada uma das suas contas.

As agendas das contas Zimbra são sincronizadas com Zimbra Desktop. Quando você adiciona, altera ou exclui um evento do Zimbra Desktop, as alterações são sincronizadas para a sua agenda na conta Zimbra.

Você pode manter uma agenda no Zimbra Desktop para cada uma das suas contas, mas os eventos de agenda não são sincronizados nas agendas da sua conta para essas contas.

Se você estiver usando outro programa de agenda, salve-o como um arquivo iCalendar (.ics) e importe-o para a sua agenda. Se você fizer alterações na agenda no Zimbra Desktop, as alterações não serão sincronizadas para a agenda. Se você recarregar a agenda .ical, a agenda no seu Zimbra Desktop é substituída e todos os eventos adicionados são perdidos.
